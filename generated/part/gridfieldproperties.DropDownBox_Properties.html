DropDownBox Properties

Choices

Specify the choices that appear in the DropDownBox.

Maximum choices

Specify the maximum number of choices to show. Set to 0 to show all choices.

Height

Specify the height of the box. If height is 1, then box appears as combo box. If height is greater than 1, then box appears as a list box.

Multi-select allowed

Specify if user can select more than one choice

In-line style

Specify style attributes that override the settings inherited from the style sheet.
