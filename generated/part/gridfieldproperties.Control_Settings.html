Initial value property

Initial state

Select True if checkbox initial state is checked.

Calculated Field Expression

Define a calculated field expression. If not blank, the field will be considered a calculated field. Javascript must be enabled on the browser for this to work.

Enable Expression

Define the Enabled Expression for the field. If the expression evaluates to true, the field will be enabled. Javascript must be enabled on the browser for this to work.

Show/Hide Expression

Define the Show Expression for the field. If the expression evaluates to true, the field will be shown. Javascript must be enabled on the browser for this to work.
