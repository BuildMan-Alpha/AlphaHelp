Row Properties

Hide row

Do not display this control in the Form. (Note: The control can still be displayed in another row if you convert that row to a freeform layout.)

Security level

If the user is logged in with a security level that is lower than the security level you specify here, the row will be hidden. Set the security level to 0 to always show this row regardless of the user's security level.

Security Groups

Select which security groups can see this row. If no groups are selected, all users can see the row.

Row label

Specify the row label

Row label in-line style

Specify style attributes that override the settings inherited from the style sheet.

Row label position

Specify where the label is positioned relative to the control.

Label prefix

Specify text or an image that should be displayed to the left of the row label. You can use HTML tags in the text.

Bubble Help Text

Enter the text to show up as Bubble Help when the user hovers over the table cell that contains this control.

Label Bubble Help Text

Enter the text to show up as Bubble Help when the user hovers over the label.

Freeform layout

Specify if this row should display just this field, or allow a freeform layout.

Freeform template

Specify the layout template for this row

Cell in-line style

Specify style attributes that override the settings inherited from the style sheet.

Break

Break property

Column Span

Set the number of columns that the control will span.
