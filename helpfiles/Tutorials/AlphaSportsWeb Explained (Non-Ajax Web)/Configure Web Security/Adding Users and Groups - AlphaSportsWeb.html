Adding Users and Groups - AlphaSportsWeb

All users who login to Alpha Sports Web have to be first registered in the user table.

  1. Display the Web Projects Control Panel.

  2. Click Security to display the Web Security dialog.

  3. Select Users and Groups and click OK to display the Web Users and Groups dialog.

  4. Display the Groups tab.

    1. Click Add Group, enter "Accounting", and click OK.

    2. Click Add Group, enter "Clerical", and click OK.

    3. Click Add Group, enter "Customers", and click OK.

    4. Click Add Group, enter "Marketing", and click OK.

  5. Display the Users tab. The Import / Export Users option will be used to add new user.

  6. Click on Import / Export Users to open the selection dialog.
  7. Select Import User Information from Table and click OK.

  8. The Import to Web Users Table dialog will open.

  9. Click for Table to Import and select the customer.dbf table.
  10. Click to open the filter builder and enter email<>"" as the filter expression.
  11. Click the Add Field button to open the field map dialog.
  12. Map the fields as follows:

    13. Field in User Table

    Field in Import table

    Userid

    Email

    Password

    Password

    Email

    Email

    Ulink

    Customer_id

  13. Click the Add Groups button to open the Select Groups Dialog dialog.
  14. Check the group Customers and click OK.
  15. Click OK to import the users from the customer table.
  16. Click on Import / Export Users to open the selection dialog.
  17. Select Import User Information from Table and click OK.

  18. The Import to Web Users Table dialog will open.

  19. Click for Table to Import and select the internal_user.dbf table.
  20. Click the Add Field button to open the field map dialog.
  21. Map the fields as follows:

    22. Field in User Table

    Field in Import table

    Userid

    Email

    Password

    Password

    Email

    Email

    Ulink

    Emp_id

  22. Click OK to import the users from the internal_users table.
  23. For each internal user, select groups other than Customer.
  24. Click OK to close the Web Users and Groups dialog

Limitations

Web publishing applications only.
