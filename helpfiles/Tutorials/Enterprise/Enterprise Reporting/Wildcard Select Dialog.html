Wildcard Select Dialog

The Wildcard Select dialog provides a way to select additional list items by entering a search string. The search string returns a list of the tables or views whose names match. You can enter letters or wildcard characters into the Search string control.

  • "?" substitutes for any single character

  • "*" substitutes for any series of characters

Selecting Replace existing selections will cause the selection of items in this dialog to replace selections made in the calling dialog. Selecting Add to existing selections will add to the selections made in the calling dialog.

Click OK when you are finished to return to the calling dialog.
