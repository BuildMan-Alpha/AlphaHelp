SQL Genie Columns Tab

The Columns tab of the SQL Genie defines the fields that your SQL SELECT statement will return. To select a field for your report.

Any time that the Execute Query button is enabled, you may click it to see a sample of the records that the query will return. At any time you may click the History button to see a list of SQL statements previously created in this session in the <span class=Screen>SQL History Dialog</span>.

Adding Columns to the Report

  1. Make a table selection from the drop down list box at the top left corner of the screen.

    1. Optionally, select "* (all columns)" in the Available Columns list and click to add them to the report.

    2. Optionally, select one or more columns from the Available Columns list and click to add them to the report.

    3. Optionally, click to add all columns from the currently selected table to the report.

    4. To remove a column from the report, select it in the Selected Columns list and click .

    5. To remove all columns from the report, click .

  2. Repeat with any other table that you selected on the Tables tab.

  3. Optionally, click Insert Expression to display the <span class=Screen>SQL Expression</span> dialog to select a column or write an expression to add to the Selected Columns list.

  4. The vertical order of columns in the Selected Columns list is their order from left to right on the report. To change the position of a column in the Selected Columns list:

    1. Select it and click to move it to the left-most position on the report.

    2. Select it and click to move it to the right-most position on the report.

    3. Select it and click to move it one position left on the report.

    4. Select it and click to move it one position right on the report.

  5. You may change properties of each column in the Selected Columns list.

Property

Description

Column

The name of a field or an expression that combines 2 or more columns. Click to display the <span class=Screen>SQL Expression</span> dialog to select a column or write an expression.

Alias

Creates an alias for the column, which is expressed as an "AS Alias " clause.

Table

The table that contains the column.

Show

When selected (TRUE), the column will display on the report.

Data type

The data type of the selected column.

Sort

Defines whether to sort the report on this column in ascending or descending order. Your choices here will appear on the Order tab of the SQL Genie and in the ORDER BY clause of the SQL SELECT statement.
