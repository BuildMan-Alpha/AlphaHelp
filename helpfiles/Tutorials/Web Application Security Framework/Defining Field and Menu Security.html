Defining Field and Menu Security

You may control access to a control of a grid or dialog component, or a menu entry of a navigation system component by specifying which groups are permitted to see it.

When defining a dialog component:

  1. Select the control in the Selected Controls list.

  2. Click in the Row Properties > Security Groups property to display the <span class=Screen>Permissions</span> dialog.

  3. Check the groups that will be granted access and click OK to continue.

When defining a grid component:

  1. Select the control in the Selected Controls list.

  2. Click in the Column Properties > Security Groups property to display the <span class=Screen>Permissions</span> dialog.

  3. Check the groups that will be granted access and click OK to continue.

When defining a navigation system component:

  1. Select the menu entry.

  2. Click in the Node Settings > Security Groups property to display the <span class=Screen>Permissions</span> dialog.

  3. Check the groups that will be granted access and click OK to continue.

Limitations

Web publishing applications only.
