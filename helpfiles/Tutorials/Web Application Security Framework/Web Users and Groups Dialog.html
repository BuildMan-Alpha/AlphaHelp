Web Users and Groups Dialog

The Web Users and Groups dialog allows you to:

  • add users

  • add groups

  • assign users to group

 

  1. Display the Users tab. For each person:

    1. Click Add User to display the Add New User dialog.

    2. Enter the appropriate values in the UserID, Password, and Confirm Password controls.

    3. Click OK.

  2. Display the Groups tab. For each group, click Add Group, enter the group's name, and click OK.

  3. Display the Users tab. For each person set the appropriate group assignments.

Supported By

Alpha Five Version 8 Professional Edition and Above

Limitations

Web publishing applications only.
