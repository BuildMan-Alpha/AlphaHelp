Edit Combo Box with Bubble Help for Choices

This example shows how to use the Combo Box %A% directive to display a CR-LF delimited list of choices as bubble help.

dim colorsas c = a5.Color_Enum()

dim colors1 as c = ""

dim colors2 as c = ""

ui_dlg_box("Select Colors",<<%dlg%

{wrap=40}

This is a standard edit-combo box. as you type, Alpha Anywhere shows you the closest matching entry.;

[.40color1^+colors];

This edit-combo box uses the %A% directive. as you type, Alpha Anywhere shows you bubble help with all possible choices.;

[%A%.40color2^+colors];

<10&OK!ok>;

%dlg%)
