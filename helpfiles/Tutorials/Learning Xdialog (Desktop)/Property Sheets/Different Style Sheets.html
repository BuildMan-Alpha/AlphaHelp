Different Style Sheets

This topic describes the process of creating a property sheet styled with a Cascading Style Sheet.

  1. This property sheet was created with Action Scripting. See Property-Sheet Style Xdialog.

  2. The style sheet was applied on the Propsheet Settings tab of the Property Sheet Builder.

  3. The designer selected "WebForm Style" in the Propsheet > Type list.

  4. The designer selected "Osx" in the Propsheet > WebForm Style list.
