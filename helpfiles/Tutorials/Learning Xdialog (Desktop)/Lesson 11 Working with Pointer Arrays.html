Lesson 11 Working with Pointer Arrays

Pointer arrays (also called "property arrays") allow you to represent "multi-columned" data in an array. For example, for a customer, you have information about the customer's name, address, phone number etc.

Note : For background, refer to the section on "Dot Variables" in the Xbasic Reference Manual.

This data can be represented in a pointer array as follows:

DIM customer3 as P

Customer1.name = "Alpha"

Customer1.address = "1 Main St."

Customer1.city = "Boston"

Customer1.state = "MA"

Customer1.zip = "02116"

Customer1.phone = "6175551212"

Customer2.name = "Beta"

Customer2.address = "2 Center St."
