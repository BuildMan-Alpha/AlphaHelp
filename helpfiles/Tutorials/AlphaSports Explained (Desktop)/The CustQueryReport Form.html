The CustQueryReport Form

The CustQueryReport form provides the user with methods for querying and selecting records for printing.

  • Selecting a print format.

  • Previewing print output.

  • Using conditional objects.

  • Using OnFlyover and OnFlyoverLeave events

  • Using Stylesheets

The CustQueryReport form demonstrates the following controls and techniques.

  • Controlling Record Operations demonstrates how to set form restrictions.

  • Creating and Using a Conditional Object demonstrates how to display and hide controls, depending on the state of a controlling variable.

  • Creating Buttons that Run Code demonstrates how to call an Action Script or a developer created script from a button event.

  • Creating Flyover Effects demonstrates how to specify flyover effects for a button.

  • Creating and Using Stylesheets demonstrates how to use stylesheets.
