Creating and Programming a Hotspot with Action Scripting

the following procedure assumes that the developer has opened the form in the Form Editor . Follow these instructions to use Action Scripting to create and format a hotspot similar to hotspot4 .

  1. Sketch the hotspot on the form.

  2. When the Define Hotspot dialog box appears, click the Use Action Scripting Editor option button.

  3. Click OK.

  1. Click Add New Action.

  2. In the Select Action dialog box, select "Form/Browse" in the Category list.

  3. Select "Open Form or Browse Layout" in the Action list.

  4. Click Next >.

  5. Select "Help" in the Layout Name list.

  6. Click Next >.

  7. Click Next > in the Open Form or Browse Layout dialog box.

  1. Click Next > and Finish.
