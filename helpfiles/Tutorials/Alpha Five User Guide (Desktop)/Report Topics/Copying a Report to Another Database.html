Copying a Report to Another Database

If you would like to copy a report to another database, it is important to understand that a report is not attached to a database. It is attached to a table. The source and destination databases must have tables with the same field names and definitions.

  1. Open the destination database that has the destination table.

  2. Attach the source table temporarily to the destination database. Select File > Add Table/Set... .

  3. In the resulting Add Tables/Sets to Database dialog in the Look in field click to navigate to and select the source table.

  4. Click Open.

  5. Click OK.

  6. Display the Report tab of the destination database.

  7. Highlight the report you want to copy, right click and select Copy to... .

  8. Select the destination table from the list and click OK.

  9. Now detach the source table. Display the Tables/Sets tab of the destination table.

  10. Right click the source table and select Drop table from database.
