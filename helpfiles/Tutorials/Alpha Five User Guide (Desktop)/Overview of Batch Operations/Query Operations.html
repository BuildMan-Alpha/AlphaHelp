Query Operations

Overview of Query Operations

The Query operation saves filter and order expressions for a table or set. The query that you create here can be seen on the Operations tab of the Control Panel and on the Saved Queries tab of the <span class=Screen>Primary Index or Query</span> dialog box.

Performing a Query Operation

  1. Display the Operations tab of the Control Panel.

  2. Click New.

  3. Select "Query Records" from the Select operation list.

  4. Select the target table or set from the Select Table/Set list.

  5. For those who know how to write Xbasic expressions, click Create to display the <span class=Screen>Query Builder</span> dialog.

  6. For those who would like to create the query interactively, click Create Using Genie to display the <span class=Screen>Query Genie</span>.

  7. To run the finished query immediately click Run.

  8. To run the finished query against a specific layout based on the selected table or set, click Run Special.

    1. Select the Layout Type.

    2. Select the Layout Name and click OK.

  9. To save the operation click and enter the name of the operation into the Save Query dialog and click OK.

Note : The next step is a one-way operation from Query Genie to Query Builder.

  1. If you created the query with the Query Genie, you may convert your inputs to filter and order expressions by clicking Convert. The results will appear in the Query Builder.
