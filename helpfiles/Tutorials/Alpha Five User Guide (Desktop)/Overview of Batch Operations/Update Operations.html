Performing an Update Operation

  1. Display the Operations tab of the Control Panel.

  2. Click New.

  3. Select "Update Operation" from the Select operation list.

  4. Select the master table (the destination of the records) from the Select Table/Set list.

  5. Click Create to display the Update Builder or Create Using Genie to display the Update Genie (see Using the Update Genie ).

  1. To define an Update operation, fill in the Update Settings table. To begin, choose the field you want to update in the Field column.

  2. In the Expression column, enter an expression, the value of which will be copied into the field. You can click the button to open the <span class=Screen>Expression Builder</span>.

  3. In the Condition column, enter the logical filter expression to indicate under which condition the update should occur. The filter expression must evaluate to TRUE for the row to be used in the update. If you leave the column blank, the row will always be used.

  4. Click to select the records to include in the Update.

  5. Click to run the operation.

You can create as many rows as Necessary. Alpha Five performs the update operation rows in the order in which they appear in the table. This lets you create expressions that refer back to updated fields. Refer to the following subsection for an example.

Using the Update Genie

  1. Select the type of update operation you would like to perform.

  2. Click Next > to continue.

  1. Enter the parameters required by the update operation.

  2. Click Next > to continue.

  1. Click Yes if you would like to add another update operation.

  2. Repeat steps 2 through 4.

  3. Click Next > to continue.

  1. If you clicked the Selected records radio button, make a selection from the list box.

  2. Enter the filtering (a logical expression ) and ordering (a character expression ) criteria required by your choice.

  3. Click Next > to continue.

  1. Indicate whether you want to run the operation now, save the operation and then run it, or view the settings of the Update Card.

  2. Click Finish.

  1. If you selected Run the Update Record operation now, (but first save the definition for future use), the Save update settings as... dialog box will appear.

  2. Enter the name of the operation into the Name field and click OK.

  3. If you selected Don't run the Update operation now, just show me how the genie has filled in the Update card, the Update Card will appear.

  4. Optionally, click Save to display the Save update settings as... dialog box.

  5. Enter the name of the operation into the Name field and click OK.

  6. Optionally, click the Run button on the toolbar to run the operation.
