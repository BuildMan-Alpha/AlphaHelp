Export Database Tables to SQL Dialog

The Export Database Tables to SQL dialog helps you export one or more Alpha Anywhere .DBF tables to a back-end SQL database.

Procedure

  1. Select a database type from the SQL Connection list.

  2. Click Connect to display the Connect to Database dialog.

    1. Navigate to and select a database in the File Name control.

    2. Enter the other values required to access the database.

    3. Click Test Connection.

    4. Click OK if the connection succeeded.

  3. Place a checkmark to the left of each table that you would like to export.

  4. Enter the name of the new table or select the name of the table you would like to overwrite in the Remote Table control.

  5. Optionally, for each remote table:

    1. Click to display the Primary Key Columns dialog.

    2. select one or more fields from the Table Columns list and click to move them (in order) to the Primary Key Columns list.

    3. Click OK to continue.

  6. Click Export.

  7. Click to exit.

Limitations

Desktop applications only.
