Working with Databases

Tables, sets, and databases are the primary structures for storing data in Alpha Five.

Your database files keep track of all of the tables, sets, toolbars, forms, and such, used for managing information. When you open an existing database, Alpha Five displays the objects (tables, forms, etc.) that make up the database in the Control Panel. With Alpha Five, you work with one database at a time.

Typically, you create different databases for unrelated uses. For example, if you keep track of information for your business and your home, you might create one database for your business and one for your home.

Good design practices suggest that:

  • Each database have its own unique folder, unless

  • Two or more databases have common tables or interact, in which case the folders should be the same.

Creating a New Database

To create a new database, either select New Database... from the Database & Tasks tab of the initial Select Database dialog or select File > New Database... from the Control Panel. The File dialog box appears where you can enter a name for the database you are creating.

You can also create a database from a . , .XLS (Microsoft Excel spreadsheet), or .MDB (Microsoft Access database) file.

  1. In the Control Panel  select File > Open Database... to display the Open Database dialog.

  2. Navigate to the directory containing the file.

  3. In the Files of type list select either:

    • .DBF (Alpha Five table)

    • .XLS (Excel spreadsheet)

    • .MDB (Access database)

  4. Click Open to display the Select Tables/Views to Import dialog.

  5. Place checkmarks next to the tables to import and click OK > Close.

If the data is in an Microsoft Excel Spreadsheet or Microsoft Access database, Alpha Five import it into a .DBF file (Alpha Five table). Finally, Alpha Five will create a database (.ADB) file to contain the table.

Opening an Existing Database

To open an existing database:

  • To browse for the database, select More Databases... from the Alpha Five screen

  • If the desired database appears in the recently opened database list, select the name and click OK.

  • If the Alpha Five screen is not available, select File > Open Database... .
