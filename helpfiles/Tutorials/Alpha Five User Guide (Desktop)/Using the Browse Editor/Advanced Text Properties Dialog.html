Advanced Text Properties Dialog

You use the Advanced Text Properties dialog to define a sequence of statements that will display text, data, buttons, or images.

To create a display statement:

  1. Select an item from the Available Controls list and click to add it to the Selected Controls list. The options are:

    • "Text"

    • "Image"

    • "Button"

    • "Justify"

    • "Space"

    • "Font"

  2. Add as many additional controls to the Selected Controls list as you need to construct the desired display statement. Each time you add a control to the Selected Controls list a properties dialog will appear to further define the control.

  3. The "Text" control displays the Add Text dialog.

    1. Enter static text with quotes ( "Last Name " ) and references to fields as expressions ( alltrim(lastname) ). Optionally, click to display use the <span class=Screen>Expression Builder</span> to compose a character expression.

    2. Optionally, set a value for word wrapping the display statement. The default value of 0 does not word wrap.

    3. Click OK to save your inputs and return to the Advanced Text Properties dialog.

  1. The "Image" control displays the Add Image dialog.

    1. Enter the name of the image into the Image Name control, or click to select an image with the <span class=Screen>Insert Image</span> dialog.

    2. Optionally, click to display use the <span class=Screen>Expression Builder</span> to compose a character expression that resolves to an image name.

    3. Optionally, resize the image by setting values in the Width and Height controls. The default values of 0 display the image at its normal size.

    4. Click OK to save your inputs and return to the Advanced Text Properties dialog.

  1. The "Button" control displays the <span class=Screen>Add Button</span> dialog.

  2. The "Justify" control displays the Add Justification dialog. By default, the justification of the display contents will be set by the parent layout.

    1. Select a justification mode. The options are:

      • "Center"

      • "Left"

      • "Right"

    2. Click OK to save your inputs and return to the Advanced Text Properties dialog.

  3. The "Space" control displays the Add Space dialog.

    1. Enter the number of spaces to insert. The default is 1.

    2. lick OK to save your inputs and return to the Advanced Text Properties dialog.

  4. The "Font" control displays the <span class=Screen>Add Font</span> dialog.

  5. Optionally, use the buttons to re-sequence the formatting steps of the display statement.

  6. Optionally, select any control in the Selected Controls list and click the Edit button to respecify its properties.

  7. Optionally, click Show Code to see the Xdialog formatting expression that you have created.

  8. Click OK to save your changes and return to the calling dialog.

Limitations

Desktop applications only.
