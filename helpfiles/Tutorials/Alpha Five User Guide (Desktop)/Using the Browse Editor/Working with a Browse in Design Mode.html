Working with a Browse in Design Mode

Changing the Browse's Dimensions

To change the height or width of a browse:

  1. Select either the bottom or right edge of the browse.

  2. Stretch the browse window to the desired size.

Removing Fields from a Browse

To remove a field:

  1. Click a column heading to highlight it.

  2. Press DELETE.

  3. The field is no longer displayed.

Adding Fields to a Browse

To add a field to a Browse:

  1. Drag the field from the <span class=Screen>Drag and Drop List</span> and drop it into the browse.
