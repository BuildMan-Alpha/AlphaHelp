Previewing a Report, Label, or Letter

Save as Dialog Box

  1. Make a selection from the Save as  list. The options are:

    • "Adobe Acrobat PDF File". Can be displayed with Adobe Acrobat Reader.
    • "Dynamic HTML File". Can be displayed with a browser.
    • "Rich Text Format File". Can be displayed with Microsoft Word or Microsoft Wordpad.
    • "Plain Text File". Can be displayed with Microsoft Notepad and many other text editors.

  1. Optionally, un-check Open file with associated application if you do not want to see the results.

  2. Enter the name of the file you are going to create in the Filename field.

  3. Optionally (V7), click Options to display the Options dialog to control more formatting and processing steps.

Send as Dialog Box

  1. Make a selection from the Select a File Type list. The options are:

    • "Adobe Acrobat PDF File"

    • "Dynamic HTML File"

    • "Rich Text File"

    • "Plain Text File"

  2. Click Options to choose the email client you want to use.

    • "Default E-Mail Client"

    • "Alpha Five E-Mail Client"

  3. If you selected "Adobe Acrobat PDF File", optionally (V7), click Click to Set PDF Options to display the Options dialog to control more formatting and processing steps.

  4. If you selected "Dynamic HTML File", optionally (V7), click Click to Set HTML Options to display the Options dialog to control more formatting and processing steps.

  5. If you selected "Rich Text File", optionally (V7), click Click to Set RTF Options to display the Options dialog to control more formatting and processing steps.
