Setting Printer Options at Run Time

You can set printer options (such as Printer name, number of copies, paper tray etc.) at run time, overriding any printer settings that are saved in the layout definition. To do this, use Action Scripting to create a script to print your Layout.

On the Options tab in the Script Genie, select the Override Printer Settings check box, and click the Set Printer Override Settings button.

Note : You can use Xbasic to specify the filename to use when you print to file.
