Defining Security Settings - AlphaSports Web

The first step when configuring the web publishing security framework for Alpha Sports Web is to define the security features you want to use.

  1. Display the Web Projects Control Panel.

  2. Click Security to display the Web Security dialog.

  3. Select Web Security Configuration and click OK to display the Security Settings dialog.

  1. Enable (check) the Security Policy > Security Active property.

  2. Enable (check) the Security Policy > Password Required property. A large number of additional properties will appear.

  3. Click in the Security Policy > Redirect Page - login property and select "Login.a5w".

  4. Click in the Security Policy > Redirect Page - insufficient permission property and select "notallowed.a5w".

  5. Click to set the Login Options > Login expiration policy to "Defined time after initial login".
  6. Click in the Login Options > Login expiration time property and select 6 months.
  7. Enable (check) the Login Options >Remember me option.

  8. Click in the Login Options > Redirect Page after Login property and select "index.a5w".

  9. Enter Alpha Five V8 into the Password Options > Password encryption key property.

  10. Enable (check) the Password Options > Allow passwiord change at login option.

  11. Click in the Password Options > Password Validation Rules property to display the Password Validation dialog.

    1. Enable (check) the Text Format > Allow digits property.

    2. Enter "6" into the Data Length > Minimum Length property.

    3. Enter "12" into the Data Length > Maximum Length property.

    4. Click OK.

  12. Enable (check) the Password Options > Password expires? option.
  13. Click in the Password Options > Password expiration time property and select 6 months.
  14. Set the Password Options > Password restricted re-use to 1.

  15. Scroll down to the Lost Data Recovery Options. Picture

  16. Click to set the Lost Data Recovery Options > Lost password action to "Recover Password ".
  17. Click to set the Lost Data Recovery Options > Data Required for password reset or rcovery to "Valid Email address".
  18. Click to set the Lost Data Recovery Options > Lost recovery method to "Send Email to user".
  19. Click to set the Lost Data Recovery Options >Lost recovery alternative method to "Create Email link on page ".
  20. Click in the Lost Data Recovery Options >Configure email to send to user with password property and accept the default layout by Clicking OK.
  21. Click in the Lost Data Recovery Options >Configure email request from user property and accept the default layout by Clicking OK.

  22. Open the customize options by expanding the option. Picture

    1. Enable (check) the Customize Options > Show external user identifier field property.

    2. Enter "__protected__user_pUser.Ulink" into the Customize Options > Session variable for identifier field property.

  23. Click Save.

  24. If you have neglected to set any required properties, a dialog similar to the following will appear. Click OK and correct these errors. Picture
