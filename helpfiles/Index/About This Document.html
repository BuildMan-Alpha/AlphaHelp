About This Document

The AlphaSports database contains a large number of sample techniques that illustrate the power and ease of use of Alpha Five. This chapter reviews selected forms, controls, scripts, and variables used in the database.

This chapter is not a comprehensive review of the design of the AlphaSports database, but it does cover the following topics.
