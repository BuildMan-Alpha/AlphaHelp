A5 V10 Feature Peek #9: Rich text

 

Any grid you build in Alpha Five Version 10 contains a built-in HTML editor for editing rich memo fields. This video demonstrates how rich memos can be edited in the grid part and the detail view part.

In the grid section, the HTML editor is configured as a popup editor. It's opened when you click on the field. And in the detail view, the HTML editor can be configured to edit "in-place." Have a look at this video to see it in action.
