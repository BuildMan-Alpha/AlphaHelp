Alpha Anywhere Feature Peek #15: Detail view
There are a variety of options where the "detail view" grid can be opened in Alpha Anywhere. In this video we've put together, you'll see the detail view opened:
 
  • on the page underneath the grid part
  • in a modal, popup window
  • in a modeless, popup window
  • in the grid part, directly under the row
  • and using the "pre-fetch" option for greater responsiveness
