Input and Output

For input/output via a web service, please refer to Data Integration. For output via report or charts, see Reports.

Supported Input

Supported Output

  • Email - Alpha Anywhere has many built in Xbasic functions for sending and receiving email.
     
  • PDF - Alpha Anywhere allows you to create PDF reports. It also allows you to create and manipulate PDF files using Xbasic and action scripting.
     
  • Microsoft Office Excel - Alpha Anywhere allows you to generate and add to Microsoft Excel Spreadsheets
     
  • Text Files (including CSV, XML, and JSON) - Alpha has built in Xbasic functions for creating text files including RTF, CSV, XML and JSON files.
