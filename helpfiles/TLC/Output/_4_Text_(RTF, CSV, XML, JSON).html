Text Output

This section points to Xbasic functions for working with a variety of file formats. Alpha Anywhere includes Xbasic functions to work with a variety of file formats. These include:

  1. Plain Text - see File Functions and Methods
  2. CSV - see the Export to ASCII Features for Grid Components
  3. JSON - see the JSON Functions documentation.
  4. XML - There are two functions that can be helpful for XML output: JSON to XML, which converts a JSON object to XML, and HTTP_POST_PAGE2, which allows you to send XML to a web service.
  5. RTF - Alpha Anywhere has a large collection of RTF Functions and Methods that allow you to programmatically create and work with rich text files.
