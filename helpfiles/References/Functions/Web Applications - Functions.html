Web Applications - Functions

Function

Description

A5_AllWebFiles()

Returns a list of all page and component files.

A5_EditComponentStyle()

Allows you to select and edit a CSS style sheet.

A5_HTML_LIST_ADO()

Returns a list of choices for a list or combo box from an ADO data source.

A5_HTML_LIST_DBF()

Returns a list of choices for a list or combo box from an DBF data source.

A5_HTML_LIST_POPULATE()

Generates code for insertion into an HTML form control.

A5_LOAD_AEX()

Loads the functions of a compiled library into an A5W web page.

A5_PROMPT_FILES_IN_DB()

Displays a selection dialog and returns a list of database and web publishing files.

A5_UNLOAD_AEX()

Unloads the functions of a compiled library from an A5W web page.

A5_WebCPOpen()

Displays the Web Projects Control Panel.

A5W_CHECKBOXES()

Generates check box code for insertion into an HTML for control.

A5W_LOAD_COMPONENT()

Loads a web component into an A5W page and returns a pointer to its data structure.

A5W_RADIOBUTTONS()

Generates radio button code for insertion into an HTML for control.

A5WS_CreateSecurityFiles()

Create empty default security files. May be used in a script or UDF.

A5WS_LockOutUserRelease()

Release a web user that is currently locked out. May be used on a Web Page, desktop script, or UDF.

A5WS_LockOutUserSet()

Lock out a user for a specified period of time. May be used on a Web Page. May be used in a desktop script or UDF.

A5WS_LogoutUser()

Remove all login information for current logged in user. May be used on a Web Page.

A5WS_Pagesecurity()

Set security for web pages. May be used on a Web Page. May be used in a script or UDF.

A5WS_SecurityQues()

Opens a list of current security questions for project. May be used in a script or UDF.

A5WS_SecuritySettings()

Open Security settings genie for project. May be used on a Web Page. May be used in a script or UDF.

LOAD_EXTERNAL_UDFS()

Loads the functions of an .ALB library into an A5W web page.

UDF_DEFINE()

Adds the Xbasic statements, which define a function, to the list of UDFs in the specified variable namespace.

UI_GET_WEB_FILE()

Displays a dialog box that allows the user to select a file from the web projects under the current database.
