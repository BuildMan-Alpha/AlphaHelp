PDF_APPEND()

Syntax

Result_Flag as L = PDF_Append ( File_1 as C File_2 as C [, Result_File as C ] )

Argument

Description

Result_Flag

.T. = Append was successful.

.F. = Append was not successful.

File_1

The name of the file that will be at the beginning of the new combined file.

File_2

The name of the file that will be at the end of the new combined file.

Result_File

Optional. Default = "". The name of the file to create. If Result_File is blank, then File_2 is appended to File_1.

Description

The PDF_APPEND() function appends File_2 to File_1, producing Result_File.

Limitations

Examples

? pdf_append("c:\inv.pdf", "c:\cus.pdf", "c:\combined.pdf")

= .T.
