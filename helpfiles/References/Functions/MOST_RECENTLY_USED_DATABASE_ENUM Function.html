MOST_RECENTLY_USED_DATABASE_ENUM()

Syntax

Database_List as C = MOST_RECENTLY_USED_DATABASE_ENUM()

Description

The MOST_RECENTLY_USED_DATABASE_ENUM() function returns a CR-LF delimited list containing the fully qualified filenames of the eight different most recently used databases. Database names are not repeated, as found in the list of most recently used database under the Alpha Anywhere File menu.

Limitations

Desktop applications only.

Example

dim cs as C

cs = most_recently_used_database_enum()

cs   ->   C:\Databases\Functions\Functions.Adb

C:\Databases\Expressions Test\Expressions Test.Adb

C:\Databases\Alphasports Official\Alphasports.Adb

C:\Databases\Travel Time\Travel Time.Adb

C:\Documents And Settings\Elarrabee\Desktop\To Delete\To Delete.Adb

C:\Documents and Settings\ELarrabee\Desktop\To delete.adb

C:\Program Files\A5v5\Samples\Alphasports\Alphasports.Adb

C:\Databases\Backup\Backup.Adb
