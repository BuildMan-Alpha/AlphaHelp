Financial Functions

Alpha Anywhere provides the following Financial functions.

Function

 

Description

FV()

V5

Returns the future value of a stream of equal Investments, invested at a fixed Interest_Rate for a certain Number_Of_Time_Periods.

FV2()

V5

Returns the future value of a stream of equal Investments, invested at a fixed Interest_Rate for a certain Number_Of_Time_Periods. Does not round up the Number_Of_Time_Periods.

TERM()

V5

Returns the number of time units required to repay a loan.

PAYMENT()

V5

Returns the fixed periodic Payment_Amount required to amortize a loan of the specified Loan_Amount at a fixed Interest_Rate over a given Number_Of_Time_Periods.

PAYMENT2()

V5

Returns the fixed periodic Payment_Amount required to amortize a loan of the specified Loan_Amount at a fixed Interest_Rate over a given Number_Of_Time_Periods.  Does not round up the Number_Of_Time_Periods.

PV()

V5

Returns the Present Value of a stream of Investments, invested at a fixed Interest_Rate for a certain Number_Of_Time_Periods.

PV2()

V5

Returns the Present Value of a stream of Investments, invested at a fixed Interest_Rate for a certain Number_Of_Time_Periods. Does not round up the Number_Of_Time_Periods.
