Addressing and Tracking Functions

Alpha Anywhere provides the following Addressing and Tracking Functions.

Method

 

Description

FEDEX_Delivery_Status()

V8

Returns the status of a shipment sent via FEDEX.

UPS_Delivery_Status()

V8

Returns the status of a shipment sent via the United Parcel Service (UPS).

USPS_City_From_ZIP()

V8

Returns the city that contains the specified ZIP code.

USPS_CityState_From_ZIP()

V8

Returns the state and city that contains the specified ZIP code.

USPS_Delivery_Status()

V8

Returns the status of a shipment sent via the United States Post Office (USPS).

USPS_State_From_ZIP()

V8

Returns the state that contains the specified ZIP code.

USPS_ZipCode_Lookup()

V8

Returns a ZIP code for a specified address.
