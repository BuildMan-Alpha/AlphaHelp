INCREMENT_VALUE()

Syntax

Output_Value as P = INCREMENT_VALUE( Variable as P )

Argument

Description

Variable

The name of a variable.

Description

INCREMENT_VALUE() increments the value referenced by Variable by one. The INCREMENT_VALUE() function operators on date (D), character (C), number (N), and Blob (B) variables. The function returns the value "1" (C), 1 (B), or 1 (N) when Variable equals "". If the variable is type date and the variable is un-initialized, the return value is {31/12/-471}.

Examples

dim cs as C

dim os as C

cs = "AB"

os = increment_value(cs)

os   ->   "AC"

dim num as N

dim onum as N

num = 12

onum = increment_value(num)

onum   ->   13.000000

dim bl as B

dim obl as B

bl = "abc"

obl = increment_value(bl)

obl   ->   ABD

dim dd as D

dd = date()

increment_value(dd)

dd -> {01/11/2003}
