AGE()

Syntax

Number_Of_Years as N = AGE( Starting_Date as D [, Ending_Date as D ] )

Argument

Description

Starting_Date

The first date.

Ending_Date

Optional. Default = DATE(). The second date to use in calculating the age.

Description

AGE() returns the integer number of years between Starting_Date and Ending_Date (default now).

Example

age({12/18/1952}) -> 49
