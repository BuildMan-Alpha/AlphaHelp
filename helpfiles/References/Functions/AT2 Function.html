AT2()

Syntax

Position as N = AT2( Find_String as C, Search_In_String as C [, Starting_Position as N ] )

Argument

Description

Position

The location of the Search_In_String + Starting_Position - 1.

Find_String

A character string.

Search_In_String

The character string to examine for occurrences of Find_String.

Starting_Position

Optional. Default = 1. The position in the string from which to start searching.

Description

AT2() returns a position integer indicating where the Find_String begins in the Search_In_String. Case sensitive. Use this function for incremental searches through a buffer.

Limitations

The Find_String argument is case sensitive.

Examples

In this case the value soft was found and its position returned.

? at2("soft", "alpha software", 3)

= 7

In this case the value soft was not found and the starting position for the next search was returned.

? at2("soft", "alpha software", 10)

= 9
