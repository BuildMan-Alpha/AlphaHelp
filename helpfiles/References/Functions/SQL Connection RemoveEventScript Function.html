SQL::Connection::RemoveEventScript()

Syntax

Result_Flag as L = .RemoveEventScript( [ Name as C ] )

Argument

Description

Result_Flag

TRUE (.T.) if the operation was successful; otherwise FALSE (.F.).

A SQL::Connection object.

Name

Optional. The name of the script. If no name is provided, the function removes the most recently added script.

Description

The RemoveEventScript() method removes an event script (see SQL Events ) from the connection.
