File Menu Functions

File menu functions also appear on the Standard Toolbar.

Function

 

Description

New

Opens a new blank document in a new tab.

Open

Displays the Open Page dialog. The display is initially filtered to show only .A5W pages, but you can change the filter to display HTML pages, or all files.

Close

If you have not saved the current document, the editor asks whether you want to save it. It then closes the tab.

Save

Saves the contents of the current tab. If you have not saved the current document, the editor displays the Save As... dialog to define the name, format, and location of the file.

Save As

Displays the Save As... dialog to define the name, format, and location of the file.

Revert to Saved

 

Discards any changes made during since the last save.

Execute Page

Publishes and previews the page without images in the LivePreview directory. This preview is after Application Server processing. This function is enabled only when the Application Server is running. Refer to Starting and Stopping the Application Server.

Preview in Browser

Opens the page with your default browser. This is without Application Server processing.

Debug Page

Opens the page in the Xbasic Script Debugger.

Print

Prints the page.

Page Properties

Displays the Page Properties dialog, which allows you to set the page's title, margins, background image, background color, text color, and link colors.

Saving a New HTML or A5W Page

To save a new page in a web project:

  1. Click File > Save As to display the Save As dialog box.

  2. Select the project from the drop-down list box at top or click New Project to create a new project.

  3. Optionally, click to create a new folder.

  4. Optionally, click to display only recently modified files.

  5. Optionally, make a selection from the Files of type list box to the list of files in the current folder. The options are:

    • "A5W Pages (*.a5w)"

    • "HTML Files (*.htm;*.html)"

    • "All Files (*.*)"

  6. Select an existing file or enter a File name and click Save.

To save a new page outside a web project.

  1. Click File > Save As to display the Save As dialog box.

  2. Select "More files" in the drop-down list box at top.

  3. Optionally, make a selection from the Save as type list box to the list of files in the current folder. The options are:

    • "A5W Pages (*.a5w)"

    • "HTML Files (*.htm;*.html)"

    • "All Files (*.*)"

  4. Optionally, click to create a new folder.

  5. Optionally, click to navigate to a parent folder.

  6. Select an existing file or enter a File name and click Save.

Setting Page Properties

To set the page properties of your HTML page through menus:

  1. Display the WYSIWYG tab of your HTML Editor window.

  2. Select File > Page Properties to display the General tab of the Page Properties dialog.

  3. Optionally, enter the name of your page in the Title field.

  4. Optionally, check Top Margin and enter the number of pixels.

  5. Optionally, check Left Margin and enter the number of pixels.

The entries you see above produce the following HTML code.

My Page

leftMargin=15

topMargin=10

>

  1. Display the Body tab of the dialog box.

  2. Optionally, check Background image enter the name of an image. Click to navigate to and select the image.

  3. Optionally, dick to select a color in the Background color, Text color, Link color, Active link color, or Visited link color fields.

The entries you see above produce the following HTML code.

text=#800000

vLink=#000000

aLink=#0000ff

link=#008000

bgColor=#ebffff

background=C:\Wallpaper\claire_081003.jpg

>

  1. Click OK to apply your selections or Cancel to discard them.

Debugging an .A5W Page

To debug the Xbasic code in a .A5W page:

  1. Click to open the Debugger.

  2. Click to start running the Xbasic code.

  3. Refer to Debugger Settings and Debugger Menu Commands for more information.

Limitations

Web publishing applications only.
