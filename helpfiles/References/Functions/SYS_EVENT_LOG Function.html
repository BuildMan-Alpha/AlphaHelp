SYS_EVENT_LOG()

Syntax

SYS_EVENT_LOG( Application as C, Event as C )

Argument

Description

Application

The name of your application.

Event

The event message. There is an optional prefix in the format :, where can be:

  • "E" = error

  • "W" = warning

  • "I" = information

  • "S" = success

  • "F" = failure

Description

The SYS_EVENT_LOG() function logs events into the Windows NT system log.

Limitations

Desktop applications only.

Example

sys_event_log("Alpha Five Application Server","I:The server has started")
