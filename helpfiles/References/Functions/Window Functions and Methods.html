Window Functions and Methods

Alpha Anywhere provides the following functions for controlling Alpha Anywhere windows.

Function

Description

A5_CLOSE_ALL_WINDOWS()

Closes all open windows except for the Control Panel.

A5_MDI_WINDOWCLASS_ADD()

Creates a new minimized window of a specified class.

A5_ISMAXIMIZED()

Returns .T. if the MDI child window has been maximized.

A5_LIST_OPEN_WINDOWS()

Returns a CR-LF delimited list of open windows.

A5_WINDOWBAR_CLOSE()

Hides the Window Bar at the bottom of the screen.

A5_WINDOWBAR_IS_OPEN()

Indicates whether the Window Bar is open.

A5_WINDOWBAR_SHOW()

Shows the Window Bar at the bottom of the screen.

A5_WINDOWBAR_TOGGLE()

Shows or hides the Window Bar at the bottom of the screen.

SYS_ID_ENUM()

Creates a CR-LF delimited character string that lists the windows currently available on your system.

SYS_ID_GET()

Obtains the Window Handle of a program that is identified by its Window_Name.

SYS_ID_HIDE()

Hides the window specified by Window_Name.

SYS_ID_MAXIMIZE()

Maximizes the window specified by Window_Name.

SYS_ID_MINIMIZE()

Minimizes the window specified by Window_Name.

SYS_ID_MOVE()

Moves the window specified by Window_Name to new coordinates.

SYS_ID_POSITION_GET()

Returns a character string containing the coordinates of the window specified by Window_Name.

SYS_ID_RESIZE()

Resizes the window specified by Window_Name to new dimensions.

SYS_ID_RESTORE()

Restores the window specified by Window_Name.

SYS_ID_SHOW()

Shows the window specified by Window_Name.

SYS_ID_SIZE_GET()

Returns a character string containing the width and height in inches of Window_Name.

Alpha Anywhere provides the following methods for controlling the application window. These methods apply to Forms, Browses, and the Status Bar. Refer to the method description for the proper way to call the method.

Function

Description

.HIDE()

Hides the window.

.ISVISIBLE()

Returns .T. (TRUE) if a window is visible; otherwise, returns .F. (FALSE).

.MAXIMIZE()

Maximizes the window.

.MINIMIZE()

Minimizes the window.

.RESTORE()

Restores the window to its size before a .MINIMIZE()or .MAXIMIZE().

.SHOW()

Displays the window.

.SIZE_GET()

Returns a character string containing the width and height in inches of the window.

.WINDOW_POSITION()

Positions a window on the screen. Has features supported only by V6.

.ZORDER()

Sets the z-order stacking position of a MDI child window.

A5.ACTIVATING()

Returns the name of the MDI window that has just got focus.

A5.DEACTIVATING()

Returns the name of the MDI window that has just lost focus.
