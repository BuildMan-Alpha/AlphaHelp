RECORD_ADD()

Syntax

Result_Flag as L = RECORD_ADD( Table_Name as C, Data_List as C [, Style as N )

Argument

Description

Result_Flag

.T. = the record was added. 
.F. = the record was not added.

Table_Name

The name of a table

Data_List

A CR-LF delimited list of the field values to add as a new record. The data can be presented in two different formats as Defined by the Style argument.

Style

An optional style selector. The default value is 1.

Style = 1

"Fieldname1 = Value1" ↩
"Fieldname2 = Value2" ↩
"FieldnameN = ValueN"

Style = 2

"Value1" ↩
"Value2" ↩
"ValueN

When entering data in an auto-increment field, use Style 1 and leave the value of the field blank as in the following line:

"Fieldname1 = "

Description

RECORD_ADD() adds a new record to Table_Name using the field values specified by Data_List.

Note : The RECORD_ADD() function opens and closes the table. You should not use TABLE.OPEN() or TABLE.CLOSE().

Example

dim cs as C

cs = "00000010" + crlf() + "Layout" + crlf() + "Report"

? cs

= 00000010

Layout

Report

record_add("category", cs, 2)
