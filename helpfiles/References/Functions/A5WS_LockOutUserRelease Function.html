Releases a locked User Value

A5WS_LockOutUserRelease()

Syntax

Result_Flag as L = A5WS_LockOutUserRelease( User_Value as C )

Argument

Description

Result_Flag

.T. = The User ID was released.

.F. = The User ID was not released.

User_Value

A locked user value.

Description

The A5WS_LockOutUserRelease() function releases a locked User Value. If the function finds no match in the locked out list, nothing happens.

Limitations

Web publishing applications only.

Example

If a user with the user value of "joe@email.com" is locked out, this will release the lock.

a5ws_LockOutUserRelease("joe@email.com")

= .T.
