Returns a list of members of a named group

A5WS_Get_Group_Assignments()

Syntax

User_List as C = A5WS_Get_Group_Assignments( Group_Name as C [, Request as P [, FlagWeb as L ]] )

Argument

Description

User_List 

CR-LF delimited list of users by userid assigned to a particular group in the current project.

Group_Name

The name of a group in the current project.

Request 

Optional. The Request system variable. This variable explicitly passes all request variables to the function.

FlagWeb

Optional.
Default = .T. when run in the web server. .F. when run in the desktop.

  • .T. = Format suitable for use in a list, checkbox, or radio control on a web page

  • .F. = CR-LF delimited list of userid values only

Description

The A5WS_Get_Group_Assignments() function returns a list of members of a named group.

Limitations

Web applications only.

Example


<%A5
dim txt as c
'Function run in a web page so output will be a CR-LF delimited list in format: groupName|groupGuid.
txt = A5WS_Get_Group_Assignments("Marketing")
?txt
%>
