DTOR()

Syntax

Angle_In_Radians as N = DTOR( Angle_In_Degrees as N )

Argument

Description

Angle_In_Degrees

A number between -180 and + 180.

Description

DTOR() converts Angle_In_Degrees (-180 to + 180) to an angle in radians (- p and + p ).

Example

? cos(0)

= 1

? cos( dtor(90) )

= 0

? cos(math_pi)

= -1
