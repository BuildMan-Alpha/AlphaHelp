Letter Functions and Methods

Alpha Anywhere provides the following Letter functions and methods.

Method

Description

A5.LETTER_ENUM()

Returns a list of letter layouts.

A5.LETTER_LOCATION()

Returns the name of the file that contains the letter layout.

LETTER.PREVIEW()

Displays the letter with selected records in the Print Preview dialog.

LETTER.SAVEAS()

Prints the letter layout  to file in PDF, HTML, RTF, or TXT formats.

PREVIEW_LETTER()

Previews letters as they would appear when printed.

PRINT_LETTER()

Creates the selected letters and then displays the Print dialog box.
