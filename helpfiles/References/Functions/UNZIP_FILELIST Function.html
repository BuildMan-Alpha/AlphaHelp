UNZIP_FILELIST()

Syntax

Extracted_Files as C = UNZIP_FILELIST( Zip_File as C [, File_List as C [, Destination_Folder as C [, Preserve_Paths as L [, Prompt_Overwrite = .T. ]]]] )

Argument

Description

Extracted_Files

A CR-LF delimited list of extracted files.

Zip_File

The path and name of the zip file.

File_List

Optional. Default = "" (all files). A CR-LF delimited llist of files to extract from the zip file.

Destination_Folder

Optional. Default = "" (uses paths in zip file). Where to place the extracted files.

Preserve_Paths

Optional. Default = .T. . Whether to preserve path information in the zip file under the Destination_Folder.

Prompt_Overwrite

Optional. Default = .T. . Whether to prompt the user when files are going to be overwritten.

Description

The UNZIP_FILELIST() function extracts all or specified files from a ZIP file to a specified directory.

Example

? unzip_filelist("c:\backup\Alphasportsa5v71.zip", "", "c:\temp\\", .f.)
