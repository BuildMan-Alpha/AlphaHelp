A5_IS_RUNTIME()

Syntax

Result_Flag as L = A5_IS_RUNTIME()

Description

The A5_IS_RUNTIME() function returns .T. if the Alpha Five Runtime or Runtime Plus version is running.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

? a5_is_runtime()

= .F.
