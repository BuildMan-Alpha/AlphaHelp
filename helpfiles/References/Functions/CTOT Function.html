CTOT()

Syntax

Short_Time_Value as Y = CTOT( Formatted_Short_Time as C )

Argument

Description

Formatted_Short_Time

A character string containing a legitimate representation of a short time value .

Description

The CTOT() function converts a time string to a Short Time value.

Examples

dim tc as C

tc = time()

? tc

= "15:47:05"

? ctot(tc1)

= 03:47:05 00 pm

? ctot("2:30 pm")

= 02:30:00 00 pm
