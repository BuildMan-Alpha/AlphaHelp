IMAGE_FILENAME_ENCODE()

Syntax

Expression_String as C = IMAGE_FILENAME_ENCODE( Input_Filename as C )

Argument

Description

Input_Filename

The fully qualified filename of the image file.

Description

Encodes an image filename with an alias.

Example

dim cs as C

cs = image_filename_encode("C:\Databases\Functions\ahv2.jpg")

cs   ->   filename_decode("ahv2.jpg")
