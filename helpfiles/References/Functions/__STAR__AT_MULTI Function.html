*AT_MULTI()

Syntax

Position as N = *AT_MULTI( Find_Strings as C, Search_In_String as C [, Occurrence as N ] )

Argument

Description

Position

The position of the first of the multiple strings in the Find_Strings argument. Zero if the characters were not found.

Find_Strings

A CR-LF delimited string containing one or more strings to find.

Search_In_String

The character string to examine.

Occurrence

Optional. Default = 1. The occurrence to find.

Description

The *AT_MULTI() function returns the position in Search_In_String of the first of the multiple strings in the Find_Strings argument. Case insensitive.

Supported By

Alpha Five Version 8 and Above

Examples

? *at_multi(comma_to_crlf("oranges,apples"),"This string contains apples and oranges")
= 22

? *at_multi(comma_to_crlf("oranges,apples"),"This string contains apples and oranges",2)
= 33

? *at_multi(comma_to_crlf("oranges,apples"),"This string contains apples and oranges",3)
= 0
