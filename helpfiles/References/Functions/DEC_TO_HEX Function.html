DEC_TO_HEX()

Syntax

Hex_Value as C = DEC_TO_HEX( Decimal_Value as N )

Argument

Description

Decimal_Value

An integer value.

Description

The DEC_TO_HEX() function returns the hexadecimal equivalent of a decimal number.

Example

dim dnum as N

dim hnum as C

dnum = 12

hnum = dec_to_hex(dnum)

? hnum ? "C"
