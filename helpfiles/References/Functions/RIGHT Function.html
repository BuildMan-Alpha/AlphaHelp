RIGHT()

Syntax

Output_String as C = RIGHT( Input_String as C, Number_Of_Characters as N )

Argument

Description

Input_String

A character string.

Number_Of_Characters

The number of characters to return.

Description

RIGHT() returns a character string containing the specified Number_Of_Characters starting from the Input_String 's right-most character.

Examples

right(LASTNAME, 4) -> "   " (4 blanks), if LASTNAME is a 10-character field containing "Smith"

right(trim(CITY), 7) -> "chester", if CITY contains "Winchester"
