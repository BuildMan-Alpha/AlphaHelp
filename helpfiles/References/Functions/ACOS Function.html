ACOS()

Syntax

Arc_Cosine as N = ACOS( Radian_Angle as N )

Argument

Description

Number

A number between - p and + p.

Description

ACOS() returns the arc cosine of a Radian_Angle. The Radian_Angle must be expressed in radians. For angles measured in degrees, use acos( dtor(degree_angle) ).

Examples

dim pi as N

dim angle as N

pi = 3.14159

angle = 45

acos(0) -> 1.57

acos(1) -> 0

acos(-1) -> 3.14

acos(pi) -> 0

acos( dtor(angle) ) -> 0.667457
