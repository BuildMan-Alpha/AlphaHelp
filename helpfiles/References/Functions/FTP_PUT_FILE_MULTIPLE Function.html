FTP_PUT_FILE_MULTIPLE()

Syntax

FTP_PUT_FILE_MULTIPLE( FTP_Address as C, User_Name as C, Password as C, Source_File_List as C, Source_Folder as C, Target_Folder as C [, Background as L ] )

Argument

Description

FTP_Address

The FTP address of the transmission.

User_Name

The user's identity.

Password

The user's password.

Source_File_List

A list of relative filenames of files to be transferred.

Source_Folder

The top level folder name where the source files are stored.

Target_Folder

The name of the folder on the target machine where the files should be transferred. If blank, the files are transferred to the root level specified by the FTP address.

Background

Optional. Default = .T.

  • .T. = File transfer takes place in the background. The progress dialog box that is displayed is modeless.

  • .F. = File transfer takes place in the foreground. The progress dialog box that is displayed is modal.

Description

The FTP_PUT_FILE_MULTIPLE() function transfers multiple files via FTP. The relative folder structure of the source files is preserved on the target computer.

Examples

For example, assume that you have a folder called " c:\mydata " which contains these files:

c:\mydata\page1.a5w

c:\mydata\page2.a5w

c:\mydata\images\image1.jpg

c:\mydata\images\image2.jpg

Here is how you could transfer these files using FTP:

dim list as C

list = <<%a%

page1.a5w

page2.a5w

images\image1.jpg

images\image2.jpg

%a%

ftp_put_file_multiple("your.ftpaddress.com", "your_username", "your_password", list, "c:\mydata", "mydata", .f.)
