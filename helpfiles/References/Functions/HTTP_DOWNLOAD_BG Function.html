HTTP_DOWNLOAD_BG()

Syntax

HTTP_DOWNLOAD_BG( Source_URL as C, Target_File as C )

Argument

Description

Source_URL

The URL of the page to download.

Target_File

The path and name of the file to create.

Description

The HTTP_DOWNLOAD_BG() function creates a background thread which uses the HTTP protocol to download from Source_URL and save the data to Target_File.

Note : When the server responds with a 302 code, HTTP_DOWNLOAD_BG() does not automatically use the new URL. The developer needs to examine result.parsed_headers.status_code, then if appropriate, try the URL provided in result.parsed_headers.location. Refer to HTTP_GET()for an example.

Examples

Download the Alphasports Explained PDF from the Learning center and save to a predetermined location.

http_download_bg("http://downloads.alphasoftware.com/Books/AlphaSports.pdf",

"c:\alphasports.pdf")
