MDY()

Syntax

Formatted_Date as C = MDY( Date_Value as D   )

Argument

Description

Date_Value

A variable containing a date value or a character string containing a legitimate representation of a date value ( ISDATE (Date_Value) = .T.).

Description

MDY() returns a Formatted_Date in the "Month DD, YYYY" format for the specified Date_Value.

Supported By

Alpha Five Version 5 and Above

Examples

mdy({12-18-95}) ->  "December 18, 1995"

mdy(BIRTHDAY) ->  "November 25, 1995", assuming BIRTHDAY contains 11-25-95

mdy(date() )-> Current system date
