HTTPD_SETTINGS_GET()

Syntax

Info as P = HTTPD_SETTINGS_GET( [ From_Memory as L ] )

Argument

Description

Info

Application Server settings.

From_Memory

Optional.

Description

The HTTPD_SETTINGS_GET() function loads the Application Server settings.

Limitations

Desktop applications only. Internal use only.

Example

? HTTPD_SETTINGS_GET()

= A5W_PageExtension = ".a5w"

access_log = "C:\Program Files\A5V8\a5w_access.log"

AllowDenyIP = "Deny"

AllowDirBrowse = 0

autostart = "0"

...
