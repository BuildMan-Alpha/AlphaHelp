CASE()

Syntax

Value as A = CASE( Expression_1 as L, Value_1 as A [,   Expression_2 as L, Value_2 as A ... [, Expression_16 as L, Value_16 as A]] )

Argument

Description

Expression_1

An expression that evaluates to .T. (TRUE) or .F. (FALSE).

Value_1

A value.

Expression_2 ... Value_2

through

Expression_16 ... Value_16

Optional. One to fifteen additional different expressions that evaluate to .T. (TRUE) or .F. (FALSE) and the corresponding values that they return.

Description

CASE() returns the value ( Value _1 ... Value _N) specified by the first expression ( Expression _ 1... Expression _ 16 ) that evaluates to .T. (TRUE).

Note : CASE() and IF() functions that return a character value will return a string with a length equal to the last string parameter.

Example

If TEMP is 30.

? case( TEMP < 32, "Cold", TEMP < 65, "Cool", TEMP < 80, "Warm", TEMP > 80, "Really Hot" )

= "Cold"

If you wanted to filter records for a limited number of specific values, you use this type of expression, which will examine the firstname field and select records containing the values "John", "Peter", and "Sally".

case( "John" $ firstname, .t., "Peter" $ firstname, .t., "Sally" $ firstname, .t. )
