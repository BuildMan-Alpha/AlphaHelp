HOURGLASS_CURSOR()

Syntax

HOURGLASS_CURSOR( On_Off_Flag as L )

Argument

Description

On_Off_Flag

  • .T. = Turns on the hourglass cursor.

  • .F. = The normal cursor is restored.

Description

HOURGLASS_CURSOR() turns on an hourglass cursor to indicate to the user that a task is being performed, and that he/she should wait until the task is complete. If On_Off_Flag is .T., the hourglass cursor is turned on. If On_Off_Flag is .F., the normal cursor is restored.

Use this command in scripts before a long operation is started.

Limitations

Desktop applications only.

Example

This script turns on the hourglass cursor before starting a long operation.

hourglass_cursor(.T.)

tbl = table.current()

query.description = "Lastname order"

query.filter = "city = 'Boston'"

query.order = "LAST_NAME"

indx = tbl.query_create()

'Operation complete. Turn off the hourglass cursor.

hourglass_cursor(.F.)
