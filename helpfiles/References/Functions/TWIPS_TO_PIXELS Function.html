TWIPS_TO_PIXELS()

Syntax

Pixels as N = TWIPS_TO_PIXELS( Twips as N )

Argument

Description

Pixels

The pixel equivalent of Twips.

Twips

A number of twips.

Description

The TWIPS_TO_PIXELS() function returns the pixel equivalent of a twips measurement.

Example

? twips_to_pixels(3000)

= 200
