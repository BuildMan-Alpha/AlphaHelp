SCRIPTEDITOR_OPEN()

Syntax

Output_String as C = SCRIPTEDITOR_OPEN( Script_Name as C [, Type as C [, Action_Script_Prompt as C ]] )

Argument

Description

Script_Name

The name of a script.

Type

Optional. Default = "Script"

  • "Script" = Opens the script specified by Script_Name.

  • "Function" = Opens the function specified by Script_Name.

Action_Script_Prompt

Optional. For internal use only.

Description

SCRIPTEDITOR_OPEN() opens the named script in the Code Editor.

Limitations

Desktop applications only.

Example

scripteditor_open("html_exists() ")
