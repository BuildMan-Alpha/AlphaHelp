FILTER_STRING_MULTI()

Syntax

Output_List as C = FILTER_STRING_MULTI( Input_List as C, Filter_List as C [, Delimiter as C [, Exclude as L ]] )

Argument

Description

Output_List

All elements of Input_List that either include or do not include all elements found in the Filter_List.

Input_List

The list to examine.

Filter_List

The list of elements that must be found or not found in Input_List.

Delimiter

Optional. Default = CR-LF. The delimiter between elements in the Input_List and Filter_List.

Exclude

Optional. Default = .F.

  • .T. = Return Input_List items that do not include elements of Filter_List.

  • .F. = Return Input_List items that include elements of Filter_List.

Description

The FILTER_STRING_MULTI() function returns all elements of an Input_List that either include or do not include all elements found in the Filter_List.

Examples

inlist = <<%a%

Alpha Software

The Alpha Project

Project Alpha

%a%

outlist = <<%b%

the

software

%b%

? filter_string_multi(inlist, outlist, crlf() , .f.)

= The Alpha Project

Alpha Software

? filter_string_multi(inlist, outlist, crlf() , .t.)

= Project Alpha
