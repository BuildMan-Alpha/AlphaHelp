A5_MEMO_EDITOR()

Syntax

Button as C = A5_MEMO_EDITOR( Text_Ptr as P, Title as C [, Readonly as L [, Insertion_Point as N ]] )

Argument

Description

Button

The name of the last button pressed.

Text_Ptr

A dot variable with a .Text sub-element.

Title

The title of the Memo Editor.

Readonly

Optional. Default = .F.

  • .T. = The text is read only.

  • .F. = The text can be edited.

Insertion_Point

Optional. Default = 1. The initial cursor location.

Description

The A5_MEMO_EDITOR() function displays the Memo Editor. It is a wrapper for A5_MEMO_EDITOR2(). It returns the name of the last button pressed.

Limitations

Desktop applications only.

Example

In this example narrative is the name of a memo field on a form.

dim response as N

response = ui_msg_box("Narrative","Do you want to type/edit a Narrative?", UI_YES_NO)

IF (response = UI_YES_SELECTED) THEN

    choice = a5_memo_editor(narrative.field_get() )

    IF choice = "save" then

        parentform.commit()

        narrative.refresh()

    END IF

END IF

This example directly updates the table.

dim tbl as P

dim response as N

dim p as P

dim p.text as C

response = ui_msg_box("Narrative","Do you want to type/edit a Narrative?", UI_YES_NO)

IF (response = UI_YES_SELECTED) THEN

    tbl = table.current()

    p.text = tbl.narrative

    choice = a5_memo_editor(p, "Narrative")

    IF choice = "save" then

        tbl.change_begin()

        tbl.narrative = p.text

        tbl.change_end(.T.)

        narrative.refresh()

    END IF

END IF
