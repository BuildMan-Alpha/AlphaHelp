SYS_KEY_DOWN()

Syntax

Result_Flag as L = SYS_KEY_DOWN( Key as C )

Argument

Description

Key

The name of a key. See SYS_KEYS_ENUM()for key names.

Description

The SYS_KEY_DOWN() function returns .T. (TRUE) if the named key is being held down.

Limitations

Desktop applications only.

Example

sys_key_down("ENTER")
