A5_DEL_FIELDS_FM_TABLE()

Syntax

Result_Flag as L = A5_DEL_FIELDS_FM_TABLE( Tablename as C, Field_List as C )

Argument

Description

Result_Flag

.T. = indicates success

.F. = inidicates failure

Tablename

The table contain the fields.

Field_List

A CR-LF delimited list of fields to remove from the table.

Description

The A5_DEL_FIELDS_FM_TABLE() function removes the fields in Field_List from Tablename.

Limitations

Desktop applications only.

Examples

? a5_del_fields_fm_table("cust_copy", "lastname")

= .T.
