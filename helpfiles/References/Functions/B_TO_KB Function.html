B_TO_KB()

Syntax

KBytes as N = B_TO_KB( Decimal_Number as N )

Argument

Description

KBytes

The equivalent number in Kbytes (multiples of 1024).

Decimal_Number

A number.

Description

The function converts bytes to kilobytes.

Example

? b_to_kb(1000000.9)

= 976.563379
