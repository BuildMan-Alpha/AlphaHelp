WUT()

Syntax

Output_String as C = WUT( Input_String as C )

Argument

Description

Output_String

The processed text string.

Input_String

A text string to capitalize.

Description

The WUT() function capitalizes the initial letter of each word in a string.

Supported By

Alpha Five Version 5 and Above

Example

? wut("Now is the time")

= "Now Is The Time"
