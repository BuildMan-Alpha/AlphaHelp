Color_Code

The RGB code for the color in the format "#R,G,B", where R, G and B are integers between 0 and 255.

Pen_Style

Optional. Default = "Solid". Values may be:

  • "Dash"

  • "Dot"

Width

Optional. Default = 1. The thickness of the line in pixels.

Description

The SET_PEN() function sets the line and border color, style, and thickness.

Limitations

Used only in the Code sections of UI_BITMAP_DRAW(), UI_SCREEN_DRAW(), and UI_PRINTER_DRAW().
