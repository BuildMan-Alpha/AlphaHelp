V6

V6

A helper function for use with FTP_SCRIPT_RUN() to construct a FTP script to transfer all of the files in a folder (including all sub-folders).

FTP_PUT_FILE()

FTP_PUT_FILE_MODAL()

V6

Sends a file in the foreground over a FTP connection.

FTP_PUT_FILE_MULTIPLE()

FTP_SCRIPT_RUN()

HTTP_DOWNLOAD()

V6

Creates a background thread which uses the HTTP protocol to download from Source_URL and save the data to Target_File.

HTTP_FETCH()

V5

DEPRECATED See HTTP_GET()

Gets the contents of a web page.

HTTP_GET()

V6

Downloads from a URL using HTTP/1.1 GET. The function supports cookies and returns a pointer with the parsed response from the server.

HTTP_PUT()

V10

Makes an HTTP PUT request.

HTTP_POST()

V6

Downloads from URL using HTTP/1.1 POST. The function returns a pointer with the parsed response from the server.

HTTP_POST_PAGE2()

V6

Extends HTTP_POST() to support XML transfers.

HTTP_UNCHUNK()

V6

Unchunks a "chunked" HTTP message.

RESOLVE_MIME_TYPE()

V7 - V11

Returns a properly formatted and legal mime type suitable for use in a page header or response header.

SYS_RESOLVE_URL()

V5

SYS_RESOLVE_URL()executes the function implied by Input_String. If Input_String is a URL, the function opens the URL in a new browser window. If it is an email address, it opens a new message in your default email application. If it is an FTP address, it opens your FTP client and attempts to open a session with the FTP host.

URLDECODE()

V5

Creates a character string by replacing the escape characters in the Encoded_URL_String with their normal ASCII equivalents.

URLENCODE()

V5

Creates a character string by converting all illegal URL address characters in the URL_String to their escape sequences.

SOCKETS.OPEN()

V6

Establishes an Internet connection.

NOTE: The SOCKETS object has been replaced by INET::Socket and INET::SSLSocket.

Alpha Anywhere provides the following socket methods. Prefix the following methods with the socket pointer returned by SOCKETS.OPEN().

Function

 

Description

<SOCKETS>.ACCEPT()

V6

Creates a Server-mode socket from an Listener-mode socket.

<SOCKETS>.ADDRESS_GET()

V6

Returns address information about the open socket.

<SOCKETS>.CLOSE()

V6

Closes a socket, if it is open.

<SOCKETS>.DATAGRAM_HANDLER()

V6

Sets up a named asynchronous handler

<SOCKETS>.DATAGRAM_KILL()

V6

Closes a datagram handler.

<SOCKETS>.DATAGRAM_SEND()

V6

Sends Data to Host at port number Port.

<SOCKETS>.DATAGRAMSEND

_WITH_RECEIPT()

V6

Sends data to a host and port, then waits for returned response.

V6

Returns .T. (TRUE) if the connection is open; .F. (FALSE) otherwise.

<SOCKETS>.OPTIONS_GET()

V6

Returns the options set for the socket.

<SOCKETS>.OPTIONS_SET()

V6

<SOCKETS>.PEEK()

V6

Reads data without removing it from the receive buffer.

<SOCKETS>.PORT_GET()

V6

<SOCKETS>.READ()

V6

V6

V6

V6

<SOCKETS>.WRITEB()

V6
