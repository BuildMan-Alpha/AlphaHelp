LOAD_LIBRARY()

Syntax

Handle as N = LOAD_LIBRARY( Library as C [, Run_Autoexec as L ] )

Argument

Description

Handle

A number from 0 to 255

Library

The path and filename of the library to load. The library will have an ".AEX" extension.

Run_Autoexec

Optional. Default = .T. (TRUE).

  • .T. = Runs the script named "autoexec" when the library is loaded.

  • .F. = Does not run the "autoexec" script.

Description

The LOAD_LIBRARY() function loads the named library (.AEX file).

Supported By

Alpha Five Version 5 and Above

Limitations

The LOAD_LIBRARY() function does not allow SCRIPT_PLAY()to run scripts.

Examples

fn = "c:\bugs\actionscript" + chr(92) + "aextest.aex"

? load_library(fn)

= 100
