TableFinder()

Syntax

TableName as C = TableFinder()

Description

The TableFinder() function displays the Table Finder dialog. Use this dialog to find tables that contain named fields.

  1. Enter part or all of the field name. You may use asterisk "*" to substitute for multiple adjacent characters or dollar "$" to substitute for any single character.

  2. Select the table of interest and click OK.

Example

? tableFinder()

= "Product sales summary"
