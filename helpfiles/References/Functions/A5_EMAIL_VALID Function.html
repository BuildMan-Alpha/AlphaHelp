A5_EMAIL_VALID()

Deprecated
This function has been deprecated. A5_IS_EMAIL_VALID() should be used as a replacement.

Syntax

Result_Flag as L = A5_EMAIL_VALID( Address as C )

Argument

Description

Result_Flag

Indicates whether the address is properly formatted.

Address

An email address.

Description

The A5_EMAIL_VALID() function determines if an email address is properly formatted. It does not validate the address itself.

Example

? a5_email_valid("john@somewhere.com")

= .T.
