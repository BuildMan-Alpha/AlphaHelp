Blob Functions and Methods

A BLOB variable is a "Binary Large OBject" variable. It can contain any data. Alpha Anywhere provides the following functions and methods for manipulating BLOB variables.

Prefix the following methods with the pointer, which is the same as the name of a BLOB variable.

Method

 

Description

.AND()

V5

Performs a bitwise AND operation on two binary objects.

.DUMP()

V5

Dumps the contents of a blob using the specification provided by the Format_String.

.NOT()

V5

Performs a bitwise NOT operation on a binary object.

.OR()

V5

Performs a bitwise OR operation on two binary objects.

.PEEK()

V5

Returns the value of a byte (8 bit) value at a Position in a blob.

.PEEKB()

V5

Returns a string of the bytes starting at Position and going for Length for the blob.

.PEEKBIT()

V5

Returns the value of the bit at Position.

.PEEKC()

V5

Returns a string of the characters starting at Position and going for Length for the blob.

.PEEKD()

V5

Returns the value of an unsigned double word (32 bit) value at a Position in a blob.

.PEEKN()

V5

Returns the value of an IEEE 8 byte floating point number at a Position in a blob.

.PEEKW()

V5

Returns the value of an unsigned word (16 bit) value at a Position in a blob.

.POKE()

V5

Sets a byte at Position in a blob to a Value.

.POKEB()

V5

Writes a blob at Position.

.POKEBIT()

V5

Sets a bit at Position in a blob to a value.

.POKEC()

V5

Sets characters in a blob starting at specified Position to the string.

.POKED()

V5

Sets a 32 bit double word at Position in a blob to a value.

.POKEN()

V5

Sets a IEEE 8 byte floating point number at Position in a blob to a value.

.POKEW()

V5

Sets a 16 bit word at Position in a blob to a value.

.REPLICATE()

V5

Writes data into a blob multiple times.

.RESIZE()

V5

Resizes a blob variable.

.ROTATE()

V5

Rotates a bitmap.

.SIZE()

V5

Returns the size of a blob in bytes.

.XOR()

V5

Performs a bitwise XOR operation on two binary objects.

The following functions and methods are useful for manipulating blobs.

Function

 

Description

BlobCompare()

V8

Indicates which of two blob variables is larger.

BlobEqual()

V8

Returns TRUE if two blob variables are identical.

CHAR_TO_BLOB()

V5

Converts a character string into a blob variable.

FILE.TO_BLOB()

V5

Reads a binary file from disk into a blob variable. The data can be in either Windows Bitmap (.BMP) or JPEG (.JPG) formats.

INCREMENT_VALUE()

V5

Increments the value of a variable by 1.

PROPERTY_FROM_BLOB()

V6

Initializes the values in a "dot" variable with a blob that was originally created using the PROPERTY_TO_BLOB() function

PROPERTY_TO_BLOB()

V6

Converts a "dot" variable to a blob variable.

STRUCT_GET()

V5

Converts the contents of a packed structure into a blob.

STRUCT_SET()

V5

Creates a packed structure based on a DECLARESTRUCT definition, and optionally reads blob data to initialize the packed structure.

<TBL>.RECORD_DATA_GET()

V5

Copies all the field values from Record_Number (or the current record if Record_Number is not specified) to a blob variable.

<TBL>.RECORD_DATA_SET()

V5

Copies all the field values from the blob variable, Record_Data, into the current record.

FILE.FROM_BLOB()

V6

Creates a Windows Bitmap (.BMP) or JPEG (.JPG) file from binary data.
