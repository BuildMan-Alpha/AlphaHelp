A5_RUN_EMMJ()

Syntax

A5_RUN_EMMJ( Table_Set_Name as C )

Argument

Description

Table_Set_Name

The name of a table or set containing a Netmailer maintenance job.

Description

The A5_RUN_EMMJ() function displays the Choose Maintenance Job dialog, which allows you to select and run a saved Netmailer maintenance job.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

a5_run_emmj("customer.dbf")
