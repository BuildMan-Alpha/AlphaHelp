A5_REPORTERRORTOUSER()

Syntax

A5_REPORTERRORTOUSER( Message as C )

Argument

Description

Message

Error message.

Description

If you have an Alpha Anywhere function that can be called by a .A5W page then it is important that the function does not try to display a UI_MSG_BOX() to report an error, as this is not allowed in a background thread.

The solution is to use the A5_REPORTERRORTOUSER() function.

  • If the user is running Version 5, it puts up a message box.

  • If the user is running Version 6 and is in the foreground thread, it puts up an error message.

  • If the user is running Version 6 and is in the background thread (i.e. in an executing .A5W page), then it generates an error that will be displayed in the user's page.

    Example

    A5_REPORTERRORTOUSER("Invalid value for this field")
