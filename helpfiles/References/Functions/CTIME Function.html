CTIME()

Syntax

Time_Equivalent as C = CTIME( Time_Value as T )

Argument

Description

Time_Equivalent

A character string representation of a time value that can be sorted. The format is YYYMMDDhhmmssP

Time_Value

A time value. For example: 02/15/2006 09:27:16 91 am

Description

The CTIME() function converts a time value into a character value that can be used in a sort expression.

Examples

? ctime(now() )

= "20060215092839365"

dim tt as T

dim na as C = "Fred"

tt = now()

? tt

= 02/15/2006 09:27:16 91 am

? na + ctime(tt)

= "Fred20060215092716912"
