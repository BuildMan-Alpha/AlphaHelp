CShortTime()

Syntax

Short_Time_Equivalent as C = CShortTime( Short_Time_Value as Y )

Argument

Description

Short_Time_Equivalent

A character string representation of a short time value that can be sorted. The format is hhmmssP

Short_Time_Value

A short time value. For example: 09:27:16 91 am

Description

The CShortTime() function converts a short time value into a character value that can be used in a sort expression.

Supported By

Alpha Five Version 7 and Above

Examples

dim st as Y

st = now()

? st

= 09:38:35 80 am

? cshorttime(st)

= "093835803"
