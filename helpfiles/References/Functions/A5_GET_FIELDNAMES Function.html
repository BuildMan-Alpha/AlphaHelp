A5_GET_FIELDNAMES()

Syntax

Field_List as C = A5_GET_FIELDNAMES( Table_Set_Name as C [, Flags as C [, Top_Level_Only as L [, Fully_Qualify_Parent_Fields as L ]]] )

Argument

Description

Field_List

A CR-LF delimited list of field names from the table or set.

Table_Set_Name

The name of a table or set, including the ".DBF" or ".SET" file name extension.

Flags

A character, numeric, logical, or date data display format. Options are:

  • "n" = field name

  • "N" = uppercased field name

  • "t" = simple field type

  • "T" = full field type

  • "W" = field width

  • "D" = field decimal

  • "R" = field rule (INCREMENT/CALCULATED=)

Top_Level_Only

Optional. Default = .F.

  • .T. = Includes top level table and one-to-one linked tables.

  • .F. = Includes all tables in the set.

Fully_Qualify_Parent_Fields

Optional. Default = .F.

  • .T. = places the table name before the names of parent table fields.

  • .F. = does not place the table name before the names of parent table fields.

Description

The A5_GET_FIELDNAMES() function returns a list of the fields in a table or set.
