A5_EDIT_FIELDRULES()

Syntax

A5_EDIT_FIELDRULES( Table_Name as C )

Argument

Description

Table_Name

The name of the table whose field rules you wish to edit.

Description

The A5_EDIT_FIELDRULES() function displays the Field Rules dialog box for the selected table.

Limitations

Desktop applications only.

Example

a5_edit_fieldrules("Customer")
