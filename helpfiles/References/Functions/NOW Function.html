NOW()

Syntax

Date_Time as T = NOW( [ Format_Flag as L ] )

Argument

Description

Format_Flag

Optional. Default = .F. If flag is .T., then the time is returned as GMT.

Description

NOW() returns the current date and time, correct to the millisecond, in the Time data type.

Contrast this with the TIME() function which returns the result as a character string. The TIME() function does not return milliseconds.

Supported By

Alpha Five Version 5 and Above

Examples

now() -> 01/16/2003 10:24:51 09 am

now(.T.) -> 01/16/2003 03:24:51 09 pm

time("MM/dd/yyyy 0h:0m:0s 2 am", now(.T.)) -> "01/16/2003 03:24:51 09 pm"

The following Xbasic sequence can be typed in the Interactive window.

dim t1 as T

dim t2 as T

t1 = now()

t2 = now()

? t2-t1

= 5.440000

dim t3 as T

t3 = time()

ERROR: variable type mismatch

(error because time() returns a character value)
