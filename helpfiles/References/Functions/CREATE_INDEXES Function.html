CREATE_INDEXES()

Syntax

Result  as P = CREATE_INDEXES( Table_Name as C, Index_Definitions as C [, Show_Xbasic as L ] )

Argument

Description

Result

A pointer variable that contains the result of the operation. Result has the following properties:

 

Description

. HasError

  • .T. = indexes could not be created

  • .F. = indexes were created

. ErrorText

If .HasError is .T., then contains the error message.

. Xbasic

Contains the Xbasic code if Show_Xbasic is .T.

Table_Name

The name of the table containing the indexes to be created.

Index_Definitions

A list of index definitions created by GET_INDEX_DEFINITIONS().

Show_Xbasic

Optional. Default = .F.

.T. = Show Xbasic code generated to create indexes.

Description

Creates indexes for a table based on a CR-LF delimited list created by GET_INDEX_DEFINITIONS().

Examples

The following example retrieves index definitions from a table, then (after they were manually deleted), recreates the indexes.

list = Get_Index_Definitions("index_entries")

? create_indexes("index_entries", list, .t.)

= ErrorText = ""

HasError = .F.

xbasic = t.index_create_begin("Entry","PADR(LEFT(ENTRY,40),40,\" \") + PADR(LEFT(SUBENTRY,40),40,\" \") + PADR(LEFT(TOPIC,40),40, \" \")","","")

t.index_add("Index_Id","INDEX_ID","","U")

t.index_create_end()
