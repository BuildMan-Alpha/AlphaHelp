A5_GENERICSEARCHREPLACE()

Syntax

A5_GENERICSEARCHREPLACE( Text_Pointer as P [, Tab_Page as C ]] )

Argument

Description

Text_Pointer

A dot variable containing the text displayed in the control.

Tab_Page

Optional. Specifies which tab on the dialog will have initial focus.

Description

The A5_GENERICSEARCHREPLACE() function displays the Find/Replace/Goto dialog box for a text object on an Xdialog form.

Limitations

Desktop applications only.

Examples

dim ptext as P

dim ptext.text as C

dim ptext.object as P

ptext.text = "some text"

ui_dlg_box("title",<<%dlg%

.20ptext;

;

%dlg%,<<%code%

if (a_dlg_button = "Find") then

    a5_genericsearchreplace(ptext)

    a_dlg_button = ""

end if

%code%)
