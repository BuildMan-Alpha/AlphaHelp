UI_GET_FILES_MULTIPLE()

Syntax

File_List as C = UI_GET_FILES_MULTIPLE( [ Dialog_Title as C [, Default_Directory as C [, File_Filter as C [, Default_File_List as C ]]]] )

Argument

Description

Dialog_Title

Optional. Default = "Select Multiple Files". The title of the dialog box.

Default_Directory

Optional. Default = "(C:\*.*)" The initial directory.

File_Filter

Optional. Default = "*.*". A filter that selects files.

Default_File_List

Optional. No default. A list of pre-selected file.

Description

The UI_GET_FILES_MULTIPLE() function prompts for filenames. Allows user to select multiple filenames.

Limitations

Desktop applications only.

Example

ui_get_files_multiple("Multiple Files", "c:\documents", "(*.txt)")
