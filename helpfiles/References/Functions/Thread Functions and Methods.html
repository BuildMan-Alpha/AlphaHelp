Thread Functions and Methods

Alpha Anywhere provides the following Thread functions and methods.

Function

Description

THREAD_COUNT()

Returns the number of running threads.

THREAD_CREATE()

Creates a named thread.

THREAD_ENUM()

Returns a list of the names of running threads.

THREAD_EVENT()

For internal use only.

THREAD_GET()

Returns the name of the current thread.

THREAD_INTERRUPT()

For internal use only.

THREAD_IS_RUNNING()

Returns TRUE if the named thread is running.

THREAD_NAME_CREATE()

Returns a unique variant of the thread name based on the suppled thread name.

THREAD_VARIABLES()

Returns a pointer to all of the variables of a specific thread.

THREAD_WAIT()

For internal use only.

THREADMON()

Opens a small window listing the running threads.

THREAD.CURRENT()

Returns a pointer to the current thread.

THREAD.GET()

Returns a pointer to the named thread.

THREAD.UI_DISABLED()

Returns the disabled status of the current thread.

XBASIC_ERROR_LOG()

Defines an error log for this thread.
