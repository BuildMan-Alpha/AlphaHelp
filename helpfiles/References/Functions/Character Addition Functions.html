Character Addition Functions

Alpha Five provides the following character addition functions.

Function

 

Description

*CONCAT_LINES Function

 V5

Appends lines to a list, adding CR-LF delimiters, and eliminating blank lines. This function is much faster than the string concatenation operators.

+ Concatenation Operator

 V5

Appends Operand2 (a character value) to Operand1. If Operand1 is not a character value, it is automatically converted to a character value before the concatenation operation.

- Concatenation with Trim Operator

 V5

Removes trailing spaces from Operand1 and then appends Operand2 to Operand1.

INCREMENT_VALUE Function

V5

Increments the value in the variable by 1.
