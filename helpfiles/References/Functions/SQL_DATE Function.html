SQL_DATE()

Syntax

Formatted_Date as C = SQL_DATE( Date as D )

Argument

Description
Formatted_Date A formatted date string.
Date A date value.

Description

The SQL_DATE() function generates a character string with the format: {YYYY-MM-DD}.

Example

? sql_date(date() )

= "{2006-03-22}"
