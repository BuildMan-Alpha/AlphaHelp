START_POLY()

Syntax

START_POLY( XCoord as N, YCoord as N )

Argument

Description
XCoord The horizontal coordinate of the first point of the polygon.
YCoord The vertical coordinate of the first point of the polygon.

Description

The START_POLY() function begins a polygon by setting its initial point.

Limitations

Used only in the Code sections of UI_BITMAP_DRAW(), UI_SCREEN_DRAW(), and UI_PRINTER_DRAW().

Example

ui_bitmap_create("test", 4, 2)

ui_bitmap_draw("test",<<%code%

inner_Rect(0, 0, 4, 2)

set_brush("green")

set_pen("black")

start_poly(2, .5)

lineto(3, 1)

lineto(1, 1.5)

poly(2, .5)

%code%)

ui_dlg_box("","{image=test}")
