A5_GENERIC_PRINT()

Syntax

A5_GENERIC_PRINT( Text_Pointer as P )

Argument

Description

Text_Pointer

A dot variable containing the text displayed in the control.

Description

The A5_GENERIC_PRINT() function prints the text in a text object in an Xdialog form.

Limitations

Desktop applications only.

Examples

dim ptext as P

dim ptext.text as C

dim ptext.object as P

ptext.text = "some text"

ui_dlg_box("title",<<%dlg%

[.20ptext];

;

%dlg%,<<%code%

if (a_dlg_button = "Print") then

    a5_generic_print(ptext)

    a_dlg_button = ""

end if

%code%)
