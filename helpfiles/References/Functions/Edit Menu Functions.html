Edit Menu Functions

Function

 

Description

Undo

Restores the edit buffer to its state before the last keystrokes.

Redo

Restores the edit buffer to its state before an Undo operation.

Cut

Removes the selected page elements to the Windows Clipboard.

Copy

Copies the selected page elements to the Windows Clipboard.

Paste

Pastes the contents of the Windows Clipboard to the cursor location.

Delete

Removes the selected page elements.

Select All

Selects all the elements in the edit buffer.

Copy to Scrapbook

Only available when the Source tab is selected. Copies the selected text to the Scrapbook library of the <span class=Screen>Code Library</span>.

Paste from Scrapbook

Pastes the currently selected entry in the Scrapbook library of the <span class=Screen>Code Library</span> at the cursor location.

Find

Only available when the Source tab is selected. Displays the <span class=Screen>Find/Replace/Goto</span> dialog, which allows the operator to search for and replace text in the edit buffer.

Check Spelling

Displays the <span class=Screen>Spell Check</span> dialog, which supports interactive spell checking of the contents of the edit buffer.

Limitations

Web publishing applications only.
