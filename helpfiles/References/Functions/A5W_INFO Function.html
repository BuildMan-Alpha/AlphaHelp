A5W_INFO()

Syntax

HTML_Code as C = A5W_INFO( [ Total as L ] )

Argument

 

Description

HTML_Code

 V6

Information about Application Server settings and variables.

Total

 V7

Optional. Default = .T.

  • .T. = Return a complete HTML page.

  • .F. = Return only formatted table data to insert into another HTML page.

Description

The A5W_INFO() function displays a summary page showing all server settings and variables. This function only works on a page submitted to the Application Server.

Limitations

Web publishing applications only.

Examples

These are several examples of how the function could be used in an A5W page.

<%a5 a5w_info() %>

...
<%a5 a5w_info(.f.) %>

...

<%a5 ? a5w_info() %>
...
<%a5
x = a5w_info()
? x %>
...
<html><body bgcolor="#ff0000">
<h2>Below is the output from a5w_info() </h2>
<%a5 a5w_info(.f.) %>
</body></html>
