BlobEqual()

Syntax

Result_Flag as L = BlobEqual( Blob1 as B, Blob2 as B )

Argument

Description

Result_Flag

.T. = Both blob variables contain the same data and have the same lengths.

.F. = The blob variables are different with respect to contents and/or length.

Blob1

A variable containing binary data.

Blob2

A variable containing binary data.

Description

The BlobEqual() function returns TRUE (.T.) if two blob variables have the same content and are the same length.

Limitations

Desktop applications only.
