RESOLVE_MIME_TYPE()

Deprecated
This function is deprecated as of V11. It will not be available in future versions.

Syntax

Mime_Type as C = RESOLVE_MIME_TYPE( File_Type as C )

Argument

Description

Mime_Type

A formatted definition of a mime type suitable for use in a page header or response header. The response is "text/html" if the File_Type cannot be identified.

File_Type

A Windows file type in the format: ". XYZ " .

Description

The RESOLVE_MIME_TYPE() function returns a properly formatted and legal mime type suitable for use in a page header or response header.

Supported By

Alpha Five Version 7 through Alpha Five Version 11

Limitations

Examples

If you preview the following page, it will display "text/xml".

<html>

<head>

<meta name="generator" content="Alpha Five HTML Editor">

<title></title>

</head>

<body>

<p> </p>

<%a5 ? RESOLVE_MIME_TYPE( ".xml" ) %>

</body>

</html>
