Label Functions and Methods

Alpha Five provides the following Label functions.

Method

 

Description

A5.LABEL_ENUM()

V5

Returns a list of label layouts.

A5.LABEL_LOCATION()

V5

Returns the name of the file that contains the label layout.

LABEL.PREVIEW()

V5

Displays the label and the selected records in the Print Preview dialog.

LABEL.SAVEAS()

V5

Prints Layout_Name to file in PDF, HTML, RTF, or TXT formats.

PREVIEW_LABEL()

V5

Previews labels as they would appear when printed.

PRINT_LABEL()

V5

Creates the selected labels and then displays the Print dialog box.
