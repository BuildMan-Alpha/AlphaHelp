ON_XBASIC_IDLE()

Syntax

ON_XBASIC_IDLE( Code as C, Flags as C )

Argument

Description

Code

The Xbasic code to be run placed into a character string.

Flags

  • "R" = If you register duplicate code, Alpha Five only executes the code once.

  • "N" = Use this flag when issuing the command from a modal dialog box.

Description

The ON_XBASIC_IDLE() function register Xbasic code that will be run the next time Alpha Five is idle. (i.e. after any scripts that are running asynchronously have finished). You would typically use this command if you have script that is running into timing problems.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

This example calls on_xbasic_idle() twice with the same code, but the code block executes only once (assuming that there was no idle after the first on_xbasic_idle() was called, which there wouldn't have been).

codeTxt = <<%code%
file.remove(tempfilename)
%code%


on_xbasic_idle(codeTxt,"R")
on_xbasic_idle(codeTxt,"R")
