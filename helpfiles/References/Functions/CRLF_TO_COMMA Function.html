CRLF_TO_COMMA()

Syntax

CR-LF_List as C = CRLF_TO_COMMA( Comma_List as C )

Argument

Description

Comma_List

A list of items, each separated by a comma.

CR-LF_List

A list of items, each separated by CR-LF codes.

Description

The COMMA_TO_CRLF() function converts a CR-LF delimited list to a comma delimited list.

Examples

dim list as C

list = <<%str%

aaron

bob

charles

david

%str%

? list

= aaron

bob

charles

david

? crlf_to_comma(list)

= "aaron,bob,charles,david"
