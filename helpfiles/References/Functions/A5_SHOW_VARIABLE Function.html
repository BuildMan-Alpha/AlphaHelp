A5_SHOW_VARIABLE()

Syntax

A5_SHOW_VARIABLE( Variable_Frame as P [, Dialog_Title as C ] )

Argument

Description

Variable_Frame

The variable or interest or a pointer to the variable frame, such as  LOCAL_VARIABLES(), GLOBAL_VARIABLES()or SESSION_VARIABLES().

Dialog_Title

Optional. The title of the resulting dialog box.

Description

The A5_SHOW_VARIABLE() displays a dialog box with tabs showing the values of each of the currently defined variables.

Limitations

Desktop applications only.

Examples

dim aa as C

aa = "Fred"

? a5_show_variable(local_variables() , "Currently Defined Variables")

dim hah as C

hah = "Hello"

a5_show_variable(hah)
