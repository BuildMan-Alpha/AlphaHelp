Type Checking Functions

Alpha Anywhere provides the following Type Checking functions.

Function

Description

ISALPHA()

Returns . T. (TRUE) if the first character of the specified Character_String is a letter; otherwise, returns .F. (FALSE).

ISBLANK()

Returns .T. (TRUE) if the field identified in the Field_Name is blank; otherwise, it returns .F. (FALSE).

ISDATE()

Returns .T. (TRUE) if Date_String is a valid date; otherwise, returns .F. (FALSE)..

ISDIGIT()

Returns .T. (TRUE) if the first character of Character_String is a number; otherwise, returns .F. (FALSE).

ISLOWER()

Returns TRUE if the first character of the Character_String is lowercase; otherwise, returns .F. (FALSE).

ISNOTBLANK()

Returns .T. (TRUE) if Field_Name contains a value; otherwise, returns .F. (FALSE).

ISNULL()

Returns .T. (TRUE) if Character_String is blank (i.e. contains a NULL value); otherwise, returns .F. (FALSE).

ISNUMBER()

Returns .T. if a character string contains only numbers; otherwise, returns .F. (FALSE).

IsShortTime()

Returns .T. if the Time_String is a valid representation of a short time value.

ISTIME()

Returns .T. if the Time_String is a valid representation of a time value.

ISUPPER()

Returns .T. (TRUE) if the first character of the Character_String is uppercase; otherwise, returns .F. (FALSE).
