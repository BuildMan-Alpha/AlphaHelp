DTOC()

Syntax

Formatted_Date as C = DTOC( Date as D [, Format as C ] )

Formatted_Date as C = DTOC( Date_Time as T [, Format as C ] )

Argument

Description

Formatted_Date

The Date_Value formatted into a character string.

Date

A variable containing a date value or a character string containing a legitimate representation of a date value ( ISDATE (Date_Value) = .T.).

Date_Time

A variable containing a date time value.

Format

Optional. Default = "MM/DD/YYYY". Refer to Predefined Date Formats.

Description

DTOC() returns a character string in the specified Format for the specified Date or Date_Time value. DTOC() is useful for indexing a table when month order is more important than chronological date order (which orders first by year, then month, then day).

If you have a multi-field index involving both date and character fields, use the CDATE() function if you want a chronological date order.

Supported By

Alpha Five Version 5 and Above

Example

dim dd as D

dd = date()

? dtoc(dd,"1-")

= "01-06-2004"

? dtoc(dd,"4")

= "January 6, 2004"
