WORD_FROM_POS()

Syntax

Word_Number as N = WORD_FROM_POS( Input_String as C, Position as N [, Word_Delimiter as C ] )

Argument

Description

Input_String

A character string.

Position

A character position within the Input_String.

Word_Delimiter

Optional. Default = space (" "). The character string that separates words.

Description

Converts the position in a text string to the number of the word in the text string. If Position occurs on a Word_Delimiter, Word_Number is 0.

Examples

word_from_pos("This is a test", 7) -> 2
