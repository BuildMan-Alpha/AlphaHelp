Dictionary Functions and Methods

Alpha Anywhere provides the following Dictionary functions and methods. The DICTIONARY.CREATE()and DICTIONARY.OPEN() methods return standard table pointers, which means that you may use other table methods to work with the dictionary.

Method

Description

A5_DICTIONARY_READ()

Reads the Xbasic code associated with an object into a character variable.

A5_DICTIONARY_WRITE()

Writes the Xbasic code associated with an object into a dictionary.

DICTIONARY.CREATE()

Creates a new data dictionary.

DICTIONARY.EXTERNAL_RECORD_CONTENT_GET()

Returns a CR-LF delimited list of filtered records from a dictionary.

DICTIONARY.OPEN()

Opens the data dictionary for the current table or set, returning an object pointer, , for the data dictionary file. A dictionary is a table, so Table Functions and Methods also apply.

DICTIONARY.STRUCTURE_COPY()

Copies structure information from one dictionary to another.

DICTIONARY.UPDATE()

Updates a data dictionary.
