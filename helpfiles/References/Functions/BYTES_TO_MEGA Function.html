BYTES_TO_MEGA()

Syntax

Formatted_Count as C = BYTES_TO_MEGA( Bytes as N [, Decimal_Places as N ] )

Argument

Description

Bytes

The count of bytes to reformat.

Decimal_Places

Optional. The number of decimal places to display.

Description

The BYTES_TO_MEGA() function returns the KB, MB, GB, or TB equivalent of a byte number.

Examples

? bytes_to_mega(65536)

= "64 KB"

? bytes_to_mega(9023482)

= "8.6 MB"

? bytes_to_mega(9023482, 4)

= "8.6055 MB"

? bytes_to_mega(390482509348, 2)

= "363.67 GB"
