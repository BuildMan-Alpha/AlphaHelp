PDF_MERGE()

Syntax

Result_Flag as L = PDF_MERGE( Base_File as C, Merge_File as C [, Result_File as C [, flagMergeAllPages as L [, flagMergeFileOnTop as L ]]] )

Argument

Description

Result_Flag

.T. = Merge was successful.

.F. = Merge was not successful.

Base_File

The file that receives content from Merge_File.

Merge_File

The content to add to Base_File.

Result_File

Optional. Default = "" (use Base_File ). Result_File has same number of pages as  Base_File. If Result_File is blank, then content is merged with Base_File.

flagMergeAllPages

Optional. Default = .T. If Merge_File has fewer pages than Base_File, flagMergeAllPages controls whether Merge_File pages are repeated so that content is merged onto all pages in Base_File.

  • .T. = Repeat pages.

  • .F. = Do not repeat pages.

flagMergeFileOnTop

Optional. Default = .F.

Description

The PDF_MERGE() function combines Base_File with Merge_File to create Result_File. If Result_File is blank, then content is merged with Base_File. Note: If you are adding a logo or watermark image, it will be the Merge_File.

Limitations

Examples

? pdf_merge("c:\inv.pdf", "c:\cus.pdf")

= .T.
