FILE_DROP_FROM_DB()

Syntax

FILE_DROP_FROM_DB( Table_Filename as C )

Argument

Description

Table_Filename

The full filename and path of the Alpha table being removed.

Description

FILE_DROP_FROM_DB() drops the table (.dbf) or set (.set) from the database (the .adb file) so that the Table_Filename no longer appears in the Control Panel.

Limitations

Desktop applications only.

Examples

This script drops a table from the Control Panel.

FILE_DROP_FROM_DB("c:\Alpha Five\data\customer.dbf")
