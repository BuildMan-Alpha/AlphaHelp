UI_POPUP_MENU_WIDTH()

Syntax

Width as N = UI_POP_MENU_HEIGHT( Menu_Name as C [, Height_Limit as N ] )

Argument

Description

Width

The width of the menu in inches.

Menu_Name

The name of a menu.

Height_Limit

Optional. Height limit for the menu.

Description

The UI_POPUP_MENU_WIDTH() function returns the width of a menu.

Limitations

Desktop applications only.

Example

? ui_popup_menu_width("menu1")

= 1.229167
