TOP_RECORDS_PCT_LIST()

Syntax

Record_List as C = TOP_RECORDS_PCT_LIST( Order_Expn as C, Percent as N [, Filter_Expn as C ] )

Argument

Description

Record_List

A list of the record numbers from the current table.

Order_Expn

Optional. Default = "" (record number order).

A character order expression that sorts selected records.

Percent

A number from 0 to 100, representing the percentage of the table's records to retrieve.

Filter_Expn

Optional. Default = ".T." (All records).

A character filter expression that evaluates to a logical value and selects records from the table.

Description

The TOP_RECORDS_PCT_LIST() function returns a list of the record numbers from the current table of the first percent records as defined the Order_Expn and Filter_Expn arguments. This function should be run from an open layout.

Supported By

Alpha Five Version 5 and Above

Example

dim frm as P

frm = form.view("Customer Information")

? top_records_pct_list("lastname", 25)

= "32 56 55 52 54 18 48 8 51 20 30 39 34 10 5 "
