A5_OPEN_HTMLFILE()

Syntax

A5_OPEN_HTMLFILE( Filename as C [, TabName as C ] )

Argument

Description

Filename

The path to and filename of the HTML file to open.

TabName

Optional. The options are:

  • "source"

  • "wysiwyg"

Description

The A5_OPEN_HTMLFILE() function opens Alpha Anywhere's HTML Editor and loads the specified file. You can specify which tab on the editor should initially be selected.

Limitations

Desktop applications only.

Example

A5_OPEN_HTMLFILE("C:\Documents\AlphaFiveHelp\User_Manual\Adding_Fields_and_Text.htm")
