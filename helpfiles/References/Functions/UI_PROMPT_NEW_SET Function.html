UI_PROMPT_NEW_SET()

Syntax

Set_Name as C = UI_PROMPT_NEW_SET( [ Default_Name as C [, Warn as L ]] )

Argument

Description

Set_Name

The name of the new set.

Default_Name

The default name to place into the Specify Set Name dialog box.

Warn

Not used.

Description

The UI_PROMPT_NEW_SET() function displays the Specify Set Name dialog box, then returns the fully qualified name of the new set.

Limitations

Desktop applications only.

Example

? ui_prompt_new_set()

= "C:\Databases\Expressions Test\test_set.set"
