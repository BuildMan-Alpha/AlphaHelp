CREATE_TABLE()

Syntax

Result_Flag as L = CREATE_TABLE( Filename as C, Field_List as C )

Argument

Description

Filename

The file name of the .DBF file to create.

Field_List

A CR-LF delimited list of field names in the format: Fieldname, type, width, decimal_places

Description

CREATE_TABLE() is a high level Xbasic function to create a new table. This function is easier to use than the low level, TABLE.CREATE_BEGIN(), TABLE.FIELD_ADD(), and TABLE.CREATE_END()methods.

See TABLE.CREATE_BEGIN()for a list of valid field type codes.

Example

Create a new table.

fields = <<%a%

fname,c,20

lname,c,20

dob,d

notes,m

%a%

create_table("c:\data\newtable.dbf", fields)
