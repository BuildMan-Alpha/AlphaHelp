*TRANSPOSE()

Syntax

Output_String as C = *TRANSPOSE( Input_String as C, firstPosition as N, secondPosition as N )

Argument

Description

Output_String

A character string.

Input_String

A non-null character string.

firstPosition

An integer greater than 0 and less than len(Input_String).

secondPosition

An integer greater than 0 and less than len(Input_String).

Description

*TRANSPOSE() returns a character string by reversing the characters in the Input_String at positions First_Position and Second_Position.

Example

? *TRANSPOSE("Edward", 3, 5)

= "Edrawd"
