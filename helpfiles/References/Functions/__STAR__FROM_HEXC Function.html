*FROM_HEXC()

Syntax

Output_String as C = *FROM_HEXC( Hex_Value as C )

Argument

Description

Hex_Value

A hexadecimal number value.

Description

The *FROM_HEXC() converts an hexadecimal string to a character value.

Supported By

Alpha Five Version 5 and Above

Example

? *FROM_HEXC("41")

=   "A"
