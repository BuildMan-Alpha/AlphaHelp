UN_PAREN()

Syntax

Output_String as C = UN_PAREN( Input_String as C )

Argument

Description

Input_String

A string expression containing parentheses.

Description

UN_PAREN() creates a character string by removing unnecessary pairs of open and close parentheses. If the Input_String has unbalanced parentheses, the function returns the original Input_String.

Example

un_paren("(a = 1) .and. (b = 2?)") -> "(a = 1) .and. (b = 2)"

un_paren("(((a = 1) .and. (b = 2)") -> "(((a = 1) .and. (b = 2)"
