EMAIL_SET_ACTIVE()

Syntax

Result_Flag as L = EMAIL_SET_ACTIVE( Profile_Name as C )

Argument

Description

Result_Flag

.T. = Profile found.
.F. = Profile not found.

Profile_Name

The name of an email profile.

Description

The EMAIL_SET_ACTIVE() function sets Profile_Name to be the active profile.

Example

? email_set_active("AT&T Broadband")

= .T.

? email_get_active()

= "AT&T Broadband"

? email_set_active("Nonexistent

Profile")

= .F.

? email_get_active()

= "AT&T Broadband"
