SAVE_TO_FILE()

Syntax

SAVE_TO_FILE( Data_String as C, Filename as C [, Append as L ] )

Argument

Description

Data_String

The data to save into the file.

Filename

The fully qualified name of a file.

Append

Optional. Default = .F. (overwrite file)

  • .T. = Append data.

  • .F. = Overwrite existing data.

Description

SAVE_TO_FILE() saves a character string to a file specified by filename. This high level command is easier than using the low level <FILE>.CREATE()and <FILE>.WRITE()methods.

Supported By

Alpha Five Version 5 and Above

Example

This script saves the contents of the variable, xx2 to a text file.

save_to_file(xx2, "f:\a5\test2.txt")
