FILES_IN_TABLE()

Syntax

File_List as C = FILES_IN_TABLE( Table_Name as C )

Argument

Description

Table_Name

The name of a table. The file extension ".dbf" is assumed.

Description

FILES_IN_TABLE() returns a CR-LF delimited character string list containing the fully qualified names of the components of the table.

Example

dim cs as C

cs = files_in_table("customer")

cs   ->   c:\databases\alphasports official\customer.dbf

c:\databases\alphasports official\customer.fpt
