*BIN_TO_RTF()

Syntax

RTF_Text as C = *BIN_TO_RTF( RTF_Binary_Text as B )

Argument

Description

RTF_Text

An ASCII encoded representation of the RTF text.

RTF_Binary_Text

The contents of a RTF memo field.

Description

The *BIN_TO_RTF() function converts binary RTF text to ASCII encoded RTF text.

Limitations

Applies to Alpha Anywhere tables only.

Examples

The following example reads RTF binary text, writes it to an HTML file, and displays it in a browser. Refer to *BIN_TO_HTML()for a picture of the original RTF memo field.

dim tbl as P

dim rtftext as C

tbl = table.open("customer")

tbl.fetch_first()

rtftext = *bin_to_rtf(tbl.rtfmemo)

? rtftext

= "{\rtf1\ansi \deff0{\colortbl;\red0\green0\blue255;\red0\green0\blue0;\red255\green0\blue0;}{\fonttbl{\f0\fswiss Arial;}{\f1\fnil Verdana;}}{{\b \f1 \cf1 \fs20 Michael Baker}{\f1 \cf1 \fs20 }{\f1 \cf2 \fs20 lives at 14 Winman Street in}{\b \f1 \cf1 \fs20 }{\b \f1 \cf3 \fs20 Natick, Massachusetts}}}"
