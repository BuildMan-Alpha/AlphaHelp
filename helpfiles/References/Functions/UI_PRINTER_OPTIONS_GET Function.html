UI_PRINTER_OPTIONS_GET()

Syntax

UI_PRINTER_OPTIONS_GET( Printer_Name as C, Options as P )

Argument

Description

Printer_Name

The name of a printer.

Options

A dot variable that returns printer options.

Description

The UI_PRINTER_OPTIONS_GET() function retrieves a list of printer options.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

dim ptr as P

dim ptr.dummy as L = .t.

ui_printer_options_get("\\doc-build\Brother MFC9600 on Ne03:", ptr)
