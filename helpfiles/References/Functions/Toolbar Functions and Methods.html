Toolbar Functions and Methods

Alpha Five provides the following Toolbar functions and methods. Use the TOOLBAR.GET() or TOOLBAR.OPEN() functions to obtain the pointer.

Method

 

Description

TOOLBAR.GET()

V5

Returns the pointer to the specified Toolbar_Name.

TOOLBAR.OPEN()

V5

Opens the specified Toolbar_Name and returns the object pointer.

.ToolbarControlEnable()

V5

Enables or disables a toolbar element.

Function

 

Description

A5_SYSTEM_TOOLBAR_SHOW()

V5

Displays a previously loaded toolbar.

A5_TOOLBAR_DEF_LOAD()

V5

Loads a previously created toolbar.

A5_TOOLBAR_OPEN()

V5

Opens and displays the named toolbar.

TOOLBAR_EDIT()

V5

Opens the Toolbar Editor for the specified Toolbar_Name.

UI_TOOLBAR()

V5

Displays a toolbar populated with the selections provided in Choice_List and returns the user's selection.
