a5_load_speed_glossary()

Syntax

a5_load_speed_glossary([C dbfname [,C OldValueFieldName [,C NewValueFieldName ]]])

Argument Description
C dbfname (optional) DBF Table to load, or '<clear>', default is system speed_glossary table
C OldValueFieldName (optional) Field name for input value
C NewValueFieldName (optional) Field name for expanded value

Description

Loads the speed glossary from disk into memory. If no value is specified for dbfname then the global, system glossary is loaded. If dbfname is specified, then the glossary is loaded from the specified table. If dbfname is '<clear>' the speed glossary is cleared from memory.

Supported by

Version 5 and above
