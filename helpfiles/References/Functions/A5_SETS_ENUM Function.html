A5_SETS_ENUM()

Syntax

Set_List as C = A5_SETS_ENUM()

Description

The A5_SETS_ENUM() function returns a CR-LF delimited list of sets in the current database.

Example

? a5_sets_enum()

= fred.set

fred2.set

inventory.set

invoice.set

invoice2.set

items-product.set

product sales.set

vendors.set
