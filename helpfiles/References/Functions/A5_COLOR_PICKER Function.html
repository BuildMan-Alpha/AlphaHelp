A5_COLOR_PICKER()

Syntax

Selected_Color as C = A5_COLOR_PICKER( [ Mode as C [, Color as C ]] )

Argument

Description

Selected_Color

The numerical value of the selected color.

Mode

Optional. The way the color value will be specified.

  • "RGB" - indicates that Color will be formatted "n1,n2,n3" or "n1+n2+n3"

  • "HEX" - indicates that Color will be formatted

Color

Optional. The initial color value to display.

Description

The A5_COLOR_PICKER() function displays the color picker dialog box and returns the value of the selected color.

Limitations

Desktop applications only.

Examples

a5_color_picker("rgb","12,34,56")
