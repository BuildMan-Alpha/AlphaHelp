Line_ReverseOrder()

Syntax

New_List as C = Line_ReverseOrder( List as C )

Argument

Description

New_List

A reordered CR-LF delimited list.

List

A CR-LF delimited list.

Description

The Line_ReverseOrder() function reverses the order of entries in a CR-LF delimited string.

Supported By

Alpha Five Version 6 and Above

Examples

col = a5.Color_Enum()

col = word(col, 1, crlf() , 5)

? col

= Black

Dark Red

Red

Pink

Rose

col = Line_ReverseOrder(col)

? col

= Rose

Pink

Red

Dark Red

Black
