HTMLENCODE()

Syntax

Encoded_String as C =   HTMLENCODE( Input_String as C [,  Strict_Flag as L] )

Encoded_String as C = HTMLENCODE( Input_String as D [, Strict_Flag as L] )

Encoded_String as C = HTMLENCODE( Input_String as N [, Strict_Flag as L] )

Argument

Description

Input_String

The string to be encoded and it can be a character, numeric or data input.

Strict_Flag

Optional. Default = .F. (FALSE). Strict_Flag controls strict encoding.

  • .F. = Only the most common characters are substituted with their HTML entity value.

  • .T. = All characters which have HTML character entities will be converted.

Description

HTMLENCODE() transforms a string in order to make it HTML-safe. Certain characters are have special meanings within HTML documents and should not be used directly. This function always returns a character string - numeric or date inputs will automatically be converted to character.

  • & becomes &

  • " becomes "e;

  • ' becomes 

  • < becomes <

  • > becomes >

This is sufficient for most all web applications and allows you to easily store HTML and other reserved characters in your database and convert them on-the-fly for display within a web page.

Example

htmlencode("") -> <HTML TAG>

htmlencode("",.T.) -> <HTML TAG>
