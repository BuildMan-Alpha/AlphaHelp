WORD_REPLACE()

Syntax

New_List as C = WORD_REPLACE( List as C, Find_Word as C, Replace_Word as C )

Argument

Description

New_List

The modified list.

Find_Word

A CR-LF delimited list of words or entries.

Replace_Word

The number of the word or entry to remove.

Description

The WORD_REPLACE() function replaces a word or entry in a CR-LF delimited list.

Examples

dim s as C

s = <<%a%

bird

cow

chicken

goat

%a%

? word_replace(s, "cow", "bull")

= bird

bull

chicken

goat
