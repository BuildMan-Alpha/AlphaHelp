LINE_INSERT()

Syntax

List as C = LINE_INSERT( List as C, String_to_Insert as C, Position as A, Before as L )

Argument

Description

List

A CR-LF delimited list.

String_to_Insert

A line to insert into List.

Position

Can either be a line number (numeric value) or a string found in an existing line. Position can include wildcard characters.

Before

  • .T. = Insertion made before Position.

  • .F. = Insertion made after Position.

Description

The LINE_INSERT() function inserts one or more lines into a CR-LF delimited list.

Supported By

Alpha Five Version 5 and Above
