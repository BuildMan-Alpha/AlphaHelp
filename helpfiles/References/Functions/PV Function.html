PV()

Syntax

Present_Value as N = PV( Investment as N, Interest_Rate as N, Number_Of_Time_Periods as N )

Argument

Description

Investment

The amount of each equal investment, which may be either positive or negative.

Interest_Rate

Interest rate per time period. Interest rate must be a positive number.

Number_Of_Time_Periods

Number of time periods (and investments), which must be a positive number.

Description

PV() returns the present value of a stream of Investment s, invested at a fixed Interest_Rate for a certain Number_Of_Time_Periods. The PV() function is used to find the amount that must be invested now to produce a given future amount.

Supported By

Alpha Five Version 5 and Above

Example

Assume a 12% per annum interest rate. An annuity pays $100 per annum for the next 30 years. To compute the present value of this payment stream, use the following expression:

PV(100,.12,30) ->  805.52
