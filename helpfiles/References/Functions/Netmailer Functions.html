Netmailer Functions

The following table contains Netmailer functions.

Function

Description

A5_RUN_NETMAILER_JOB()

Run a saved Netmailer job with additional filter and order criteria.

A5_RUN_EMMJ()

Choose a Netmailer Maintenance Job from a table or set and run it.

A5_EMAIL_VALID()

Returns .T. if an address is correctly formed, .F. otherwise.
