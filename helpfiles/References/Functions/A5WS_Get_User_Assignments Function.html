Returns a list of groups to which a user belongs

A5WS_Get_User_Assignments()

Syntax

Group_List as C = A5WS_Get_User_Assignments( User as C [, Request as P ] )

Argument

Description

Group_List

A list of groups to which the specified user belongs.

User

The userid of a person in the user table.

Request 

Optional. The Request system variable. This variable explicitly passes all request variables to the function. Required if run from a web page.

Description

The A5WS_Get_User_Assignments() function returns a list of the groups to which a person belongs.

Limitations

Web applications only

Example

? A5WS_Get_User_Assignments("doris")

= "Marketing"
