SPACE()

Syntax

Output_String as C = SPACE( Number_Of_Spaces as N )

Argument

Description
Number_Of_Spaces The number of space characters to place into the output string.

Description

Returns a character string containing the specified Number_Of_Spaces.

SPACE() creates a string with an arbitrary number of spaces. You typically use SPACE() when constructing character strings by concatenating pieces together, because it enables you to insert blank spaces between the pieces.

Supported By

Alpha Five Version 5 and Above

Examples

trim("Paul     ") + SPACE(1) + "Mahoney" -> Paul Mahoney
