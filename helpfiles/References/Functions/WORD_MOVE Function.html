WORD_MOVE()

Syntax

New_List as C = WORD_MOVE( List as C, From_Position as N, To_Position as N )

Argument

Description

New_List

The modified list.

List

A CR-LF delimited list of words or entries.

From_Position

The position of the word or entry to move.

To_Position

The new position of the word or entry.

Description

The WORD_MOVE() function moves a word or entry from one position to another in a CR-LF delimited list.

Examples

dim s as C

s = <<%a%

bird

cow

chicken

goat

%a%

? word_move(s, 2, 4)

= bird

chicken

goat

cow
