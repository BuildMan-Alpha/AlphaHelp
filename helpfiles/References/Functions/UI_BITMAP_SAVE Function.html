UI_BITMAP_SAVE()

Syntax

UI_BITMAP_SAVE( Bitmap_Name as C )

Argument

Description

Bitmap_Name

The name of a bitmap in memory.

Description

The UI_BITMAP_SAVE() method saves a named bitmap to a blob variable.

Limitations

Desktop applications only.

Example

dim b1 as B

ui_bitmap_create_from_screen("test", 0, 0, 4, 1)

b1 = ui_bitmap_save("test")

ui_bitmap_draw("george", <<%code%

ui_bitmap_load("george", b1)

%code%)

ui_dlg_box("","{image=george}")
