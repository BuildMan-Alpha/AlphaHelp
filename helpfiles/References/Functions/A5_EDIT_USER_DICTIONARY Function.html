A5_EDIT_USER_DICTIONARY()

Syntax

A5_EDIT_USER_DICTIONARY()

Description

The A5_EDIT_USER_DICTIONARY() function displays the Edit User Defined List for Spellchecker dialog, allowing a user to edit the user defined list of words for the dictionary.

Limitations

Desktop applications only.

Example

A5_EDIT_USER_DICTIONARY()    

See Also

USER_DICTIONARY_ADD(), USER_DICTIONARY_DELETE()
