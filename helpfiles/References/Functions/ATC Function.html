ATC()

ATC() is a case insensitive version of the AT()function.

Syntax

Position as N = ATC( Find_String as C, Search_In_String as C [, Number as N ] )

Argument

Description

Find_String

A character string.

Search_In_String

The character string to examine for occurrences of Find_String.

Number

Optional. Default = 1. The integer number specifying which occurrence to return. -1 Specifies the last occurrence of the Find_String. If omitted, the first occurrence of the Find_String will be returned.

Description

ATC() returns a position integer indicating where the n th occurrence of the Find_String begins in the Search_In_String. If the Find_String is not found, ATC() returns 0. n is specified through the Number argument. Case insensitive. See AT() for a case-sensitive version of this function.

Refer to the AT()function for more information and additional examples.

Limitations

The Find_String argument is case sensitive.

Examples

atc("dog",PRODUCT) -> 10, if PRODUCT contains "Dinky-Di Dog Food"

atc("def","ABCDEF") ->  4
