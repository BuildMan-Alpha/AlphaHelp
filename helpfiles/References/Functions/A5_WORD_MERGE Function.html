A5_WORD_MERGE()

Syntax

Result_Flag as L = A5_WORD_MERGE( Mode as C, Merge_Data as C [, Template as C ] )

Argument

Description

Result_Flag

.T. = Export succeeded.

.F. = Export failed.

Mode

"new" = displays the Microsoft Mail Merge dialog box.

"template" = requires the Template argument.

Merge_Data

The name of a text file containing comma delimited data.

Template

Optional. The name of a template.

Description

The A5_WORD_MERGE() function exports data to Microsoft Word 's Mail Merge function. The function can use an existing template or open the Microsoft Word 's Mail Merge dialog box to create a template on the fly.

Limitations

Desktop applications only.

Example

a5_word_merge("new", "data.txt")
