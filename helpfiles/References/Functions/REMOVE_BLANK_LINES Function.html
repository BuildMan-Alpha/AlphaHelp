REMOVE_BLANK_LINES()

Syntax

Compacted_List as C = REMOVE_BLANK_LINES( Input_List as C )

Argument

Description

Input_List

A CR-LF delimited character string list containing 0 or more lines with NULL content.

Description

REMOVE_BLANK_LINES() removes all lines in a CR-LF delimited list that have NULL ("") content. Lines that contain invisible characters, such as spaces (" ") will not be removed.

Example

dim cs as C

cs = "abc" + crlf() + "" + crlf() + "def" + crlf()

? cs

= abc

def

CS = remove_blank_lines(cs)

? cs

abc

def
