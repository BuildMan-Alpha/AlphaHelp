WORDATC()

Syntax

Position as N = WORDATC( Word as C, Input_String as C [, Word_Delimiter as C ] )

Argument

Description

Word

The word to find in Input_String.

Input_String

A character string.

Word_Delimiter

Optional. Default = space (" "). The character string that separates words.

Description

WORDATC() returns the word number of Word in Input_String. Case sensitive.

If the optional Word_Delimiter parameter is not specified, a word is any sequence of one or more characters separated either by a space or the start or end of the string.

Limitations

The Word argument is case sensitive.

Examples

wordatc("test", "This is a test") -> 4

dim fruits as C

fruits = "apple" + crlf() + "banana" + crlf() + "grape" + crlf() + "peach" + crlf() + "plum"

wordatc("banana", fruits, crlf() ) ? 2

wordatc("Banana", fruits, crlf() ) ? 0
