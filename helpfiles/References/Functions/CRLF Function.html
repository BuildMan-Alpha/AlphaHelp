CRLF()

Syntax

CRLF( [ Count as N ] )

Argument

Description

Count

Optional. Default = 1. The number of CR-LF character sequences to generate.

Description

The CRLF() function inserts the carriage return and line feed characters into a string.

Example

crlf(3)
