MASK()

Syntax

Formatted_String as C = MASK( Unformatted_String as C, Template_String as C)

Argument

Description

Unformatted_String

A character string.

Template_String

A character string containing formatting and space characters. Any non-space character may be used for formatting.

Description

MASK() inserts characters from Unformatted_String into Template_String to replace all of the spaces in Template_String and returns the newly formatted string. Non blank characters in Template_String are included in the output Formatted_String. To include a space in the Formatted_String, use the pipe character ( | ) in the Template_String.

Example

? mask("6175551212", "(  )|   -   ")

= "(617) 555-1212"
