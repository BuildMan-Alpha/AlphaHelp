A5_FIELDSIZES()

Syntax

Size_Info as P = A5_FIELDSIZES( Field_Type as C )

Argument

Description

Size_Info

A dot variable with two elements:

.Decimal = number of decimal characters

.Length = length in characters

Field_Type

The name of a field type.

Description

The A5_FIELDSIZES() function returns information about the size of table fields that are not character or numeric.

Examples

? a5_fieldsizes("time")

= decimal = 0

Length = 17

? a5_fieldsizes("date")

= decimal = 0

Length = 8
