Email Functions and Methods

Alpha Five provides the following email functions and methods for using Alpha Five's built-in email program.

Function or Method

Description

A5_EMAIL_FILTER_BUILDER()

Displays the Email Filter Genie.

A5_EMAIL_PROFILES_ENUM()

Returns a list of email profiles.

A5_EMAIL_VALID()

Reports whether an email address is properly formatted.

A5_IS_EMAIL_VALID()

Reports whether one or more email addresses are properly formatted.

A5_RUN_NETMAILER_JOB()

Runs a Netmailer job, optionally hiding the user interface.

EMAIL_CLIENT()

Opens the Alpha Five email client allowing you to send an email message.

EMAIL_CLIENT_EXTERNAL()

Opens the default email client and fills in any of the optional arguments that you specify.

EMAIL_GET_ACTIVE()

Returns the name of the active email profile.

EMAIL_GET_FIELD()

Returns a field from the header of an email message.

EMAIL_PROFILE_LOAD()

Loads a named profile from the system registry into a dot variable.

EMAIL_PROFILE_SAVE()

Saves a named profile from a dot variable to the system registry.

EMAIL_PROFILE_VALID()

Determines whether an email profile has complete and acceptable values.

EMAIL_READER_OPEN()

Starts the email reader program.

EMAIL_RECEIVE()

Receives an email message using Alpha Five's built-in email features.

EMAIL_SAVE_ATTACH()

Saves attachments to an email message.

EMAIL_SELECT_PROFILE()

Allows the user to select an email profile.

EMAIL_SEND()

Sends an email message using Alpha Five's built-in email features.

EMAIL_SEND2()

Sends an email message using Alpha Five's built-in email features.

EMAIL_SET_ACTIVE()

Sets the active email profile.

SET.SEND()

Emails a set using either the Alpha Five internal email client or a third party email client.

SYS_RESOLVE_URL()

Executes the function implied by Input_String. If Input_String is a URL, the function opens the URL in a new browser window. If it is an email address, it opens a new message in your default email application. If it is an FTP address, it opens your FTP client and attempts to open a session with the FTP host.

TABLE.SEND()

Emails a table using either the Alpha Five internal email client or a third party email client.

UI_EMAIL_SETTINGS()

Displays the Email Settings dialog box.

UI_GET_EMAIL_TEMPLATES()

Displays the Retrieve Email Message dialog box and returns the selected profile.

Alpha Five provides the following functions for sending email via SMTP mail servers.

Method

Description

EMAIL_SMTP_CLOSE()

Closes a connection with an SMTP email server.

EMAIL_SMTP_OPEN()

Opens a connection with an SMTP email server.

EMAIL_SMTP_SEND()

Sends an email message.

email_send_noprofile()

Sends an email message, doing its own setup and teardown of the SMTP connection. Can use Alpha Five or .NET Framework SMTP.

Alpha Five provides the following functions for retrieving email from POP3 mail servers.

Method

Description

EMAIL_POP_CLOSE()

Closes the connection with a POP3 email server.

EMAIL_POP_HEADERS()

Retrieves the header of an email message.

EMAIL_POP_LIST()

Returns an array containing byte counts for each message on the server.

EMAIL_POP_OPEN()

Opens a connection to a POP3 server and logs in.

EMAIL_POP_RETR()

Retrieves a single message from a POP3 server.

EMAIL_POP_RETR_UI()

Displays the Download Email dialog box and returns a count of messages on your email server that match your filter specifications.
