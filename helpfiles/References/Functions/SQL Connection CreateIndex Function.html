SQL::Connection::CreateIndex()

Syntax

Result_Flag as L = <SQL::Connection>.CreateIndex( <TableDef> as SQL::TableInfo, <IndexDef> as SQL::IndexInfo )

Argument

Description

Result_Flag

TRUE (.T.) if the operation was successful; otherwise FALSE (.F.).

<SQL::Connection>

A SQL::Connection object.

<TableDef>

A SQL::TableInfo object.

<IndexDef>

A SQL::IndexInfo object.

Description

The CreateIndex() method creates an index on the specified table in the currently connected database using the TableDef and IndexDef specifications.
