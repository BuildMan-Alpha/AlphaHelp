EXTENT_WIDTH()

Syntax

Width as N = EXTENT_WIDTH()

Argument

Description

Width

The width of the print image.

Description

The

EXTENT_WIDTH()

function is available for use inside of the code section of the UI_PRINTER_DRAW() function. It returns the width of the print image in inches.

Limitations

Used only in the code section of UI_PRINTER_DRAW().

Example

This section of code uses EXTENT_WIDTH() to determine the width of the print image being created.

page_description = <<%code%

...

'this tells me how big the page is in inches

page_width = extent_width()

page_height = extent_height()

...

%code%
