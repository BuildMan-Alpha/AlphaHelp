*FIRST()

Syntax

Result_Flag as L = *FIRST( Search_In_String as C, Find_String as C )

Argument

Description

Search_In_String

A non-null character string.

Find_String

A character string.

Description

*FIRST() returns .T. (TRUE) if Find_Stringis found at the start of Search_In_String.

Note: Do not confuse this function, which operates on a character expression, with the summary function FIRST() , which operates on a group of records.

Limitations

The Find_Stringargument is case sensitive.

Example

*first(LASTNAME, "MC") -> TRUE, if the LASTNAME field starts with the character string "MC"
