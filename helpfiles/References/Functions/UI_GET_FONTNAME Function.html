UI_GET_FONTNAME()

Syntax

UI_GET_FONTNAME( [ Title as C [, Default_Font as C [, Style as C ]]] )

Argument

Description

Title

Optional. Default = "elect Fontname". The title of the dialog box.

Default_Font

Optional. Default = none. The default font selection.

Style

Optional. Default = "". The style of the display.

  • "" = dialog box

  • "a" = drop-down list

Description

The UI_GET_FONTNAME() function displays the Select Fontname dialog box.

Limitations

Desktop applications only.

Example

ui_get_fontname("Fonts", "Arial", "a")
