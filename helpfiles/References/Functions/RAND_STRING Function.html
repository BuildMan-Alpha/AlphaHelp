RAND_STRING()

Syntax

Random_String as C = RAND_STRING( Length as N)

Argument

Description

Length

The number of characters to place into Random_String.

Description

RAND_STRING() returns a random string of characters between a and z. The string has Length characters.

Examples

rand_string(4) ->  "abnm"

The next time the function is evaluated, it will produce a different string.
