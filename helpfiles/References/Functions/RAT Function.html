RAT()

Syntax

Position as N = RAT( Find_String as C, Search_In_String as C [, Number as N ] )

Argument

Description

Find_String

The character string to find.

Search_In_String

The character string to examine for occurrences of Find_String.

Number

Optional. Default (1). The instance of the Find_String to find.

Description

The RAT() function searches backwards through the Search_In_String looking for the n th occurrence of the Find_String ; where n is specified through the Number argument. If the Find_String is found, then RAT returns its Position, a numeric value that equals the number of characters from the end of the Search_In_String at which the Find_String starts.  If no match is found, RAT returns 0.  Number is an optional argument, and if omitted the first occurrence of the Find_String will be returned.

Limitations

The Find_String argument is case sensitive.

Examples

? rat("word", "word1 word2 word3", 1)

= 5

? rat("word", "word1 word2 word3", 2)

= 11

? rat("word", "word1 word2 word3", 3)

= 17

? rat("word", "word1 word2 word3", 4)

= 0

? rat("O'Neill", "Timothy P. O'Neill")

= 7

? rat("case", "CaSe SeNsItIve")

= 0
