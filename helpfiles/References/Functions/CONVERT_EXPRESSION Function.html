CONVERT_EXPRESSION()

Syntax

Output_String as C = CONVERT_EXPRESSION( Expression as C, Convert_Expression_Flags as C [, Tablename as C [, Variable_Frame as P]] )

Argument

Description

Expression

An expression.

Convert_Expression_Flags

  • "V" = Expand vars. Substitute values for variable names.

  • "C" = Expand calcs. Substitute values for calculated value names

  • "L" = Force long names. Expand object names to their full addresses.

  • "S" = Force short names. Remove object name address information.

Tablename

Optional. Default = the current table. The table context of the variables or objects. The name of a table.

Variable_Frame

An expression that defines the scope of the variables. See Variable Data Types.

Description

CONVERT_EXPRESSION() returns a character string by converting elements of an expression based on flags.

Example

The following example substitutes the value of the variable var->name for the expression "var->name".

dim shared name as C

name = "ed"

filter = "myname = var->name"

convert_expression(filter, "V") -> myname = "ed"
