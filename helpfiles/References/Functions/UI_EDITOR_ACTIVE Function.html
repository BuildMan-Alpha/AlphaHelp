UI_EDITOR_ACTIVE()

Syntax

Result_Flag as L = UI_EDITOR_ACTIVE()

Argument

Description

Result_Flag

  • .T. = An instance of the Code Editor is running.

  • .F. = The Code Editor is not running.

Description

The UI_EDITOR_ACTIVE() function indicates whether an instance of the Code Editor is running.

Limitations

Desktop applications only.

Example

? ui_edit_active()

= .T.
