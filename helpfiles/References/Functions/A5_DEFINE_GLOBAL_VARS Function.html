A5_DEFINE_GLOBAL_VARS()

Syntax

A5_DEFINE_GLOBAL_VARS()

Description

The A5_DEFINE_GLOBAL_VARS() function displays the Global Variables dialog box, which allows you to edit, add, or delete global variables.

Limitations

Desktop applications only.

Example

a5_define_global_vars()
