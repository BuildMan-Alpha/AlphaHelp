SQL_TableExport()

Syntax

Result_Flag as L = SQL_TableExport( Conn as SQL::Connection as C, Create_Table as L, Source_DBF_Table as C, Owner as C, Destination_Table as C, BYREF Result_Message as C, Filter as C, Order as C, QueryFlags as C, ExportFieldList as C, Arguments as SQL::Arguments, CreateStatement as C, ShowProgress as L, AllowCancel as L, StatusBarFormatString as C )

Argument

Description

Result_Flag

TRUE (.T.) if the operation was successful; otherwise FALSE (.F.).

Conn

A SQL::Connection object.

Create_Table

.T. = Create a new table.

Source_DBF_Table

The path to and name of the Alpha Anywhere table to export.

Owner

The owner of the back-end database.

Destination_Table

The name of the table to create in the back-end database.

Result_Message

A character variable that will receive any error message.

Filter

Default = "" (all records). A logical expression that selects the records to export.

Order

Default = "" (record number order). A character expression that orders the exported records.

QueryFlags

Default = "". Refer to Query Options.

ExportFieldList

Default = "". Either a simple list of CR-LF or comma delimited fields or a list of fields AND expressions. If the ExportFieldList contains expressions, then it must be CR-LF delimited.

Arguments

Default = NULL_VALUE(). Values used in the filter or order expressions. These values may come from variables or from user input at run time.

Create_Statement

Default = "". A SQL CREATE TABLE statement.

ShowProgress

Default = .T.  Indicates whether to display a progress bar.

AllowCancel

Default = .T.  Indicates whether the progress bar contains a cancel button.

StatusBarFormatString

Default = "". The formatting string that defines the appearance of the progress bar.

Description

The SQL_TableExport() function exports an Alpha Anywhere table to a back-end database.

Limitations

Desktop applications only.

Example

dim fieldlist as C

dim tblPointer as P

dim cn as sql::connection

dim message as C

 

fieldlist = <<%txt%

fullname = ut(firstname) + ", " + ut(lastname)

lastname

firstname = upper(firstname)

date = date()

number = 123*23

%txt%

 

tblPointer = table.OpenMap("customer",fieldlist)

flag = cn.open("{A5API=Excel,FileName='c:\customer_alphadao.xls'}")

if flag = .f. then

    ui_msg_box("Error",cn.CallResult.text)

    end

end if
 
flag = sql_tableexport(cn,.f.,tblPointer,"","Customer",message)
