UI_GET_EMAIL_TEMPLATES()

Syntax

Output_String as C = UI_GET_EMAIL_TEMPLATES( [ External as L ] )

Argument

Description

Output_String

The name of the selected email template. The value will be NULL if the user did not select a template or clicked Cancel.

External

Optional. Default = .F.

  • .T. = Display Select Email Template dialog box to select a .TXT file.

  • .F. = Display a list of internal email templates.

Description

The UI_GET_EMAIL_TEMPLATES() function prompts for the name of an email template. Email templates can be saved from within the Alpha Five email client.

Limitations

Desktop applications only.

Example

dim prof as C

prof = ui_get_email_templates()

? prof

= "Ed"
