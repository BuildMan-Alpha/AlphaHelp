STRIP_TRAILING_CHAR()

Syntax

Output_String as C = STRIP_TRAILING_CHAR( Input_String as C, Remove_Char as C )

Argument

Description
Output_String The processed string.
Input_String The string to examine.
Remove_Char The character to remove.

Description

The STRIP_TRAILING_CHAR() function removes the specified character from the end of the input string.

Examples

? strip_trailing_char("this is text...",".")

= "this is text.."
