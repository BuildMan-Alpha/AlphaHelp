CALCULATED_FIELD_GET()

Syntax

 as P = CALCULATED_FIELD_GET( Variables as P, Name as C )

Argument

Description

Variables

A pointer to the variable scope. See Variable Data Types and Scope

Name

The name of the calculated value.

Description

The CALCULATED_FIELD_GET() function returns a pointer to a calculated value.

Limitations

Desktop applications only.

Examples

dim ptr as P

dim pp as P

pp = local_variables()

ptr = calculated_field_get(pp, "all_days_average")

See Also

CALCULATED_FIELD_DELETE(), CALCULATED_FIELD_ENUM(), CALCULATED_FIELD_SET()
