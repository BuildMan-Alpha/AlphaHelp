FEDEX_Delivery_Status()

Syntax

Status as C = FEDEX_Delivery_Status( Tracking_Number as C )

Argument

Description

Status

The status of a shipment sent through FEDEX.

Tracking_Number

The tracking number of a shipment sent through FEDEX.

Description

The FEDEX_Delivery_Status() function contacts the FedEx web site and looks up the status of a shipment sent through FedEx.

Note : Use of this function requires an active Internet connection.

Supported By

Alpha Five Version 8 and Above

Limitations

Desktop applications only.

Example

? fedex_delivery_status("9612014730443905198600")
= "Delivered"
? fedex_delivery_status("9612478019051986000000") 'invalid number
= ""
