File Functions and Methods

Alpha Five provides the following FILE methods. Prefix the following methods with the FILE keyword.

Function

 

Description

FILE.APPEND()

V8

 Appends character or blob data to a file.
FILE.APPEND_LINE()

V8

 Appends character data to a file.

.ATTRIBUTES_SET()

V5

Sets the attributes for filename.

.ATTRIBUTES_GET()

V5

Gets the attributes of filename.

.COPY()

V5

Copies Source Filename to the Destination Filename.

.COPY2()

V6

Copies a file to a new location, creating the target folder(s) if they do not exist.

.COPY_FOLDER()

V6

Copies files and subfolders from one folder to another.

.CREATE()

V5

Creates and opens a new file, returning the object pointer reference to the file.

.CREATE_TIME()

V5

Returns the date/time that Filename was created.

.DEPENDENCIES()

V5

Determines if the master file, which is based on transaction files, is out of date.

.DIR_CREATE()

V5

Creates a new directory.

.DIR_CREATE_RECURSE()

V6

Creates a new directory, creating all the intermediate directories that are required.

.DIR_GET()

V5

Gets the current directory.

.DIR_PUT()

V5

Moves to a directory.

.DIR_REMOVE()

V5

Deletes a directory.

.DRIVE_SPACE_AVAIL()

V5

Returns the number of bytes of available space on Drive.

.DRIVE_SPACE_TOTAL()

V5

Returns the size in bytes of Drive.

.DRIVE_TYPE()

V5

Returns the type of Drive.

.DRIVES_GET()

V5

Returns a list of all of the logical drives.

.EXISTS()

V5

Determines if a file exists on disk.

.FILENAME_PARSE()

V5

Returns the specified component (drive, path, name or extension) of Filename.

.FROM_BLOB()

V6

Creates a file from binary data.

.FROM_PROPERTY()

V6

Creates a file from a pointer (dot) variable.

.FROM_STRING()

V6

Creates a file from a character string.

.FTP_PARSE()

V6

 

.LONGNAME_TO_

SHORTNAME()

V6

Converts Windows long path names to the old style 8.3 Windows filename format.

.MACHINES_GET()

V5

Returns a list of network drives, machines, and printers.

.OPEN()

V5

Opens a data file for input or output and returns the object pointer to the file.

.READ_TIME()

V5

Returns the date/time that Filename was last read.

.REMOVE()

V5

Erases a particular file, removing it from the disk.

.RENAME()

V5

Changes the name of a file.

.SEND()

V5

Sends multiple files via the current MAPI email client, or the built-in Alpha Five email client.

.SHORTNAME_TO_

LONGNAME()

V5

Converts an old style 8.3 Windows filename to a long name.

.SIZE()

V5

Returns the size of a file.

.TEMP_CREATE()

V5

Creates a temporary data file with a unique name.

.TEMP_PATH_GET()

V5

Returns the path to the temporary directory.

.TO_BLOB()

V5

Reads a binary file from disk into a blob variable.

.TO_PROPERTY()

V6

Reads a file created by the FILE.FROM_PROPERTY() method and returns its contents as a string.

.TO_STRING()

V5

Reads a file from disk into a text variable.

.UNZIP()

V5

Extracts the files that match the mask from Zip_Filename, and places the files in the Output_Directory.

.UPDATE_TIME()

V5

Returns the date/time that filename was last updated.

.WATCH_CREATE()

V5

Allows you to monitor the activity in a particular folder on your computer, and automatically execute Xbasic code when events occur.

.WATCH_DESTROY()

V5

Removes a watch set by FILE.WATCH_CREATE().

.WATCH_ENUMERATE()

V5

Lists the "watches" set by the FILE.WATCH_CREATE() method.

.ZIP()

V5

Adds a file or files to a zip file called Zip_Filename.

.ZIPLIST()

V5

Returns a character string with all of the files in Zip_Filename, formatted according to the Format_String.

Prefix the following methods with the object pointer. Use the FILE.OPEN() to retrieve the object pointer.

Function

 

Description

.BYTES_GET()

V5

Returns the number of bytes in a file.

.CHARS_GET()

V6

Returns the number of characters in a file.

.CLASS()

V5

Returns the class of the file.

.CLOSE()

V5

Closes the file referenced by the object pointer.

.EOF()

V5

Returns TRUE if the end of the file.

.FLUSH()

V5

Flushes output held in the disk cache, to assure all output is stored safely in the file.

.IS_VALID()

V5

Determines if the file pointer is valid.

.POS_GET()

V5

Returns the position in the file of the file pointer.

.READ()

V5

Retrieves a specified Number of Characters from a file starting at the current file offset.

.READ_LINE()

V5

Retrieves text data from the current file offset to the end of the line.

.READB()

V5

Retrieves a specified Number of Bytes from a file starting at the current file offset.

.SEEK()

V5

Sets the position of the file pointer to a Byte Offset (measured in bytes) in relation to the beginning of the file.

.TEXT_ENCODING()

V6

Returns the text encoding method.

.WRITE()

V5

Transfers the contents of a Text string to the location of the current file pointer in the open file.

.WRITE_LINE()

V5

Writes a line of text data (Text Line) to the location of the current file pointer.

.WRITEB()

V5

Transfers the contents of a blob to the location of the current file pointer.

Alpha Five provides the following useful file functions.

Function

 

Description

A5_AddTempFile()

V6

Registers a temporary file that Alpha Five will delete when you exit Alpha Five.

A5_FILE_REMOVE()

V5

Deletes a file.

A5_FileAttributeHelp()

V5

Displays the FileFind Attribute Codes dialog and returns the selected attribute codes.

A5_FileAttributeHelper()

V6

Displays the Select File Attribute dialog and returns the selected attribute codes.

A5_Get_FilesUpdatedSince()

V6

Returns a list of files updated since a specified time.

A5_PROMPT_FILES_IN_DB()

V7

Displays a selection dialog and returns a list of database and web publishing files.

A5_SYNCHRONIZE_FOLDERS()

V6

Returns a dot variable containing lists of the files and folders in Folder1 and Folder2 that have different update times.

BOF()

V5

Tests to see if the record pointer is at the first record of the current table. If it is, the function returns .T. (TRUE); otherwise, it returns .F. (FALSE).

COPY_FILES()

V6

Copies a list of files from a source folder to a destination folder.

COPY_FOLDER()

V6

Copies all of the files in the Source_Folder to the Target_Folder.

EOF()

V5

Returns .T. (TRUE) if the record pointer is at the end of the file for the table specified by its Table Handle; otherwise, it returns .F. (FALSE).

FILE_COPY()

V5

Copies Source Filename to Destination Filename.

FILE_GET_DATE_TIME()

V5

Returns the date and/or time on which the file specified by filename was last updated.

FILE_GET_SIZE()

V5

Returns the size of a file.

FILES_IN_SET()

V5

Creates a CR-LF delimited character string containing the fully qualified names of the components of the set.

FILES_IN_TABLE()

V5

Creates a CR-LF delimited character string list containing the fully qualified names of the components of the table.

FILE_IS_OPEN()

V6

Determines whether a specified file is open.

GET_FROM_FILE()

V5

Opens a file specified by filename and reads the contents of the file.

LIST_FILES_IN_SET()

V5

Creates a CR-LF delimited character list containing the fully qualified names of the components of the set.

SAVE_TO_FILE()

V5

Saves a character string to a file.

UI_GET_FILE()

V5

Allows the user to select a path and filename. Has features only supported by V6.

UI_GET_FILES_

MULTIPLE()

V5

Allows the user to select files.

UNZIP_FILELIST()

V6

Extracts all or specified files from a ZIP file to a specified directory.

UNZIP_FILES()

V5

UNZIP_FILES()extracts files from Zipfile_Name, that match the Mask specification, to the Destination_Folder.

ZIP_FILES()

V5

A high level utility function that add a list of files to a zip file.
