ISDATE()

Syntax

Result_Flag as L = ISDATE( Date_String as C )

Argument

Description

Date_String

A character string that represents a date. Refer to Date and Time Format Elements for more about valid date formats.

Description

ISDATE() returns .T. (TRUE) if Date_String is a valid date; otherwise it returns .F. (FALSE).

Supported By

Alpha Five Version 5 and Above

Example

? isdate("2/1/99")

= .T.

? isdate("2/29/99")

= .F.
