Core Functions for Writing Expressions

The following list of functions is recommended as the starting point for the programmer who is writing Xbasic expressions. The programmer should also know how to use the Xbasic Language.

Function

Description

Character Functions

ALLTEXT()

Returns a character string of all of the field values in the current record.

ALLTRIM()

Removes leading and trailing blanks from a Input_String.

ASC()

Returns the integer ASCII_Value of the first letter in the specified Character_String.

AT()

Returns a position integer indicating where the Nth occurrence of the Find_String begins in the Search_In_String. Case insensitive.

ATC()

Returns a position integer indicating where the Nth occurrence of the Find_String begins in the Search_In_String. Case sensitive.

CHR()

Returns the Single_Character corresponding to the integer ASCII_Value.

CONTAINS()

Returns .T. (TRUE) if all words in the Find_Words string appear at least once in the Search_String.

LEFT()

Returns a character string containing the left-most Number_Of_Characters of the Input_String.

LEN()

Returns an integer equal to the number of characters in the Input_String.

LOWER()

Converts the specified character string to lowercase.

LTRIM()

Removes any specified leading characters in the Input_String.

OCCURS()

Returns the number of times the Find_String is found in the Search_In_String.

QUOTE()

Places quotes around Input_String and returns the result as a string.

RTRIM()

Removes any specified trailing characters in the Input_String.

STRITRAN()

Replaces every occurrence of Find_String in Input_String with a Replace_String. Case insensitive.

STRTRAN()

Replaces every occurrence of Find_String in Input_String with a Replace_String. Case sensitive.

STR()

Converts a Number to a right-justified character string.

SUBSTR()

Returns a Substring of a Input_String. The Substring begins at the specified Starting_Position and continues for the specified Number_Of_Characters.

UPPER()

Converts all the text in a Input_String to uppercase.

VAL()

Converts a Character_String to a numeric value.

WORD()

Returns one or more words, specified by the Word_Number and Word_Count and delimited by Word_Delimiter, from the supplied Input_String.

WORDAT()

Returns the word number of Word in Input_String. Case insensitive.

WORDATC()

Returns the word number of Word in Input_String. Case sensitive.

Character Functions

More functions for manipulating characters and strings.

Date and Time Functions

ADD_BUS_DAYS()

Adds Add_Days number of business days to a Starting_Date and returns an Ending_Date.

ADDMONTHS()

Adds the specified Number_Of_Months to the Starting_Date.

ADDYEARS()

Adds the specified Number_Of_Year s to a Date and returns an Ending_Date.

AGE()

Returns the integer number of years between Starting_Date and Ending_Date (default now).

CDOW()

Returns a character string containing the name of the day of the week for the specified Date.

CMONTH()

Returns the Month_Name in the specified Date

CTOD()

Converts a character string containing date information and returns a Date value.

CYEAR()

Returns a character string containing the year specified in Date_Value in the form "YYYY".

DATE()

Returns todays date.

DATE_VALUE()

Returns a Date value for the specified year, month and day.

DAY()

Returns an integer corresponding to the Date's day of the month.

MONTH()

Returns an integer equal to the month number (1-12) of the specified Date_Value.

NOW()

Returns the current date/time, correct to the millisecond, in the Time data type.

TIME()

Returns a Formatted_String containing the system time.

TOTIME()

Returns a formatted character string containing a time based on the specified Number_Of_Seconds past midnight.

YEAR()

Returns a four digit integer equal to the year portion of the specified Date_Value.

Expression Evaluation Functions and Methods

BETWEEN()

Returns .T. (TRUE) if Expression is greater than or equal to Lower_Value and less than or equal to Higher_Value.

CASE()

Returns the value ( Value _1 ... Value _16) specified by the first expression ( Expression _ 1 ... Expression _ 16 ) that evaluates to .T. (TRUE).

CCVALID()

Returns .T. (TRUE) if a field contains a valid Credit_Card_Number.

EVAL()

Evaluates Expression and returns the result of that expression.

IIF()

Evaluates the Logical_Expression. If the expression evaluates to TRUE, the True_Value is returned, otherwise the False_Value is returned.

Evaluation Functions

More evaluation functions.

File Functions and Methods

EOF()

Returns .T. (TRUE) if the record pointer is at the end of the file for the table.

LINE_COUNT()

Returns the number of lines in a CR-LF delimited character string list.

Mathematical Functions

CEILING()

Returns the smallest integer that is greater than or equal to the specified Number.

FLOOR()

Returns the largest integer that is either less than or equal to the specified Number.

INT()

Truncates the decimal portion of the specified Number and returns the remaining integer.

Mathematical Functions

More mathematical and trigonometric functions.

Query Functions and Methods

CURRENT_FILTER_EXPN()

Returns the filter expression of the primary table in a session.

CURRENT_ORDER_EXPN()

Returns the order expression of the primary table in a session.

Query Functions and Methods

More query functions.

Record Functions

AVERAGE()

Returns the average value of the Expression evaluated for a group of records.

COUNT()

A summary function that returns the number of records in a group of records.

MAXIMUM()

Returns the maximum value of the Expression evaluated for a group of records.

MINIMUM()

Returns the minimum value of the Expression evaluated for a group of records.

RECCOUNT()

Returns the number of records in the specified table.

RECNO()

Returns the record number of the current record in the specified table.

RUN_COUNT()

Returns the running count of records of the Expression evaluated from the first record to the current record.

RUN_TOTAL()

Returns the total value of the Expression evaluated from the first record to the current record.

TOTAL()

Returns the total value of the Expression evaluated for a group of records.

Record Functions

More record functions.

Report Functions and Methods

PAGECOUNT()

Returns the number of pages in a report.

PAGENO()

Returns the current page number of a printed layout.

Report Functions and Methods

More functions for printing reports.

Type Checking Functions

ISALPHA()

Returns . T. (TRUE) if the first character of the specified Character_String is a letter; otherwise, returns .F. (FALSE).

ISDATE()

Returns .T. (TRUE) if Date_String is a valid date; otherwise it returns .F. (FALSE).

ISNUMBER()

Returns .T. (TRUE) if a character string contains only numbers; otherwise returns .F. (FALSE).
