EMAIL_PROFILE_VALID()

Syntax

Result_Flag as L = EMAIL_PROFILE_VALID( Profile_Name as C )

Argument

Description

Result_Flag

TRUE (.T.) = the profile has all necessary information

FALSE (.F.) = the profile is incomplete or has invalid values

Profile_Name

The name of the email profile to evaluate.

Description

The EMAIL_PROFILE_VALID() function checks the specified email profile and returns TRUE (.T.) if it has all necessary information; otherwise, it returns FALSE (.F.).

Examples

? email_profile_valid("Default")

= .T.

? email_profile_valid(email_get_active() )

= .T.
