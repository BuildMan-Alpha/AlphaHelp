A5_ADD_FIELDS_TO_TABLE()

Syntax

A5_ADD_FIELDS_TO_TABLE( Table_Name as C, Field_List as C )

Usage

Desktop applications only

Argument

Description

Table_Name

The name of the table to receive the new field(s).

Field_List

A CR-LF delimited list of one or more field entries. Each entry has the same format as when adding a new field through the Restructure Table dialog box. The format for each field entry as is as follows:

Field_Name as C, Field_Type as C [, Width as N [, Decimals as N ]].

Argument

Description

Field_Name

A name starting with a letter (A-Z or a-z) and composed of letters, numbers (0-9), and underscore (_) characters.

Field_Type

One of the following values:

  • "activex" = ActiveX

  • "bitmap" = Windows Bitmap image field

  • "C" = character

  • "D" = date

  • "E" = exponent

  • "image" = image file reference

  • "jpeg" = JPEG image field

  • "L" = logical

  • "N" = numeric

  • "M" = memo

  • "ole" = OLE

  • "R" = RTF memo

  • "T" = time

  • "Y" = short time

Width

Required for character and numeric fields. The total width of the field, including, if numeric, decimal point and decimal character positions.

Decimals

Only applicable to Numeric. The number of decimal character positions.

Description

The A5_ADD_FIELDS_TO_TABLE() function adds field(s) to an existing table.

Limitations

Example

dim tbl as P

tbl = table.open("product")

? tbl.fields_get()

= 7.000000

a5_add_fields_to_table("product", "junk,C,0")

? tbl.fields_get()

= 8.000000
