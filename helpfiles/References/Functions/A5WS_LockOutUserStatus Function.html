Determines the locked out status of a user value

A5WS_LockOutUserStatus()

Syntax

Result as C = A5WS_LockOutUserStatus( UserValue as C)

Argument

Description

Result

Returns blank value if 'UserValue' is not locked out or lock out time has expired. Will return 'locked out until' time as character value if user is locked out.

UserValue

A user value to be checked.

Description

The A5WS_LockOutUserStatus() function determines the locked out status of a user value.

Limitations

Web publishing applications only.

Example

This example shows that the user "joe@email.com" is locked out until Jan 1, 2007 at 10:00 pm.

?a5ws_LockOutUserStatus("joe@email.com")

= "01/01/2007 10:00:00 00 pm"
