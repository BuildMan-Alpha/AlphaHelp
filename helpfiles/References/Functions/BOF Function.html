BOF()

Syntax

Beginning_Of_File_Flag as L = BOF()

Description

BOF() returns .T. (TRUE) if the record pointer is at the first record of the current table; otherwise, it returns .F. (FALSE).

Supported By

Alpha Five Version 5 and Above

Example

bof() -> TRUE if the record pointer is at the first record of the current table.

bof() -> FALSE if the record pointer is not at the first record of the current table.
