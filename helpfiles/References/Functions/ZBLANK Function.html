ZBLANK()

Syntax

Formatted_String as C = ZBLANK( Number as N [, Length as N [, Decimal_Places as N ]] )

Argument

Description

Number

The number to format.

Length

Optional. Default (10). The number of characters in the Formatted_String.

Decimal_Places

Optional. Default (0). The number of characters after the decimal point in the Formatted_String. If you do not specify decimal places, STR() rounds the Number to the nearest whole number.

Description

ZBLANK() converts a Number to a formatted right justified character string, returning a blank (empty) string if the number is zero.

Examples

zblank(DISCOUNT) ->   "         ", if DISCOUNT is 0

len3? -> 10

len0? -> 10

zblank(14.3, 10, 3) -> "   14.300"
