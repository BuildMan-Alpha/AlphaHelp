TABLE_REMOVE()

Syntax

TABLE_REMOVE( Filename as C )

Argument

Description

Filename

The path and filename of a .DBF file.

Description

The TABLE_REMOVE() function removes all the files associated with a Alpha Five table.

Limitations

Desktop applications only.

Example

table_remove("C:\Databases\mydb\my_database.Dbf")
