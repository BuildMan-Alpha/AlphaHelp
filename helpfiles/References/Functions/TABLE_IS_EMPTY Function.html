TABLE_IS_EMPTY()

Syntax

Result_Flag as L = TABLE_IS_EMPTY( Table_Name as C )

Argument

Description

Result_Flag

  • .F. = Table is not empty.

  • .T. = Table is empty.

Table_Name

The name of an Alpha Table table.

Description

The TABLE_IS_EMPTY() function determines whether a table contains any records.

Example

dim tbl as P

tbl = table.open("customer")

? table_is_empty(tbl)

= .F.
