Creates default security files for a web publishing project

A5WS_CreateSecurityFiles()

Syntax

Result_Flag as L = A5WS_CreateSecurityFiles()

Argument

Description

Result_Flag

.T. = Indicates that security files were created.

.F. = Indicates that security files were not created.

Description

The A5WS_CreateSecurityFiles() function creates default security files for a web publishing project. A5WS_CreateSecurityFiles() has limited use for an end user, because,

if no security files have been created, Alpha Five automatically creates security files as part of the publishing process. If run in Version 8, A5WS_CreateSecurityFiles() will not overwrite any existing security files, just create default files if none exist. If run in Version 7, the function will always overwrite any existing security files and create default files with security turned off.

Limitations

Web publishing applications only.

Example

? A5WS_CreateSecurityFiles(

= .T.
