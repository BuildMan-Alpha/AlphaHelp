WRAP_TEXT_HEIGHT()

Syntax

Height as N = WRAP_TEXT_HEIGHT( Text as C, Width as N [, Offset as N ] )

Argument

Description

Text

The text to draw on the bitmap.

Width

The width of the rectangle within the text is wrapped.

Offset

Optional. Default = 0. The offset from the left edge of the bitmap where the text begins.

Description

WRAP_TEXT_HEIGHT() returns the height in inches of the text.

Supported By

Alpha Five Version 5 and Above

Limitations

Used only in the Code sections of UI_BITMAP_DRAW(), UI_SCREEN_DRAW(), and UI_PRINTER_DRAW().
