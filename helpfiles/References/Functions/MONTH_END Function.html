MONTH_END()

Syntax

Last_Day as D = MONTH_END( Date_Value as D   )

Argument

Description

Date_Value

A date constant or variable.

Description

MONTH_END() returns the date of the last day of the month in which date occurs.

Examples

? month_end({7/12/2002})

= {7/31/2002}
