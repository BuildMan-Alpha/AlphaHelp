CONVERT_TYPE()

Syntax

Output_Value as A = CONVERT_TYPE( Input_Value as A, Data_Type as C )

Argument

Description

Input_Value

A variable containing character, numeric, date, or logical data.

Data_Type

  • "C" = Character

  • "N" = Numeric

  • "D" = Date

  • "L" = Logical

  • "T" = Time (date time value)

Description

CONVERT_TYPE() converts variables from one data type to another, preserving the data if possible.

Examples

convert_type("yes", "l") ?.T.

convert_type(345, "c") -> "345"
