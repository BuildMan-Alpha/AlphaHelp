Security Functions

Alpha Anywhere provides the following functions to maintain user and group assignments.

Function

Description

A5_ADD_NEW_GROUP()

Adds a new group.

A5_ADD_NEW_USER()

Adds a new user to the User Account File.

A5_ASSIGN_USER_TO_GROUPS()

Adds or drops a user from groups.

A5_CHANGELOGGEDONUSER()

Displays the Change Logged On User dialog box, allowing you to log onto Alpha Anywhere as a different user.

A5_CHANGE_USER_PASSWORD()

Changes a user's password.

A5_CHANGE_PERMISSION()

Changes a user password and associated object permissions.

A5_Decrypt_Binary()

Decodes data encoded by A5_Encrypt_Binary().

A5_Decrypt_String()

Decodes text encode by A5_Encrypt_String().

A5_GetAttemptedUser()

Returns the name of the last user unable to logon.

A5_DELETE_GROUP()

Deletes a group.

A5_DELETE_USER()

Deletes a user.

A5_Encrypt_Binary()

Encodes binary data using a variety of 64 bit encryption algorithms.

A5_Encrypt_String()

Encodes text using a variety of 64 bit encryption algorithms.

A5_GetAttemptedUser()

Returns the name of the last user unable to logon.

A5_GET_GROUP_ASSIGNMENTS()

Retrieves a CR-LF delimited list of users assigned to a particular group.

A5_GET_GROUPS()

Retrieves a CR-LF delimited list of all the groups that have been defined.

A5_GET_GROUP_ASSIGNMENTS()

Retrieves a CR-LF delimited list of users assigned to a particular group.

A5_GET_USER_ASSIGNMENTS()

Retrieves a CR-LF delimited list of groups to which a user is assigned.

A5_GET_USERS()

Retrieves a CR-LF delimited list of all defined users.

A5_GROUP_HAS_PERMISSION()

An internal function that indicates whether a group ID has permission to an object.

A5_HAS_PERMISSION()

Tests whether a user has the specified permission to an object.

A5_SEC_GET_GID()

Returns a group ID from a group name.

A5_SecGroupHasPermission()

Indicates whether a group name has permission to an object.

A5_Security_Option()

A high level function that calls a variety of security dialogs.

A5_USER_GROUPS_DIALOG()

Displays the Users and Groups dialog box, if user and group security is enabled, and if the user is logged on with the Master Password.

A5_USERBELONGSTO()

Indicates whether a User belongs to a Group or, if a User is not specified, if the specified Group exists.

API_GETUSERS()

Returns a CR-LF delimited list of users of a computer or domain.

USER_GROUPS()

Creates a CR-LF delimited list of the groups that the current user is a member of.

USER_NAME()

Returns the name of the current user.

Alpha Five provides the following security functions and methods.

Function or Method

Description

A5_SecurityEnabled()

Determines whether a database requires a password.

BP_BUTTON_PASSWORD()

Displays the Set Button Password dialog box, which allows you to assign a password to the button.

DB_HAS_PASSWORD()

Determines whether a database requires a password.

FILE.WATCH_CREATE()

Allows you to monitor the activity in a particular folder on your computer, and automatically execute Xbasic code when events occur.
