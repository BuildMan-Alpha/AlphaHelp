*TO_HEX()

Syntax

Value as N = *TO_HEX( Integer_Number as N )

Argument

Description

Integer_Number

An integer value to be converted to hexadecimal format.

Description

The *TO_HEX() function converts a number to an hexadecimal string.

Supported By

Alpha Five Version 5 and Above

Example

? *to_hex(123)

= "0000007B"
