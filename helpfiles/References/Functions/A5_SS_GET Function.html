A5_SS_GET()

Syntax

Result_Flag as L = A5_SS_GET( Stylesheet_Ptr as P, Stylesheet_Name as C )

Argument

Description

Result_Flag

.T. = The pointer was successfully created.

.F. = The pointer was not created.

Stylesheet_Ptr

A pointer variable that becomes the stylesheet pointer.

Stylesheet_Name

The name of the stylesheet. Use A5_STYLESHEET_ENUM()to get a list of styles.

Description

The A5_SS_GET() function creates a pointer to the specified stylesheet.

Limitations

Desktop applications only.

Example

dim ssh as P

? a5_ss_get(ssh, "Avant Garden")

= .T.
