A5_RUN_SAVED_ZIP_JOB()

Syntax

Return as P = A5_RUN_SAVED_ZIP_JOB( Saved_Settings as C [, Method as C [, To as C [, Subject as C [, Message as C [, Zip_Filename as C [, Exclude_Files as L ]]]]]] )

Argument

Description

Saved_Settings

Settings created though the Zip Database dialog box. The Saved_Settings name is specified when you click the Save Settings button

Method

  • "Zip" = Creates a zip file and saves it to disk.

  • "Send" = You can specify the optional address fields.

To

Optional. The recipient of the emailed database.

Subject

Optional. The subject line on the email.

Message

Optional. The message of the email.

Zip_Filename

Optional. The zip file to attach to the message.

Exclude_Files

Optional. Default = .F.

  • .T. = Select all files except the specified database files.

  • .F. = Select the specified database files.

Description

The A5_RUN_SAVED_ZIP_JOB() function runs a saved Zip job. Zip settings are created using the Zip Database dialog box (accessed from the Tools > Zip Database command).

Using this function, a developer could easily create a saved zip setting called "application_files_only" that zipped up all of the application files (omitting all data files), which he could then send to all of his clients. The entire operation could be completely automated.

Example

dim ptr as P

ptr = a5_run_saved_zip_job("expressions_test_zip", "Send", "me@alphasoftware.com", "The latest Expressions Test database", "The database is attached")
