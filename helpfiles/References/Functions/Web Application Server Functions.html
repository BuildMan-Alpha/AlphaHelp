Web Application Server Functions

Alpha Anywhere provides the following functions for controlling the Web Application Server.

Function

 

Description

A5_COUNT_WEBSESSIONS()

V7

Returns the number of active sessions.

A5W_INFO()

V6

Displays a summary page showing all server settings and variables.

SERVER.RESET()

V7

Deletes all server variables.

SESSION.RESET()

V6

Resets the current session by clearing out all variables that are stored in the current client session.

HTTPD_SERVER_IS_RUNNING()

V6

Indicates whether the Web Application Server is running.

HTTPD_SERVER_RESTART()

V6

Restarts the Web Application Server.

HTTPD_SERVER_START()

V6

Starts the Web Application Server.

HTTPD_SERVER_STOP()

V6

Stops the Web Application Server.

HTTPD_SERVER_TOGGLE()

V6

Toggles the state of the Web Application Server (starts a stopped server, stops a running server).
