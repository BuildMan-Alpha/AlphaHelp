VARIABLE_CONTEXT_NAME()

Syntax

Frame as C = VARIABLE_CONTEXT_NAME( Variable_Pointer as P )

Argument

Description

Frame

The name of the variable frame, if any.

Variable_Pointer

A pointer to the variable.

Description

The VARIABLE_CONTEXT_NAME() function returns the name of the variable frame (if any) of the referenced variable.

Supported By

Alpha Five Version 5 and Above

Example

? variable_context_name(global_variables() )

= "Global"
