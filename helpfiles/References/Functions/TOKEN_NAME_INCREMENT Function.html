TOKEN_NAME_INCREMENT()

Syntax

Unique_Filename as C = TOKEN_NAME_INCREMENT( Filename as C [, Max_Length as N ] )

Argument

Description

Filename

The name of a file.

Max_Length

Controls the maximum length of the resulting Unique_Filename.

Description

TOKEN_NAME_INCREMENT() returns a unique file name derived from Filename by adding a sequence number.

Example

output_file = "customer"

while file.exists(:a5.get_shared_path() + chr(92) + output_file + ".pdf")

    output_file = token_name_increment(output_file)

end while
