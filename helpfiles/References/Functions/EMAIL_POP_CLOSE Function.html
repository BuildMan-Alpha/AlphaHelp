EMAIL_POP_CLOSE()

Syntax

Result_Flag as L = EMAIL_POP_CLOSE( pSocket as P )

Argument

Description

Result_Flag

.T. = The connection was closed gracefully.
.F. = The connection was forced closed with .CLOSE().

pSocket

A pointer variable that is populated by EMAIL_POP_OPEN().

Description

The EMAIL_POP_CLOSE() function attempts to gracefully close a connection with a POP3 server.

Example

? email_pop_open(pSocket, "mail.mycompany.com", "john_doe", "sneaky")

= .T.

? email_pop_close(pSocket)

= .T.
