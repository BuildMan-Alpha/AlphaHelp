Object Functions and Methods

Alpha Anywhere provides the following Object functions.

Function

Description

CURRENT_OBJECT()

Returns the full object name (including window) of the object that currently has focus.

IS_OBJECT()

Returns .T. if Object Pointer, or Object Address references a valid object.

OBJ()

Returns an object pointer to the object specified by Object Address.

OBJECT_NAME_

EXPAND()

Takes a library object name (such as a form, report, browse, append, copy, toolbar etc.), and expands the name to a fully qualified name that includes the object dictionary.

OBJECT_NAME_

EXPAND_EXT()

Takes a external library object name (such as a form, report, browse, append, copy, toolbar etc.), and expands the name to a fully qualified name that includes the object dictionary.
