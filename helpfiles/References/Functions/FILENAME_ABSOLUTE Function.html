FILENAME_ABSOLUTE()

Syntax

Fully_Qualified_Filename as C = FILENAME_ABSOLUTE( Relative_Filename as C [, Reference_Path as C ] )

Argument

Description

Relative_Filename

A relative path to the file. If NULL, the current database path is assumed.

Reference_Path

Optional. Default = path of current database. A path to use as a starting point in composing the absolute filename.

Description

FILENAME_ABSOLUTE() returns a character string after converting a relative filename into an fully qualified filename. In the following example, the database path is "C:\Databases\Functions".

Supported By

Alpha Five Version 5 and Above

Example

dim cs as C

cs = filename_absolute("input_data.txt")

cs   ->"C:\Databases\Functions\input_data.txt"

cs = filename_absolute("input_data.txt", "c:\Databases")

cs   -> "c:\Databases\input_data.txt"
