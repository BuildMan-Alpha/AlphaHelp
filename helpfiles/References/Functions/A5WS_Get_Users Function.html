Returns a list of the userid values in the user table

A5WS_Get_Users()

Syntax

User_List as C = A5WS_Get_Users( [ Request as P [, FlagWeb as L ]] )

Argument

Description

User_List

A list of the userid values in the user table.

Request

The Request system variable. This variable explicitly passes all request variables to the function.

FlagWeb

Optional. Default = .F. .

  • .T. = Format suitable for use in a list control on a web page

  • .F. = CR-LF delimited list

Description

The A5WS_Get_Users() function returns a list of the userid values in the user table.

Limitations

Web applications only.

Example

? A5WS_Get_Users()

= alfred

betty

charles

doris
