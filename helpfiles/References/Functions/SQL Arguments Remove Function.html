SQL::Arguments::Remove()

Syntax

Result_Flag as L = .Remove( Name as C )

Argument

Description

Result_Flag

TRUE (.T.) if the operation was successful; otherwise FALSE (.F.).

A SQL::Arguments object.

Name

The unique name property of an argument. See the SQL::Argument object.

Description

The Remove() method removes an argument from a SQL::Arguments object.

Examples

dim args as SQL::Arguments

args.add("city", "Boston")

args.add("state", "MA")

? args.ArgumentNumber("state")

= 2

? args.remove("city")

= .T.

? args.ArgumentNumber("state")

= 1
