DIR_REMOVE_RECURSE()

Syntax

Success_Flag as L = DIR_REMOVE_RECURSE( Starting_Directory_Name as C [, Confirmation as L ] )

Argument

Description

Success_Flag

Indicates whether the function was able to delete anything.

Starting_Directory_Name

The initial directory to delete. This argument cannot be equal to "".All sub-directories and files below this level are deleted. The directory does not need to exist.

Confirmation

Optional. Default = .F.

  • .T. = Suppresses all dialogs.

  • .F. = Displays a dialog lists the directory that is about to be deleted and states that deleted flies cannot be recovered. Displays a warning dialog is displayed if the selected directory does not exist.

Description

The DIR_REMOVE_RECURSE() function recursively deletes a directory and everything in it. This is similar to deltree in DOS or rmdir /q /s at the NT/XP command line.

Caution : This function is dangerous. Using it at the C:\ prompt could potentially delete everything on your disk and ruin your day.

Supported By

Alpha Five Version 5 and Above

Examples

Success_Flag = DIR_REMOVE_RECURSE( "c:\temp\\" )
