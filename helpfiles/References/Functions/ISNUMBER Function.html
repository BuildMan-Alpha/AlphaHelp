ISNUMBER()

Syntax

Result_Flag as L = ISNUMBER( Character_String as C [, Allow_Decimal_Separator as L [, Allow_Thousands_Separator as L ]] )

Argument

 

Description

Character_String

 V5

A character string.

Allow_Decimal_Separator

 V7

Optional. Default = .F.

  • .T. = allow decimal separators

  • .F. = do not allow decimal separators

Allow_Thousands_Separator

 V7

Optional. Default = .F.

  • .T. = allow thousands separators

  • .F. = do not allow thousands separators

Description

ISNUMBER() returns .T. (TRUE) if a Character_String contains only numbers; otherwise, it returns .F. (FALSE).

Supported By

Alpha Five Version 5 and Above

Examples

isnumber("123")

= .T.

isnumber("123a")

= .F.

? isnumber("123,345.23")

= .F.

? isnumber("123,345.23", .t.)

= .F.

? isnumber("123,345.23", .t., .t.)

= .T.
