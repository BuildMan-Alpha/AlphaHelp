A5_ISMAXIMIZED()

Syntax

Status_Flag as L = A5_ISMAXIMIZED()

Argument

Description

Status_Flag

.T. = The MDI child window is maximized.

.F. = The MDI child window is not maximized.

Description

The A5_ISMAXIMIZED() function returns .T. if the MDI child window has been maximized.

Limitations

Desktop applications only.

Example

? a5_ismaximized()

= .F.
