BITMAP_COLORS_GET()

Syntax

Color_Data as C = BITMAP_COLORS_GET( Bitmap_Data as B, Color_Format as C [, Preserve_Native_Format as L ] )

Argument

Description

Color_List

A profile of the image data.

Bitmap_Data

An image read into a blob variable.

Color_Format

Specifies the data to be returned. Allowable values are:

  • "R" = red

  • "G" = green

  • "B" = blue

  • "C" = count

  • "I" = palette index

Preserve_Native_Format

Optional. Default = .F. (FALSE).

Description

The BITMAP_COLORS_GET() function returns profile information about a bitmap (.BMP) image.

Supported By

Alpha Five Version 5 and Above

Example

dim ff as P

dim bb as B

dim cc as C

ff = file.open("C:\Documents\Xbasic Reference\Images\logo.bmp",FILE_RO_EXCLUSIVE)

bb = ff.readb(100000)

cc = bitmap_colors_get(bb, "r",.t.)
