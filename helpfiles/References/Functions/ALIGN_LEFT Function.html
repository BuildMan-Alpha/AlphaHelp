ALIGN_LEFT()

Syntax

Output_String as C = ALIGN_LEFT( Input_String as C )

Argument

Description

Output_String

A list containing entries with leading space characters.

Input_String

A new list with leading space characters removed.

Description

The ALIGN_LEFT() function trims space characters from the left side of entries in a list.

Example

dim x as C

dim y as C

x = " word" + crlf() + "another word"

? align_left(x)

=

word

another word
