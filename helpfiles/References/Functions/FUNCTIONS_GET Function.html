FUNCTIONS_GET()

Syntax

Function_List as C = FUNCTIONS_GET( Types as C, Limitations as C )

Argument

Description

Function_List

A CR-LF delimited list of function names and prototypes.

Types

"*" = all types.

Limitations

  • "F" = Functions

  • "G" = Graphics

  • "X" = Xbasic

Description

The FUNCTIONS_GET() function returns a list of functions.

Example

dim flist as C

flist = functions_get("*", "F")

fcount = line_count(flist)

? fcount

= 1158
