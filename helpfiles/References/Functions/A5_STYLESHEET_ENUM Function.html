A5_STYLESHEET_ENUM()

Syntax

Style_List as C = A5_STYLESHEET_ENUM()

Description

The A5_STYLESHEET_ENUM() function returns a list of form styles.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

dim ssl as C

ssl = a5_stylesheet_enum()
