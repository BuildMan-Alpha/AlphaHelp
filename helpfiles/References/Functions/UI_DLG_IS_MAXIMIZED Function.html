UI_DLG_IS_MAXIMIZED()

Syntax

Result_Flag as L = UI_DLG_IS_MAXIMIZED( Title as C )

Argument

Description

Result_Flag

Indicates whether the dialog box is maximized.

  • .T. = is maximized

  • .F. = is not maximized

Title

The name of the dialog box.

Description

The UI_DLG_IS_MAXIMIZED() function indicates whether the specified modeless dialog is maximized.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

The following example creates a modeless dialog box, then tests to see if it is maximized.

ui_modeless_dlg_box("Visible Xdialog", <<%dlg%

<20,3Click Here>

%dlg%)

? ui_dlg_is_maximized("Visible Xdialog")

= .F.
