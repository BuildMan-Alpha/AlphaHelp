THREAD_NAME_CREATE()

For internal use only.

Syntax

Unique_Name as C = THREAD_NAME_CREATE( Thread_Name as C )

Argument

Description

Unique_Name

A unique thread name based on the supplied thread name.

Thread_Name

The name of a thread.

Description

The THREAD_NAME_CREATE() function returns a unique variant of the thread name based on the suppled thread name.

Limitations

Desktop applications only. No user interface related commands are allowed in a thread. Using any command that attempts to create or manipulate any type of user interface from within a thread will cause Alpha Anywhere to hang. Examples of such invalid functions include debug() , ui_*, StatusBar.*, ControlPanel.*, trace.*, etc. .
