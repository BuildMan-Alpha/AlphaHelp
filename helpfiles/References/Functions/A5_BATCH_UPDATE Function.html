A5_BATCH_UPDATE()

Syntax

A5_BATCH_UPDATE()

Description

The A5_BATCH_UPDATE() function displays the Update Fields dialog box when a form or browse is open.

Example

Note : To make this example work from the Interactive window, you must first select Interactive > Select Interactive Window Session ... > Customer Information.

dim frm as P

frm = form.view("Customer Information")

a5_batch_update()
