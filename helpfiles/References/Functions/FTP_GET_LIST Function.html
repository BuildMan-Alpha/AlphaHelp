FTP_GET_LIST()

File_List as C = FTP_GET_LIST( Address as C, User as C, Password as C [, Directory as C [, Just_Names as L ]] )

Argument

Description

Address

The FTP address of the transmission.

User

The user's identity.

Password

The user's password.

Directory 

Optional. Default = The directory of the Address. The path of the directory to examine.

Just_Names

Optional. Default = .F.

  • .T. = The names of the files

  • .F. = Owner, group, size, date, and name of files

Description

The FTP_GET_LIST() function retrieves a list of files in a directory over a FTP connection.

Note : The format of the directory listing varies from server to server.

Examples

This example retrieves only file names.

dim fl as C

fl = ? ftp_get_list("www.alphasoftware.com", "alphadoc2", "@lph@5", "/products/review", .t.)

? fl

= A1BookingScreen.jpg

A1ControlCentre.jpg

A1Invoice.jpg

Action_Script_ADO_ODBC_Data.jpg

Action_Script_View_Xbasic.jpg

...
