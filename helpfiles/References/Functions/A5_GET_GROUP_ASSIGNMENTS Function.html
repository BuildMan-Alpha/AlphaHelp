A5_GET_GROUP_ASSIGNMENTS()

Syntax

User_List as C = A5_GET_GROUP_ASSIGNMENTS( Master_Password as C, Group_Name as C )

Argument

Description

Master_Password

The master password for the database/

Group_Name

The name of the group.

Description

The A5_GET_GROUP_ASSIGNMENTS() function allows the Administrator to retrieve a CR-LF delimited list of users assigned to a particular group.

Limitations

Desktop applications only.

Example

? a5_get_group_assignments("mpassword", "Engineering")
