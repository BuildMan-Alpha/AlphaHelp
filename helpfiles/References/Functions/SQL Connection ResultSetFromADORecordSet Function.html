SQL::Connection::ResultSetFromADORecordSet()

Syntax

 as SQL::ResultSet = ResultSetFromADORecordSet( OpenRecordSet as OLE::ADODB.RecordSet [, Connection as OLE::ADODB.Connection ] )

Argument

Description

 

OpenRecordSet

 

Connection

 

Description

The SQL::Connection::ResultSetFromADORecordSet()creates a SQL::ResultSet object for an open ADO Recordset. This object can be passed to SQL update functions.

The Record set (and connection if provided) will be closed when the ResultSet goes out of scope.
