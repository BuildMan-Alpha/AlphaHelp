SQL_QueryImport()

Syntax

ResultObject as P = SQL_QueryImport( Connection as SQL::Connection, Query as C, Local_Filename as C [, TableInfo as SQL::TableInfo [, ShowProgress as L

[, AllowCancel as L  [, AddTableToDatabase as L [, ArgumentsPassedIn as SQL::Arguments  [, StatusBarFormatString as C]]]]]] )

 

 

Argument

Description

ResultObject

Returns the result of the action. The following properties are returned:

.HasError = .T. if there was an error, .F. if the query was imported
.ErrorText = contains the error text if there was an error
.UserCancelled =  .T. if the user cancelled before the import completed
 

Connection

A SQL::Connection object.

Query

A SQL SELECT query that selects records from the target database.

Local_Filename

The path and name of the local Alpha Five .DBF table to create.

TableInfo

Optional. A SQL::TableInfo object. If you do not need to specify a value for this argument enter null_value().

ShowProgress

Optional. Indicates whether to show a progress dialog. Default is .T.

AllowCancel

Optional. Indicates whether the user is allowed to cancel the import before it has completed. Default is .T.

AddTableToDatabase

Optional. Indicates whether the local table should be added to the local Database (i.e. the .adb file). Default is .T. (If you don't need the table to be part of the local Database, you can save time by setting this flag to .F.)

ArgumentsPassedIn

If your query references any arguments, you can pass in values for these arguments. If you do not need to specify a value for this argument, enter null_value().
   

Description

The SQL_QueryImport() function imports a table from a back-end SQL database into an Alpha Five .DBF table.

Supported By

Alpha Five Version 8 Enterprise Edition

Limitations

Desktop applications only.

Example

dim cn as sql::connection

dim cs as c

cs = "Your connection string goes here"

dim sqlquery as c

sqlquery = "select * from customers where city = :whatCity"

dim args as sql::arguments

args.set("whatCity","London")

 

dim flag as l

flag = cn.open(cs)

if flag = .f. then

     ui_msg_box("Error","Could not connect: " + cn.callresult.text)

     end

end if

 

dim resultP as p

resultP = SQL_QueryImport(cn,sqlquery,"c:\data\customers.dbf",null_value(),.t.,.t.,.t.,Args)
