CLOSE_IF_EXISTS()

Syntax

CLOSE_IF_EXISTS( Name as C )

Argument

Description

Name 

The name of a layout.

Description

The CLOSE_IF_EXISTS() function closes a layout if it is open. This function will also close reports when in design mode.

Limitations

Desktop applications only.

Example

dim ptr as P

ptr = form.view("Customer Information")

? close_if_exists("Customer Information")
