Xbasic_Stack_Dump()

Syntax

Stack_Info as C = Xbasic_Stack_Dump()

Argument

Description

Stack_Info

 

Description

The Xbasic_Stack_Dump() function returns a stack dump.

Limitations

Desktop applications only.

Example

dim stack as C

dim current_function as C

stack = xbasic_stack_dump()

current_function = word(stack,1,crlf() )

? current_function

= ui_dlg_event("@=a5_script_editor_internal",event)
