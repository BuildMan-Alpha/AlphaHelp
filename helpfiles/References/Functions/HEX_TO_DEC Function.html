HEX_TO_DEC()

Syntax

Decimal_Value as N = HEX_TO_DEC( Hex_Value as C )

Argument

Description

Hex_Value

A hexadecimal value.

Description

The HEX_TO_DEC() function returns the integer equivalent of a hexadecimal value.

Supported By

Alpha Five Version 5 and Above

Example

dim dnum as N

dim hnum as C

hnum = "D"

dnum = hex_to_dec(hnum)

? dnum ? 13.000000
