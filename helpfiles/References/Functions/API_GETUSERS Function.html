API_GETUSERS()

Syntax

User_List as C = API_GETUSERS( [ Location as C [, Domain as L ]] )

Argument

Description

User_List

A CR-LF delimited list of users.

Location

Optional. Default = the name of the current computer. The name of a computer or a domain.

Domain

Optional. Default = .F.

  • .T. = Location refers to a domain.

  • .F. = Location refers to a computer.

Description

The API_GETUSERS() function returns a CR-LF delimited list of users of a computer or domain.

Limitations

Desktop applications only.

Example

? api_getusers("alphasoftware")
