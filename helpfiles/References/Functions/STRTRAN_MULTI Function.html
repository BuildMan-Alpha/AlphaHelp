STRTRAN_MULTI()

Syntax

Output_String as C = STRTRAN_MULTI( Search_String as C, Find_List as C, Replace_List as C )

Argument

Description

Output_String

The character string produced by the replacement operation.

Search_String

The character string to search.

Find_List

A list of character strings to look for.

Replace_List

A list of character strings to replace found instances of character strings on the Find_List. Elements of the Find_List and Replace_List are paired.

Description

STRTRAN_MULTI() creates a character string by replacing occurrences of entries in the Find_List found in the Search_String with the corresponding entries in the Replace_List.

Limitations

The Find_List argument is case sensitive.

Example

dim ss as C

dim rs as C

dim ts as C

ts = "ab1 ab2 cd1 cd2"

ss = "ab" + crlf() + "cd" + crlf()

rs = "AB" + crlf() + "CD" + crlf()

stritran_multi(ts, ss, rs)   ->   "AB1 AB2 CD1 CD2"
