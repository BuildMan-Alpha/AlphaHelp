LOAD_EXTERNAL_UDFS()

Syntax

LOAD_EXTERNAL_UDFS( ALB_Filename as C )

Argument

Description

AEX_Filename

The path to and name of the .ALB library file.

Description

The LOAD_EXTERNAL_UDFS() function loads the functions of an .ALB library into an A5W web page.

Examples

<%a5

load_external_udfs("LibraryName.alb")

%>
