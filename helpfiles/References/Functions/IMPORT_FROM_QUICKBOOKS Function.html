IMPORT_FROM_QUICKBOOKS()

IMPORT_FROM_QUICKBOOKS( ) is a part of QLinker, which is available as an optional add-in for Alpha Anywhere.

Syntax

IMPORT_FROM_QUICKBOOKS( Type as C, Source_Filename_or_DSN as C, Table_to_Create as C, SQL_Statement as C [, Password as C [, User_Name as C ]] )

Argument

Description

Type

Constant: "QUICKBOOKS"

Source_Filename_or_DSN

Full filename and path of "qlinker.mdb" found in the "QLinker" folder under the Alpha installation directory.

Table_to_Create

Full filename and path of the Alpha table to create.

SQL_Statement

An SQL expression that selects the files to take from "qlinker.mdb" to create the new Alpha table.

Password

Where required by QuickBooks: QuickBooks password.

User_Name

Where required by QuickBooks: QuickBooks user name.

Description

IMPORT_FROM_QUICKBOOKS() imports a table from QuickBooks and creates an Alpha table.

Example

Import_From_Quickbooks("QUICKBOOKS","C:\Program Files\A5V5\QLinker\qlinker.mdb","C:\Databases\Expressions Test\Invoice.dbf","select refnumber , billaddress_city , billaddress_state , txndate from invoice","","")

->   C:\Databases\Expressions Test\Invoice.dbf
