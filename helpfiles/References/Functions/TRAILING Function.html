TRAILING()

Syntax

Separation_Characters as C = TRAILING( Input_String as C, Word_Number as N )

Argument

Description

Input_String

A string expression containing words separated by repeating characters.

Word_Number

Specifies the word in the Input_String   to examine .

Description

TRAILING() returns the characters that separate the Word_Number word in Input_String from the next word.

Example

trailing("now...is,,,,the:::time",2) ->  ",,,,"

trailing("now...is,,,,the:::time",1) -> "..."
