UNMASK()

Syntax

Un_Formatted_String as C = UNMASK( Formatted_String as C, Template_String as C )

Argument

Description

Formatted_String

A character string generated by the MASK() function.

Template_String

A character string containing formatting and space characters. Any non-space character may be used for formatting.

Description

UNMASK() removes the Template_String characters from the Formatted_String, which may have been created by MASK(). If the Template_String is not represented exactly in the Formatted_String, the function returns only the portion of the Formatted_String that matches the Template_String.

Example

unmask("(617) 555-1212", "(  )|  -    ") -> "6175551212"

unmask("(617) 555-1212", "(  )  -    ") -> "", because TemplateString is not identical to the template in CharacterExpression
