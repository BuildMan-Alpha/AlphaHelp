Print and PDF Functions

and Methods

Alpha Anywhere provides the following Print functions and methods.

Functions

Description

A5_GENERIC_PRINT()

Prints the text in a text object in an Xdialog form.

A5_PRINT_

CURRENT_RECORD()

Displays the Print Current Record dialog box.

NEW_PAGE()

Causes the printer to start a new page.

PDF_APPEND()

Combines two PDF files to create a third file.

PDF_ENCRYPT()

Encrypts and adds password(s) to a PDF page.

PDF_FromBitmap()

Creates a PDF file from a bitmap.

PDF_MERGE()

Merges two PDF files. Typically used to place a watermark on the pages of a PDF file.

PDF_PAGECOUNT()

Returns the number of pages in a PDF file.

PREVIEW_BROWSE()

Creates the selected browse and then previews it.

PREVIEW_FORM()

Creates the selected form and then previews it.

PREVIEW_LABEL()

Creates the selected labels and then previews them.

PREVIEW_LETTER()

Creates the selected letters and then previews them.

PREVIEW_REPORT()

Creates the selected report and then previews it.

PRINT_BROWSE()

Creates the selected browse and then displays the Print dialog box.

PRINT_FORM()

Creates the selected forms and then displays the Print dialog box.

PRINT_LABEL()

Creates the selected labels and then displays the Print dialog box.

PRINT_LETTER()

Creates the selected letters and then displays the Print dialog box.

PRINT_REPORT()

Creates the selected report and then displays the Print dialog box.

PRINT_TO_FAX()

Selects the appropriate object, merges it with the selected data, and displays the Print dialog box.

UI_GET_PRINT_

OR_PREVIEW()

Displays the Print Method dialog box and returns the user's selection.

UI_PRINTER_

DEFAULT_GET()

Returns the name of the default Windows printer.

UI_PRINTER_

OPTIONS_GET()

Retrieves a list of printer options.

UI_PRINTER_

SETTINGS_GET()

Retrieves the configuration settings of a printer.

UI_PRINTERS_GET()

Returns a list of printers installed on the current machine.

Methods

Description

BROWSE.PREVIEW()

Previews the print formatted browse on the screen.

FORM.PREVIEW()

Creates the selected forms and then displays a preview of the forms.

LABEL.PREVIEW()

Creates the selected labels and then displays a preview of the labels.

LETTER.PREVIEW()

Creates the selected letters and then displays a preview of the letters.

REPORT.PREVIEW()

Creates the selected report and then displays a preview of the report.
