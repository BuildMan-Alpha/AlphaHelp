Regular Expressions Functions and Methods

Alpha Anywhere provides the following functions and methods that use the Regex search engine.

Function

Description

FILEFIND.GREP()

Executes a regular expression against the contents of the files in the current directory that match a specified file pattern and returns a list of filenames and text.

FILEFIND.GREP_RECURSE()

Executes a regular expression against the contents of the files in the current directory and its subdirectories that match a specified file pattern and returns a list of filenames and text.

REGEX_ESCAPE()

Formats special characters so they can be processed by REGEX functions.

REGEX_GREP()

Returns all lines or elements in the target text or list that match a regular expression

REGEX_MATCH()

Searches Text for a match to Search_Expresssion returns .T. (TRUE) or .F. (FALSE).

REGEX_MERGE()

Searches Text for one or more matches to Search_Expresssion and conditionally performs replacement operations.

REGEX_SPLIT()

Searches Text for one or more matches to Search_Expresssion and returns each matching string on its own line.
