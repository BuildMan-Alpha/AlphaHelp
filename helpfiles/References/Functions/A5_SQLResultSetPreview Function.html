A5_SQLResultSetPreview()

Syntax

A5_SQLResultSetPreview( RS as SQL::ResultSet [, Title as C [, Modal as L [, Width as N [, Height as N [, Maxrows as N [, TableName as C [, ResultSetIsEmpty as L ]]]]]]] )

Argument

Description

RS

A SQL::ResultSet object.

Title

Optional. Default = "". The title of the dialog box.

Modal

Optional. Default = .T. Sets the mode of the dialog box.

  • .T. = Modal

  • .F. = Modeless

Width

Optional. Default = 100. The width of the dialog box in characters.

Height

Optional. Default = 20. The height of the dialog box in lines.

Maxrows

Optional. Default = 100. Not changeable. The number of rows of data to display.

TableName

Optional. Default = "Table ". Used for the dialog title if Title is NULL.

ResultSetIsEmpty

Optional. Default = .F.

Description

The A5_SQLResultSetPreview() function displays up to 100 rows of data stored in a SQL::ResultSet object.

Limitations

Desktop applications only.

Example

DIM conn as SQL::Connection

DIM cs as C

dim qry as SQL::Query

dim SQL as C

DIM rs as SQL::ResultSet

cs = "{A5API='Access', A5Syntax='Access', FileName='C:\Program Files\a5v6\MDBFiles\Alphasports.mdb', UserName='Admin'}"

SQL = "select * from customer"

if (conn.Open(cs)) then

    if (qry.Parse(SQL)) then

        if (qry.Execute(conn)) then

            rs = qry.ResultSet

            A5_SQLResultSetPreview(rs)

        end if

    end if

end if
