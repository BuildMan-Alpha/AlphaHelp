UI_GET_PASSWORD()

Syntax

Result_String as C = UI_GET_PASSWORD( Title as C, Caption_Text as C [, Default_String as C [, Password_Character as C ]] )

Argument

Description

Title

The title of the dialog box.

Caption_Text

The caption next to the data entry field.

Default_String

Optional. Default = "". The default value to display.

Password_Character

Optional. Default = the actual characters typed. The character to display as the user types.

Description

UI_GET_PASSWORD() creates and displays a dialog box to prompt for a password.

The Result String will be empty ("") if the Cancel button is selected.

Limitations

Desktop applications only.

Example

Prompts the user for a password and beeps if the password is incorrect.

password = "Alpha5"

user = ui_get_password("Password","Enter the password","","&")

if user = "" then

    end

end if

if user == password then

    ui_msg_box("Password","Accepted.")

else

    ui_beep()

    ui_msg_box("Password", "Password invalid.")

    end

end if
