A5_ZAP_TABLE()

Syntax

A5_ZAP_TABLE( Table_Name as C )

Argument

Description

Table_Name

The name of the table containing the records to remove.

Description

The A5_ZAP_TABLE() function removes records from the named table but preserves the table structure.

Example

? a5_zap_table("test")

= ""
