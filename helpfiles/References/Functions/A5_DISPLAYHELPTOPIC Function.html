A5_DISPLAYHELPTOPIC()

Syntax

Result_Flag as L = A5_DISPLAYHELPTOPIC( Topic as C [, Type as C [, Help_Project as C ]] )

Argument

Description

Result_Flag

.T. = Topic was found

.F. = Topic was not found

Topic

The topic or index entry to use to retrieve a page from the help text. A table of contents reference must contain both the folder and filename of the topic.

Note : To position the cursor to a topic and bookmark, append " #bookmark_name " after the ".htm ".

Note : References to index entries must be top level entries.

Type

Optional. Default = "index_entry".

  • "toc" = match table of contents entry

  • "index_entry" = match index entry

Help_Project

Optional. The default name and location for the webhelp is "http://support.alphasoftware.com/alphafivehelp/index.htm". The CHM file is in the Alpha Anywhere program folder.

Description

The A5_DISPLAYHELPTOPIC() function attempts to retrieve the selected topic from either the local compiled help file (.CHM format) stored in the Alpha Anywhere program directory or from the webhelp located at Help_Project. The choice is made by the user setting specified under Help > Help Source.
