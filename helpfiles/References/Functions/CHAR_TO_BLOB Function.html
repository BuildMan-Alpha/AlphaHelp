CHAR_TO_BLOB()

Syntax

OutputData as B = CHAR_TO_BLOB( Input_String as C )

Argument

Description

Input_String

A character string.

Description

CHAR_TO_BLOB() returns the blob equivalent of the original character Input_String.

Supported By

Alpha Five Version 5 and Above

Example

dim ch as C

dim chb as B

ch = "Rosemary"

chb = char_to_blob(ch)

chb   ->   Rosemary
