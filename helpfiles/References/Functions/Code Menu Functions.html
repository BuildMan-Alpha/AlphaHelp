Code Menu Functions

All functions are active on the Source tab only and when Xbasic code is selected.

Function

 

Description

Format Xbasic Code

 

Applies automatic formatting, including structured indents.

In/Outdent Code > Indent Code

 

Inserts a tab at the left margin.

In/Outdent Code > Outdent Code

 

Removes a tab from the left margin.

In/Outdent Code > Left Align Code

 

Moves all code to the left margin.

Comment/Un-comment Code > Comment Out

 

Inserts the comment character at the left margin.

Comment/Un-comment Code > UnComment Out

 

Removes the comment character from the left margin.

Transform

Displays the <span class=Screen>Transform Code</span> dialog, which applies special transformations to selected code.

Limitations

Web publishing applications only.
