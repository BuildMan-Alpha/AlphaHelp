*PLAIN_TO_HTML()

Syntax

HTML_Text as C = *PLAIN_TO_HTML( Plain_Text as C )

Argument

Description

HTML_Text

HTML encoded text with default font, style, and size attributes.

Plain_Text

Plain un-encoded ASCII text.

Description

The *PLAIN_TO_HTML() function converts plain ASCII text into HTML text.

Supported By

Alpha Five Version 5 and Above

Example

dim html_text as C

dim plain_text as C

plain_text = "George lived on top of a mountain."

html_text = *plain_to_html(plain_text)

? html_text

=

George lived on top of a mountain.
