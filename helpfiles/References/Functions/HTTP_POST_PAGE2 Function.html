HTTP_POST_PAGE2()

Syntax

Page_Text as C = HTTP_POST_PAGE2( URL as C [, Body as C [, Include_Headers as L [, Timeout as N ]]] )

Argument

Description

Page_Text

The HTML text of the selected page. Null if the server cannot be contacted.

URL

The address of the server that will receive your data.

Body

Optional. Default = " ". The parameters you wish to send. The size of this field is unlimited.

Include_Headers

Optional. Default = .F. (do not return headers). Allows the programmer to have the function return the headers along with the response body.

Timeout

Optional. Default = . Specifies the time, in seconds, after which the function should time out if a complete response has not been received from the server.

Description

The HTTP_POST_PAGE2() function extends the HTTP_POST Function to support XML transfers. The HTTP_POST() function downloads from URL using HTTP/1.1 POST.

Note : HTTP_POST_PAGE2() uses a Microsoft component named IXMLHTTPRequest. If the certificate does not match, the object aborts the connection.

HTTP_POST_PAGE2() is not server-safe and cannot be used in an A5W environment.
