A5_Run_VBScript()

Syntax

Result as C = A5_Run_VBScript( VBScript as C )

Argument

Description

Result

Error returned by the VBScript interpreter.

VBScript

VBScript code.

Description

The A5_Run_VBScript() function runs VBScript code and returns VBScript error messages.

Examples

This code runs a VBScript, which opens a message box with a greeting.

dim vbscript as C

 

vbscript = <<%a%
dim name
dim greeting
name = "Tom"
greeting = "Hello " & name & vbcrlf & "This is VBScript code inside Alpha Anywhere"
msgbox greeting
%a%
a5_run_VBScript(vbscript)

The code below has an intentional syntax error. In this case, the code does not compile and A5_Run_VBScript() returns the error text.

dim vbscript as C

 

vbscript = <<%a%
dim name
dim greeting
name = "Tom"
greeting = "Hello " & name & vbcrlf & "This is VBScript code inside Alpha Anywhere" &
msgbox greeting
%a%

result = a5_run_VBScript(vbscript)
if result <> ""
    ui_msg_box("script error", result, UI_STOP_SYMBOL)
end if
