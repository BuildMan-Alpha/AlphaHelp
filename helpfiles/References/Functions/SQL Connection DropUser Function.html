SQL::Connection::DropUser()

Syntax

Result_Flag as L = <SQL::Connection>.DropUser( UserName as C )

Argument

Description

Result_Flag

TRUE (.T.) if the operation was successful; otherwise FALSE (.F.).

<SQL::Connection>

A SQL::Connection object.

UserName

The user's login name.

Description

The DropUser() method removes the named user from the currently connected database or server.

Supported By

Alpha Five Version 8 Enterprise Edition

Limitations

Example

dim conn as SQL::Connection

dim connString as C

cs = "{A5API=Access,FileName='C:\Program Files\A5V8\MDBFiles\Alphasports.mdb',UserName='Admin'}"

if .not. conn.open(cs) then

    ui_msg_box("Error", conn.CallResult.text)

    end

end if

if .not. conn.DropUser("Edward") then

    ui_msg_box("Error", conn.CallResult.text)

end if

conn.close()
