A5_GetArgumentValue()

Syntax

C value a5_getArgumentValue(C argName)

Argument Description
Returned value Run-time value of argument
argName Name of argument to evaluate

Description

Used in report writer to extract values of arguments at runtime so that they can be included in the report itself.

Gets the value of an argument. This function is used in a calculated field in a Layout when you want to print the current value of an argument.
