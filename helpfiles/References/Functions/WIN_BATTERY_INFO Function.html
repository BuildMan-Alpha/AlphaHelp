WIN_BATTERY_INFO()

Syntax

 as P = WIN_BATTERY_INFO( [ Computer as C ] )

Argument

Description

A pointer variable containing information about a computer's battery.

Computer

Optional. Default = "." (local computer). The name of the computer to analyze.

Description

The WIN_BATTERY_INFO() function retrieve information about the computer's battery.

Limitations

This function does not work with Windows 98 and Windows ME.

Examples

? WIN_BATTERY_INFO()

= Availability = 3

BatteryRechargeTime =

BatteryStatus = 3

Caption = "Internal Battery"

Chemistry =
