A5_IS_EMAIL_VALID()

Syntax

Result as L = A5_IS_EMAIL_VALID( Email_Address as C )

Argument

Description

Result

.T. = Indicates that the email address format is correct.

.F. = Indicates that the email address format is incorrect.

Email_Address

One or more email addresses separated by commas (V6) or semicolons (V6 and V7).

Description

The A5_IS_EMAIL_VALID() function reports whether email addresses are properly formatted.

Supported By

Alpha Five Version 6 and Above

Example

? a5_is_email_valid("joe@@nonsense.com")

= .F.
