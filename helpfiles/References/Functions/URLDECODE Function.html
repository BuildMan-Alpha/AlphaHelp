URLDECODE()

Syntax

Output_String as C = URLDECODE( Encoded_URL_String as C )

Argument

Description

Encoded_URL_String

An encoded URL address produced by URLENCODE()or similar function.

Description

URLDECODE() decodes text that has been urlencoded. The function creates a character string by replacing the escape characters in the Encoded_URL_String with their normal ASCII equivalents.

Example

dim home_page as C

home_page = "/Development&Documentation/usersElarrabee/index.html"

http_get("http://www.server.com/ed.asp?home_page=" + urlencode(home_page))
