Performance Functions

Alpha Anywhere provides the following functions that will be useful to the Alpha Anywhere application developer.

Method

 

Description

CPUCount()

V8

Returns the number of available CPUs in the computer.

PrimaryProcessor()

V8

Returns the index of the primary processor.

SelectBestProcessor()

V8

Assigns the process affinity to the highest CPU index (0, 1, ...) with the lowest number of occurrences of alpha5.exe, a5desktopserver.exe, and a5applicationserver.exe.

SetPrimaryProcessor()

V8

Sets the processor that will run Alpha Anywhere.
