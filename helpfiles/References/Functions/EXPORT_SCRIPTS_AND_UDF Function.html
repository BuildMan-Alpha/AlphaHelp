EXPORT_SCRIPTS_AND_UDF()

Syntax

EXPORT_SCRIPTS_AND_UDF( [ Filename as C ] )

Argument

Description

Filename

Optional. The name of a text file to contain your scripts and functions. If no filename is specified, then the function prompts for filename.

Description

The EXPORT_SCRIPTS_AND_UDF() function exports all scripts and user defined functions (UDFs) to a file. If no filename is specified, Alpha Anywhere chooses a unique name based on current database.

Tip : Put EXPORT_SCRIPTS_AND_UDF() in autoexec script to backup scripts and UDFs automatically every time a database is opened.

Example

export_scripts_and_udfs("c:\myfunctions.txt")

import_scripts_and_udfs("c:\myfunctions.txt")
