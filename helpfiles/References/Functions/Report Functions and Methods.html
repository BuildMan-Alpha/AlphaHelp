Report Functions and Methods

Alpha Anywhere provides the following Report functions.

Function

Description

A5_REPORT_SAVED()

Allows users of Runtime+ to save named reports.

AVERAGE()

Returns the Average_Value of the Expression evaluated for a group of records.

BLANK_DUPLICATE()

Removes repeating instances of a field value that occur in a sequence of two or more records.

COUNT()

Returns the number of records in a group of records.

CURRENT()

Returns the value of a field or an expression based on field contents for the current record.

DUPLICATE()

Returns the number of duplicate records for the Expression evaluated for a group of records.

FIRST()

Returns the first value of the Expression evaluated for a group of records.

LAST()

Returns the last value of the Expression evaluated for a group of records.

LAYOUT_DELETE()

Deletes the named Form, Browse, Label, Letter, or Report.

LAYOUT_RENAME()

Renames or copies the named Form, Browse, Label, Letter, or Report.

MAXIMUM()

Returns the Maximum_Value of the Expression evaluated for a group of records.

MINIMUM()

Returns the Minimum_Value of the Expression evaluated for a group of records.

PAGE_AVERAGE()

Returns the average value for a field on a page of a report.

PAGE_COUNT()

Returns the count of field entries on a page of a report.

PAGE_MAXIMUM()

Returns the maximum value for a field on a page of a report.

PAGE_MINIMUM()

Returns the minimum value for a field on a page of a report.

PAGE_TOTAL()

Returns the total value for a field on a page of a report.

PAGECOUNT()

Returns the number of pages in a report.

PAGENO()

Returns the current page number of a printed layout.

PREVIEW_REPORT()

Previews a report.

PRINT_REPORT()

Creates the selected report and then displays the Print dialog box.

*parent_report_pagecount()

Returns the number of pages in the current report's parent. Useful in a custom table of contents.

Alpha Anywhere provides the following Report methods.

Function

Description

A5.REPORT_ENUM()

Returns a list of reports.

A5.REPORT_LOCATION()

Returns the dictionary location of a report.

REPORT.FAX()

Generates a report and queues it for transmission using the Windows Fax Service.

REPORT.OUTPUT_FILE_OVERRIDE()

Sets the output file to Filename for all subsequent print operations.

REPORT.PREVIEW()

Displays the report with selected records in the Print Preview dialog.

REPORT.PRINTER_OVERRIDE()

Sets the printer to printer_name for all subsequent print operations.

REPORT.SAVEAS()

Prints the report to file in PDF, HTML, RTF, or TXT formats.

REPORT.SEND()

Emails the report in PDF, HTML, RTF, or TXT formats.
