A5_MEMO_EDITOR2()

Syntax

Status as P = A5_MEMO_EDITOR2( Text_Ptr as P, Title as C [, Readonly as L [, Insertion_Point as N ]] )

Argument

Description

Status

A dot variable containing the .insertion_point and .lastbutton elements.

Text_Ptr

A dot variable with a .Text sub-element.

Title

The title of the Memo Editor.

Readonly

Optional. Default = .F.

  • .T. = The text is read only.

  • .F. = The text can be edited.

Insertion_Point

Optional. Default = 1. The initial cursor location.

Description

The A5_MEMO_EDITOR2() function displays the Memo Editor. and returns the last button pressed and the location of the cursor.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Examples

dim ptr as P

dim ptr.text as C

dim answer as P

answer = a5_memo_editor2(ptr, "Editor")

? answer

= insertion_point = 9

lastbutton = "save"
