UI_GET_PRINT_OR_PREVIEW()

Syntax

Selection as C = UI_GET_PRINT_OR_PREVIEW( [ Default_Selection as C ] )

Argument

Description

Selection

Returns either "Print" or "Preview".

Default_Selection

Optional. Default value for the dialog box. Either "Print" or "Preview".

Description

The UI_GET_PRINT_OR_PREVIEW() function displays the Print Method dialog box and returns the user's selection.

Limitations

Desktop applications only.

Example

? ui_get_print_or_preview()

= "Preview"
