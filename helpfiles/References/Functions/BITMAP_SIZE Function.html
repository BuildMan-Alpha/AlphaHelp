BITMAP_SIZE()

Syntax

Dimensions as P = BITMAP_SIZE( Filename as C, Units as C )

Argument

Description

Dimensions

A dot variable containing .height and .width properties.

Filename

The name of the Windows Bitmap (.BMP) or JPEG image file to examine.

Units

The unit of measurement. The options are:

  • "inches"

  • "cm"

  • "pixels"

Description

The BITMAP_SIZE() function returns a dot variable containing the dimensions of an image file.

Examples

dim ptr as P

ptr = bitmap_size("C:\test2.jpg", "pixels")

? ptr.height

= 1024

? ptr.width

= 768
