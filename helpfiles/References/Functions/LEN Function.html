LEN()

Syntax

Length as N = LEN( Input_String as C)

Argument

Description

Input_String

A character string.

Description

LEN() returns an integer equal to the number of characters in the Input_String.

The length of a NULL (empty) string is 0.

Examples

' if FIRSTNAME is defined as a 15-character field

? len(FIRSTNAME)

= 15

' if FIRSTNAME is a variable that contains "Steve"

? len( trim(FIRSTNAME) )

= 5
