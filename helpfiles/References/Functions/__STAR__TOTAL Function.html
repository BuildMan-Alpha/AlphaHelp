*TOTAL()

Syntax

Value as N = *TOTAL( List as C )

Argument

Description

Value

The total of a list of numeric entries.

List

A CR-LF delimited list of numeric entries.

Description

The *TOTAL() function totals a list of numeric entries.

Example

dim list as C

list = <<%a%

1

4

2

10

%a%

? *total(list)

= 17
