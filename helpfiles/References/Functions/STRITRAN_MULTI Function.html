STRITRAN_MULTI()

Syntax

Output_String as C = STRTIRAN_MULTI( Input_String as C, Find_Strings as C, Replacement_Strings as C )

Argument

Description

Output_String

The character string produced by the replacement operation.

Input_String

The character string to search.

Find_Strings

The string to find in the Input_String.

Replace_Strings

Default = "" (NULL). The string that replaces the Find_String.

Description

STRTIRAN_MULTI() is similar to STRITRAN() , except that Alpha Five performs multiple search and replace operations on the Input_String, one for each row in Find_Strings.

Find_Strings is a CR-LF delimited list of words to search for. Replacement_Strings is a CR-LF delimited string of replacement words. The first line in Replacement_Strings is used as the replacement for the first line of Find_Strings, and so on. STRTIRAN_MULTI() is not case sensitive.

Supported By

Alpha Five Version 5 and Above

Examples

Assume that Find_Strings contains the following list of words:

He

sold

cars

businesspeople

Assume that Replacement_Strings contains the following list of words:

She

leased

trucks

farmers

Assume that Search_String contains "He sold cars to businesspeople in California"

stritran_multi(Input _ String, Find_Strings, Replacement_Strings) -> "She leased trucks to farmers in California"
