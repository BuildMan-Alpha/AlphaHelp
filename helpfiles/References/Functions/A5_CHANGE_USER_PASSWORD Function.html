A5_CHANGE_USER_PASSWORD()

Syntax

Result_Flag as L = A5_CHANGE_USER_PASSWORD( User_Name as C, Old_Password as C, New_Password as C )

Argument

Description

Result_Flag

.T. (TRUE) if the change was successful. .F. (FALSE) otherwise.

User_Name

The user's name.

Old_Password

The old password.

New_Password

The new password.

Description

The A5_CHANGE_USER_PASSWORD() function allows an administrator to change a user's password.

Limitations

Desktop applications only.

Example

a5_change_user_password("jpjones", "never_give", "up_the_ship")
