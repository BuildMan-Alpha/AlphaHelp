GMT()

Syntax

Formatted_Short_Time as C = GMT( [ Format_String as C ] )

Argument

Description

Format_String

Optional. Default = "hh:mm:ss". The format for the time value.

Description

GMT() returns Greenwich Mean Time as a Formatted_Short_Time. For information on the optional Format_String parameter, see Time Format Codes.

Example

gmt() -> "19:31:18"
