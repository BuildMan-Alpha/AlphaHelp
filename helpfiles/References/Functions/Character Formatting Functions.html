Character Formatting Functions

Alpha Anywhere provides the following character formatting functions.

Function

 

Description

*ADDSLASHES()

 V5

Adds backslash characters before quote characters, so they can be used in expression strings.

*TREE_FROM_OUTLINE()

 V6

Converts an indented list into data formatted for display in a tree control.

*TREE_TO_OUTLINE()

 V6

Converts data formatted for a tree control into an indented list.

ALLTRIM()

 V5

Removes leading and trailing blanks from a Input_String.

ANSI_TO_OEM()

 V5

Converts characters from the ANSI (Windows) character set to the OEM (DOS) character set.

DELIVERY_CODE()

 V5

Returns the delivery point code for Address and Zip.

FORMAT_DATA()

 V5

Formats a character, date, numeric, or logical value.

LEFT()

 V5

Returns a character string containing the specified Number_Of_Characters starting from the left-most character of the Input_String.

LTRIM()

 V5

Removes any specified leading characters in the Input_String.

MASK()

 V5

Inserts characters from Unformatted_String into Template_String to replace all of the spaces in Template_String.

OEM_TO_ANSI()

 V5

Converts characters from the OEM (DOS) character set to the ANSI (Windows) character set.

PADC()

 V5

Pads a Unformatted_String on both sides with the Pad_String and returns a string of the specified Length.

PADL()

 V5

Pads a Unformatted_String on the left with a Pad_String, and returns a string of the specified Length.

PADR()

 V5

Pads a Unformatted_String on the right with a Pad_String, and returns a string of the specified Length.

QUOTE()

 V5

Inserts quotes around Input_String.

RAND_STRING()

 V5

Returns a random string of characters between a and z.

REGEX_ESCAPE()

 V5

Formats data for processing by REGEX functions.

REMSPECIAL()

 V5

Strips all non-alphabetic and non-numeric characters, including spaces, from a string.

REPLICATE()

 V5

Creates a character string by replicating a Repeat_String Repeat_Count number of times.

RIGHT()

 V5

Returns a character string containing the specified Number_Of_Characters starting from the Input_String 's right-most character.

RTRIM()

 V5

Removes any specified trailing characters in the Input_String.

S_QUOTE()

 V6

Quotes and formats data for use in expressions.

SPACE()

 V5

Returns a character string containing the specified Number_Of_Spaces.

STR()

 V5

Converts a Number to a right-justified character string.

STUFF()

 V5

Alters Input_String by replacing the specified Number_Of_Characters, beginning at the specified Starting_Position, with the Insert_String.

TAGGED_PATTERN()

 V5

Parses the Input_String (into "tags", or "fields") using the format specified by the Input_Format string and creates a formatted character string based on the format specified by Output_Format.

TRANSFORM()

 V5

Returns a formatted character string containing the specified Input_String modified according to the instructions of the Formatting_String.

TRIM()

 V5

Removes trailing blanks from a Input_String.

UNMASK()

 V5

Removes the Template_String characters from the Formatted_String created by MASK().

UNQUOTE2()

 V5

Removes one level of quotation marks, but never removes the last level, which is necessary for character string data.

UN_PAREN()

 V5

Removes unnecessary pairs of open and close parentheses.

UT()

 V5

Converts all the text to uppercase, and removes trailing blanks from a Input_String.

WORD_TAGGED_PATTERN()

 V5

Parses each "word" (or "record") in the input string (into "tags", or "fields") using the format specified by the Input_String string and creates an output result string based on the format specified by Output_Format.

WORD_WRAP()

 V6

Breaks a string into multiple lines of maximum length Line_Length characters.

ZBLANK()

 V6

Converts a Number to a formatted right justified character string.
