CMTOINCHES()

Syntax

Inches as N = CMTOINCHES( Centimeters as N [, Decimal as N ] )

Argument

Description

Inches

The inch equivalent of the Centimeters value.

Centimeters

The number of centimeters to convert to inches.

Decimal

Optional. Default = 2. The number of decimal positions to return.

Description

The CMTOINCHES() function converts a centimeter value to an inch value.

Examples

? cmtoinches(10,5)

= 3.93701
