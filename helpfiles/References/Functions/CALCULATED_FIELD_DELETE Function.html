CALCULATED_FIELD_DELETE()

Syntax

CALCULATED_FIELD_DELETE( Variables as P, Name as C )

Argument

Description

Variables

A pointer to the variable scope. See Variable Data Types and Scope.

Name

The name of the calculated field.

Description

The CALCULATED_FIELD_DELETE() function removes a calculated variable.

Limitations

Desktop applications only.

Examples

dim ptr as P

ptr = local_variables()

calculated_field_delete(ptr, "New")

See Also

CALCULATED_FIELD_SET(), CALCULATED_FIELD_GET(), CALCULATED_FIELD_ENUM()
