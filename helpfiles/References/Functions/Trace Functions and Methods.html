Trace Functions and Methods

Alpha Anywhere provides the following Trace methods and functions.

Function

 

Description

TRACE.CLEAR()

V5

Clear the Trace window.

TRACE.SHOW()

V5

Display the Trace window.

TRACE.WRITE()

V5

Sends a text string to the current line of the Trace window.

TRACE.WRITELN()

V5

Sends a text string to the current line of the Trace window and then moves the insertion point to the next line.

A5_IS_TRACE_WINDOW_OPEN()

V5

Indicates whether the Trace window is open.

A5_TOGGLE_TRACE_WINDOW()

V5

Opens and closes the Trace window.

TRACELN()

V5

Writes the value of an expression to the Trace window.
