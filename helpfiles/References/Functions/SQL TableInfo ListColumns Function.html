SQL::TableInfo::ListColumns()

Syntax

Column_List as C = SQL::TableInfo::ListColumns( [ Include_Types as L ] )

Argument

Description

Column_List

A CR-LF delimited list of column names.

Include_Types

Optional. Indicates whether to include Alpha Five data type, length, and decimal points.

Description

The SQL::TableInfo::ListColumns() function lists the columns for the current table. If IncludeTypes is set to true, the list also includes the Alpha data type and the length and precision in comma separated format.
