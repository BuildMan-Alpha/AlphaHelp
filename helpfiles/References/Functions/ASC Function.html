ASC()

Syntax

ASCII_Value as N = ASC( Character_String as C, [ Positive as L ] )

Argument

 

Description

Character_String

V5

A character string.

Positive

V6

Optional. Default = .F.

  • .T. = Return a positive number for high-order characters.

  • .F. = Return a negative number for high-order characters.

Description

ASC() returns the integer ASCII_Value of the first letter in the specified Character_String. The inverse function of ASC() is CHR().

Examples

dim FIRSTNAME as C

FIRSTNAME = "Borg"

? asc(FIRSTNAME)

= 66

? asc( chr(67) )

= 67

? asc("N")

= 78

? asc( chr(252) )

= -4

? asc(chr(252), .t.)

= 252
