A5_DUPLICATE_SET()

Syntax

A5_DUPLICATE_SET( Set_Name as C )

Argument

Description

Set_Name

The name of the set to duplicate.

Description

The A5_DUPLICATE_SET() function displays the Duplicate Set dialog box, which allows you to copy a set.

Limitations

Desktop applications only.

Example

a5_duplicate_set("invoice")
