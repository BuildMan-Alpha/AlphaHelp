SQL_ConnectionStringDialog()

Syntax

Connection_String as C = SQL_ConnectionStringDialog( [ Create_DB_Button as L ] )

Argument

Description

Connection_String

A connection string for the selected database.

Create_DB_Button

Optional. Default = .T. Indicates whether to display the create database button.

Description

The SQL_ConnectionStringDialog() function displays the Create SQL Connection String dialog, which helps you define a connection string for a SQL database.

Example

? SQL_ConnectionStringDialog(.t.)

= "{A5API='Access',A5Syntax='Access',FileName='C:\Databases\Northwind\northwind.mdb',UserName='Admin'}"
