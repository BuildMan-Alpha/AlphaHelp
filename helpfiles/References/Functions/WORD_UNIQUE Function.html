WORD_UNIQUE()

Syntax

Output_String as C = WORD_UNIQUE( Input_String as C, Delimiter as C )

Argument

Description

InputString

A string containing a series of delimited words or values.

Delimiter

The delimiter that separates the words or values.

Description

The WORD_UNIQUE() function returns a list of unique words in a delimited string.

Examples

s = "a" + crlf() + "ab" + crlf() + "abc" + crlf() + "ab" + crlf() + "a"

? s

= a

ab

abc

ab

a

? word_unique(s, crlf() )

= a

ab

abc
