TAN()

Syntax

Tangent as N = TAN( Radian_Angle as N )

Argument

Description
Radian_Angle A number between - p and + p.

Description

Returns the tangent of a Radian_Angle. For angles measured in degrees, use TAN(DTOR(  Degree_Value )).

Examples

tan(0) -> 0

tan(.999)) -> 1.55
