A5_EXTRACT_ADVANCE()

Syntax

Result_String as C = A5_EXTRACT_ADVANCE( Input_String as C, Start_String as C, End_String as C [, Include_Flags as L [, Case_Sensitive as L ]] )

Argument

Description

Result_String

The data found between Start_String and End_String.

Input_String

The data to process.

Start_String

The text that bounds Result_String on the left.

End_String

The text that bounds Result_String on the right.

Include_Flags

Optional. Default = .F.

  • .F. = Does not return the Start_String and End_String.

  • .T. = Returns the Start_String and End_String.

Case_Sensitive

Optional. Default = .F.

  • .F. = The search for Start_String and End_String is not case sensitive.

  • .T. = The search for Start_String and End_String is case sensitive.

Description

The A5_EXTRACT_ADVANCE() function will find and return the text between two character strings, optionally including the two character strings.

Example

? a5_extract_advance("this is a string with several words", "string", "several", .f., .f.)

= " with "
