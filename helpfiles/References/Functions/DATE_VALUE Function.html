DATE_VALUE()

Syntax

Date_Value as D = DATE_VALUE( Year as N, Month as N, Day as N )

Argument

Description

Year

The four digit integer representation of a year.

Month

The one or two digit integer representation of a month.

Day

The one or two digit integer representation of a day of the month.

Description

DATE_VALUE() returns a Date_Value for the specified year, month and day. This function is useful because it works regardless of what regional settings (e.g. U.S., United Kingdom, etc.) are in effect.

Examples

date_value(2000,2,1) -> {2/1/2000} (assuming regional settings are set for the U.S).

date_value(2000,2,1) -> {1/2/2000} (assuming regional settings are set for the U.K).
