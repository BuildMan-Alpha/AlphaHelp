A5_MDAC_VERSION()

Syntax

Version_Number as C = A5_MDAC_VERSION()

Description

The A5_MDAC_VERSION() function returns the version number of the installed MDAC ( Microsoft Data Access Components ). If MDAC is not installed, the function returns a blank string.

MDAC is required for the IMPORT_ODBC_TABLE() function.

Examples

? A5_MDAC_VERSION()

= "2.81.1117.0"
