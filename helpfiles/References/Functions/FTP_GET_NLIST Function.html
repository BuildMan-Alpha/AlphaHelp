FTP_GET_NLIST()

Syntax

File_List as C = FTP_GET_NLIST( Address as C, User as C , Password as C, Pattern as C )

Argument

Description

File_List

A list of files located in the specified directory.

Address

The FTP address of the transmission.

User

The user's identity.

Password

The user's password.

Pattern

The directory of the Address and the specification for the file types to retrieve.

Description

The FTP_GET_NLIST() function returns a list of files in a directory.

Supported By

Alpha Five Version 6 and Above

Example

dim fl as C

fl = ftp_get_file("www.alphasoftware.com", "alpha5", "password", "learningcenter.asp")
