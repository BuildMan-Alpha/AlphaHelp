ISTIME()

Syntax

Result_Flag as L = ISTIME( Time_String as C )

Argument

Description

Result_Flag

.T. = Indicates that the Time_String can be interpreted as a valid time.

.F. = Time_String can not be interpreted as a valid time.

Time_String

A character representation of a time value.

Description

The ISTIME() function returns .T. if the Time_String is a valid representation of a time value.

Note : A Time value includes a date component.

Examples

? istime("10/15/2003 10:00:00 00 am")

= .T.

? istime("10/15/2003")

= .T.

? istime("10:00:00 00 am")

= .F.
