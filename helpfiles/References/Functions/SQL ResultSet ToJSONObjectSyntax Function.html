SQL ResultSet::ToJSONObjectSyntax()

Syntax

C Result = <SQL::ResultSet>.rsToJSON([,N rows [,N startRow [,L flagForceObjectNamesToUppercase [,L forceToChar [,C dateTimeFormat [,C dateFormat ]]]]]])

Argument Description
N rows (optional) number of rows to use
N startRow (optional) first row to use
L flagForceObjectNamesToUppercase (optional) Uppercase names if .T.
L forceToChar (optional) If .T. (default) single-quote numeric and logical values
C dateTimeFormat (optional) Format string for dateTime values, as used by the time Function, e.g. "yyyy-MM-dd"
C dateFormat (optional) Format string for date values, as used by the time Function

Description

Takes an AlphaDAO resultset and converts the data to a JSON object. More efficient than the rsToJSON Function.

Limitations

Examples

dim cn as sql::connection
cn.Open("::Name::Northwind")
cn.Execute("select * from orders")
rs=cn.ResultSet
txt=rs.ToJSONObjectSyntax(10)
showvar(txt)
