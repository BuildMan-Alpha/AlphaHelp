*ARRAY_APPEND()

Syntax

*ARRAY_APPEND( Array as A, New_Value as A )

Argument

Description

Array

The array that receive the new value.

New_Value

The new value to add to Array.

Description

The *ARRAY_APPEND() function re-dimensions an array and assigns a value to the new array element.

Supported By

Alpha Five Version 5 and Above

Examples

dim b[1] as C

*array_append(b, "b1")

*array_append(b, "b2")

? b

= [1] = "b1"

[2] = "b2"

dim c[1] as N

*array_append(c, 123)

*array_append(c, 456)

? c

= [1] = 123

[2] = 456

See Also

Array Functions and Methods

Instead of using *array_append(), the preferred syntax is to use [] with no array index,

e.g.

dim a[0] as n
a[] = 0
a[] = 0
a[] = 1
