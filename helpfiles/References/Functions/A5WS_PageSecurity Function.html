Displays the Page Security Assignment dialog

A5WS_PageSecurity()

Syntax

A5WS_PageSecurity( [ Page_Name as C ] )

Argument

Description

Page_Name

The name of a page.

Description

The A5WS_PageSecurity() function displays the Page Security Assignment dialog.

Limitations

Web applications only.
