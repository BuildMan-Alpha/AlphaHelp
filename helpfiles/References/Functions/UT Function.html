UT()

Syntax

Output_String as C = UT( Input_String as C )

Argument

Description

Input_String

A character string.

Description

Converts all the text to uppercase, and removes trailing blanks from a Input_String.

Examples

ut("assorted parts    ") -> "ASSORTED PARTS"

ut(L_NAME) + upper(F_NAME) -> "DUDLEYBETH   ", if L_NAME contains "Dudley    " and F_NAME contains "Beth   "
