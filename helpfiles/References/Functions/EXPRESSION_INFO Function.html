EXPRESSION_INFO()

Syntax

Info as C = EXPRESSION_INFO( Expression as C, Property as C )

Argument

Description

Expression

An Xbasic expression referring to a table or set in the current session.

Property

The type of information returned:

Property

Information Returned

type

Returns, C - character, N - numeric, D - date, or L - logical, depending on the type of the expression.

ref_table

Returns "T" if the expression refers to any fields from a table. A simple expression (e.g. "1 + 2") would return "F" since there are no fields from tables involved.

ref_children

Returns "T" if the expression involves fields from a child table.

ref_one_to_manys

Returns "T" if the expression involves field from a one-to-many child table.

Description

EXPRESSION_INFO() returns information about an Expression. The type of information returned is specified by Property. The expression must refer to a table or set in the current session.

Supported By

Alpha Five Version 5 and Above
