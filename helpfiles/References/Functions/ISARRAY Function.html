ISARRAY()

Syntax

Result_Flag as L = ISARRAY( Data_Object as P )

Argument

Description

Result_Flag

.T. = Object is an array.

.F. = Object is not an array.

Data_Object

A variable of any type.

Description

The ISARRAY() function determines whether an object is an array.

Supported By

Alpha Five Version 6 and Above

Examples

dim ptr as P

? IsArray(ptr)

= .f.

dim ptr2[1] as P

? IsArray(ptr2)

=.t.

? arraytype(ptr2)

= "P"
