USPS_State_From_ZIP()

Syntax

State as C = USPS_State_From_ZIP( Zip5 as C )

Argument

Description

State

The name of the state containing the specified ZIP code.

Zip5

A 5 or 9 digit ZIP code. The function truncates 9 digit codes to 5 digits.

Description

The USPS_State_From_ZIP() function contacts the USPS web site and looks up the state containing the specified ZIP code.

Note : Use of this function requires an active Internet connection.

Note : ZIP codes are used by the US Postal Service (USPS) for addressing mail in the United States.

Limitations

Example

? usps_state_from_zip("01803")
= "MA"

? usps_state_from_zip("12345")
= "NY"
