CSTATE()

Syntax

State_Name as C = CSTATE( State_Code as C )

Argument

Description

State_Code

The two character code for a US state or Canadian province.

Description

CSTATE() converts a two-character string containing a State_Code to its full State_Name.

Note : Canadian province codes and names can also be processed with this function.

Supported By

Alpha Five Version 5 and Above

Examples

cstate("NY") "New York"

cstate(STATE) "New York", if STATE contains "NY"

cstate(STATE_PROV) "Saskatchewan", if STATE_PROV contains "SK"
