FTP_FILE_XFER()

Syntax

FTP_FILE_XFER( Address as C, User as C, Password as C, Source_File as C, Destination_File as C, Get_Put as C [, Modal as L [, Title as C ]] )

Argument

Description

Address

The FTP address of the transmission.

User

The user's identity.

Password

The user's password.

Source_File

The file to send.

Destination_File

The file to save remotely.

Get_Put

Specifies the operating mode.

  • "get" = The specified file is downloaded from the server to the local system.

  • "put" = The specified file is uploaded to the server from the local system.

Modal

Optional. Default = .T.

  • .T. = The displayed dialog is modeless.

  • .F. = The dialog is modal, and the script pauses till the FTP transfer is complete.

Title 

Optional. Default = "FTP Transfer". The title of the dialog box.

Description

The FTP_FILE_XFER() function is a lower-level function that is wrapped by FTP_GET_FILE()and FTP_PUT_FILE().
