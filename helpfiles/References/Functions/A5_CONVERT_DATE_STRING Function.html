A5_CONVERT_DATE_STRING()

Syntax

Output_Date as C = A5_CONVERT_DATE_STRING( Input_Date as C, Input_Format as C, Output_Format as C, Separator as C )

Argument

Description

Output_Date

The reformatted date string.

Input_Date

The original data string, containing values for year, month, and day.

Input_Format

A character string containing the letters "Y", "M", and "D" that indicates the sequence of input values.

Output_Format

A character string containing the letters "Y", "M", and "D" that indicates the sequence of output values.

Separator

The character(s) to place between the year, month, and day values.

Description

The A5_CONVERT_DATE_STRING() function reformats a date string.

Example

? a5_convert_date_string("12/31/05", "MDY", "YMD", "-")

= "05-12-31"
