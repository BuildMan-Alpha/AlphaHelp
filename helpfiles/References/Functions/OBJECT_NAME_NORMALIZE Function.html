OBJECT_NAME_NORMALIZE()

Syntax

Actual_Layout_Name as C = OBJECT_NAME_NORMALIZE( Layout_Name as C   )

Argument

Description

Actual_Layout_Name

The name that Alpha Anywhere will give to a layout object when it is instantiated.

Layout_Name

The formal name of a layout object.

Description

The OBJECT_NAME_NORMALIZE() function returns the name that Alpha Anywhere will give to a layout object when it is instantiated.

Say you have a form called "Main - Menu". When the first instance of this form is opened, Alpha Anywhere creates an object called "Main___Menu" because dashes and spaces cannot be used in object names. This function allows you to know how Alpha Anywhere will name the object.

Limitations

Desktop applications only.

Example

? object_name_normalize("customer information")

= "customer_information"
