*LINE_INSERT()

Syntax

New_String as C = *LINE_INSERT( Old_String as C, Insert_Text as C, Start_Line as N )

Argument

Description

New_String

The modified text containing line insertion(s).

Old_String

The original unmodified text.

Insert_Text

The text to insert.

Start_Line

The number of the first line to replace.

Description

The *LINE_INSERT() function inserts line(s) into a string starting at the line position defined by Start_Line.

Examples

lines = <<%str%

one

two

three

four

%str%

? *line_insert(lines, "TWO AND A HALF", 3)

= one

two

TWO AND A HALF

three

four
