STR()

Syntax

Formatted_String as C = STR( Number as N [, Length as N [, Decimal_Places as N [, Format as C ]]] )

Argument

Description

Number

The number to format.

Length

Optional. Default (10). The number of characters in the Formatted_String. Refer to Number Formats for more information.

Decimal_Places

Optional. Default (0). The number of characters after the decimal point in the Formatted_String. If you do not specify decimal places, STR() rounds the Number to the nearest whole number.

Format

Optional. Default (""). The pattern to use when formatting Formatted_String. Values for Format are:

Format Character

Meaning

$

Includes a leading currency symbol.

%

Displays a number as a percentage.

,

Separates thousands with the thousands separator.

X

Formats the number in words (e.g. for checkbook applications).

Description

STR() converts a Number to a right-justified character string. The optional Length parameter determines the length of the string, and the optional Decimal_Places parameter specifies the number of decimal places. The length must allow space for decimals, the decimal point, and a - or + sign.

Note : Because STR() returns a string with a right-justified number, you may want to use the LTRIM() function to remove leading spaces before using the string in other operations.

Examples

str(SALARY) -> "     25500", if SALARY contains 25500

str(SALARY/10000,8,4) -> "    2.5500"

str(23434.34,12,2,"$,") -> " $23,434.34"

You may need to order the records in a table on multiple fields that are of
different types. For example, you may want to create an index to display records in DEPARTMENT order, and within DEPARTMENT by SALARY.

DEPARTMENT + str(SALARY)

Alpha Anywhere enables you to create new text strings by concatenating existing strings.
The individual pieces to be concatenated must all be character strings. You, therefore, have to convert numeric fields to character strings before they can be concatenated to other fields. The following expression, for example, returns a date in the format
"Month DD, YYYY" (e.g., November 11, 1968).

cmonth(BIRTHDATE) + " " + ltrimday(BIRTHDATE?) + ", " + ltrimyear(BIRTHDATE?)
