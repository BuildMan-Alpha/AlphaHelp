CSTATE_ABREVLOOKUP()

Syntax

Abbreviation as C = CSTATE_ABREVLOOKUP( State_Province as C )

Argument

Description

Abbreviation

Two letter abbreviation.

State_Province

The full name of a US state or Canadian province.

Description

The CSTATE_ABREVLOOKUP() function returns the two character abbreviation for a US state or Canadian province.

Examples

? CSTATE_ABREVLOOKUP("Vermont")

= "VT"
