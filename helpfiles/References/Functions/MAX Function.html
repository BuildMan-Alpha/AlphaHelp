MAX()

Syntax

Largest_Number as N = MAX( ListNumber as N, Number2 as N )

Argument

Description

ListNumber

A number.

Number2

A number.

Description

MAX() returns the larger of two numbers: ListNumber or Number2.

Supported By

Alpha Five Version 5 and Above

Example

max(MATH, VERBAL) -> 600, if MATH contains 600 and VERBAL contains 400
