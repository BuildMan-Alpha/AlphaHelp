A5_ADD_NEW_GROUP()

Syntax

Result_Flag as L = A5_ADD_NEW_GROUP( Master_Password as C, Group_Name as C )

Argument

Description

Result_Flag

.T. = the group was successfully added.
.F. = the group was not added.

Master_Password

The master password for the database.

Group_Name

The name of the new group.

Description

The A5_ADD_NEW_GROUP() function allows the administrator to add a new group. Requires that the database first be password protected.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

? a5_add_new_group("mpassword", "Customer Support")

= .T.
