A5W_REPORT_SAVEAS Function

Syntax

result as c = a5w_report_saveAs(LayoutName as c,Saveas_format = "PDF", Filter = "",Order = "", Filename = "", globalVariables as p , PrintOptions = null_value())

Argument

Description

LayoutName

Report to print

Saveas_format

type of tile to create (PDF/Excel/HTML/RTF/text)

Filter

Filter expression to apply to report

Order

order expresion to apply to report detail.

Filename

Name of file to create.

globalVariables

pointer to the global variables()

PrintOptions

PDF Driver options

Arguments

Optional SQL Arguments (if required by the report datasource).

Options

Other options

Description

A5w_report_saveas is the method to call to generate a Report from the Application Server.

Example

a5w_report_saveAs("customer List","PDF","","","c:\data\custlist.pdf",global_variables())
