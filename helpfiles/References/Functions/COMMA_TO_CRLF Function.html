COMMA_TO_CRLF()

Syntax

CR-LF_List as C = COMMA_TO_CRLF( Comma_List as C )

Argument

Description

CR-LF_List

A list of items, each separated by CR-LF codes.

Comma_List

A list of items, each separated by a comma.

Description

The COMMA_TO_CRLF() function converts a comma delimited list to a CR-LF delimited list.

Supported By

Alpha Five Version 6 and Above

Examples

? comma_to_crlf("A,B,C")

= A

B

C
