A5_TOOLBAR_OPEN()

Syntax

Toolbar_Name as C = A5_TOOLBAR_OPEN( Toolbar_Definition_Name as C [, Global_Flag as L ] )

Argument

 

Description

Toolbar_Name

V7

The name of the toolbar that you attempted to open.

Toolbar_Definition_Name

V7

The name of the toolbar to open.

Global_Flag

V8

Optional. Default = .F.

  • .T. = Global toolbar (all toolbars in V7 and earlier are global)

  • .F. = Local toolbar

When Global_Flag is .F., Alpha Five will automatically do the following:

  • Hide the toolbar when the window from which the toolbar was launched looses focus

  • Show the toolbar when the window from which the toolbar was launched regains focus

  • Close the toolbar when the window from which the toolbar was launched is closed.

Description

The A5_TOOLBAR_OPEN() function opens and displays the named toolbar. You must use Toolbar_Name to close the toolbar with UI_MODELESS_DLG_CLOSE()when you are finished with it.

Note : If you are converting an existing V7 application to V8, you will have to change any calls to A5_TOOLBAR_OPEN() so that they pass in .T. as the value for the Global_Flag variable.

Limitations

Desktop applications only.

Example

? a5_toolbar_open("Reports")

= "Reports1"

if ui_modeless_dlg_exist("Reports1") then

ui_modeless_dlg_close("Reports1")

end if
