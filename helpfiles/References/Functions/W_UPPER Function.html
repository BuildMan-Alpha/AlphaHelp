W_UPPER()

Syntax

Output_String as C = W_UPPER( Input_String as C )

Argument

Description

Input_String

A character string.

Description

Capitalizes the first character of each word in the supplied Input_String.   W_UPPER() does not change the case of any other characters.

Example

? w_upper("delta alpha gamma")

= "Delta Alpha Gamma"

dim wrd as c = "DELTA GAMMA ALPHA"

? w_upper( lower (wrd) )

= "Delta Gamma Alpha"
