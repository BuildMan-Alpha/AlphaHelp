WORD_WRAP()

Syntax

New_String as C = WORD_WRAP( Old_String as C, Line_Length as N )

Argument

Description

New_String

The word wrapped version of Old_String.

Old_String

A string containing words in standard sentence format.

Line_Length

The maximum number of characters to place on a line.

Description

The WORD_WRAP() function breaks a string into multiple lines of maximum length Line_Length characters.

Example

dim s as C

s = "bird cow chicken goat tree shrub mouse flower"

? word_wrap(s, 30)

= bird cow chicken goat tree

shrub mouse flower
