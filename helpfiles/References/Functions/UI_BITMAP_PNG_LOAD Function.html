UI_BITMAP_PNG_LOAD Function

Syntax

ui_bitmap_png_load(name as c,png_data as b)

Argument

Description

name

The name of the bitmap to create.

png_data

The blob variable containing the PNG image data.

Description

Load a bitmap from a PNG blob into the bitmap cache, returns TRUE if succeeded.

Example

ui_bitmap_png_load("rainy", file.to_blob("C:\Windows\winsxs\x86_microsoft-windows-gadgets-weather_31bf3856ad364e35_6.1.7600.16385_none_4db0b909695af8f9\docked_black_rainy.png"))

ui_dlg_box("show rainy","{image=rainy}")
