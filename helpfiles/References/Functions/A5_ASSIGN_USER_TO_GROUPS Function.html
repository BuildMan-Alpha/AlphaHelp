A5_ASSIGN_USER_TO_GROUPS()

Syntax

A5_ASSIGN_USER_TO_GROUPS( Master_Password as C, User_Name as C , Group_Names as C [, Action as C ] )

Argument

Description

Master_Password

The master password for the database.

User_Name

The name of the user.

Group_Names

A semi_colon (;) delimited list of groups.

Action

Optional. Default = "add".

  • "add" = add user to groups

  • "drop" = remove user from groups

Description

The A5_ASSIGN_USER_TO_GROUPS() function allows an administrator to add or drop a user from groups.

Limitations

Desktop applications only.

Example

a5_add_new_group("mpassword", "Sales")

? a5_assign_user_to_groups("mpassword", "John", "Sales")

= .T.
