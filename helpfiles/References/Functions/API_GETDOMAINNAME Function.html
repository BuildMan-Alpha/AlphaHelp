API_GETDOMAINNAME()

Syntax

Domain_Name as C = API_GETDOMAINNAME()

Argument

Description

Domain_Name

The name of your network domain, if any.

Description

The API_GETDOMAINNAME() function returns the name of your network domain.

Limitations

Desktop applications only.

Examples

? API_GetDomainName()

= "ALPHASOFTWARE"
