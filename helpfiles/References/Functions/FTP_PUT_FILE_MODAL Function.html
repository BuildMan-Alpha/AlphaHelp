FTP_PUT_FILE_MODAL()

Syntax

FTP_PUT_FILE_MODAL( Address as C, User as C, Password as C, Source_File as C, Destination_File as C )

Argument

Description

Address

The FTP address of the transmission.

User

The user's identity.

Password

The user's password.

Source_File

The file to send.

Destination_File

The file to save remotely.

Description

The FTP_PUT_FILE() function sends a file in the foreground over a FTP connection.

Limitations

Desktop applicatons only.

Examples

ftp_put_file_modal("www.alphasoftware.com", "alpha5", "password", "c:\learningcenter.asp","/technical_documentation/learningcenter.asp")
