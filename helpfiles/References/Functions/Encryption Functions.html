Encryption Functions

Alpha Anywhere provides the following Encryption functions and methods.

Function or Method

 

Description

*CRC()

V5

Generates a 32 bit Cyclic Redundancy Code (CRC) from a text string.

*MD5()

V6

Takes as input a message of arbitrary length and produces as output a 128-bit "fingerprint" or "message digest" of the input.

API_UUIDCREATE()

V5

Returns a Universally Unique Identifier (UUID...otherwise known as a Globally unique ID or GUID).

BASE64DECODE()

V5

Takes as input base-64 encoded character data of arbitrary length and produces a decoded string.

BASE64ENCODE()

V5

Takes as input binary data of arbitrary length and produces an encoded string in base-64 format.

BASE64ENCODE_FILE()

V5

Returns a string in base-64 format encoded from binary data of arbitrary length.

CCVALID()

V5

CCVALID()returns .T. (TRUE) if a field contains a valid Credit_Card_Number.

CHKDIGIT()

V5

Returns a Checksum_Character that is the checksum digit for a string of numbers contained in the Input_String parameter.

DECRYPT_STRING()

V5

Takes a character string encrypted by the ENCRYPT_STRING() function and produces a plain text version of it.

DEFAULT_ENCRYPTION_

KEY_SET()

V5

Sets the default key that Alpha Anywhere will use to encrypt and decrypt tables.

ENCRYPT_STRING()

V5

Takes an character string of arbitrary length and produces an encrypted version of it. Note: Consider using the more secure A5_Encrypt_String Function.

HTMLENCODE()

V6

Transforms a string in order to make it HTML-safe.

MD5()

V6

Takes as input a message of arbitrary length and produces as output a 128-bit "fingerprint" or "message digest" of the input.

<TBL>.DECRYPT()

V5

Decrypts a table.

<TBL>.ENCRYPT()

V5

Encrypts a table.

<TBL>.ENCRYPTED()

V5

Returns .T. (TRUE) if the table is encrypted.

URLDECODE()

V5

Creates a character string by replacing the escape characters in the Encoded_URL_String with their normal ASCII equivalents.

URLENCODE()

V5

Encodes text so that it can be safely used as Part of a URL.

UUDECODE()

V5

Creates binary (blob) data by decoding a text string encoded by the UUENCODE() function.

UUENCODE()

V5

Creates a character string by converting the format of binary data to 7 bit ASCII.
