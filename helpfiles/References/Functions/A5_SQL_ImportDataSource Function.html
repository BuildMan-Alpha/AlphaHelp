A5_SQL_ImportDataSource()

Syntax

Result as P = A5_SQL_ImportDataSource( Data_Source as P, Table_Name as C [, Add_Table as L [, Arguments as SQL::Arguments [, Options as P [, Open_Connection as L [, Show_Progress as L [, Allow_Cancel as L ]]]]]] )

Argument

Description

Result

A dot variable that describes the result of the operation.

Element

Type

Description

.error

L

.T. indicates that an error occurred

.errorText

C

A text description of any error that occurred.

Data_Source

A dot variable that describes the data to be retrieved. See Data Source Data Structure.

Table_Name

The name of the table to refresh. The path of the current database is assumed.

Add_Table

Optional. 

Arguments

Optional. Default = NULL_VALUE(). Arguments that retrieve value(s) from variable(s) or prompts for value(s) at runtime. Only applicable to SQL Reports. Refer to SQL::Arguments Object.

Options

Optional. Default = NULL_VALUE(). Sets filter (WHERE) and order (ORDER BY) expressions for a query against a passive-linked table. A pointer dot variable with 2 elements.   Refer to Using the Options Argument.

Element

Type

Description

.filter

C

Adds to the WHERE clause in the underlying SQL expression.

.order

C

Replaces the ORDER BY clause in the underlying SQL expression.

Open_Connection

Optional. Default = NULL_VALUE().

Show_Progress

Optional. Default = NULL_VALUE().

Allow_Cancel

Optional. Default = NULL_VALUE().

Description

The A5_SQL_ImportDataSource() function imports the data specified by a Data Source name to an Alpha Anywhere table.

Limitations

Desktop applications only.
