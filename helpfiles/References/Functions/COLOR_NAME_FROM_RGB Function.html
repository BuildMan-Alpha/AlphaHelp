COLOR_NAME_FROM_RGB()

Syntax

Name as C = COLOR_NAME_FROM_RGB( Red as C, Green as C, Blue as C )

Argument

Description

Name

The Alpha Five color name, if it exists.

Red

The red value. A character string containing a number between 0 and 255.

Green

The green value. A character string containing a number between 0 and 255.

Blue

The blue value. A character string containing a number between 0 and 255.

Description

The COLOR_NAME_FROM_RGB() function returns an Alpha Five color name (if it exists) for an RGB value.

Supported By

Alpha Five Version 7 and Above

Example

? color_name_from_rgb("128", "128", "128")

= "Gray-50"
