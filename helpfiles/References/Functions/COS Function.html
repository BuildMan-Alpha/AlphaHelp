COS()

Syntax

Cosine as N = COS( Radian_Angle as N )

Argument

Description

Radian_Angle

A number between - p and + p.

Description

COS() returns the cosine of a Radian_Angle. For angles measured in degrees, use COS(DTOR( Degree_Angle )).

Examples

cos(0) -> 1

cos( dtor(90) ) -> 0
