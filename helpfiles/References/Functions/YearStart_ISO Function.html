YearStart_ISO()

Syntax

First_Week as D = YearStart_ISO( Date as D )

Argument

Description

First_Week

The first day of the ISO year for a given year.

Date

The date to be analyzed.

Description

The YearStart_ISO() function returns the first day of the ISO year.

Supported By

Alpha Five Version 7 and Above

Examples

? yearstart_iso(2005)
= {01/03/2005}

 

? yearstart_iso(2004)
= {12/29/2003}
