Date_FirstDayOfMonth()

Syntax

First Date as D = Date_FirstDayOfMonth ( [ ReferenceDate as D ] )

Argument

Description

First Date

The first date of a month.

ReferenceDate

Optional. Default = today. The date to use when calculating the first day of the month.

Description

The Date_FirstDayOfMonth() function returns the first day of a month.

Example

? Date_FirstDayofMonth()

= {06/01/2006}

? Date_FirstDayofMonth({7/4/06})

= {07/01/2006}
