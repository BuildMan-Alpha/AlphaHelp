WORD_IN_COMMON()

Syntax

Common_List as C = WORD_IN_COMMON( List1 as C, List2 as C [, Delimiter as C ] )

Argument

Description

Common_List

The words found in both lists, separated by the delimiter string.

List1

A character string list of words.

List2

A character string list of words.

Delimiter

Optional. Default = CR-LF. The character string that delimits words in each list.

Description

Returns a list of words in common between List1 and List2. The words in List1 and List2 are separated by Delimiter. This function is typically used in Xbasic programs to find a list of CR-LF delimited words that two strings have in common.

Example

The following Xbasic script defines two lists of CR-LF-delimited words, then finds the common words.

list1 = <<%a%

red

green

yellow

blue

purple

orange

%a%

list2 = <<%a%

red

orange

cyan

%a%

? word_in_common(list1, list2, crlf() )

= "red

orange"

dim a as C = "one,two,three,four"

dim b as C = "two,four"

? word_in_common(a, b, ",")

= "two,four"
