A5_EDIT_SPEED_GLOSSARY()

Syntax

A5_EDIT_SPEED_GLOSSARY()

Description

The A5_EDIT_SPEED_GLOSSARY() function displays the Speed Typing Glossary dialog box, which allows you to add, edit or delete entries.

Limitations

Desktop applications only.

Example

a5_edit_speed_glossary()
