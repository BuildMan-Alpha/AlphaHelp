MRU_QUERY()

Syntax

MRU as P = MRU_QUERY( Set_or_Table_Name as C )

Argument

Description

Set_or_Table_Name

The name of a table or set with the ".set" or ".dbf" extension.

Description

MRU_QUERY() returns a pointer variable containing the most recent values for query.filter, query.flags, and query.order.

Supported By

Alpha Five Version 5 and Above

Example

dim cs as P

cs = mru_query("code.set")

? cs

=   filter = ACTION <> ""

flags = ""

order = "NAME + ACTION"

? cs.order

= "NAME + ACTION"
