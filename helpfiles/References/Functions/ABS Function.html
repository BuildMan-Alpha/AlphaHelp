ABS()

Syntax

Value as N = ABS( Number as N )

Argument

Description

Number

A number.

Description

ABS() returns the absolute value of a Number or numeric expression. The absolute value of a number is its unsigned value.

Examples

abs(-32) -> 32

abs(SCORE1 - SCORE2) -> 25, if SCORE1 is 50 and SCORE2 is 75
