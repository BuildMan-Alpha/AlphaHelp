SORTSTRING_BY_LENGTH()

Syntax

New_List as C = SORTSTRING_BY_LENGTH( Old_List as C [, Direction as C ] )

Argument

Description

New_List

The new sorted list.

Old_List

The original CR-LF delimited list.

Direction

Optional. Default = Ascending.

Description

The SORTSTRING_BY_LENGTH() function sorts a CR-LF delimited list by the length of its entries.

Examples

dim s as C

s = <<%a%

#a

#ab

#abc

#abcd

#af

#%a%

? sortstring_by_length(s)

= a

af

ab

abc

abcd
