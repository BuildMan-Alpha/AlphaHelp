LOG()

Syntax

Natural_Log as N = LOG( Number as N )

Argument

Description

Number

A number.

Description

LOG() returns the natural logarithm (base e) of the specified Number.

Examples

log(2.718281828459045) -> 1.0000 ' natural log of e
log(10) -> 2.3026
nAmt = log(2) / log(10) 'compute log base 10 of 2
?nAmt
= 0.301029995663981
