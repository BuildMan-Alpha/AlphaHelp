LASTWORD()

Syntax

Word as C = LASTWORD( Input_String as C [, Delimiter as C ] )

Argument

Description

Word

The last word of the Input_String.

Input_String

A string with multiple words or entries separated by a delimiter.

Delimiter

Optional. Default = " ". The delimiter that separates the words or entries.

Description

The LASTWORD() function returns the last word or entry in a string.

Examples

dim s1 as C

dim s2 as C

s1 = "one two three four"

s2 = "one" + crlf() + "two" + crlf() + "three" + crlf() + "four"

? lastword(s1, " ")

= "four"

? lastword(s2, crlf() )

= "four"
