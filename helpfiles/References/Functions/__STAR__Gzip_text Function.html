*Gzip_text()

Syntax

B result *GZIP_TEXT( text as c)

Argument Description
result Compressed string, as blob
text String to compress

Description

Compress string using gzip.

Compresses a text variable. Result is a blob. The amount of compression will depend on the size of the input string. Compressing a small string might result in blob that is actually larger than the input string.

Limitations

Examples

dim bigtext as c
bigtext = replicate( "This is a test of the gzip library", 20 )
b = *GZIP_TEXT (bigtext )
? b.size()
=
48 
? len( bigtext )
=
680 
bigtext2 = *gunzip_text (b )
? len ( bigtext2 ) = 680 
? bigtext = bigtext2
=.
T.
