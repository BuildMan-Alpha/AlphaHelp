UI_EDITOR_GET()

Syntax

 as P = UI_EDITOR_GET()

Argument

Description

A pointer to the currently open Code Editor with focus.

Description

The UI_EDITOR_GET() function returns a pointer to the currently open Code Editor with focus.

Limitations

Desktop applications only.

Example

dim ptr as P

ptr = ui_editor_get()
