UI_TOP_MENU_HEIGHT()

Syntax

Height as N = UI_TOP_MENU_HEIGHT( Menu_Bar as C [, Width_Limit as N ] )

Argument

Description

Height

The height of the menu in inches.

Menu_Bar

The name of the menu.

Width_Limit

Optional. The maximum width of the menu.

Description

The UI_TOP_MENU_HEIGHT() function gets the height of a top menu (with optional width restriction).

Limitations

Desktop applications only.

Example

? UI_TOP_MENU_HEIGHT("Invoice")

= 0.270833
