SQL::Schema::TablesReferencing()

Syntax

TIL as SQL::TableInfoList = TablesReferencing(Table as SQL::TableInfo)

Argument Description
TIL SQL::TableInfoList contains a single Table property that is a collection of the tables that reference the Table argument
Table The Table for which the method supplies the referencing tables

Description

Returns a set of TableInfo items for tables that reference the table passed in
