UI_PROMPT_NEW_TABLE()

Syntax

Table_Name as C = UI_PROMPT_NEW_TABLE( [ Default_Name as C [, Warn as L ]] )

Argument

Description

Set_Name

The name of the new table.

Default_Name

The default name to place into the Specify Table Name dialog box.

Warn

Optional. Default = .F.

.F. = Does not warn you if you will replace an existing table.

.T. = Warns you if you will replace an existing table.

Description

The UI_PROMPT_NEW_TABLE() function displays the Specify Table Name dialog box, then returns the fully qualified name of the new table.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

ui_prompt_new_table("Test", .t.)

= "C:\Databases\Expressions Test\Test.dbf"
