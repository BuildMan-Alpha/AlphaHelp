A5_CreatePassiveLinkTable()

Syntax

 as P = A5_CreatePassiveLinkTable( Data_Source_Name as P [ Connection_Name as SQL::Connection [, Suppress_Error_Messages as L [, Add_Table_to_DB as L ]]] )

Argument

Description

A pointer to the new table. The pointer has these subelements.

.ErrorText = character error message

.HasError = logical error flag

Data_Source_Name

Optional. Default = NULL_VALUE(). A pointer containing a .DataSource property.

Connection_Name

Optional. Default = NULL_VALUE(). A SQL::Connection object.

Suppress_Error_Messages

Optional. Default = .F.

  • .F. = Display error messages.

  • .T. = Hide error messages.

Add_Table_to_DB

Optional. Default = .T.

  • .T. = Add table to database

  • .F. = Just create table, but do not add it to database.

Description

The A5_CreatePassiveLinkTable() function displays the Specify DataSource dialog. With this dialog you specify a SQL SELECT query that retrieves data from a SQL database.

Limitations

Desktop applications only.
