OCCURS_WORDS()

Syntax

Count as N = OCCURS_WORDS( Word as C, Search_In_List as C )

Argument

Description

Word

The word to find.

Search_In_List

A CR-LF delimited list of words to examine.

Description

OCCURS_WORDS() returns the number of times Word appears in Search_In_String.

Supported By

Alpha Five Version 5 and Above

Example

dim cs as C

cs = form_objects_get("Category_Form")

cs   ->   SUBCATEGORY

SUBCATEGORY_LBL

CATEGORY

CATEGORY_LBL

ID

ID_LBL

Tables

Tables:CATEGORY

num = occurs_words("CATEGORY", cs, crlf() )

num -> 1.000000
