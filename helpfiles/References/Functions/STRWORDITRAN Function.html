STRWORDITRAN()

Syntax

Output_String as C = STRWORDITRAN( Input_String as C, Find_Word as C [, Replacement_Word as C [, Start_Word as C [, Occurrences as N ]]] )

Argument

Description

Input_String

A character string.

Find_Word

The word to find in the Input_String.

Replacement_Word

Optional. Default = "" (NULL). The word that replaces the Find_Word.

Start_Word

Optional. Default = 1. Specifies which (and subsequent) occurrences will be replaced. A value N for the optional Start_Word begins the replacement at the N th occurrence of Find_Word.

Occurrences

Optional. Default = All. Specifies the maximum number of occurrences of Find_String to replace with Replace_String.

Description

Looks for a Find_Word in a Input_String, and replaces every occurrence of it with a Replacement_Word.  If Replacement_Word is omitted, Find_Word is replaced with NULL characters. A value N for the optional Start_Word begins the replacement at the N th occurrence of Word. A value for the optional Occurrences specifies the maximum number of occurrences of Word to be replaced.  STRWORDITRAN() is case insensitive.

Note : You can use STRWORDTRAN() with memo fields.

Supported By

Alpha Five Version 5 and Above

Examples

strworditran("partnership is the most important part of part of a successful relationship", "part", "element") -> "partnership is the most important element of element of a successful relationship"

STRTRAN("partnership is the most important part of part of a successful relationship", "part", "element") -> "elementnership is the most important element of element of a successful relationship"

strworditran("She sells sea shell on the sea shore","sea","ocean", 2, 1) -> "She sells sea shell on the ocean shore"
