SQL::Schema::TablesReferencedBy()

Syntax

TIL as SQL::TableInfoList = TablesReferencedBy(Table as SQL::TableInfo)

Argument Description
TIL SQL::TableInfoList contains a single Table property that is a collection of the tables referenced by the Table argument
Table The Table for which the method supplies the referenced tables

Description

Returns a set of TableInfo items for tables referenced by the table passed in

Limitations
