A5_DICTIONARY_READ()

Syntax

Code as C = A5_DICTIONARY_READ( Dictionary_File as C, Object_Type as C, Object_Name as C )

Argument

Description

Code

Xbasic code.

Dictionary_File

The name of a dictionary file containing a toolbar.

Object_Type

  • "tlbr" = Toolbar

  • "menu" = Menu

Object_Name

The name of the toolbar object.

Description

The A5_DICTIONARY_READ() reads the Xbasic code associated with a toolbar or menu from a dictionary file.

Limitations

Desktop applications only.

Example

The following script reads the contents of a toolbar named mainmenu into a character variable.

dict = "C:\Program Files\A5v6\Samples\Alphasports\Alphasports.Alb"

tb = a5_dictionary_read(dict, "tlbr", "mainmenu")

? tb

=

...

The following script writes the code ( newtb ) for a toolbar named mainmenu into the dictionary file Alphasports.Alb.

a5_dictionary_write(dict, "tlbr", "mainMenu", newtb)
