ROUND_DOWN()

Syntax

Result as N = ROUND_DOWN( Value as N, Decimal_Places as N )

Argument

Description

Result

A version of the number truncated to Decimal_Places decimal places.

Value

The original number.

Decimal_Places

The number of decimal places to keep.

Description

The ROUND_DOWN() function rounds a number down to the specified number of decimal places.

Examples

? round_down(pi)
= 3
? round_down(pi,1)
= 3.1
? round_down(pi,2)
= 3.14
? round_down(pi,3)
= 3.141
? round_down(pi,4)
= 3.1415
? round_down(pi,5)
= 3.14159
