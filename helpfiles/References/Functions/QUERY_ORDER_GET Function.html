QUERY_ORDER_GET()

Syntax

Order_Expression as C = QUERY_ORDER_GET( Saved_Query_Name as C )

Argument

Description

Saved_Query_Name

The name of a saved query.

Description

QUERY_ORDER_GET() returns the order expression from a saved query operation. The saved query is listed in the Operations tab of the Control Panel.

Example

order = query_order_get("Search for overdue customers")
