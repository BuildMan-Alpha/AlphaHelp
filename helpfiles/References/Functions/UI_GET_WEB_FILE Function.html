UI_GET_WEB_FILE()

Syntax

Selected_File as C = UI_GET_WEB_FILE( Title as C [, File_Types as C [, Default_Choice as C [, Default_Ext as C ]]])

Argument

Description

Selected_File

The file that the user selected.

Title

The title of the dialog box.

File_Types

Optional. Default = "" (all types). The file types to display.

  • ".a5w" = A5W Pages

  • ".html" = HTML Pages

  • ".htm" = HTML Pages

  • ".jpg" - Images

  • ".bmp" = Images

Default_Choice

Optional. Default = "". The default file name selection.

Default_Ext

Optional. Default = "". The default file type.

Description

The UI_GET_WEB_FILE() function displays a dialog box that allows the user to select a file from any web project under the current database.

Limitations

Desktop applications only.

Examples

? ui_get_web_file("Folders in Project")
