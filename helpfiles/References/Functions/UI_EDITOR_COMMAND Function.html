UI_EDITOR_COMMAND()

Syntax

UI_EDITOR_COMMAND( Command as C )

Argument

Description

Command

One of the following commands.

  • "EDIT_CUT"

  • "EDIT_COPY"

  • "EDIT_DELETE"

  • "EDIT_SELECT_ALL"

  • "EDIT_PASTE"

  • "EDIT_UNDO"

Description

The UI_EDITOR_COMMAND() function executes an editor command as if it were executed from the menus.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

In the following example the word "keyword" was put into the clipboard, then the function was run.

? ui_editor_command("EDIT_PASTE")

keyword

= ""
