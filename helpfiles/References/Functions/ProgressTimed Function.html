ProgressTimed()

Syntax

ProgressTimed( Specified as N )

Argument

Description

Specified

The number of seconds until the progress bar completes.

Description

The ProgressTimed() function displays a progress dialog box that completes in Specified seconds.

Supported By

Alpha Five Version 6 and Above

Limitations

Desktop applications only.

Example

The following script displays a progress bar that completes in 10 seconds.

progresstimed(10)

See Also
