Logical .XOR.

Syntax

Result as L = Expression1 .XOR. Expression2

Argument

Description

Expression1

A value or an Xbasic expression that evaluates to a value.

Expression2

A value or an Xbasic expression that evaluates to a value.

Description

When a logical XOR (.XOR.) is placed between two logical expressions, the result is TRUE if only one of the expressions on either side of the .XOR. is TRUE. This is a bitwise logical operation. The following truth table shows all the possible outcomes of an .OR. operation:
