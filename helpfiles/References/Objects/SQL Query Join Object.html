SQL::Query::Join Object

The SQL::Query::Join object fully describes the joining of a parent table to a referenced table. Key properties include .JoinType and (if an outer join) the .OuterJoinType.

Methods

The SQL::Query::Join object has the following methods.

Method

Description

::Parse()

Accepts an optional text string and parses it. If the parse is successful, the elements of the SQL statement will be available immediately as properties of the object.

::Reset()

Reinitializes the object as if it had just been declared.

Note : Joins can nest. A SelectTableReference can contain a join and that join has a table reference which can also contain a join and so on.

Limitations

Internal use only.
