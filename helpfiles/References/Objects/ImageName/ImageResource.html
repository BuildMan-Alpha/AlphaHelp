ImageResource Object

Description

The ImageResource object is useful when you want to create a logical image name and then map it to the physical image names defined in Alpha Five. If you use logical image names, then you don't have to change your code if you want to change the image that it uses, you just have to change the image map.

Methods

The ImageResource object has these methods:

Method

Type

Description

ImageResource.Get()

P

Creates an ImageResource object.

<ImageResource>.CleanupImages()

P

Unloads all resource bitmaps (i.e. file, system or dictionary).

<ImageResource>.GetBitmapNameMap()

P

Gets the bitmap alias dictionary, for directly translating bitmap aliases.

Supported By

Alpha Five Version 8 and Above
