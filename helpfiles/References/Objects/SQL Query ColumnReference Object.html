SQL::Query::ColumnReference Object

The SQL::Query::ColumnReference object describes a reference to a table column. It occurs wherever a column is referenced in a select list or value or logical expression. Note that it does not include the alias. The alias is a variable of the SQL::Query::SelectItem object.
