SQL::Query::GroupBy Object

The SQL::Query::GroupBy object describes the GROUP BY and HAVING expressions of a SELECT statement. The GROUP BY clause contains an optional ALL keyword and a list of column expressions upon which to group the result set. The optional HAVING clause is evaluated for each aggregated row to determine whether the summary will be included in the output.
