OLE::ADODB.Connection

The OLE::ADODB.Connection object supports reading from an ADO compatible database, such as  Microsoft Access. For more information about this object and programming techniques, refer to http://www.w3schools.com/ado/default.asp and http://msdn.microsoft.com/library/default.asp?url=/library/en-us/dnanchor/html/anch_dataaccess.asp.
