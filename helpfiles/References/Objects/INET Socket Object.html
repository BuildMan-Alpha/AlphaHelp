The INET::Socket object has these methods:

Method

Description

Accept()

Accept a new connection (the socket must already be listening).

Close()

Close an open connection.

Connect()

Open a connection to the requested host and port.

GetOptions()

Get one or more option values in a single call.

Listen()

Open a socket to listen for incoming requests.

Peek()

Read received characters from into stringNote.

Query_TCP()

Returns .T. (TRUE) if the TCP connection described by the arguments matches an active connection with the requested state.

Read()

Read characters from socket into string.

ReadBinary()

Read bytes from socket into a BLOB variable.

ReadLine()

Reads a line from the receive buffer.

ReadUntil()

Reads from the input stream until the delimiter is encountered or a timeout occurs.

SendFile()

'Send the file named.

SetOptions()

Set one or more options in a single call.

Write()

Write string data.

WriteBinary()

Write binary data.
