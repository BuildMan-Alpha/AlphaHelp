SQL::List Object

All properties that are arrays support the following properties and methods. Since an array may support several types of objects, you must check an item for token type (using the TargetTokenType property) before referencing its properties. The list items may be accessed using array syntax.

All functions take a zero-based position index. Array syntax is one-based.

Methods

The SQL::List object has the following methods.

Method

Description

::Add()

Inserts an item at the end of the list. The item must be a valid type for the list.

::InsertBefore()

Inserts an item prior to the specified position. The item must be a valid type for the list.

::Item()

Returns the item at the specified index.

::Remove()

Removes the item at the specified position.

::Reset()

Completely empties the list.
