SQL::Arguments Object

The SQL::Arguments object is a collection of arguments to be passed to a SQL function.

Properties

The SQL::Arguments object has the following properties.

Property

Type

Mode

Description

Arguments

P

R

An array of SQL::Argument objects.

Count

N

R

The number of arguments.

Methods

Method

Description

::Add()

Add a new argument.

::ArgumentNumber()

Get the sequence number (subscript) of an argument from the name.

::Clear()

Removes all arguments.

::Find()

Returns an argument by name.

::Remove()

Remove an argument.

::Set()

Sets or adds an argument value and optionally the usage.

::SetNull(Name as C, Type as C [, Usage as SQL::ArgumentUsage])

Sets a value to NULL and optionally set the usage.

Example

dim conn as SQL::Connection

dim sql as C

dim vCity as C

dim args as SQL::Arguments

vCity = ui_get_text("City", "Show Companies in what city?")

sql = "select lastname, firstname from customer where bill_city = :city Order By Company"

if .not. conn.open("{A5API=Access,FileName='C:\Program Files\a5v8\MDBFiles\Alphasports.mdb', UserName='Admin'}")

    ui_msg_box("Error", conn.CallResult.text)

    end

end if

if .not. args.Set("city", vCity)

    end

end if

if .not. conn.execute(sql, args)

    ui_msg_box("Error", conn.CallResult.text)

    end

end if

sql_resultset_preview(conn.resultset)
