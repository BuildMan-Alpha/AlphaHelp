{STARTUP}

Syntax

{STARTUP= Event }

Argument

Type

Description

Event

C

The name of an event to fire at startup.

Description

The {STARTUP} command defines an event to fire when the Xdialog is loaded.

Limitations

Desktop applications only

Example

ui_dlg_box( "Examiner" , <<%dlg%

{startup=change}

{region}

...

%dlg% , <<%code%

if a_dlg_button = "change" then

  a_dlg_button = ""

...
