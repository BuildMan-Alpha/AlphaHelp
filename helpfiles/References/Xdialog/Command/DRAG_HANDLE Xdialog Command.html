{DRAG_HANDLE}

Syntax

{DRAG_HANDLE= Bitmap_Name }

Argument

Type

Description

Bitmap_Name

C

 The name of a bitmap to use as the drag handle.

Description

The {DRAG_HANDLE} command is used in modeless xdialogs that are designed to be used as toolbars (i.e. they have the {DOCKABLE} command in them). The command specifies which bitmap to use for the "drag handle" when the toolbar is docked. Both a5_toolbar_drag_horiz and a5_toolbar_drag_vert are built in bitmaps. You are free to use any bitmap that you want. The bitmap is repeated to fill the width or height of the toolbar.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only

Example

ui_modeless_dlg_box("Drag Handle", <<%dlg%

{dockable}

{drag_handle=a5_toolbar_drag_horiz}

{drag_handle=a5_toolbar_drag_vert}

{font=Arial,10,B}

{lf};

<20,2Close!B1> {sp} <20,2Open!B2>;

{lf};

%dlg%)
