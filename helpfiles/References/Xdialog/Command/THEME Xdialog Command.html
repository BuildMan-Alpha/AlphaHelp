{THEME}

Syntax

{THEME= Mode }

Argument

Type

Description

Mode

C

  • "xp" = Enables XP Themes.

  • "classic" = Disables XP Themes.

Description

The {THEME} command enables or disables XP Themes.

Supported By

Alpha Five Version 6 and Above

Limitations

Desktop applications only

Examples

{theme=xp}

{theme=classic}
