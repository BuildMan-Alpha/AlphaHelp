{YPAD}

Syntax

{YPAD= Height }

Argument

Type

Description

Height

N

Sets line spacing.

Description

The {YPAD} command sets the spacing between lines in points.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only

Example

ui_dlg_box("ddd",<<%dlg%

{ypad=12}

fred;

tom;

harry;

%dlg%)
