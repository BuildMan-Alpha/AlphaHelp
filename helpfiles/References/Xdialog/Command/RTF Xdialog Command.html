{RTF}

Syntax

{RTF= [ %Directives% ] Width [, Columns ] Label }

Argument

Type

Description

%Directives%

C

Refer to Xdialog Language Directives and Operators.

Width

N

An integer value for the maximum number of characters to display on one line. The actual number displayed may be less because of word wrap.

Columns

N

Optional. Default = 1. An integer value for the number of rows to display.

Label

C

The label of the RTF control.

Description

The {RTF} command places an RTF text box on the Xdialog.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only

Example

dim r as P

dim a as C

r = rtf.create(" ")

r.insert("Initial text to appear in RTF field. Right click and select Properties or display RTF Editor to change font attributes.", 1)

a = r.rtf_text

ui_dlg_box("Xdialog Sample", <<%dlg%

{rtf=a}<15,2Display Editor>;

%dlg%)

r.rtf_text = a

a5_rtf_editor2(r)
