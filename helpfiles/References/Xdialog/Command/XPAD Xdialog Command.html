{XPAD}

Syntax

{XPAD= Width }

Argument

Type

Description

Width

N

The width of the column in characters, by default based on the Arial 8 point type face.

Description

The {XPAD} commands sets the horizontal space between columns.

Limitations

Desktop applications only

Example

ui_dlg_box("{XPAD}",<<%dlg%

{xpad=12}

{region}

a|b|c;

{endregion}

%dlg%)
