{UNITS}

Syntax

{UNITS= Unit }

Argument

Type

Description

Unit

C

There are five allowed values:

  • "D" = default (Arial, 8 point)

  • "P" = pixels

  • "I" = inches

  • "C" = centimeters

  • "F" = use current font for line and character spacing

Description

The {UNITS} command specifies the unit of measurement to be used when sizing Xdialog controls.

var1 = "this is a test"

ui_dlg_box("",<<%dlg%

{units=F}

{font=arial,20}

{text=50,2:var1};

Version 6 Note

The {units} command does not work for all control types. In Version 6, to allow developers to more precisely line up controls, many of the Xdialog controls can now have their sizes specified in inches, in addition to "characters" and "lines". To indicate that a unit is "inches", you use the " (double quote).

ui_dlg_box("Test",<<%dlg%

[2.90",5"selected^#list]

%dlg%)

'creates a list box that is 2.9 inches wide and 5 inches high. Or,

ui_dlg_box("Test",<<%dlg%

{Embedded=5",3":windowname}

%dlg%)

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only
