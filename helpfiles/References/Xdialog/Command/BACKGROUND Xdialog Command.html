{BACKGROUND}

Syntax

{BACKGROUND= Color }

Argument

Type

Description

Color

C

The name of a color or an RGB format.

Description

The {BACKGROUND} command specifies the background color the dialog.

Limitations

Desktop applications only

Example

format = <<%dlg%

{background=Green}{stretch=bottom}

%dlg%

ui_dlg_box("Simple Xdialog", format)
