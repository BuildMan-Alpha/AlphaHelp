{STICKTO}

Syntax

{STICKTO= [ Horizontal ], [ Vertical ] }

Argument

Type

Description

Horizontal

C

Optional. The allowed values are: "Left", "Right".

Vertical

C

Optional. The allowed values are: "Top", "Bottom".

Description

The {STICKTO} command defines which side(s) of the dialog box the control or region should stick to when the dialog box is stretched.

Version 6 Note

If you have this command:

{stretch=width}{stickto=bottom}

In v6 it is interpreted as : {stretch=width,top}, which will give you the intended result. Similarly, if you have:

{stretch=width,height}{stickto=bottom}

This is interpreted as: {stretch=width,height,top}.

Version 5 Note

If you have this command:

{stretch=width}{stickto=bottom}

would be interpreted as: {stickto=bottom}, which is not what you would have intended. Similarly, in V5, the commands:

{stretch=width,height}{stickto=bottom}

would be interpreted as: {stickto=bottom}.

Limitations

Desktop applications only

Example

ui_dlg_box("Stretch",<<%dlg%

{stretch=width}

{stickto=right,bottom}

%dlg% )
