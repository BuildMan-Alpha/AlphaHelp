{SP}

Syntax

{SP= Characters }

Argument

Type

Description

Characters

N

The number of spaces to insert. Characters may be a fractional number, such as 1.25.

Description

One or more Button objects may appear inside the Format section of a UI_DLG_BOX() function.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only

Example

ui_dlg_box("Xdialog Sample", <<%dlg%

{lf};

<15,2OK>{sp=5}<15,2Cancel>;

%dlg%)
