{ICON}

Syntax

{ICON= Bitmap_Name }

Argument

Type

Description

Bitmap_Name

C

The name of the bitmap.

Description

The {ICON} command is used in MDI modeless xdialogs to specify the name of the bitmap to use as the icon on the title bar.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only

Example

{icon="A5_ball_green"}
