{YSIZE}

Syntax

{YSIZE= Lines }

Argument

Type

Description

Lines

N

The number of lines (based on the default font, which is Arial, 8 point) to use as the default spacing between two lines on the Xdialog. The number may be fractional.

Description

The {YSIZE} command sets the default vertical spacing between controls.

Limitations

Desktop applications only

Example

result=ui_dlg_box("Title",<<%dlg%

{ysize=2}

First name:| [.32firstname];

Last name:| [.32lastname];

Address 1:| [.32address1];

Address 2:| [.32address2];

City:| [.32City];

State:| [State];

Zip:| [Zip];

{lf};

%dlg%)
