{HYPERLINK}

Syntax

{HYPERLINK= Label [ ! Event ][ ? Enabled ] }

Argument

Type

Description

Label

C

 The label of the button. The optional "= Variable " syntax allows you to use the contents of a variable as the button label.
Event

C

 The name of the event generated by the button.
Enabled

L

 Optional. See Enabled Syntax.

Description

{HYPERLINK} creates a button that looks like a hyperlink.

Supported By

Alpha Five Version 8 and Above

Limitations

Desktop applications only

Examples

dim button1Enabled as L = .t.

dim dlg_body as C

dlg_body = <<%dlg%

When button 1 has focus, the Windows 'hand' cursor is used:;

{hyperlink=Button1!Button1_clicked?Button1Enabled};

{lf};

(button1Enabled) Enable button?;

%dlg%

ui_dlg_box("Cursors",dlg_body,<<%code%

if left(a_dlg_button,6) = "button" then

    ui_msg_box("Event",a_dlg_button )

    a_dlg_button = ""

end if

%code%)
