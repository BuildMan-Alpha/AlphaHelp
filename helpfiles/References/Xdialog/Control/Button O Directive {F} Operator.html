Button O Directive {F} Operator

Syntax

{F = Fontname [, Size [, Flags ]] }

Argument

Type

Description

Fontname

C

The name of the font (e.g. Arial). Not quoted.

Size

N

Optional. Point size.

Flags

C

Optional. Indicates style. Not quoted.

  • B = bold

  • I = italic

  • U = underlne

Description

The {F} operator is used in a button's owner draw (O) directive and its {@@} operator to specify font and style. If used, the {F} operator must come first after the owner draw directive.

Example

This example creates a multi-line label for a button. Note that +U and -U start and stop the underline attribute. See Owner Draw O Directive Command Summary.

button1 = "{F=+U}press me{LF}{F=-U}now"

ui_dlg_box("Hello",<<%dlg%

<%O={@@}%20,5=button1>

%dlg%)

Refer to Button O Directive {@@} Operator.
