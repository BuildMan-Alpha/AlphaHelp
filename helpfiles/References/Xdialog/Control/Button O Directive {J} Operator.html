Button O Directive {J} Operator

Syntax

{J = Justification }

Argument

Type

Description

Justification

C

Alignment of the button label. Possible values are:

  • L = Left (default)

  • C = Center

  • R = Right

Description

The {J} operator is used in a button's owner draw (O) directive and its {@@} operator to specify the alignment of the button label.
