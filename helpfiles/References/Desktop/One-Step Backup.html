One-Step Backup

 Alpha Five allows you to backup your database with a single menu selections by selecting the Tools, Backup/Restore Database, One-step Backup command. Before you can run a one-step backup, you must first have saved backup settings.

To save backup settings:

  1. Select Tools > Backup/Restore Database > Backup Database... .

  2. Make your selections on the dialog.

  3. Press the Save Settings button.

  4. To run the one-step backup, select Tools > Backup/Restore Database > One-step Backup.

You can also run One-Step Backup from Xbasic using the Onestep_Backup() command.

Limitations

Desktop applications only.
