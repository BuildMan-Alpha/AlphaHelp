Disappearing Indexes

If Alpha Anywhere cannot evaluate an index tag expression it will delete the index tag from the collection assigned to the given table. Possible reasons that an expression cannot be evaluated include:

  • the use of reserved words as Names for variables

  • the use of variables that have not yet been initialized, or which contain values that Alpha Anywhere is not expecting

  • a corrupted record with bad data
