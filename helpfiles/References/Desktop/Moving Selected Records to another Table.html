Moving Selected Records to another Table

When you are viewing a form or browse, you can move the currently selected set of records.

  1. Select Records > Move Selected Records.

  2. Pick the target table from the Target table list.

  3. Optionally, click to view the Xbasic code generated by this procedure.

  4. Click OK to move the currently selected records to the target table and delete them from the current table or click Cancel to exit without any action.
