Exporting and Importing Scripts and Functions

You can share scripts or transfer them to other databases by exporting and importing them.

To export a script

  1. Display the Code tab of the Control Panel.

  2. Select Code > Export... or Function > Export... or Script > Export... to display the Export Scripts and/or Functions dialog box.

  3. Select the Scripts, Functions, or Scripts and Functions radio buttons.

  4. Enter the name of the file to export in the Filename field or click to navigate to and select a file to replace.

  5. Optionally, click Export All to display the Export Scripts and Functions to ASCII File dialog box.

  6. Optionally, click Export Selected... to display the Select Scripts and Functions dialog box.

  7. Place checkmarks to the left of each script or function that you want to export.

  8. Optionally, click the Select All or UnSelect All buttons to make your choices.

  9. Click OK.

To import a script

  1. Display the Code tab of the Control Panel.

  2. Select Code > Import... or Function > Import... or Script > Import... to display the Import Scripts and/or Functions dialog box.

  3. Click to navigate to and select the file to import.

  4. Click Import to display the Import from dialog box.

  5. Place checkmarks to the left of each script or function that you want to import.

  6. Optionally, select a script or function and click Preview to display the Preview dialog box.

    1. Select the script and click Preview Script.

    2. Select the function and click Preview Function.

    3. Click to exit.

  7. Optionally, click the Select All or UnSelect All buttons to make your choices.

  8. Click Import.
