Determining if a Record Has Been Changed

The following technique will determine if a record has changed before it has been saved.

Examples

dim t1 as P

dim t2 as P

dim rec as N

dim curr_rec as C

dim init_rec as C

dim flag_dirty as L

t1 = table.open("customer")

t2 = table.open("customer")

' fetch record from t1

t1.change_begin()

t1.lastname = "simpson"

rec = t1.recno()

' determine if a change has been made to any field

curr_rec = property_to_string(t1)

t2.fetch_goto(rec)

init_rec = property_to_string(t2)

if (init_rec <> curr_rec) then

    flag_dirty = .T.

else

    flag_dirty = .F.

end if
