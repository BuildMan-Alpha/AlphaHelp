ADO Record Set Genie

The ADO Record Set Genie allows you to create an Xdialog form that is bound to an ADO data source. The Xdialog form will allow you to view, edit, delete, and enter records. The genie is available while the Code Editor is open.

Select Code > More Genies > ADO Record Set to display the ADO Genie. The genie leads you through the process of:

  • selecting the OLE DB Provider

  • selecting the Database

  • entering user name and password (if necessary)

  • building the connection string

The result is the Xbasic code to insert into an Xdialog form.

Supported By

Alpha Five Version 6 and Above
