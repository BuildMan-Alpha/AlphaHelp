Disabling the Maximize and Minimize Buttons

Disabling the maximize, minimize, and other window menu functions prevents the user from closing or resizing it. The following procedure disables the functions circled in red.

  1. Open the form in design mode.

  2. Select Form > Form Properties.

  3. Display the Window tab of the Form Properties dialog box.

  4. Optionally, clear Sizable, System menu, Minimize button, and Maximize button.

  5. Display the Menus/Toolbars tab.

  6. Select "" under Menus View and Menus View right click.

  7. Click OK and save your design changes.
