Customizable Bubble Help

Customizable bubble help allows you to create bubble help that more closely matches your application requirements. You can specify the font, point size, and style of the text used in bubble help. You can also specify a different background color, and use all of the other capabilities of the Xdialog syntax, including the ability to display embedded images.

To access the Bubble Help dialog, select a form object, right click and select Properties > Help Entry > Edit ....

You can preview the bubble help by clicking the Preview button.
