Creating Layouts

When creating a new layout, you can either copy the design of an existing layout, create one from scratch, or edit the Default Browse or Default Form.

To create a new layout:

  1. In the Control Panel, click the tab corresponding to the type of layout you want to create.

  2. Click the New button on the toolbar.

  3. Alpha Anywhere opens a Genie to guide you through creating the new layout.

Copying a Layout

To copy an existing layout:

  1. In the Control Panel, right-click on the layout you want to copy. From the right-click menu, select Duplicate.

  2. Alpha Anywhere prompts you to name the new copy.

  3. Give the layout a name and click OK to save.

Deleting a Layout

To delete a layout:

  1. Right click the layout in the Control Panel.

  2. Select Delete.

  3. Click OK.
