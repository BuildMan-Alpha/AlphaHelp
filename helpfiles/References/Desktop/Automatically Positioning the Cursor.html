Automatically Positioning the Cursor

You can specify where the cursor appears during edits.

  1. In the Control Panel select the table and click Table > Edit Field Rules.

  2. Display the Data Entry tab.

  3. In the Position cursor at drop-down list box, choose one of the following options:

    • "End of selected text" - Use this option if the value in the field is generally replaced during edits. Highlighted entries are automatically deleted as soon as you start typing a new word.

    • "Start of unselected text" - Use this option for fields in which you generally edit existing values.

    • "End of unselected text" - This is the preferred option for fields in which you enter notes. You can easily add text to the end without deleting text already in a field.
