Creating Labels for a Set

If you are creating a Label layout for a set, and your set contains one or more one-to-many links, by default you will get only one Label for each primary record in your set. If you want to get a separate Label for each child linked to the primary table, you can choose a different addressee table.

To set the addressee table:

  1. Choose Label > Properties....

  2. When the Properties dialog box appears, click the Label tab.

  3. Then choose the appropriate table in the Addressee Table drop-down list box.

For example, you have a set in which individual sales representatives are linked to multiple customers in the customer table. If you want to print a different Label for each customer, instead of each representative, you can choose Customer from the Addressee Table drop-down list box.
