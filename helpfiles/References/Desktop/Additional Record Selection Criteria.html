Additional Record Selection Criteria

When you are printing the data in a form or browse, the Print Layout Genie, Print Genie , Query Genie , and Query by Expressio n give you the ability to further refine the records that you will select.

These sub-selections are particularly useful for mailing list applications in which you want to send a mailing to a subset of your list. The ability to select records randomly is also very useful in statistical sampling applications.

Criteria

Comments

No additional selection criteria

Print the records already selected by the form or browse filter. To see the current filter, right click on the layout in Design Mode, select Properties..., and display the Filter/Order tab.

Current record only

The record that you are viewing in a form or the current line in a browse.

Currently selected records

 

Sub-Selection from currently selected records

If you have applied an index or query to the layout, you can choose a sub-selection option:

Option

Comment

Top N Selected Records

Enter the number of records to select.

Bottom N Selected Records

N Records Selected at Random

Top N% Selected Records

Enter the percentage of records to select.

Bottom N% Selected Records

N% of Records Selected at Random

Every Nth Selected Record

Enter the interval between records.

Order criteria, no additional Filter criteria

Enter an Order Expression. Click to display the Order Builder.

Records selected by a Filter Expression

Add an additional Filter Expression. Click to display the Filter Builder.

Optionally, enter an Order Expression. Click to display the Order Builder.

Optionally, pick a Sub-select option.

Records selected by a saved Query definition

Select a Query Name. Click to see a list of saved queries. Optionally, pick a Sub-select option. Check Save filter/order expression if you want Alpha Five to remember the filter expression, and NOT the name of the saved Query Operation.

Marked/Un-Marked Records

Select either the Marked Records or Un-Marked Records radio buttons. Optionally, clear Search in only if you want to work with the entire table. Optionally, enter an Order Expression. Click to display the Order Builder.

Range of Record Numbers

Enter the First record number and Last record number.

Range of Key Values

Click Key to select either an Index, a Query, or a Field to use as a key. Click to see a list of key values for the First ke y and Last key fields. Click First or Last to have Alpha Five automatically use the first and last key values.

Selected Record Numbers

Enter a comma delimited list of record numbers. Place a hyphen between pairs of numbers to indicate a range.

Selected Key Values

Click Key to select either an Index, a Query, or a Field to use as a key. Click to see a list of key values. Enter a delimited list of key values. Optionally, select a different delimiter than "," from the Delimiter list.
