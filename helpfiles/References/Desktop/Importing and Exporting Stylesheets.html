Importing and Exporting Stylesheets

Alpha Five lets you import and export Stylesheets. The exported Stylesheet is saved as a text file, with an optional JPEG preview image.

Exporting a Stylesheet

To export a Stylesheet:

  1. In the Stylesheets dialog, select the Stylesheet you want to export.

  2. Click the Export button.

  3. When the Export Stylesheet dialog box appears, enter a File Name for the text file.

  4. Select the Export JPEG Preview Image if you want to create a preview image for the Stylesheet.

  5. Click the Export button.

Importing a Stylesheet

To import a Stylesheet:

  1. In the Stylesheet Gallery, click the Import button.

  2. When the Import Stylesheet dialog box appears, enter or browse for the Stylesheet (*.txt) you want to import.

  3. Click the Import button.
