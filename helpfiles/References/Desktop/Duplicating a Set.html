Duplicating a Set

When you duplicate a set, you create a new set definition that has the same structure, and contains the same field rules, layouts, operations, etc. that were associated with the original set.

Note : The tables that the set references are not duplicated.

To duplicate a set with the Duplicate Set dialog box:

  1. Right click on a set in the Control Panel, and select Utilities > Duplicate.

  2. Or, click Set > Utilities > Duplicate ... .

  3. Enter the new set name either by:

    1. Typing it into the New Set Name field

    2. Clicking Browse and selecting a set to replace from the New Set Name dialog box.

  4. Click OK to conclude or Cancel to discard your inputs.
