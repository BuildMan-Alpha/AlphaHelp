Combo Box Properties Most Frequently Used

Combo boxes in Alpha Five have a wide range of customizable properties:

Category

Property

Values

Description

Setup

Read Only

True / False

When True, the user cannot enter a value or make a selection. Used to protect fields from unauthorized input.

Fly-over Effects

True / False

Formatting changes that occur when the cursor is above the control.

Hide

True / False

When True, the control is invisible. Used to make a control disappear when it is not relevant.

Format

Width

Integer

The display width of the data.

Decimal Places

Integer

Formatting of decimal numbers.

Format Style

See Control

Text formatting that occurs when the cursor leaves the combo box.

Choices

User Defined

Developer enters a list of choices.

CR-LF delimited text list formatted as:

display-value1|save-value1
display-value2|save-value2
etc.

Computed Automatically

Values from a table

Any field from any table.

An expression that results in a list

Refer to the Expression property.

One of many lists that Alpha Five maintains internally

See Control

Display

See Control

Available only when Choices are "User Defined". Many combinations of text and/or bitmap images.

Expression

Any Xbasic expression

Available only when Choices are "Computed Automatically" and Source is "Expression". An Xbasic expression or function that returns a CR-LF formatted text list.

Watch Expression

Any Xbasic expression, but typically a field.

Used only when choices are computed automatically. Alpha Five monitors this expression for a change in its value, which triggers a repopulation of the choices list.

Help Entry

Bubble Help

Text

Short text description.
