Creating an Inline Style from an Object

In layout Design Mode:

  1. Right-click the object and select Styles > New Style. The Create New Style dialog box appears.

  2. Enter a unique name for the style, and click OK.

Create New Style Dialog Box

Now the new style is available to any object of the same type on the same layout.

Applying an Existing Inline Style

In layout Design Mode:

  1. Right-click an object, select Styles, and select the style name from the list.

  2. The selected style is applied to the object.
