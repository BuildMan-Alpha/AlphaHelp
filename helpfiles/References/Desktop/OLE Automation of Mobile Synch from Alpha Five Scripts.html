OLE Automation of Mobile Synch from Alpha Anywhere Scripts

Requesting synchronization from a Alpha Anywhere Web Page (A5W)

  1. On the form from which you want to request the synchronization, add a request as follows (see MainMenu.a5w):

<form action="MobileSync.a5w"><input type="submit" value="Mobile Sync"></form>

  1. The page requested will need to do the following (see MobileSync.a5w):

dim MobileSync as OLE::MSync.Sync

dim MobileContext as OLE::MSync.SyncOption

MobileContext.Load()

MobileSync.SetOptionObject(MobileContext)

MobileSync.DoSync()

Determining the Current User Name for MobileServer

The MobileServer prefixes the ODBC connection name with " <user name>_ ". In order to connect to the database Alpha Anywhere needs to determine the name of the current Mobile Server user.

Solution:

Use the OLE object MSync.SyncOption to determine the user name (see DemoNavigator.a5w):

dim MobileContext as OLE::MSync.SyncOption

MobileContext.Load()

session.A5WebDemo.MobileServerUserName = MobileContext.UserName

Example

The code below uses MSync.SyncOption to determine the current user and then builds a connection string for the database.

' Get the logged in user from Oracle using the

' automation object MSync.SyncOption

dim MobileContext as OLE::MSync.SyncOption

MobileContext.Load()

MobileServerUserName = MobileContext.UserName

' Create the connection string based on the user name

connectionstring = "{a5api=odbc, a5syntax=oraclelite, datasource=" + MobileServerUserName + "_A5WebDemo, username=A5WebDemo, password=A5WebDemo}"
