Create Linked Table Dialog

The Create Linked Table dialog helps you create a linked table.

Procedure

  1. Select a type of back-end database from the Connection Type list.

  2. Click Connect to display the Connect to Database dialog.

    1. Navigate to and select a database in the File Name control.

    2. Enter the other values required to access the database.

    3. Click Test Connection.

    4. Click OK if the connection succeeded.

  3. The default list of data sources shows only tables. Optionally, check Show Views, Show Linked Tables, and Show System Tables.

  4. Select the remote back-end table or view that you wish to be the data source in the Table list.

  5. Optionally, click Arguments to display the

    1.  

  6. Optionally, click Advanced to display the Create Linked Table - Advanced Settings dialog.

  7. Enter the path and name of the local Alpha Five table in the Local File control. You may also click to navigate to, name, or select an existing table.

  8. Optionally, make a selection from the Prompt Required list. The options are:

    • "None"

    • "Password only"

    • "User Name and Password"

  9. Optionally, check Server-Side Dictionary.

  10. Click Create.

  11. Click Close to exit.

Supported By

Alpha Five Version 9 Enterprise Edition

Limitations

Desktop applications only.
