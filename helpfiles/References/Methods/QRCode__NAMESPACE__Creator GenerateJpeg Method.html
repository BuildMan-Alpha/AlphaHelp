QRCode::Creator GenerateJpeg Method

Syntax

PUBLIC STATIC GenerateJpeg AS B (text AS C, flags AS C)

Argument

Description

text

QRCode input

flags

Correction level can be L,M,Q or H (L=7%, M=15%, Q=25%, H=30%).

Description

Return a blob containing the contents of a .JPEG formatted QRCode image.
