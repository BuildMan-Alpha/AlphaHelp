QUERY.DELETE()

Syntax

Result_Flag as L = QUERY.DELETE( Query_Name as C )

Argument

Description

Result_Flag

Indicates whether the query was deleted.

Query_Name

The name of a saved query. The query must exist, or an error results.

Description

The QUERY.DELETE() method displays a confirmation dialog, then deletes a saved query.

Examples

? query.delete("junk")

= .T.
