.RECNO_LIST_GET()

Syntax

Record_List as C = .RECNO_LIST_GET( Count as N [, Value as N ] )

Argument

Description

Record_List

A space delimited list of Count record numbers from the index referenced by . If the count is negative, the record numbers are retrieved from the end of the index.

Count

The number of records to return.

Value

Optional. Default = 1. Every Nth record in the index is returned. For example, if Count is 100, and Value is 5, a total of 20 record numbers are returned.

Description

The .RECNO_LIST_GET() method retrieves Count record numbers from the index referenced by . The primary use of the function is in Alpha Five's "Top n Records" and "Top n% Records" queries. If references record number order, a NULL string is returned.

Supported By

Alpha Five Version 5 and Above

Example

This script the name of the table associated with the index.

dim tbl as P

dim i as P

dim string as C

tbl = table.open("customer")

query.order = "lastname"

query.filter = ".T."

i = tbl.query_create()

string = i.recno_list_get()
