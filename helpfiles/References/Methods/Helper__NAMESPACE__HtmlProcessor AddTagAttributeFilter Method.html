<p class=A5>Description</p>
<p>
Add a tag attribute filter to an HTMLProcessor object. You can add as many tag attribute filters as you want.
The value expression gets called everytime an attribute that matches the filter is found.<br>
<br>
The example script processes all the IMG tags 'src' attributes in processed HTML to point be prefixed with 'http://myimages.net/'.<br>
<br>
The result from running the script is this:<br>
<i>
&lt html&gt<br>
&lt body&gt<br>
&lt img src="http://myimages.net/Logo.png" &gt<br>
&lt img src="http://myimages.net/Signature.png" &gt<br>
&lt /body&gt<br>
&lt /html&gt<br>
</i><br><p class=A5>Example</p>
