{Dialog.Object}.getStateInfo Method

Syntax

{Dialog.Object}.getStateInfo()

Description

Gets the data from the Dialog Component's 'state' object in the form of a text string of name value pairs, delimited with & characters.

TIP: If, in your Javascript code, you simply want to read a value from the state object, you do NOT need to use this method. You simply read the value directly from the state object. See example below.



//Get the state info as a string of name value pairs
var stateData = {dialog.object}.getStateInfo();

//Read the 'myvar1' property from the state info object
var x = {dialog.object}.stateInfo'myvar1';

//You can also set a property in the state info object directly. 
{dialog.object}.stateInfo'myvar1' = 'alpha';
