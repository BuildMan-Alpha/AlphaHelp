FILE.OPEN()

Syntax

> as P = FILE.OPEN( Filename as C, Open_Mode as N )

Argument

Description

Filename

The name of the file.

Open_Mode

The Open Mode system variables are:

Variable

Function

FILE_RO_EXCLUSIVE

Read Only (Exclusive)

FILE_RW_EXCLUSIVE

Read or Write (Exclusive)

FILE_RO_SHARED

Read Only (Shared)

FILE_RW_SHARED

Read or Write (Shared)

Description

The OPEN() method opens a data file for input or output and returns the object pointer to the file. The Open_Mode parameter determines how the file is to be accessed.

You must supply an existing Filename with the appropriate Open_Mode system variable.

Note : When you delete , Alpha Five closes the file.

Example

Open any file and preview the first 20 characters.

dim file_pointer as P

dim filename as C

dim text as C

filename = ui_get_file("Choose File", "Text File (*.txt)", "", "X")

if filename = "" then

    end

end if

file_pointer = file.open(filename, FILE_RW_SHARED)

text = file_pointer.read(20)

trace.writeln("First 20 chars: " + text)

file_pointer.close()
