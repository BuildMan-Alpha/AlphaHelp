TOOLBAR.OPEN()

Syntax

 as P = TOOLBAR.OPEN( Toolbar_Name as C )

Argument

Description

Toolbar_Name

The name of the toolbar.

Description

The TOOLBAR.OPEN() method opens the specified Toolbar_Name and returns an object pointer to the toolbar.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

tbar = toolbar.open("My Own Toolbar")
