TABLE.FILENAME_GET()

Syntax

Filename as C = TABLE.FILENAME_GET( Table_Name as C )

Usage

Desktop applications only

Argument

Description

Table_Name

Table_Name is an entry in the Tables/Sets tab on the Control Panel.

Description

The TABLE.FILENAME_GET() method returns the filename of Table_Name. Contrast this with .FILENAME_GET() method which returns the filename of the table referenced by the object pointer.

Note : If you are working with a shadow database (created by a Network Optimization operation), Filename will be the file name of the shadow table. If you want the filename of the real table, use the TABLE.ACTUAL_FILENAME_GET() method.

Example

This script returns the filename of the customer table.

filename = table.filename_get("customer")
