A5.FONTSIZE_ENUM()

Syntax

Size_List as C = A5.FONTSIZE_ENUM( Font_Name as C )

Argument

Description

Font_Name

The name of an installed font.

Description

The A5.FONTSIZE_ENUM() method returns a list of point sizes available for a named font.

Supported By

Alpha Five Version 5 and Above
