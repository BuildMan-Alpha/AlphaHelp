.ADDRESS_GET()

Syntax

Address as C = .ADDRESS_GET( Flag as C )

Argument

Description

Address

The requested address.

Flag

Which address to return.

  • "L" = Local IP address

  • "R" = Remote IP address

Description

The .ADDRESS_GET() method returns address information about the open socket.

Supported By

Alpha Five Version 6 and Above
