OLE and ActiveX Error Methods

Error Functions

The functions below retrieve error information from the last automation call (property get, property set, method). If the OLE object returns an exception, the error information is stored in the object and can be retrieved by using the following functions.

Prefix the following methods with the name of the object returned by the OLE.CREATE() method.

Method

Description

.LASTERROR_CODE()

General OLE Error code

.LASTERROR_MINORCODE()

An error code specific to the control.

.LASTERROR_SOURCE()

The OLE object returning the error (some objects use other objects internally).

.LASTERROR_DESCRIPTION()

The description of error. In many cases, this is the only useful information returned. For example: ADO returns text indicating a logon failure, but does not set the Minor Code field.

.LASTERROR_HELPFILE()

The name of the help file that may contain additional information to resolve the error.

.LASTERROR_HELPCONTEXT()

The context id within the help file specific to the error encountered.
