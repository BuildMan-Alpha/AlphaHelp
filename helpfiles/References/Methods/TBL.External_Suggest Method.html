Table.External_Suggest()
Returns a list of suggestions from a field or expression in a table.

Syntax
C Suggestions = Table.External_Suggest(C tablename, C expr,C value [,N limit [, C more_text [, C filter ]]])

Similar to .Suggest(), but does not require a pointer to an open table. This method will automatically open the table in order to get the suggestions.

See <tbl>.Suggest() for an example.
