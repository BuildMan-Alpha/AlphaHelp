STATUSBAR.SET_TEXT()

Syntax

STATUSBAR.SET_TEXT( Text as C )

Argument

Description

Text

The text to display in the Status Bar.

Description

The STATUSBAR.SET_TEXT() method displays the specified text on the Status Bar.

Limitations

Desktop applications only.

Example

Displays the status of a task.

tbl = table.open("c:\a5\a_sports\customer")

iterations = 0

tbl.fetch_first()

while .not. tbl.fetch_eof()

    tbl.fetch_next()

    iterations = iterations + 1

    statusbar.SET_text("Working - records processed: " + str(iterations,5,0))

    'Put in code that does something to each record here

end while

statusbar.clear()
