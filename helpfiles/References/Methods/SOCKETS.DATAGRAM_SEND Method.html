<SOCKETS>.DATAGRAM_SEND()

Syntax

<SOCKETS>.DATAGRAM_SEND( Host as C, Port as N, Data as C )

Argument

Description

Host

Host address.

Port

Port number.

Data

Data to send.

Description

The SOCKETS>.DATAGRAM_SEND() method sends Data to Host at port number Port.

Supported By

Alpha Five Version 6 and Above

Example

Sockets.datagram_send("127.127.127.127", 4557, "ui_msg_box('', 'Hello ' + time()) " + crlf())
