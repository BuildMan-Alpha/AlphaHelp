CLIPBOARD.REGISTER_TYPE()

Syntax

Registered_Type as C = CLIPBOARD.REGISTER_TYPE( Clipboard_data_type as C )

Argument

Description

Clipboard_data_type

The name for the type of data stored in the clipboard.

Description

The CLIPBOARD.REGISTER_TYPE() method allows the user to specify a name for the type of data stored on the clipboard. This can be used for cutting and pasting user defined data formats across programs or even within Alpha Five.

Limitations

Desktop applications only.

Example

This piece of code puts a record from the Invoice Items table onto the clipboard. The data type is recorded as ?Invoice Item' so that a piece of code that pastes data can determine the format of the data on the clipboard

item_type = :Clipboard.register_type("Invoice Item")

item_data = items.record_data_get()

:Clipboard.set_data(item_data, item_type)

This snippet of code will paste an Invoice Item record into the Invoice Items table, but only if the recorded clipboard format type matches "Invoice Item".

if :Clipboard.has_data(item_type) then

   item_data = :Clipboard.get_data(item_data,item_type)

   items.enter_begin()

       items.record_data_set(item_data)

       items.invoice_number = invoice_header.invoice_number

   items.enter_end(.T.)

end if
