<StringScanner>.SkipToDigit()

Syntax

Result_Flag as L = <StringScanner>.SkipToDigit()

Argument

Description

Result_Flag

True if an digit was found.

Description

The <StringScanner>.SkipToDigit() function moves the offset to the next digit (0-9). The offset does not move if it is already pointing to a digit.

Examples

dim scanner as P

dim txt as C

txt = <<%text%

This is wonderful prose written

by a technical 12 writer of note.

%text%

scanner = stringscanner.create(txt)

? scanner.skiptodigit()

= .T.

? scanner.GetToOffset()

= This is wonderful prose written

by a technical

? scanner.GetRemainder()

= 12 writer of note.
