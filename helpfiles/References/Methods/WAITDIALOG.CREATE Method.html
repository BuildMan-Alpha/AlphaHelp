.CREATE()

Syntax

.CREATE( Message_Rows as N, Style as C )

Argument

Description

Message_Rows

The number of lines to allow below the progress bar for text messages, maximum of 3.

Style

  • "Percent" - a standard progress bar display that shows the degree of completion

  • "Repeating" = the progress bar builds from the left, disappears, and then builds again

  • "Bounce" = the progress bar shuttles left to right, then right to left, and repeats continuously

Description

The .CREATE() method creates and displays the {WAITDIALOG} dialog box with progress bar.

Limitations

Desktop applications only.

Examples

See {WAITDIALOG} for an example.

See Also
