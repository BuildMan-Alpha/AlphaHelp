FILEFIND.LIST_NEW_FILES()

Syntax

FILEFIND as C = FILEFIND.LIST_NEW_FILES( Path as C, Pattern as C [, Date_String as C [, Time_String as C [, AM_PM as C ]]] )

Argument

Description

Path

The starting folder.

Pattern

A pattern to select file names using wildcard characters like ?*' and ?? '.

Date_String

Optional. Default = Today. The date on or after which the file was modified. Note: The date must be in MM/DD/YY format.

Time_String

Optional. Default = None. The time on or after which the file was modified.

AM_PM

Optional. Default = None. Enter one of the following strings.

  • "AM"

  • "PM"

Description

The FILEFIND.LIST_NEW_FILES() method recursively searches folders and sub-folders and returns a list of files that match a pattern that were modified on or after a specified date and time.

Example

dim lst as C

dim cnt as N

lst = filefind.List_New_Files("c:\\", "*.txt", "3/1/03")

cnt = line_count(lst)

? cnt

= 38.000000
