.PEEKBIT()

Syntax

New_Blob as B = .PEEKBIT( Position as N )

Argument

Description

Position

The position of bit to return.

Description

The .PEEKBIT() method returns the bit at Position.

Example

The following example shows the bits that make the ASCII character A.

b1 = char_to_blob("A")

for i = 8 to 1 step -1

?b1.peekbit(i)

next i
