.FIND_LAST()

Syntax

 as P = .FIND_LAST()

Description

The .FIND_LAST() method returns the last key value.

Example

dim t as P

dim i as P

dim x as P

t = table.open("customer")

i = t.index_primary_put("Lastname")

x = i.find_last()

? x

= Key = "Wong                                             "

Record = 28.000000
