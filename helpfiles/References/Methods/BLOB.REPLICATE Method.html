.REPLICATE()

Syntax

.REPLICATE( Blob_Data as B, Count as N )

Argument

Description

Blob_Data

A blob variable with data that you wish to write into .

Count

The number of times to write Blob_Data.

Description

The .REPLICATE() method initializes and writes Blob_Data Count times.

Example

dim b1 as B

dim b2 as B

b2 = char_to_blob("d")

b1.replicate(b2, 4)

? b1

= dddd
