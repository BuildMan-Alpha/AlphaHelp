OLE.Alias_Object()

Syntax

 as P = OLE.Alias_Object()

Argument

Description

An OLE object.

Description

The OLE.Alias_Object() method creates an OLE object to be used with the <OLE>.SET() method.

Examples

a = ole.Alias_Object()
a.Set("MyOwnName","dhtmledit.dhtmledit")
h.class = "MyOwnName"

dim h.object as P
ui_modeless_dlg_box("x","{activex=100,20h}")
