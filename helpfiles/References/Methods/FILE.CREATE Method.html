FILE.CREATE()

Syntax

 as P = FILE.CREATE( Filename as C, Open_Mode as N )

Argument

Description

Filename

The file name, including drive and path.

Open_Mode

The Open Mode system variables are:

Variable

Function

FILE_RO_EXCLUSIVE

Read Only (Exclusive)

FILE_RW_EXCLUSIVE

Read or Write (Exclusive)

FILE_RO_SHARED

Read Only (Shared)

FILE_RW_SHARED

Read or Write (Shared)

Description

The FILE.CREATE() method creates and opens a new file, returning a object pointer reference to the file.

Note : When you delete , Alpha Five closes the file.

Example

Ask user to type a new filename.

filename = ui_get_file("Create a file","Text Files (*.txt)", "c:\a5\defaults.txt","N")

if (filename = "") then

    end

end if

Create the file.

file_pointer = file.create(filename, FILE_RW_EXCLUSIVE)

Write first line of data to the file.

for i = 1 TO 5

    file_pointer.write("A5_" + ltrim( str(i) ) )

next i

'New line

file_pointer.write_line("")

'Write another line of data

for i = 1 TO 5

    file_pointer.write("VAR" + ltrim( str(i) ) )

next i

file_pointer.flush()

file_pointer.close()
