A5.TOOLBAR_ENUM()

Syntax

Toolbar_List as C = :A5.TOOLBAR_ENUM( [ Flag as N ] )

Argument

Description

Flag

Optional.

  • NULL (Default)

  • 1 = Full paths and filenames of the sets.

Description

The A5.TOOLBAR_ENUM() method returns a CR-LF delimited list of all toolbars in the database.

Supported By

Alpha Five Version 5 and Above
