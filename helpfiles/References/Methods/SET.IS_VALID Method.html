.IS_VALID()

Syntax

Result_Flag as L = .IS_VALID()

Description

The .IS_VALID() method returns .T. (TRUE) if the set has a valid structure; otherwise, it returns .F. (FALSE).

Supported By

Alpha Five Version 5 and Above

Example

dim ptr as P

ptr = set.open("invoice")

? ptr.is_valid()

= .T.
