<FILE>.WRITE()

Syntax

Bytes_Written as N = <FILE>.WRITE( Text as C [, Number_of_Bytes as N ] )

Argument

Description

Bytes_Written

The number of bytes written to the file.

Text

A variable or expression containing character data.

Number_of_Bytes

Optional. Default = all. The number of bytes to write.

Description

The <FILE>.WRITE() method transfers the contents of a Text string to the location of the current file pointer in the open file referenced by the object pointer, <FILE>. The optional Number_of_Bytes parameter specifies the number of bytes (or characters) from Text that will be written to the file.

Supported By

Alpha Five Version 5 and Above

Examples

This script creates a defaults file with 4 groups of 4 characters, VAR1 .. VAR4.

file_pointer = file.create("C:\a5\defaults.txt", FILE_RW_EXCLUSIVE)

for i = 1 TO 4

    file_pointer.write("VAR" + ltrim( str(i) ) )

next i

file_pointer.flush()

file_pointer.close()
