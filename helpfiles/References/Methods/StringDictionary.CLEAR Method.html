.CLEAR()

Syntax

.CLEAR()

Argument

Description

Data_Value

Character data to be saved in the dictionary.

Key_Value

A unique character value.

Description

The .CLEAR() method Removes all keys from a StringDictionary list.

Supported By

Alpha Five Version 7 and Above

Examples

dim sd as StringDictionary

sd.set("abc", "3")

? sd.get("abc")

= "3"

sd.clear()

? sd.get("abc")

= ""
