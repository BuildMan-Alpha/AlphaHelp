Field Rules Methods

Alpha Five provides the following methods for modifying Field Rules using Xbasic. Use the <TBL>.RULES_GET() method to create the pointer.

Function

 

Description

<Field_Rules>.Change_Begin()

V5

Indicates that you are about to change the value of a field rule.

<Field_Rules>.Change_End()

V5

Indicates that you are finished changing the value of one or more field rules and want to save the changes.

<Field_Rules>.Mode_Get()

V5

Determines the current data entry mode of the pointer.
