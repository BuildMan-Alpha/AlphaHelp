FILE.DRIVE_TYPE()

Syntax

Drive_Type as C = FILE.DRIVE_TYPE( Drive as C )

Argument

Description

Drive_Type

Valid types are:

  • "Removable"

  • "Fixed"

  • "CD-ROM"

  • "Remote"

Drive

The letter of the drive.

Description

The FILE.DRIVE_TYPE() method returns the type of Drive.

Examples

Type the following in the Interactive window:

file.drive_type("a:") -> "Removable"

file.drive_type("c:") -> "Fixed"

file.drive_type("g:") -> "CD-ROM"

file.drive_type("A:\") ? ERROR  '(because \" has a special meaning in Alpha Five. The backspace character is used to define quotes within a quoted string.)

file.drive_type("A:\ ") -> "Removable"
