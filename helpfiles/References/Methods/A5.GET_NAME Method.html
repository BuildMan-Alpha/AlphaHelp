A5.GET_NAME()

Syntax

Database_Filename as C = A5.GET_NAME()

Description

The A5.GET_NAME() method returns the filename of the current database.

Supported By

Alpha Five Version 5 and Above

Example

? a5.Get_Name()

= "C:\Databases\Expressions Test\Expressions_Test.Adb"

? file.filename_parse(a5.get_name(),"NE")

= "Expressions_Test.Adb"
