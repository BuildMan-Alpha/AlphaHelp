<OBJECT>.FILTER_GET()

Syntax

Filter_Expression as C = <OBJECT>.FILTER_GET()

Description

The <OBJECT>.FILTER_GET() method applies to:

  • Browses (for <OBJECT> use either the <BROWSE> pointer or the name of the browse)

  • Forms (for <OBJECT> use either the <FORM> pointer or the name of the form)

The <OBJECT>.FILTER_GET() method returns the current filter (a logical ex<x>pression ) for the layout.

Limitations

Desktop applications only.

Example

This example shows the state of the CustQueryReport form after a Query by Form operation on the Lastname field.

dim ptr as P

ptr = form.load("CustQueryReport")

ptr.show()

? ptr.filter_get()-> (LASTNAME > "b")
