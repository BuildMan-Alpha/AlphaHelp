TABLE.DELETE_FIELDS()

Syntax

TABLE.DELETE_FIELDS( Table_Name as C, Field_List as C )

Argument

Description

Table_Name

The name of the table that is losing fields.

Field_List

A CR-LF delimited list of field(s).

Description

The TABLE.DELETE_FIELDS() method deletes fields from an existing table. Field_List is a CR-LF delimited string containing the names of fields to be deleted.

All of the existing data in Table_Name is preserved when the fields are deleted (except, or course for the data in the fields being deleted).

Example

This script deletes several the Lastname field from the Customer Information table.

table.delete_fields("Customer Information", "Lastname")
