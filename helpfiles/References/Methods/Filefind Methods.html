Filefind Methods

The FILEFIND.FIRST()and FILEFIND.GET() methods work with the files in a single directory. The FILEFIND.GET_RECURSE()and FILEFIND.LIST_NEW_FILES() methods search the starting directory and all sub-directories found below it.

Prefix the following methods with the FILEFIND keyword.

Function

 

Description

.FIRST()

V5

Creates the object pointer that can be used to find all of the files in a directory.

.GET()

V5

Returns a list of all the files that match a pattern in a directory.

.GET_RECURSE()

V5

A high level wrapper for FILEFIND.GET() that recursively searches sub-directories to find all files that match a pattern.

.GREP()

V6

Executes a regular expression against the contents of the files in the current directory that match a specified file pattern and returns a list of filenames and text.

.GREP_RECURSIVE()

V6

executes a regular expression against the contents of the files in the current directory and its sub-directories that match a specified file pattern and returns a list of filenames and text.

.LIST_NEW_FILES()

V5

Recursively searches folders and sub-folders and returns a list of files that match a pattern that were modified on or after a specified date and time.

Prefix the following methods with the object pointer.  FILEFIND.FIRST() returns the pointer.

Function

 

Description

.CREATION_TIME()

V5

Returns the time the file was created.

.EOF()

V5

Returns .T. if the last .EOF() method moved past the last file in the list of files referenced by the object pointer.

.GET_RECURSE()

V5

A high level wrapper for FILEFIND.GET()that recursively searches subdirectories to find all files that match a pattern.

.IS_ARCHIVE()

V5

Returns .T. if the file is an archive file.

.IS_DIRECTORY()

V5

Returns .T. if the file is a directory.

.IS_HIDDEN()

V5

Returns .T. if the file is hidden.

.IS_SYSTEM()

V5

Returns .T. if the file is a system file.

.LAST_ACCESS_TIME()

V5

Returns the last time the file was accessed.

.LAST_WRITE_TIME()

V5

Returns the last time the file was written to.

.NAME()

V5

Returns the name of the current file in the list of files.

.NEXT()

V5

Moves to the next file in the list of files
