.GET_ROW_POSITIONS()

Syntax

Position_List as C = .GET_ROW_POSITIONS( Format as C [, Style as P ] )

Argument

Description

Position_List

 

Format

A data format, as selected from the Select Format dialog box.

Style

Internal use only.

Description

The .GET_ROW_POSITIONS() method returns a CR-LF delimited string of row ending positions in inches.

Limitations

Desktop applications only.
