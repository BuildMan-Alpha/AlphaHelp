.TYPE_GET()

Syntax

Type as N = .TYPE_GET()

Description

The .TYPE_GET() method returns the type of the index referenced by the object pointer. The index type can be:

Type of Index File

Number

a tag in an index file

2

a query list

6

record number order

-1

Supported By

Alpha Five Version 5 and Above

Example

dim tbl as P

dim qry as P

tbl = table.open("test")

query.filter = "ID <> ''"

query.options = "D"

qry = tbl.query_create()

? qry.type_get()

= 6
