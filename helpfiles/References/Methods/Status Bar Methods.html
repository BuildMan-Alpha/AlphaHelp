Status Bar Methods

Alpha Anywhere provides the following Status Bar methods. Prefix each method with the keyword "STATUSBAR".

Function

Description

.ABORT_CHECK()

Returns TRUE (.T.) if user hit an abort area on the Status Bar.

.ABORT_RESET()

Resets the abort state. Do before abort loop and after an abort is canceled.

.CLEAR()

Displays the STATUSBAR robot, then clears the Status Bar.

.HIDE()

Hides the Status Bar window.

.ISVISIBLE()

Reports whether or not the Status Bar window is visible.

.PERCENT()

Displays a bar graph on the Status Bar showing the percent completion of a task.

.REFERENCE_EXPRESSION_GET()

Return the reference expression to reconstruct a reference to the object.

.ROBOT()

Displays a broken line bar graph on the Status Bar indicating that a task is being performed.

.SET_TEXT()

Displays the specified text on the Status Bar.

.SHOW()

Shows the Status Bar window.

.VARIABLES()

Returns a pointer to the variables defined at the Status Bar level.
