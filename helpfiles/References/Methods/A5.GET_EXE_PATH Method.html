A5.GET_EXE_PATH()

Syntax

Path as C = :A5.GET_EXE_PATH()

Description

The A5.GET_EXE_PATH() method returns the path of the Alpha Five executable.

Note : If you are going to reference a file, you have to add the backslash at the end of the path.

Supported By

Alpha Five Version 5 and Above

Example

? A5.GET_EXE_PATH()

= "C:\Program Files\A5V6"

? a5.Get_exe_Path()+ "\\"

= "C:\Program Files\a5v6\"
