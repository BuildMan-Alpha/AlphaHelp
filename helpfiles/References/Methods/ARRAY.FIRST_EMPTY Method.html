<ARRAY>.FIRST_EMPTY()

Syntax

Index as N = <ARRAY>.FIRST_EMPTY()

Description

The <ARRAY>.FIRST_EMPTY() method returns the index of the first NULL element in an array.

Example

Assume an array "A" contains the following elements:

Element

Value

A[1]

Orange

A[2]

Banana

A[3]

Apple

A[4]

Pineapple

A[5]

Mango

A[6]

  

A[7]

  

A[8]

  

? A.first_empty()

= 6 ' because element 6 is the first NULL (empty) element.
