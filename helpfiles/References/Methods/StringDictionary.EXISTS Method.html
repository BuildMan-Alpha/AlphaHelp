.EXISTS()

Syntax

Result_Flag as L = .EXISTS( Key_Value as C )

Argument

Description

Result_Flag

  • .T. = The specified entry exists.

  • .F. = The specified entry does not exist.

Key_Value

The key value of an entry in a StringDictionary list.

Description

The .EXISTS() method determines whether a specified entry is in the StringDictionary list.

Supported By

Alpha Five Version 7 and Above

Examples

dim sd as StringDictionary

sd.set("abc", "3")

? sd.exists("abc")

= .T.

sd.clear()

? sd.exists("abc")

= .F.
