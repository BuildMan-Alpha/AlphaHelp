<TBL>.ENCRYPT()

Syntax

Result_Flag as L = <TBL>.ENCRYPT( [ Encryption_Key as C ] )

Argument

Description

Encryption_Key

Optional. Default = default encryption key set by the DEFAULT_ENCRYPTION_KEY_SET() function.

The character string to use as the encryption key.

Description

The <TBL>.ENCRYPT() method encrypts the table referenced by the object pointer, <TBL>.  Result_Flag is .T. if the operation is successful. If Encryption_Key is not specified, Alpha Five uses the default encryption key. The default encryption key can be changed using the DEFAULT_ENCRYPTION_KEY_SET() function. Alpha Five requires exclusive access to the table in order to encrypt it.

Supported By

Alpha Five Version 5 and Above

Example

Encrypt a table using the default key.

dim t as P

t = table.open("customer")

t.encrypt()

t.close()

Encrypt a table using a specific key.

dim t as P

t = table.open("customer")

t.encrypt("swordfish")

t.close()
