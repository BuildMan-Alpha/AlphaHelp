<array>.age Method

N age = <array>.age()

Get the array age semaphore, which reflects the number of changes made to the array since its creation.

Example

dim arrc[0] as c
arrc[arrc.append()] = "fred"
? arrc.age()
= 1
arrc[arrc.append()] = "joe"
? arrc.age()
= 2
? arrc.dump()
= fred
joe
