.PRIMARY_GET()

Syntax

Table_Name as C = PRIMARY_GET( Set_Name as C )

Argument

Description

Set_Name

The name of the set.

Description

The .PRIMARY_GET() method retrieves the name of the primary table for set.

Supported By

Alpha Five Version 5 and Above

Example

dim ptr as P

dim prim as P

ptr = set.open("invoice")

? ptr.primary_get("invoice")  -> "INVOICE_HEADER.DBF"
