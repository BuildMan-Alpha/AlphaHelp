A5.DEACTIVATING()

Syntax

Window_Name as C = A5.DEACTIVATING()

Description

The A5.DEACTIVATING() method tells you the name of the MDI window that has just lost focus.

Note : You can change a modeless Xdialog into a MDI window by adding the {MDICHILD} command.

Supported By

Alpha Five Version 6 and Above

Limitations

Desktop applications only.

Example

? a5.DeActivating()

= "ControlPanel"
