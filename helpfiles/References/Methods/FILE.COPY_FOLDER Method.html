FILE.COPY_FOLDER()

Syntax

Result_Flag as L = FILE.COPY_FOLDER( Source_Folder as C, Target_Folder as C [, Only_Copy_If_Newer as L ] )

Argument

Description

Source_Folder

The path of the folder to copy.

Target_Folder

The path of the folder to receive the copied data.

Only_Copy_If_Newer

Optional. Default = .F.

.T. = Only copies newer files.

.F. = Copies all files

Description

The FILE.COPY_FOLDER() method copies all files and subfolders from Source_Folder to Target_Folder.

Examples

file.Copy_Folder("c:\documents\alphafivehelp", "c:\documents\temp", .t.)
