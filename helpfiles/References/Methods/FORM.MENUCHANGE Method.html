.MENUCHANGE()

Syntax

.MENUCHANGE( Menu_Name as C )

Argument

Description

Menu_Name

The name of previously defined menu.

Description

The .MENUCHANGE() method displays the named menu.

Limitations

Desktop applications only.

Example

The following example loads the query menu.

dim ptr as P

ptr = form.load("Customer Information")

ptr.menuchange("query")
