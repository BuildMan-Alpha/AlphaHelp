FILE.TO_STRING()

Syntax

Data as C = FILE.TO_STRING( Filename as C )

Argument

Description

Filename

The full path specification of the file.

Description

The FILE.TO_STRING() method reads a file from disk into a text variable, Data.

Example

dim cc as C

cc = file.to_string("C:\Frame.Txt")

? len(cc)

= 1777.000000
