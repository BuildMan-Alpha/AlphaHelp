.FIND_FIRST()

Syntax

 as P = .FIND_FIRST()

Description

The .FIND_FIRST() method returns a pointer to the first key value.

Supported By

Alpha Five Version 5 and Above

Example

dim tbl as P

dim qry as P

tbl = table.open("test")

query.filter = "ID <> ''"

query.options = ""

qry = tbl.query_create()

? qry.find_first()

= Key = "11111-01"

Record = 1
