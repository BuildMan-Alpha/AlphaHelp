.RECORDS_GET()

Syntax

Number_of_Records as N = .RECORDS_GET()

Description

The .RECORDS_GET() method retrieves the Number_of_Records selected by the query list.

Example

dim tbl as P

dim qry as P

tbl = table.open("test")

query.filter = "ID <> ''"

query.options = "D"

qry = tbl.query_create()

? qry.records_get()

= 6
