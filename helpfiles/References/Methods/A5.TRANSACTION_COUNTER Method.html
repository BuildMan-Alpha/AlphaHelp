A5.TRANSACTION_COUNTER()

Syntax

Transaction_Count as N = A5.TRANSACTION_COUNTER( [ Object_Name as C ] )

Argument

Description

Object_Name

Optional. The name of an A5 object. See Object Types for possible values.

Description

The A5.TRANSACTION_COUNTER() method fires an event every time an Alpha Five window opens or closes.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

Xdialog forms use A5.TRANSACTION_COUNTER()for the window bar. The Xdialog that displays the window bar has the following code, which calls the redraw_bar event every time an Alpha Five window opens or closes.

{watch=a5.transaction_counter()!redraw_bar}
