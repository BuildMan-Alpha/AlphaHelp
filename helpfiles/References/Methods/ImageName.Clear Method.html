.Clear()

Syntax

.Clear()

Description

The .Clear() method removes all image name entries from the string dictionary.

Supported By

Alpha Five Version 8 and Above

Examples

dim ir as P

dim nm as P

ir = ImageResource.get()

nm = ir.GetBitmapNameMap()

nm.Initialize(<<%str%

image_1=$a5_folder

image_2=$a5_folder_open

%str%)

? nm.exists("image_2")

= .T.

nm.clear()

? nm.exists("image_2")

= .F.
