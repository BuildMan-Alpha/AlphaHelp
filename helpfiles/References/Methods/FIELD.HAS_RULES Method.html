.HAS_RULES()

Syntax

Result_Flag as L = .HAS_RULES()

Description

The .HAS_RULES() method returns TRUE (.T.) if the field has loaded rules; otherwise, it returns .F. (FALSE).

Example

dim ptr as P

dim tptr as P

dim fptr as P

ptr = form.load("Customer Information")

ptr.show()

tptr = Table.current()

fptr = tptr.field_get(1)

? fptr.has_rules()

= .T.

fptr = tptr.field_get(2)

? fptr.has_rules()

= .F.
