<ARRAY>.CLEAR()

Syntax

<ARRAY>.CLEAR( [ Start as  N [, End as N ]] )

Argument

Description

Start

Optional. Default = 1. The number of the first element to clear.

End

Optional. Default = Last element. The number of the last element to clear.

Description

The <ARRAY>.CLEAR() method clears the contents of a single dimensional array. The optional Start and End arguments restrict the portion of the array that is cleared out.

Example

To remove blank entries from an array, use the &ltARRAY>.DELETE() method and the &ltARRAY>.INITIALIZE() method. For example:

Temp = A.DUMP()

A.clear(1, A.SIZE())

A.INITIALIZE(Temp)
