A5.ACTIVATING()

Syntax

Window_Name as C = A5.ACTIVATING()

Description

The A5.ACTIVATING() method tells you the name of the MDI window that has just gotten focus.

Note: You can change a modeless Xdialog into a MDI window by adding the {MDICHILD}command.

Limitations

Desktop applications only.

Example

? a5.Activating()

= "Code_Editor"
