FORM.CREATE()

Syntax

 as P = FORM.CREATE( Table_Set_Name as C, Format as C [, Style as P ] )

Argument

Description

A pointer to the form that you have created.

Table_Set_Name

The name of a table or set.

Format

Default text that appears on the form.

Style

Optional.  Default = "Alpha Five Default". A pointer to a stylesheet. Use A5_STYLESHEET_ENUM()to get a list of styles.

Description

The FORM.CREATE() method creates a form layout for the selected table or set and then displays the form in View mode.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

dim ptr as P

dim ssh as P

dim ssl as C
