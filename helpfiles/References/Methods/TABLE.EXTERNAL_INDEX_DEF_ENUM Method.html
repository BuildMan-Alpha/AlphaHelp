TABLE.EXTERNAL_INDEX_DEF_ENUM()

Syntax

Information_List as C = TABLE.EXTERNAL_INDEX_DEF_ENUM( Table_Name as C, Format as C )

Argument

Description

Table_Name

The name of a table.

Format

  • "E" = Error

  • "P" = Filename

  • "O" = Order

  • "F" = Filter

  • "X" = All flags

  • "U" = Unique

  • "D" = Descending

  • "I" = Insensitive

  • "T" = Type

Description

The TABLE.EXTERNAL_INDEX_DEF_ENUM() method returns lists of information about table indexes.

Example

? table.external_index_def_enum("product","N")

= "DescriptioProduct_I0Product_IdVendor"

? table.external_index_def_enum("invoice_header", "P")

= "C:\DATABASES\EXPRESSIONS TEST\INVOICE_HEADER.CDXC:\DATABASES\EXPRESSIONS TEST\INVOICE_HEADER.CDXC:\DATABASES\EXPRESSIONS TEST\INVOICE_HEADER.CDXC:\DATABASES\EXPRESSIONS TEST\INVOICE_HEADER.CDXC:\DATABASES\EXPRESSIONS TEST\INVOICE_HEADER.CDX"
