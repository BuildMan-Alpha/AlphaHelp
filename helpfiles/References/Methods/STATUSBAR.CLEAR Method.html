STATUSBAR.CLEAR()

Syntax

STATUSBAR.CLEAR()

Description

The STATUSBAR.CLEAR() method clears the Status Bar.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

This script displays the STATUSBAR robot, then clears the Status Bar.

for i = 1 to 1000

    statusbar.robot()

next

statusbar.clear()
