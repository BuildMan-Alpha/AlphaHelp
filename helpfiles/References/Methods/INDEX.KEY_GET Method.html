.KEY_GET()

Syntax

 as P = .KEY_GET()

Description

The .KEY_GET() method returns the current key value.

Example

dim t as P

dim i as P

dim b as P

dim b.key as C

t = table.open("customer")

i = t.index_primary_put("Lastname")

b = i.key_get()

? b

= Key = "Graham                                             "

Record = 1.000000
