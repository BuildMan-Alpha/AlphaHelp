CONTROLPANEL.SETFOCUS()

Syntax

CONTROLPANEL.SETFOCUS( Object_Name as C [, class=Name as C ] )

Argument

Description

Object_Name

The name of a Control Panel object.

class=Name

Optional. The name of class of objects. The following values are available.

  • "BROWSE"

  • "FORM"

  • "CODE"

  • "LABEL"

  • "LETTER"

  • "OPERATION"

  • "REPORT"

  • "TABLE"

Description

The CONTROLPANEL.SETFOCUS() method sets focus to the selected object.

Note : You must have something already selected on the same Control Panel tab window for the function to work.

Example

ControlPanel.SetFocus("edit_customer_info", "form")

controlpanel.setfocus("invoice_items", "table")
