<TBL>.RECORD_CONTENT_GET()

Syntax

Record_List as C = <TBL>.RECORD_CONTENT_GET( Expression as C [, Order as C [, Filter as C ]] )

Argument

Description

Record_List

A CR-LF delimited list of values, one per record.

Expression

An expression that references one or more fields from the table.

Order

Optional. Default = the current order.

Filter

Optional. Default = the current filter.

Description

The <TBL>.RECORD_CONTENT_GET()retrieves a CR-LF delimited list of the fields specified by Expression. This method is faster than TABLE.EXTERNAL_RECORD_CONTENT_GET() when used with a table that is already open.

Example

dim tbl as P

tbl = table.open("customer")
tbl.query_create("", "bill_state_region = 'ny'")
? tbl.record_content_get("lastname + bill_state_region")
= Feld NY
Dubi NY
Phillips NY
Carr NY
Barber NY
Bernstein NY
Baker NY
Harris NY

? tbl.record_content_get("lastname + bill_state_region", "lastname", .t.")
