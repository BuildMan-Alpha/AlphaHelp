A5.FONT_ENUM()

Syntax

Font_List as C = A5.FONT_ENUM( [ Allow_Raster as L ] )

Font_List as C = A5.FONT_ENUM( [ Font_Type_Code as N ] )

Argument

Description

Allow_Raster

Optional. Default = all fonts.

  • .T. = include raster fonts

  • .F. = do not include raster fonts

Font_Type_Code

Optional. Default = all fonts. An integer number.

Description

The A5.FONT_ENUM() method lists all of the available fonts.

Supported By

Alpha Five Version 5 and Above
