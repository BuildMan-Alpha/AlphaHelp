.PEEKB()

Syntax

New_Blob as B = .PEEKB( Position as N, Length as N )

Argument

Description

Position

The byte starting position for the read operation.

Length

The number of bytes to copy.

Description

The .PEEKB() method returns a portion of a second blob variable.

Example

dim b1 as B

dim b2 as B

b1 = char_to_blob("AB")

b2 = b1.peekb(1, 1)

? b2

= A
