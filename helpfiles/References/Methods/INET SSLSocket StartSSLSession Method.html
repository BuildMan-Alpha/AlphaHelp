INET::SSLSocket::StartSSLSession()

Syntax

Result_Flag as L = StartSSLSession( INET::Socket OpenSocket )

Argument

Description

Result_Flag

TRUE (.T.) if the operation was successful; otherwise FALSE (.F.).

OpenSocket

An INET::Socket object created with INET::Socket::Accept().

Description

The INET::Socket OpenSocket method begins an SSL session with a socket that is already opened. If successful, the socket remains secure until closed. If unsuccessful, both sockets close. After this call you should no longer reference OpenSocket.

Supported By

Alpha Five Version 8 Enterprise Edition
