FILE.TEMP_PATH_GET()

Syntax

Temp_Dir as C = FILE.TEMP_PATH_GET()

Argument

Description

Temp_Dir

The path to the temporary directory.

Description

The FILE.TEMP_PATH_GET() method returns the path to the temporary directory.

Example

? file.temp_path_get()

= "C:\DOCUME~1\ELARRA~1\LOCALS~1\Temp\"

? file.shortname_to_longname(file.temp_path_get())

= "C:\Documents and Settings\BBlatz\Local Settings\Temp\"
