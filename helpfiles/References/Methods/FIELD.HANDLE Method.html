.HANDLE()

Syntax

Handle as N = .HANDLE()

Description

The .HANDLE() method returns the old style Handle for the field.

Example

dim tbl as P

dim fptr as P

tbl = table.open("customer")

fptr = tbl.field_get("customer_id")

? fptr.name_get()

= "CUSTOMER_ID"

? fptr.handle()

= 65543.000000
