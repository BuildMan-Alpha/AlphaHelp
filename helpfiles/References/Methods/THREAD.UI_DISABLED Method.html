THREAD.UI_DISABLED()

Syntax

Thread_Status as L = THREAD.UI_DISABLED()

Argument

Description

Thread_Status

.T. = The thread is disabled.

.F. = The thread is not disabled.

Description

The THREAD.UI_DISABLED() method returns the disabled status of the current thread.

Limitations

Desktop applications only.

Examples

? thread.ui_disabled()

= .F.
