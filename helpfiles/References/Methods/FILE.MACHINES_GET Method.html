FILE.MACHINES_GET()

Syntax

Network_Resource_List as C = FILE.MACHINES_GET( Flags as C )

Argument

Description

Network_Resource_List

A list of all of the shared drives, machines and printers available to your on your network.

Flags

Flags are:

  • "S" = Storage

  • "P" = Printers

  • "D" = Include domains

  • "F" = Format

Flags can include any of the directives, separated by any delimiter.

Format

  • "C" = Comment

  • "P" = Provider

  • "L" = Localname

  • "T" = Type

  • "B" = Base

  • "M" = Machine

  • "R" = Resource

Description

The FILE.MACHINES_GET() method returns a list of all of the shared drives, machines and printers available to your on your network.
