<ARRAY>.SIZE()

Syntax

Size as N = <ARRAY>.SIZE( [ Dimension as N ] )

Size as N = Object_Address.CHOICE.LIST.SIZE()

Argument

Description

Dimension

Optional. Default = 1. The index of the array dimension.

Object_Address

The address of a list or combo box on a form.

Description

The <ARRAY>.SIZE() method returns size of an array.

Supported By

Alpha Five Version 5 and Above

Example

The following shows how to get the size of a multi-dimensional array.

dim xa[3,5] as C

?xa.size(1)

= 3.000000

?xa.size(2)

= 5.000000

?xa.dimensions()

= 2.000000

The following example gets the size of the choice array and then re-dimensions it, adding 4 new elements.

size = :customer:payment.choice.list.size()

size = size + 4

:customer:payment.choice.list.redim(size)
