TABLE.GET_MEMO_EXTENSION()

Syntax

Extension as C = .GET_MEMO_EXTENSION( Table_Filename_Extension as C )

Argument

Description

Extension

The filename extension that would be used for the table's memo file.

Table_Filename_Extension

The three character file type extension associated with any Alpha Five table.

Description

The TABLE.GET_MEMO_EXTENSION() method returns the extension of the associated memo file for a particular Alpha Five object type.

Example

? table.get_memo_extension(".dbf") -> ".fpt"

? table.get_memo_extension(".ddd") -> ".ddm"

? table.get_memo_extension(".alb") -> ".alm"

? table.get_memo_extension(".set") -> ".sem"
