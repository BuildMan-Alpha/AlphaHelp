<TBL>.GET_MEMO_EXTENSION()

Syntax

Extension as C = <TBL>.GET_MEMO_EXTENSION( Table_Filename_Extension as C )

Argument

Description

Extension

The filename extension that would be used for the table's memo file.

Table_Filename_Extension

The three character file type extension associated with any Alpha Five table.

Description

The <TBL>.GET_MEMO_EXTENSION() method returns the filename extension that would be used for the table's memo file.

Example

? ptr.get_memo_extension(".ddd")

= ".DDM"

? ptr.get_memo_extension("set")

= "SEM"

? ptr.get_memo_extension(".dbf")

= ".FPT"
