{dialog.Object}.refreshDataSeries Method

Syntax

{dialog.Object}.refreshDataSeries(seriesNames);

Description

Refreshes one or more data series. seriesName can be the name of a single data series, or an array of data series names.
When a data series is refreshed, all controls that are bound to that series are automatically updated. For example, if chart1 is bound to series1 and series2, then refreshing series1 will cause chart1 to be updated. Since chart1 is also bound to series2, series2 will then also be automatically updated, etc.

{dialog.Object}.refreshDataSeries('Series1');

//Update multiple series
{dialog.Object}.refreshDataSeries('Series1','Series2');
