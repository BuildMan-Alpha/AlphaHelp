.TABLE_GET()

Syntax

 as P = .TABLE_GET()

Description

The .TABLE_GET() method returns a pointer to the table that the query list is based on.

Example

This script finds the name of the table associated with the pointer.

dim tbl as P

dim qry as P

dim ptr as P

tbl = table.open("test")

query.filter = "ID <> ''"

query.options = "D"

qry = tbl.query_create()

ptr = qry.table_get()

? ptr.name_get()

= "TEST"
