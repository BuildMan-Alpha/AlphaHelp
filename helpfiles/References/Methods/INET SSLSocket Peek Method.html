INET::SSLSocket::Peek()

Syntax

Result_Flag as L = INET::SSLSocket::Peek( Data BYREF as C, LengthRequested as N )

Argument

Description

Result_Flag

TRUE (.T.) if the operation was successful; otherwise FALSE (.F.).

Data

The data read by the method.

LengthRequested

The number of characters to read.

Description

The INET::SSLSocket::Peek() method reads characters into a string. Peek does not remove the characters from the receive buffer.
