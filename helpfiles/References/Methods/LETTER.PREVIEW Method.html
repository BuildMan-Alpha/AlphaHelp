LETTER.PREVIEW()

Syntax

Letter_Name as C = LETTER.PREVIEW( Letter_Name as C [, Filter_Expression as C [, Order_Expression as C [, Modal as L [, Session_Modal as L ]]]] )

Argument

Description

Letter_Name

The name of the letter to preview.

Filter_Expression

Optional. Default = All records. A logical expression that selects records from the table.

Order_Expression

Optional. Default = Record number order. A character expression that orders the records that are selected. This expression must be less than 240 characters long.

Modal

Optional. Default = .F.

  • .T. = The user cannot change focus without closing the dialog.

  • .F. = The user can change focus without closing the dialog.

Session_Modal

Optional. Default = .T.

  • .T. = This means that any script that calls the LETTER.PREVIEW() method will pause until the Print Preview window is closed. However, the preview window itself is not a modal dialog box. The user is free to select other windows in Alpha Five.

  • .F. = The script does not pause until the Print Preview window is closed.

Description

The Letter.PREVIEW() method displays the Preview dialog box showing the selected letter.
