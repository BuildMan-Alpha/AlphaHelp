<ARRAY>.MATCHING()

Syntax

Element_List as C = <ARRAY>.MATCHING( Filter as C )

Argument

Description

Element_List

A CR-LF delimited list of array elements matching Filter.

Filter

A character filter expression that selects elements in an array.

Description

The <ARRAY>.MATCHING() methods returns a CR-LF delimited list of elements from a single dimensional array that match a filter.

Supported By

Alpha Five Version 6 and Above

Example

dim p[10] as P

p[1].fname="Selwyn"

p[1].lname="Rabins"

p[2].fname="Lenny"

p[2].lname="Forziati"

p[3].fname="Cian"

p[3].lname="Chambliss"

p[4].fname="Richard"

p[4].lname="Rabins"

? p.matching("lname=\"Rabins\"")

= [1]

[4]

Note : The output includes the square brackets around each index
