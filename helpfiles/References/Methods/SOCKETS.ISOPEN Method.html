.ISOPEN()

Syntax

Result_Flag as L = .ISOPEN()

writes String to the socket

Argument

Description

Result_Flag

.T. (TRUE) = The connection is open.

.F. (FALSE) = The connection is closed

Description

The .ISOPEN() method returns .T. (TRUE) if the connection is open; .F. (FALSE) otherwise.
