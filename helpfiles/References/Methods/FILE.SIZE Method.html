FILE.SIZE()

Syntax

Byte_Size as N = FILE.SIZE( Filename as C )

Argument

Description

Filename

The name of the file.

Description

The FILE.SIZE() method returns the size of a file.

Example

? file.size("C:\Databases\AlphaMovies\Customers.Ddd")

= 4149
