Script Functions and Methods

Alpha Anywhere provides the following script functions.

Function

Description

A5.SCRIPT_ENUM()

Returns a list of saved scripts.

A5_SETUP_ASK_VARS()

Allows you to add "ask variable" support to your own User Defined Functions.

A5.UDF_ENUM()

Returns a list of user defined functions.

DEBUG()

Turns the Alpha Anywhere Debugger on or off.

ADDIN.RUN()

Runs a script loaded with LOAD_LIBRARY().

EXPORT_SCRIPTS_AND_UDF()

Exports scripts and functions to a text file.

IMPORT PACKAGE

Makes the functions in package_name available in the calling script.

IMPORT_SCRIPT_UDF()

Imports scripts and functions previously exported with EXPORT_SCRIPTS_AND_UDF().

LOAD_LIBRARY()

Loads an .AEX file containing scripts and functions.

ON_CONDITION()

Registers some code to be executed when a logical_expression is true.

PROFILER_BEGIN()

Insert the at the point that you want to begin profiling a script.

PROFILER_END()

Insert the at the point that you want to end profiling a script.

SCRIPT_LOAD()

Returns a character string or blob that contains the contents of Script Name.

SCRIPT_PLAY()

Runs a Global Script.

SCRIPT_PLAY_LOCAL()

Runs a Global Script, but also inherits the variable name-space of the calling script.

SCRIPT_SCHEDULE()

Schedules a script for playback at a specific time.

SCRIPT_SCHEDULE_NAME_GET()

Returns the name of the script that is associated with Timer.

SCRIPT_SCHEDULE_TIME_GET()

Returns the play back time of the script that is associated with Timer.

SCRIPT_SPAWN()

Runs the named script.

SCRIPTEDITOR_OPEN()

Opens the named script in the Code Editor.

SLEEP()

Pauses the execution of a script for the specified number of Seconds.

UDF_LOAD()

Returns a character string or blob that contains the contents of a global function specified by Function Name.

UI_FREEZE()

Prevents the screen from updating while a script is running.

UI_SELECT_SCRIPT()

Displays the Pick the script to play dialog box.

UI_SELECT_SCRIPT_POPUP()

Displays one or more script names as menu selections.

VARIABLES_ENUM()

Returns a CR-LF delimited string of variable names in the variable frame referenced by Pointer.

UI_YIELD()

Provides Alpha Anywhere with time to refresh the screen display.

WAIT_UNTIL()

Causes a script to stop processing until logical_expression is .T..

XBASIC_ASYNCHRONOUS()

Causes the script to wait until after other Alpha Anywhere processes have been completed, not including screen refreshes.

XBASIC_SYNCHRONOUS()

Forces a script to run in the foreground thread.

XBASIC_WAIT_FOR_IDLE()

Causes the script to wait for other Alpha Anywhere processes and screen refreshes to complete before proceeding.
