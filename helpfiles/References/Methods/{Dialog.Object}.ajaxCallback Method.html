{Dialog.Object}.ajaxCallback Method

Syntax

{Dialog.Object}.ajaxCallback(,,xbasicFunction,,additionalData)

Description

Does an Ajax callback to a server. Data from the Dialog is submitted to the server. It is recommended that you use Action Javascript to write the code for you.
Note: The method has several blank arguments in order to keep the method prototype the same as for the Grid component.

xBasicFunction - the name of the Xbasic function you want to call on the server.
additionalData - (optional) - any additional data you want to submit. The data in the controls on the Dialog is automatically submitted. This can be used to submit additional data. Specify a string of name/value pairs. e.g. 'data1=value1&data2=value2'

Use Action Javascript to write the code for you. Dialog Component
