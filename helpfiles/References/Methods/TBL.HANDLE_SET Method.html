<TBL>.HANDLE_SET()

Syntax

<TBL2> as P = <TBL>.HANDLE_SET( Number as N )

Argument

Description

Number

An integer number.

Description

The <TBL>.HANDLE_SET() method sets the handle number for a table.

Supported By

Alpha Five Version 5 and Above

Example

dim tbl as P

dim tbl2 as P

tbl = table.open("invoice menu")

? tbl.handle()

= 4.000000

tbl2 = tbl.handle_set(8)

? tbl2.handle()

= 8.000000
