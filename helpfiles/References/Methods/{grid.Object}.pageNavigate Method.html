{grid.Object}.pageNavigate Method

Syntax

{grid.Object}.pageNavigate(target)

Description

Navigates to the specified page. Target can be an absolute page number (e.g. '23'), or 'first', 'prev', 'next' or 'last'.
You can also specify 'next-loop' and 'prev-loop'. If you are on the last pagd and you specify 'Next-Loop', the Grid will display to the first page. If you are on the first page and you specify 'Prev-Loop', the Grid will display the last page.
//Navigate to the last page
{grid.Object}.pageNavigate('last');

//Navigate to page 3
{grid.Object}.pageNavigate('3');
