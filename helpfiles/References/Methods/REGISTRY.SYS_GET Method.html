REGISTRY.SYS_GET()

Syntax

Data as C = REGISTRY.SYS_GET( Entry as C )

Argument

Description

Data

The value read from the Registry.

Entry

The registry entry name and full path.

Description

The REGISTRY.SYS_GET() method retrieves the value of a registry entry.

Examples

The following example returns the Windows directory on your computer.

ui_msg_box("", "" + system_root())

FUNCTION system_root as C ()

    system_root = Registry.sys_get("HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\CurrentVersion\SystemRoot")

    IF (system_root = "") then

       system_root = Registry.sys_get("HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\SystemRoot")

    END IF

END FUNCTION
