PropertyGrid.ExpandItem()

Syntax

PropertyGrid.ExpandItem( Name as C, [ Expand as L ] )

Argument

Description

Name

The name of a category in a property grid.

Expand

Optional. Default = .T.

  • .T. = Expand category

  • .F. = Close category

Description

The PropertyGrid.ExpandItem() method expands or contracts a named category within a property grid created with the Xdialog {PROPGRID} command.

Note : Only the {PROPGRID} command can create the PropertyGrid object.

Supported By

Alpha Five Version 8 and Above

Example

Refer to {PROPGRID} for an example.
