.Change_Begin()

Syntax

.Change_Begin()

Description

The .Change_Begin() method indicates that you are about to change the value of a field rule. Use <TBL>.RULES_GET() to create the pointer.

Limitations

Desktop applications only.

Examples

dim tbl as P

dim fr as P

tbl = table.open("invoice_header")

fr = tbl.rules_get()

fr.change_begin()

fr.date.skip = "F"

fr.change_end(.t.)

tbl.close()
