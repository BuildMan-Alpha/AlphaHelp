.CHARS_GET()

Syntax

Character_Count as N = .CHARS_GET()

Argument

Description

Character_Count

The number of characters (as opposed to bytes) in a file.

Description

The .CHARS_GET() method returns the number of characters (as opposed to bytes) in files, taking into account the character encoding method.

Examples

h = file.open("c:\r_broken.bin", FILE_RO_SHARED)

? h.bytes_get()

= 38

? h.chars_get()

= 18

? h.read(19)

= x this is a test

? h.text_encoding()

= "UTF16LE"
