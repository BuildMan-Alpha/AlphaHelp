SET.ERASE()

Syntax

SET.ERASE( Set_Name as C, Safety as L )

Argument

Description

Set_Name

The name of the set.

Safety

  • .T. = Delete the set.

  • .F. = Do not delete the set.

Description

The SET.ERASE() method deletes Set_Name if Safety is .T. (TRUE).

Example

set.erase("Invoice.set", .T.)
