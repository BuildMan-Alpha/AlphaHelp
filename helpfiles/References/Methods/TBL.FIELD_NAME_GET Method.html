<TBL>.FIELD_NAME_GET()

Syntax

Field_Name_List as C = <TBL>.FIELD_NAME_GET()

Description

The <TBL>.FIELD_NAME_GET() method returns the names of the fields in a table.

Example

dim tbl as P

tbl = table.open("customer")

? tbl.field_name_get()
