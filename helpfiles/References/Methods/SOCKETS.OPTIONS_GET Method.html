.OPTIONS_GET()

Syntax

Status_List as C = .OPTIONS_GET( Option_List as C )

Argument

Description

Status_List

 

Option_List

 

Description

The .OPTIONS_GET() method retrieves a list of the status of each of the options that a socket supports.

Examples

x = s.options_get(<<%str%

'listening'=A

'linger'=L

'keep alive'=K

'Reuse address'=R

'Dont Route'=N

'Delay'=D

'Debug'=X

'input buffer'=I

'output buffer'=O

'timeout'=T

%str%)

? x

= listening=F

linger=F

keep alive=F

Reuse address=F

Dont Route=F

Delay=F

Debug=F

input buffer=8192

output buffer=8192

timeout=-1
