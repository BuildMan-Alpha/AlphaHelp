APPEND.RUN_SILENT()

Syntax

Code_Text as C = APPEND.RUN_SILENT( Operation as C [, Filter as C ] )

Argument

Description

Operation

The name of a previously created append operation.

Filter

Optional. Default = "". An additional character filter expression for the transaction table that selects records to append.Optional.

Description

The APPEND.RUN_SILENT() method appends data using a previously defined append operation.

Note : If you specify an optional filter argument, the filter is applied in addition to any filter that may have been defined as Part of the Operation. (You can check to see if a filter is defined as Part of an Operation by editing the saved operation, and selecting the "Select records" menu item.)

Limitations

Desktop applications only.

Examples

The following example runs the Append Customer saved operation and displays a warning dialog box.

append.run_silent("Append Customer", "Lastname < 'Z'")
