TABLE.FIELDNAME_CHANGE()

Syntax

TABLE.FIELDNAME_CHANGE( Table_Filename as C, Old_Fields as C, New_Fields as C )

Argument

Description

Table_Filename

The fully qualified file name of the table.

Old_Fields

A CR-LF delimited list of old field names.

New_Fields

A CR-LF delimited list of new field names.

Description

The TABLE.FIELDNAME_CHANGE() function allows you to change field names through Xbasic.

Important: If any of the fields whose name you change are included in any index expressions, those indexes will be dropped.

Note : Alpha Five version 5 pads the new field name with spaces to a length of 10 characters.

Examples

dim old_names as C

dim new_names as C

old_names = "ID" + crlf()+ "Reference"

new_names = "ID_Seealso" + crlf()+ "Description"

table.fieldname_change("C:\Databases\Help_Database\Seealso.Dbf", old_names, new_names)
