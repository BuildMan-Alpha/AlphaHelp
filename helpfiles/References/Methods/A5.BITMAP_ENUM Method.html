A5.BITMAP_ENUM()

Syntax

Bitmap_Name_List as C = :A5.BITMAP_ENUM( [ Flags as N ] )

Argument

Description

Flags

Optional. Default = NULL. Bitmaps in the current database.

The following values are possible.

  • 256 = Bitmaps in the bitmap library.

  • 512 = Alpha Five internal bitmaps (used for toolbars and menus).

  • 1024 = System bitmaps.

Description

The A5.BITMAP_ENUM() method returns a CR-LF delimited list of bitmaps.

Supported By

Alpha Five Version 5 and Above
