THREAD.GET()

Syntax

 as P = THREAD.GET( Name as C )

Argument

Description

Name

The name of the thread set by the THREAD_CREATE() function.

Description

The THREAD.GET() method retrieves a pointer to a thread.

Supported By

Alpha Five Version 6 and Above

Limitations

Desktop applications only.

Examples

dim tptr as P

tprt = thread.get("noisy")
