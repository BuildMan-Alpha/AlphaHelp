<BROWSE>.RECORDS_GET()

Syntax

Record_Count as N = <BROWSE>.RECORDS_GET()

Argument

Description

Record_Count

The number of records in the browse's current query list.

Description

The <BROWSE>.RECORDS_GET() method returns the number of records in a browses's current query list.

Limitations

Desktop applications only.

Example

dim ptr as P

ptr = browse.view("CustomerBrowse")

? ptr.Records_Get()

= 60.000000
