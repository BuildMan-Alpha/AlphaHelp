.LAST_ACCESS_TIME()

Syntax

Access_Time as T = .LAST_ACCESS_TIME()

Description

The .LAST_ACCESS_TIME() method returns the time the file was last accessed.

Supported By

Alpha Five Version 5 and Above

Example

dim ptr as P

ptr = filefind.first("c:\*.txt", FILE_FIND_NORMAL)

? ptr.last_access_time()

= 02/27/2003 04:51:35 12 pm

? ptr.last_write_time()

= 02/27/2003 04:51:35 12 pm

? ptr.name()

= "c:\Button.Txt"
