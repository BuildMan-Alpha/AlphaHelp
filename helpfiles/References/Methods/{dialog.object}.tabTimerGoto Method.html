{dialog.object}.tabTimerGoto Method

Syntax

{dialog.object}.tabTimerGoto(tabId,target);

Description

Advances to the target tab pane. target can be 'first', 'previous', 'next' or 'last'. Target can also be an explicit tab pane number.
(Applies only to tab objects for which the method for selecting the active tab has been set to 'timer'.)
{dialog.object}.tabTimerGoto('TAB_1','next');
