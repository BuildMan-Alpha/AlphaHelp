FILE.FROM_BLOB()

Syntax

Byte_Count as N = FILE.FROM_BLOB( Filename as C, Data as B )

Argument

Description

Filename

The path and name of the file to create.

Data

The binary data to write.

Description

The FILE.FROM_BLOB() method creates a file from binary data.

Example

dim cc as C

dim bb as B

cc = file.to_string("C:\Garden\achillea.bmp")

bb = char_to_blob(cc)

file.from_blob("c:\temp\flower.bmp", bb)
