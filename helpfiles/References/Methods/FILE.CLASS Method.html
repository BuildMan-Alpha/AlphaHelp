.CLASS()

Syntax

File_Class as C = .CLASS()

Argument

Description

File_Class

The class of the file.

Description

The .CLASS() method returns the class of a file.

Supported By

Alpha Five Version 5 and Above

Example

dim fp as P

fp = file.open("C:\Form_Methods.Txt", FILE_RO_EXCLUSIVE)

? fp.class()

= "File"
