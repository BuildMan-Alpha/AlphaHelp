TOOLBAR.GET()

Syntax

 as P = TOOLBAR.GET( Toolbar_Name as C )

Argument

Description

Toolbar_Name

The name of the toolbar.

Description

The TOOLBAR.GET() method returns a pointer to the specified Toolbar_Name.

TIP : A toolbar's name can be determined by undocking the toolbar so that it is floating, and checking the title that appears in the toolbar title bar.

Limitations

Desktop applications only.

Example

The following disables the 3 rd control on the default toolbar in Form View.

tb = toolbar.get("Form View")

tb.controlenable(3,.f.)
