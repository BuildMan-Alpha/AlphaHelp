RESPONSE.REDIRECT()

Syntax

RESPONSE.REDIRECT( URL as C )

Argument

Description

URL

The replacement URL to use.

Description

The RESPONSE.REDIRECT() method redirects the client (web browser) to the specified URL. A redirect can not open a new browser window.

A RESPONSE.REDIRECT()command does not stop execution of the page. If you want to prevent the Application Server from sending additional output to the browser, your script should include an END command.

If you do a response.redirect("page2.a5w"), this is a relative redirect so the browser will continue using the current protocol, either HTTP or HTTPS. But if you use response.redirect("http://server/page2.a5w"), the browser will now use HTTP because you have explicitly told it to do so.

Note : Response variables are local to the page (script) and must be passed to any functions defined on the page.

Examples

response.redirect("http://www.alphasoftware.com")
