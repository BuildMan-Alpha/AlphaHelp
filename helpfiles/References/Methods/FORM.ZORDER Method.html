.ZORDER()

Syntax

.ZORDER( Position as C )

Argument

Description

Position

The stacking position of the MDI window.The allowable values for position are:

  • "top"

  • "bottom"

  • "after: window_name "

Description

The .ZORDER() method sets the window's z-order stacking position.This is the order in which open MDI child windows are stacked on top of each other.

A typical use of this command would be to create a form that acts as "wallpaper". i.e. a form that acts like the Windows desktop, which allows you to set wallpaper.

Limitations

Desktop applications only.

Examples

dim ptr as P

dim ptr2 as P

ptr = form.view("customerInfo")

ptr2 = form.view("invoices")

ptr2.zorder("after:customerinfo")
