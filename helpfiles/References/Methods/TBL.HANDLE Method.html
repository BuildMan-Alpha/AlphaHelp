<TBL>.HANDLE()

Syntax

Handle as N = <TBL>.HANDLE()

Description

The <TBL>.HANDLE() method returns the old style (Version 1) handle when using a new style (Version 3 and above) Xbasic object. This allows the Xbasic programmer to reuse Version 1 code on tables that they opened using the newer object methods.

Supported By

Alpha Five Version 5 and Above

Example

Newer style xbasic.

dim tbl as P

dim shared db_handle as N

tbl = table.open("Invoice_header")

' Call a version one script that reads a shared db_handle

db_handle = tbl.handle()
