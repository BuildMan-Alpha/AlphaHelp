.Mode_Get()

Syntax

Mode as N = .Mode_Get()

Argument

Description

Mode

Possible modes are:

  • 0 = Read mode

  • 1 = Change mode

Description

The .Mode_Get() method determines the current data entry mode of the pointer.

Limitations

Desktop applications only.

Examples

dim tbl as P

dim fr as P

tbl = table.open("invoice_header")

fr = tbl.rules_get()

? fr.mode_get()

= 0

fr.change_begin()

? fr.mode_get()

= 1

fr.change_end(.t.)

tbl.close()
