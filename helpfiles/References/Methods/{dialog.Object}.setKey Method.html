{dialog.Object}.setKey Method

Syntax

{dialog.Object}.setKey(alias, value , row);

Description

If the Dialog has been bound to tables, sets the primary key value for the specified table alias. If row is not specified, the alias is a top level alias. Otherwise, row indicates the row in a Repeating Section.
The key value is always a Javascript array of character data (even if the primary key is a single column). If value is set to 'undefinedkey' the key is deleted.
//Set the primary key for the INVOICE_ITEMS alias in row 3 of the Repeating Section to '23','11'.; {dialog.Object}.setKey('INVOICE_ITEMS','23','11',3);
