DDE Methods

DDE methods are maintained for backwards compatibility. Developers building new applications should use the OLE Automation Methods instead of DDE. Future versions of Alpha Anywhere may not support DDE.

Alpha Anywhere provides the following DDE methods. Prefix the following methods with the DDE keyword.

Method

 

Description

.OPEN()

V5

Initiates dynamic data exchange (DDE) with another active Windows application.

.RECEIVE()

V5

Establishes dynamic data exchange (DDE) with another active Windows application, and then requests an item of information which is returned as a Data String.

.SEND()

V5

Establishes dynamic data exchange (DDE) with another active Windows application and then sends an item of information to that application.

.TERMINATE_ALL()

V5

Closes all open DDE channels.

Alpha Anywhere provides the following methods. The following methods must be prefixed with the pointer. Use DDE.OPEN() to retrieve the pointer.
