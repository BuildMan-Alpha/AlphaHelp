.LAST_PRESSED()

Syntax

Button_Name as C = .LAST_PRESSED()

Description

The .LAST_PRESSED() method returns the name of the last button pressed when a form is opened as a dialog. See ((Using the FORM.VIEW__ method|Using the FORM.VIEW() method)) for details.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

:form.view("Which_Report","Dialog")

last_button = :which_report.last_pressed()

:which_report.close()
