A5.JOIN_ENUM()

Syntax

Join_Operation_List as C = :A5.JOIN_ENUM( [ Flag as N ] )

Argument

Description

Flag

Optional. Default = NULL.

2 = Full paths and filenames of the join operations.

Description

The A5.JOIN_ENUM() method returns a carriage return-line feed separated list of all of the join operations in the database.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.
