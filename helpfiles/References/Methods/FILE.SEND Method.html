FILE.SEND()

Syntax

FILE.SEND( Filenames as C [, Email_Address as C [, Options as C [, Zipname as C [, Comment as C [, Use_A5_Email_Client as L [, Store_in_Outbox as L [, Show_Client as L ]]]]]]] )

Argument

Description

Filenames

A string of comma-delimited file names. You must specify the full drive and path of each file in filenames. Wildcard specifications can be used in the list of file names.

Email_Address

Optional. Destination email address.

Options

Optional. If you specify more than one file, you must use the "C" or "Z" option flag to compress the files into a single zip file. If wildcards are used, the "C" option flag must be specified.

Zipname

Required if wildcards are used. The name of the zip file to create. Otherwise the zip file has the same name as the first file in the list.

Comment

A comment about the files that goes into the subject line.

Use_A5_Email_Client

Optional. Default = .F. (FALSE)

  • .T. = Use Alpha Five email client.

  • .F. = Use default MAPI email client.

Store_in_Outbox

Optional. Only if Use_A5_Email_Clien t is .T.

  • .T. = A copy of the message is stored in the Alpha Five email outbox (typically A_EMAIL.DBF in the Alpha Five program folder).

  • .F. = A copy of the message is not stored in the Alpha Five email outbox.

Show_Client

Optional. Default = .T. (TRUE). Only if the optional Email_Address is specified and Use_A5_Email_Client is .T.

Determines whether the Alpha Five email client is displayed before sending files in the case where you select the Alpha Five email client option.

  • .T. = Show Alpha Five email client before sending email.

  • .F. = Do not show Alpha Five email client.

Description

The FILE.SEND() method sends multiple files via the current MAPI email client, or the built-in Alpha Five email client.

If, for example, Use_A5_Email_Client is .F., and the current MAPI email client is Microsoft Outlook, Alpha Five will open Outlook. Similarly, if the email client is Eudora, or Netscape Mail, Alpha Five will open the appropriate email program.

Example

This script will send two files. It compress the files first into a zip file called myfiles.zip.

dim filelist as C

Filelist = "c:\data\file1.txt,c:\data\file2.txt"

file.Send_FILEs(filelist,"","C","myfiles")
