.FIND_NEXT()

Syntax

.FIND_NEXT()

Form_Name.FIND_NEXT()

Description

The .FIND_NEXT() method repeats the Find by Form, finding the next record.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

This example finds the second record in the Customer table of AlphaSports that has a lastname value equal to "McDonald".

dim frm as P

frm = form.view("Customer Information")

frm.find_by_form()

frm.lastname.activate()

frm.lastname.value = "McDonald"

frm.find_run()

frm.find_next()

frm.find_exit()
