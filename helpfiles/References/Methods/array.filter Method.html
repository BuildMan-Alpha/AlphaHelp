<array>.filter Method


<array>.filter(C expression, L exclude)

Delete entries from an array that don't match a filter expression if exclude is false (the default), or do match if exclude is true.

Example

Filter only

dim arr[0] as p
arr.initialize_properties("name,age:n",<<%code%
#Fred,37
#Joe,28
#John,22
#Jones,44
#%code%)
? arr.size()
= 4
arr.filter("name='fred'")
? arr.size()
= 1


Filter exclude

dim arr[0] as p
arr.initialize_properties("name,age:n",<<%code%
#Fred,37
#Joe,28
#John,22
#Jones,44
#%code%)
? arr.size()
= 4

arr.filter("name='fred'",.t.)
? arr.size()
= 3
