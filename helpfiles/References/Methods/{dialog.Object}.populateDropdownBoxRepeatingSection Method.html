{dialog.Object}.populateDropdownBoxRepeatingSection Method

Syntax

{dialog.Object}.populateDropdownBoxRepeatingSection(controlName,data,clearExistingData);

Description

Same as the .populateDropdownBox() method, but operates on a control that is in a Repeating Section. The choices in the DropdownBoxes in every row in the Repeating Section are updated.

See {dialog.object}.populateDropdownBox() Dialog Component
