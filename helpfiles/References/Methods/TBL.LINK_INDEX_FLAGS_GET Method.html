.LINK_INDEX_FLAGS_GET()

Syntax

OutputString as C = .LINK_INDEX_FLAGS_GET()

Description

The .LINK_INDEX_FLAGS_GET() method returns the options set when linking two tables in a set.

Supported By

Alpha Five Version 5 and Above

Example

dim st as P

dim tb as P

st = set.open("invoice")

tb = table.current(2)

? tb.link_index_flags_get()

= "DU"
