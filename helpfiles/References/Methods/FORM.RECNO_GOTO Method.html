.RECNO_GOTO()

Syntax

.RECNO_GOTO( Record_Number as N )

Argument

Description

Record_Number

The record order number of the desired record.

Description

The .RECNO_GOTO() method makes the specified record number current.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

The following example displays the Inventory form with record number 6.

dim sptr as P

sptr = form.load("Inventory")

sptr.Recno_goto(6)

sptr.show()
