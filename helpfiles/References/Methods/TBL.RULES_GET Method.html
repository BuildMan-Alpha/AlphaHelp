<TBL>.RULES_GET()

Syntax

<Field_Rules> as P = <TBL>.RULES_GET()

Description

The <TBL>.RULES_GET() method returns a pointer to a Fields Rule object that contains field rule information for the selected table.

Supported By

Alpha Five Version 5 and Above

Example

dim tbl as P

dim fr as P

tbl = table.open("invoice_header")

fr = tbl.rules_get()

? fr

= V Change_Begin() 'Begin changes to the field rules.

V Change_End([L Commit]) 'End changes to the field rules.
