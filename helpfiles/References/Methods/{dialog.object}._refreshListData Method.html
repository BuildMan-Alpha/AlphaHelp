{dialog.object}._refreshListData Method

Syntax

{dialog.object}._refreshListData(listId);

Description

Refreshes the data in a List control. For example, if the list is based on a SQL query, a callback is made to re-execute the query. Contrast this method with the .refresh() method which simply redraws the List using the existing List data. 

{dialog.object}._refreshListData('MYLIST1');
