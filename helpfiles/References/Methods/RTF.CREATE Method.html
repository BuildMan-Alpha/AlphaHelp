RTF.CREATE()

Syntax

 as P = RTF.CREATE( [ Text_Data as C ] )

Argument

Description

Text_Data

The initial text to put into an RTF object.

Description

The RTF.CREATE() method creates a new RTF object and returns a pointer to it.

Example

dim ptr as P

ptr = rtf.create("Alpha!")
