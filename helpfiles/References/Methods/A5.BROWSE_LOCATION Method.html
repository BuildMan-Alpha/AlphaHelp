A5.BROWSE_LOCATION()

Syntax

Filename as C = A5.BROWSE_LOCATION( Browse_Name as C )

Argument

Description

Filename

The name of the file that contains the browse.

Browse_Name

The name of a browse.

Description

The A5.BROWSE_LOCATION() method returns the name of the file that contains the named browse.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Example

? a5.browse_location("Product List")

= "c:\databases\expressions test\product.ddd"
