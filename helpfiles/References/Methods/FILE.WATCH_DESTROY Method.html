FILE.WATCH_DESTROY()

Syntax

FILE.WATCH_DESTROY( Watchname as C )

Argument

Description

Watchname

The name of a watch created with FILE.WATCH_CREATE().

Description

The FILE.WATCH_DESTROY() function removes a watch set by FILE.WATCH_CREATE().

Example

file.watch_create("temp_dir", "c:\temp", "F", "my_alert_function()")

? file.watch_enumerate()

= temp_dir c:\temp F

file.watch_destroy("temp_dir")    

See Also

FILE.WATCH_CREATE(), FILE.WATCH_ENUMERATE(), FILE.DEPENDENCIES()

Supported By
