FILE.FROM_STRING()

Syntax

FILE.FROM_STRING( Filename as C, Data as C  [, Encoding as C ])

Argument

Description

Filename

The path and name of the file to create.

Data

The character data to write into the file.

Encoding

Set to 'UTF8' for a UTF8 encoded file. This adds the standard UTF8 prefix characters. (V12 Build 2288 and above).

Description

The FILE.FROM_STRING() method creates a file from character data.

Note : The target directory must exist or the function will fail.

Example

dim cc as C

cc = file.to_string("C:\Frame.Txt")

? len(cc)

= 1777.000000

file.from_string("c:\new_frame.txt", cc)
