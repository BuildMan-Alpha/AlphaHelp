A5.SYSTEM_MODE_GET()

Syntax

Mode_Settings as C = A5.SYSTEM_MODE_GET( System_Mode as C )

Argument

Description

System_Mode

To get a list of System Mode values, type: ?  A5.SYSTEM_MODE_GET() in the Interactive Window.

Description

The A5.SYSTEM_MODE_GET() method returns the settings associated with various system modes.

Example

? A5.System_Mode_Get("data_entry_style")

= "Modeless"

? A5.System_Mode_Get("interactive_auto_help")

= "complete,help,color"

? A5.System_Mode_Get("editor_bubble_help")

= "enable"
