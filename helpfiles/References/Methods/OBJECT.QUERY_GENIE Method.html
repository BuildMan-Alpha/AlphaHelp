<OBJECT>.QUERY_GENIE()

Syntax

<OBJECT>.QUERY_GENIE( [ Query_Name as C ] )

Argument

Description

Query_Name

Optional. The name of a saved query operation.

Description

The <OBJECT>.QUERY_GENIE() method applies to:

  • Browses (for <OBJECT> use either the <BROWSE> pointer or the name of the browse)

  • Forms (for <OBJECT> use either the <FORM> pointer or the name of the form)

The <OBJECT>.QUERY_GENIE() method displays the Query Genie dialog box.

Limitations

Desktop applications only.

Example

dim ptr as P

ptr = form.load("Travel Time")

ptr.Query_Genie()
