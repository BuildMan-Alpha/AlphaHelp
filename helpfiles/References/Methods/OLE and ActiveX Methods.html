OLE and ActiveX Methods

Alpha Anywhere provides the following ActiveX/OLE methods.

Method

 

Description

<OLE>.SET()

V7

Defines an alias for an ActiveX control.

OLE.AddLicence()

V7

Adds a ProgID and associated license key to the global license cache.

OLE.Alias_Object()

V7

Creates an OLE automation object to be used with <OLE>.SET().

OLE.CREATE()

V5

Creates an OLE automation object.

OLE.CreatePicture()

V7

Creates an IPictureDisp object.

OLE.GetLicense()

V7

Returns the key associated with a ProgID in the global cache.

OLE.GETOBJECT()

V6

Uses an existing OLE automation object, if available. Otherwise, it creates a new one.

OLE.LoadDeveloperLicense()

V7

Returns the key associated with a ProgID on a developer's machine.

OLE.RemoveLicense()

V7

Removes a ProgID and associated license key from the global cache.
