A5 Enumeration Methods

Alpha Five provides the following enumeration methods. These various methods return lists of the names of system objects.

Function

 

Description

A5.APPEND_ENUM()

V5

Returns a CR-LF delimited list of the names of the saved append operations stored in the currently open .adb file.

A5.BITMAP_ENUM()

V5

Returns a CR-LF delimited list of bitmaps.

A5.BROWSE_ENUM()

V5

Returns a CR-LF delimited list of all saved browses.

A5.COPY_ENUM()

V5

Returns a CR-LF delimited list of all of the copy operations in the database.

A5.EXPORT_ENUM()

V5

Returns a CR-LF delimited list of all of the export operations in the database.

A5.FORM_ENUM()

V5

Returns a CR-LF delimited list of all saved forms.

A5.IMPORT_ENUM()

V5

Returns a CR-LF delimited list of all of the import operations in the database.

A5.JOIN_ENUM()

V5

Returns a CR-LF delimited list of all of the join operations in the database.

A5.LABEL_ENUM()

V5

Returns a CR-LF delimited list of all saved label layouts.

A5.LETTER_ENUM()

V5

Returns a CR-LF delimited list of all saved letter layouts.

A5.LIBRARY_ENUM()

V5

Returns a CR-LF delimited list of all .ALB files.

A5.MARK_ENUM()

V5

Returns a CR-LF delimited list of all of the mark operations in the database.

A5.MENU_ENUM()

V5

Returns a CR-LF delimited list of all saved menus.

A5.POST_ENUM()

V5

Returns a CR-LF delimited list of all of the post operations in the database.

A5.QUERY_ENUM()

V5

Returns a CR-LF delimited list of all of the query operations in the database.

A5.REPORT_ENUM()

V5

Returns a CR-LF delimited list of all saved report layouts.

A5.SCRIPT_ENUM()

V5

Returns a CR-LF delimited list of all saved scripts.

A5.SET_ENUM()

V5

Returns a CR-LF delimited list of all sets in the database.

A5.SUMMARIZE_ENUM()

V5

Returns a CR-LF delimited list of all of the Summary operations in the database.

A5.TABLE_ENUM()

V5

Returns a CR-LF delimited list of all tables in the database.

A5.TOOLBAR_ENUM()

V5

Returns a CR-LF delimited list of all toolbars in the database.

A5.UDF_ENUM()

V5

Returns a CR-LF delimited list of all user defined functions (UDFs) in the database.

A5.UPDATE_ENUM()

V5

Returns a carriage return-line feed separated list of all of the saved update operations in the database.

A5.XTAB_ENUM()

V5

Returns a carriage return-line feed separated list of all of the crosstab (XTAB) operations in the database.

A5_EMAIL_PROFILES_ENUM()

V5

Returns a list of email profiles.

A5_GET_OPERATIONS()

V5

Returns a list of the saved operations of a specified type.

A5_LIST_OPEN_WINDOWS()

V5

Returns a CR-LF delimited list of open windows.

A5_SETS_ENUM()

V5

Returns a list of sets in the current database.

A5_STYLESHEET_ENUM()

V5

Returns a list of form styles.

A5_TABLE_ENUM_ACTUAL()

V5

Returns a list of tables that have valid filenames.
