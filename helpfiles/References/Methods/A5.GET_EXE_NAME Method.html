A5.GET_EXE_NAME()

Syntax

Name as C = A5.GET_EXE_NAME()

Argument

Description

Name

The Alpha Anywhere executable's base name.

Description

The A5.GET_EXE_NAME() method returns the base name of the Alpha Anywhere executable.

Example

? a5.get_exe_name()

= "alpha5.exe"
