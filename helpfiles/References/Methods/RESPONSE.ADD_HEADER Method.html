RESPONSE.ADD_HEADER()

Obsolete
This method is obsolete and has been removed from the software as of Version 11. See Replacing Response.Add_Header in V11.


Syntax

RESPONSE.ADD_HEADER( Header_Value as C )

Argument

Description

Header_Value

An information listing at the top of an HTTP request or response.

Description

The RESPONSE.ADD_HEADER() method function adds an HTTP header to the response. The last header added will be the first header sent back to the client.

Note : Response variables are local to the page (script) and must be passed to any functions defined on the page.

Example

reponse.add_header("Connection: close")

reponse.add_header("Allow: GET, HEAD, POST")

The following commands will inhibit page caching.

Response.AddHeader("CacheControl: no-cache")
Response.AddHeader("Pragma: no-cache")
Response.AddHeader("Expires: -1")
