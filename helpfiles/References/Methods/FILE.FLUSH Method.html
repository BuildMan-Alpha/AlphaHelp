<FILE>.FLUSH()

Syntax

<FILE>.FLUSH()

Description

The <FILE>.FLUSH() method flushes output held in the disk cache, to assure all output is stored safely in the file referenced by the object pointer <FILE> before the file is closed.

Example

text = ui_get_text("Text", "Enter text:", "Value")

file_pointer = file.create("c:\a5\flush.txt", FILE_RW_EXCLUSIVE)

for i = 1 TO 50

    file_pointer.write(text)

next i

file_pointer.flush()'write any buffered text to disk

file_pointer.close()
