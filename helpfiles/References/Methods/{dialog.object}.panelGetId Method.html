{dialog.object}.panelGetId Method

Syntax

{dialog.object}.panelGetId(panelName,panelPart);

Description

Gets the ID of the specified 'part' of a Panel. The 'part' can be 'header', 'body', or 'footer'. Getting the ID of a Panel part is useful if you want to set properties (such as the Inner HTML) of the Panel. 

var panel = {dialog.object}.panelGetActive();
var id = {dialog.object}.panelGetActive(panel,'body');
id.innerHTML = 'some new text';
