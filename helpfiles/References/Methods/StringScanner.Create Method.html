StringScanner.Create()

Syntax

 as P = StringScanner.Create( Text as C )

Argument

Description

A new scanner object that supports operations against Text.

Text

The text that you want to process

Description

The StringScanner.Create() function creates a new scanner object that supports operations against Text.

Examples

dim scanner as P

dim txt as C

txt = file.to_string("C:\Documents\AlphaFiveHelp\AlphaFiveHelp.hhc")

scanner = stringscanner.create(txt)
