.CREATION_TIME()

Syntax

Creation_Time as T = .CREATION_TIME()

Description

The .CREATION_TIME() method returns the time the file was created.

Example

dim ptr as P

ptr = filefind.first("c:\*.txt")

? ptr.creation_time()

= 02/27/2003 10:05:50 65 am

? ptr.name()

= c:\Button.Txt"
