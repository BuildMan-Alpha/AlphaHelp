A5.SYSTEM_COLOR_SET()

Syntax

A5.SYSTEM_COLOR_SET( System_Color_Name as C, Color as C  )

Argument

Description

System_Color_Name

The new color alias.

Color

The name of a system color.

Description

The A5.SYSTEM_COLOR_SET() method creates a color alias.

Example

a5.System_Color_Set("Error", "Red")
