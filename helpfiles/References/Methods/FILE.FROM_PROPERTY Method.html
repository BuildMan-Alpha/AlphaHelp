FILE.FROM_PROPERTY()

Syntax

FILE.FROM_PROPERTY( Filename as C, Data as P )

Argument

Description

Filename

The path and name of the file to create.

Data

A pointer variable containing the data to write.

Description

The FILE.FROM_PROPERTY() method creates a property file from a dot variable.

Example

Dim a[1] as P

a[1].name = "Fred"

a[1].city = "Boston"

a[1].age = 23

a[].name = "Tom"

a[..].city = "NY"

a[..].age = 35

file.From_property("c:\my_property.txt", a)
