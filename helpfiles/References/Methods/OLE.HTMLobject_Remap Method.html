OLE.HTMLobject_Remap()

Syntax

Modified_HTML as C = OLE.HTMLobject_Remap( HTML as C )

Argument

Description

Modified_HTML

HTML source with a clsid value for Alpha Five Version 6.

HTML

HTML source with a clsid value for Alpha Five Version 7 or higher.

Description

The GUID in the HTML string (clsid:EE3218B4-5E36-4435-93D6-FB563D7BAD16) is the version 6 GUID. Each new version of Alpha Five has different GUIDs for internal Alpha Five controls. This function will translate the GUID into the correct GUID for the version of A5 that you are running.

Supported By

Alpha Five Version 7 and Above

Example

html = <<%str%

...

if version()>=7 then

    html = ole.HTMLobject_Remap(html)

end if
