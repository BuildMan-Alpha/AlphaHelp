<TBL>.QUERY_DETACH()

Syntax

<TBL>.QUERY_DETACH( [ Query_Name as C ] )

Argument

Description

Query_Name

Optional. The name of a query.

Description

The <TBL>.QUERY_DETACH() method detaches the current or a named query.

Supported By

Alpha Five Version 5 and Above

Example

dim tbl as P

dim inx as P

tbl = table.open("customer")

inx = tbl.query_primary_put("lastname")

? inx.records_get()

= 60.000000

tbl.query_detach("lastname")
