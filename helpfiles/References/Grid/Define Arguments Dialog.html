Define Arguments Dialog

You use Arguments with the List View Genie to pass parameters to the Define SQL Statement dialog box. To create an argument:

  1. Click Insert New Argument.

  2. Enter the name of the argument into the Argument Name > Name field.

  3. Select the data type of the argument in the Argument Name > Data-type list. The options are:

    • "Character"

    • "Data"

    • "Logical"

    • "Numeric"

  4. Define the type of source of the argument's value in the Argument Name > Get argument value from list. The options are:

    • "Always prompt for value"

    • "Get value from variable"

  5. If you selected to get a value from a variable in step 4, select the variable's scope in the Variable > Variable Scope list. The options are:

    • "Local Variable"

    • "Session Variable"

    • "Global Variable"

  6. Enter or select the name of the variable in the Variable > Variable Name list.

  7. Optionally, select "True" in the Variable > Prompt for value if variable not found list if you want to prompt the user for a value at run time.

  8. Click OK to continue.
