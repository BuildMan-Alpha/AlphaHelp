Character Operators

Character operators are used between character expressions to yield character results. Alpha Five supports the character operators + (concatenation) and - (concatenation with trim).

>Concatenation (+)

This operator concatenates (combines) two character strings into a single new character string. In the resulting character string, the entire value stored in the first field is positioned before the value from the second field.

For example, if Firstname and Lastname are both 15 character fields:

FIRSTNAME + LASTNAME --> "Beverly        Brine         "

>Concatenation with Trim (-)

This operator strips trailing blanks from the first field and then concatenates it with the second field to form a single new character string.

For example:

FIRSTNAME - LASTNAME --> "BeverlyBrine           "

FIRSTNAME - " " + LASTNAME - "" --> "Beverly Brine"
