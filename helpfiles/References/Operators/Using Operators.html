Using Operators

An operator is a symbol you use to represent a mathematical, relational, logical, or string operation. For example, the asterisk (*) represents the mathematical operation of multiplication.

You can combine functions, field names, and constants with operators to form more complex expressions.

Most operators use the following syntax:

expression operator expression

Each data type (Character, Date, Logical, And Numeric) has a corresponding list of operators so that you can combine multiple values of that type together into a single value. The resulting data type is determined by the operator and the data type of its operands.
