^ (Exponent Operator)

Syntax

Result as N = Operand1 as N ^ Operand2 as N

Argument

Description

Operand1

A numeric value greater than or equal to zero.

Operand2

A numeric value.

Description

The ^ (exponent) operator raises Operand1 to the Operand2 power.

Examples

? 5^2

= 25
