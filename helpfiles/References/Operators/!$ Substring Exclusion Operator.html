!$ (Substring Exclusion Operator)

Syntax

Result as L = Find_String as C !$ Search_String as C

Argument

Description

Find_String

The character string that you are looking for.

Search_String

The character string to examine.

Description

The !$ operator determines whether the character expression on the left of the operator is not contained anywhere within the character expression on the right. Operations involving the !$ operator return a logical result (i.e., TRUE or FALSE). The string comparison is not case sensitive.

Example

dim Company as C = "The Computer Store"

? "Computer" !$ Company

= .F.

? "computer" !$ Company

= .F.

"Alpha" !$ Company

= .T.
