Emailing Scripts and Functions

To email a script or function:

  1. Display the Code tab of the Control Panel.

  2. Select Code > Send... to display the Print Scripts and Functions dialog.

  3. Optionally, click Scripts or Functions to limit your selection.

  4. Optionally, click Send All.

  5. Optionally, click Send Selected... to display the Select Scripts and Functions dialog box.

    1. Place checkmarks to the left of scripts and functions that you want to email.

    2. Optionally, click Select All or Un-Select All to speed your selection process.

    3. Click OK to email or Cancel to quit without action.
