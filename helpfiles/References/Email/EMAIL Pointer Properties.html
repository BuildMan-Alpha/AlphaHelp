Pointer Properties

Prefix the following properties with the pointer. Use EMAIL.OPEN()to retrieve .

Account Information

Property

Description

Example

.CLIENT_HOSTNAME The mail server used to send mail (SMTP) mail.isp.com
.CLIENT_IDENTITY Your email address Bill@isp.com
.CLIENT_PASSWORD Your email account password mypassword
.CLIENT_USERNAME Your username selwyn
.ERROR Contains any error messages reported by the mail server.  
.MAIL_HOSTNAME The mail server used to receive mail. (POP) mail.isp.com
.PORT The port to use. Generally this can be 0 0

Receiving Email

Property

Description

Example

.RECEIVE_ATTACHMENTS_FIELD The field where attachments are stored. This field will contain the attachment filenames. The actual attachments are in a folder called "Attachments" relative to the table into which email is read. Attachments
.RECEIVE_AUTODELETE Controls whether email is deleted from the server when it is read. .f.
.RECEIVE_BCC_ADDRESS_FIELD The field where the Bcc addresses are stored. Bcc
.RECEIVE_CC_ADDRESS_FIELD The field where the CC addresses are stored. CC
.RECEIVE_DATE_FIELD The field where the email date is stored. Sent_DATE
.RECEIVE_FROM_ADDRESS_FIELD The field where the address of the sender is stored. From
.RECEIVE_LIMIT The number of messages Alpha Anywhere should read from the server when the .RECEIVE() method is executed. -1 indicates all messages. -1
.RECEIVE_MESSAGE_BODY_FIELD The field where the message body is stored. Body
.RECEIVE_REPLY_FIELD The field where the email reply-to address is stored. Reply
.RECEIVE_SORT_DATE_FIELD   Sort_Date
.RECEIVE_SUBJECT_FIELD The field where the email subject is stored. Subject
.RECEIVE_TO_FIELD The field where the To addresses are stored. To

Sending Email

Property

Description

.SEND_ATTACHMENTS The filename of any attachment to be sent with the message.
.SEND_BCC A comma delimited list of BCC addresses to which email is to be sent.
.SEND_CC A comma delimited list of CC addresses to which email is to be sent.
.SEND_CONTENTS The body of the email message.
.SEND_SUBJECT The subject of the email message.
.SEND_TO A comma delimited list of email addresses to which email is to be sent.
.TEXT_TYPE Set to "HTML" for HTML email, or "" (i.e. blank) for regular text email.
.XMAILER The Xmailer field identifies the mailer. It is contained in the message header.
