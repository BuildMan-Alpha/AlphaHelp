Favicon

The icon in the top left corner of a web page is called the Favicon, short for favorites icon. By default, it is found on a web site at /favicon.ico. Most browsers look for this file.

To use your own icon, create a favicon.ico icon and place it in your webroot. If you don't have one, the Alpha Web Application Server will supply the Alpha icon.

If you don't want to display your own icon and you don't want the Alpha icon displayed either, you can disable the functionality in the Application Server Control Panel, Advanced tab, Built-in Resources section.
