Defining an Alpha Five Query

  1. If you selected "Alpha Five .dbf Tables" in Selecting the Data Source :

    1. Select whether the data will come from a Table or View. A view is similar to an Alpha Five set and may contain one or more tables.

    2. If you believe that the field list is not up to date, click to refresh the field list.

    3. If you selected "Table" in step 1, continue with Define an Alpha Five Table Query.

    4. If you selected "View" in step 1, continue with Define an Alpha Five View Query.

    Note : Views display one-to-one relationships, not one-to-many relationships.

  2. If you selected "Remote Database accessed via AlphaDAO" in Selecting the Data Source, continue with Define an AlphaDAO SQL Query.

  3. If you selected "Remote Database accessed via ADO" in Selecting the Data Source :

    1. If you want to connect to a Microsoft Access database, continue with Define an Access Query.

    2. If you want to connect to an ODBC compatible database (such as MySQL or Oracle ), continue with Define a SQL Query.

    3. If you want to connect to an ADO compatible database (such as Microsoft SQL Server ), continue with Define an ADO Query.

Limitations

Web publishing applications only.
