Creating a New Page

Creating a New Alpha Five Web Page

Alpha Anywhere (.A5W) pages may contain ordinary HTML, JavaScript, plus Xbasic and web components.

  1. Select the project in the Web Projects list to which you wish to add the web page.

  2. Click to display the New File dialog.

  3. Select Alpha Five Web Page (A5W) and click OK.

  4. The HTML Editor appears. Refer to Building ASW Web Pages for additional instructions.

Creating a New HTML Page

Regular HTML (.HTM) pages may contain ordinary HTML and JavaScript.

  1. Select the project in the Web Projects list to which you wish to add the HTML page.

  2. Click to display the New File dialog.

  3. Select HTML Page and click OK.

  4. The HTML Editor appears.

Using Other Editors to Edit Pages

After you have created an A5W page, you may edit its content in other editors, such as Windows Notepad and Adobe Dreamweaver.

  1. To edit a web page with another editor:

    1. Right click the page listing in the Web Projects Control Panel.

    2. Select Open With and select one of the listed editors.

  2. To edit a web page with the default editor:

    1. Select Open With > Choose Program... to display the Open with... dialog.

    2. Select "" in the list. The program that opens is the one associated with the file type of the page.

    3. Click Open.

  3. To add a new program to the list:

    1. Select Open With > Choose Program... to display the Open with... dialog.

    2. Click in the Program control.

    3. Select "" in the list.

    4. Navigate to the editor's program file.

    5. Click Open.

    6. Optionally, change the description of the program of the Name control.

    7. Click Open.

  4. To remove a program from the list:

    1. Select Open With > Choose Program... to display the Open with... dialog.

    2. Select the program in the list.

    3. Click Delete.

    4. Click Open.
