Activating the Application Server

After you have installed your copy of the Application Server software to a server computer, you will need to activate it.

  1. Start the Application Server by clicking Start > All Programs > Alpha Anywhere Web App Server > Alpha Anywhere Web App Server.

  2. The Application Server runs as a background task, so the only indication that you will that it is running is the icon that appears on the task bar of your screen.

  3. Click this icon to display the Application Server Control Panel.

  4. Display the Session tab.

  5. Click Add to display the New License dialog box.

  6. Enter the license number that you received in the email from Alpha Software and click OK.

  7. Select your new license in the Licensed Sessions list and click Activate.

  8. Click Start Server.

  9. Click Close.

The Application Server is ready to be used.
