Creating Tab Controls on a Dialog Component

Tabbing Styles

Dialog components, grid components with columnar formats, and grid detail view parts may use tabs to divide content between multiple "pages". The following dialog places Contact and Mailing information on two pages using the Single Band style.

The two panes of the tab control at different moments. The SingleBand Style allows you to directly open any pane, one at a time. When one tab pane expands, all other tab panes collapse.

The Tab style allows users to directly access any pane. The pane labels (at bottom), which display the panes, can also be at the top.

The two panes of the tab control at different moments. The Tab Style allows you to directly open any pane, one at a time. The tab pane buttons can be either at the top or the bottom of the tab control

Another style of tabbing is know as "Genie Style". The user uses the < Back and Next > buttons to view different tabs.

The two panes of the tab control at different moments. The Genie Style forces you to access tab panes sequentially, from first to last.

The Bands style is similar to the Single Band style, except that all panes may also be close or open at the same time.

The Bands style allows the user to display or collapse all panes at the same time.

Defining Tabs

To define tabs on a dialog component:

  1. Display the Form Controls page.

  2. Click Insert Layout Command... , which appears below the Control Types list.

  3. Select "[Tab Control]" and click OK.

  4. In the Control Types list position the [Tab Control] entry above all controls that it will contain.

  5. Click Insert Layout Command... , select "[Tab Pane]" and click OK.

  6. Position the [Tab Pane].1 entry below the [Tab Control].1 but above all controls that it will contain.

  7. Click Insert Layout Command... , select "[Tab Pane]" and click OK.

  8. Position the [Tab Pane].2 entry below the controls of [Tab Pane].1 but above all controls that it will contain.

  9. Repeat steps 7 and 8 for additional tab panes.

  10. Select the [Tab Control] object in the Select Controls list.

  11. Click to display the HTML Layout dialog.

  12. In the Type field select "Tabs", "Genie", "Bands", or "SingleBands".

Setting Tab Style Tab Control Properties

If you chose "Tabs" in step 12:

  1. Select whether to place the tab controls on the "Top" or "Bottom" in the Tab Settings > Align field.

  2. Optionally, in the Tab Settings > Inline Style field clickto display the <span class=Screen>Style Editor</span>. Here you can define HTML settings that will override the selected style sheet.

  3. Optionally, set the width of the tab control in the Sizing > Width field. Otherwise, the size of the control will change as its contents change. The tab control will be left aligned within its frame.

  4. Optionally, set the height of the tab control in the Sizing > Height field. The tab control will be vertically centered within its frame.

Setting Genie Style Tab Control Properties

If you chose "Genie" in step 12:

  1. Select whether the control will have a title.

  2. Optionally, in the Genie Settings > Inline Style field click to display the <span class=Screen>Style Editor</span>. Here you can define HTML settings that will override the selected style sheet.

  3. Optionally, change the tile of the "Previous" button in the Genie Settings > Previous button label field.

  4. Optionally, change the tile of the "Next" button in the Genie Settings > Next button label field.

  5. Optionally, set the width of the tab control in the Sizing > Width field. Otherwise, the size of the control will change as its contents change. The tab control will be left aligned within its frame.

  6. Optionally, enable the progress bar in the Genie Settings > Progress Bar field. This device gives you a graphical view of which tab in the sequence of tabs that you are viewing.

  7. If you enable the progress bar, additional properties will appear.

    1. Select the progress bar style.

    2. Optionally, change the size and style of the progress bar cells that represent unvisited panes.

    3. Optionally, change the size and style of the progress bar cells that represent visited panes.

  8. Optionally, set the height of the tab control in the Sizing > Height field. The tab control will be vertically centered within its frame.

Setting Bands Style Tab Control Properties

If you chose "Bands" or "SingleBands" in step 12:

  1. Select whether to display tab separators in the Bands Settings > Separator field.

  2. Optionally, set the width of the tab control in the Sizing > Width field. Otherwise, the size of the control will change as its contents change. The tab control will be left aligned within its frame.
