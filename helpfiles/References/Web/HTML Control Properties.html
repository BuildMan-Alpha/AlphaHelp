HTML Control Properties

The HTML Controls that you place on an HTML page have many properties that you can read and write.

Form Tags

The <FORM>...</FORM> tags surround one or more <INPUT>, <SELECT>...</SELECT> , or <TEXTAREA>...</TEXTAREA> tags. A form also has a submit button that instructs the browser to send the contents of these tags to the Application Server. The general syntax for the <FORM> tag is:

<FORM METHOD=" Get | Post " [ ACTION=" action " ]  [ other options ] > [ <INPUT> ]  [ <SELECT>...</SELECT> ]  [ <TEXTAREA>...</TEXTAREA> ] </FORM>

Refer to an HTML reference for a complete description of the <FORM> tag. The METHOD argument has two possible values:

  • Get = You may save (bookmark) and refresh URLs formatted through a HTTP GET command. Some devices may restrict the total length of a URL to 128 characters, which may trim the arguments appended by a GET command.

  • Post = Used for forms that submit a large amount of data. There is no length restriction with POST commands. You cannot bookmark the arguments sent by a HTTP POST command.

Input Tags

The <INPUT> tag accepts user input in a variety of formats. The general syntax for the tag is:

<INPUT type=" style " name=" name " value=" value " [ size=" size " ] [ maxlength=" maxchars " ] [ checked ] [ style=" arguments " ] >

Select and Option Tags

The <SELECT> and <OPTION> tags present the user with a number of options in a list format. The general syntax for the tag is:

<SELECT name=" name " [ size=" size " ]  [ multiple ] >

<OPTION [ selected ] >value</OPTION>

...

<OPTION [ selected ] >value</OPTION>

</SELECT>

Description

C

  • "Text" - normal display of text.

TextArea Controls

The <TEXTAREA> tag produces a multi-line text box. The general syntax for the tag is:

<TEXTAREA name=" name " [ rows=" rows " ] [ cols=" cols " ] [ wrap=" wrap " ] [ style=" arguments " ] > value </TEXTAREA>

Refer to an HTML reference for a complete description of the <TEXTAREA> tag.

Property
