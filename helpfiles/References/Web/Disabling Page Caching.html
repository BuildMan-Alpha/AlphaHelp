Disabling Page Caching

The HTTP protocol specifies a number of headers to control caching of web pages. You can add the following code to your pages to set all of those headers:

Examples

<%a5

Response.Add_Header("Expires: Sun, 14 Jan 1968 12:01:00") 'use any date in the past Response.Add_Header("Pragma: no-cache")

Response.Add_Header("Cache-Control: private")

%>
