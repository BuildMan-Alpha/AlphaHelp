Insert table

Displays the Table Properties dialog, which allows you to specify table characteristics, then inserts the table at the cursor location.

Insert row

Place the cursor in a cell below where the row is to be inserted.

Insert column

Place the cursor in a cell to the right of where the column is to be inserted.

Insert cell

Place the cursor in a cell to the right of where the cell is to be inserted. This action inserts the

Delete row

Place the cursor in a cell in the row to be deleted. The row and its contents will be deleted.

Delete column

Place the cursor in a cell in the column to be deleted. The column and its contents will be deleted.

Delete cell

Place the cursor in the cell to be deleted. The cell and its contents will be deleted.

Merge cells

Place the cursor in the left-most of two or more adjacent cells in the same row. Hold the SHIFT key and move the cursor to the right-most cell. The contents of the cells will be combined. A COLSPAN=n command, where "n" represents the number of cells, will be inserted into the

command.
