Disabling the Back Key

To disable the back key on a page, place the following code immediately after the <head> tag. Any Xbasic code on the page follows after this JavaScript code. After the user navigates to the next page the back button will be disabled.

<script language="JavaScript">

    <!-- javascript:window.history.forward(1); //-->

</script>
