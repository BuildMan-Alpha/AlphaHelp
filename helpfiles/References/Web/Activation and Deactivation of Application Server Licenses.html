Activation and Deactivation of Application Server Licenses

The Sessions tab of the Application Server Control Panel (click Tools > Application Server > Sessions ) allows you to add and remove licenses for your Application Server. Each license allows a specific number of simultaneous user sessions on your server. Multiple licenses may be installed on a single server in order to allow a larger number of sessions.

After a license is added in the Application Server Control Panel, it must be activated before it is usable. To activate a license:

  1. Display the Sessions tab of the Application Server Control Panel.

  2. Make sure your Internet connection is enabled. Your computer must be connected to the Internet in order for the activation process to be completed. If your computer does not typically have Internet access, it must be temporarily connected to the Internet before you can activate a Application Server license.

  3. Select the license in the Licence number list.

  4. Click the Activate button. An activation request will be sent from your computer to Alpha Software's activation server. as long as that license is not active on another computer, it will be activated on your computer.

Once you have activated your license, it cannot be activated on any other computer. It can however be added to any number of other computers. This would allow you to store a copy of the license on another server, but not actually use that license while it was active on the first server.

To use your license on a different computer, for example moving it to your server from your development workstation:

  1. One the first computer display the Sessions tab of the Application Server Control Panel.

  2. Make sure your Internet connection is enabled.

  3. Select the license in the Licence number list.

  4. Click the Deactivate button.

  5. On the second computer display the Sessions tab of the Application Server Control Panel.

  6. Click Add and enter the license from the first computer.

  7. Select the license in the Licence number list.

  8. Click the Activate button.
