Adding Xbasic to HTML Pages

The .A5W pages processed by the Application Server contain a mixture of HTML and Xbasic code. There are two equally valid ways of adding the Xbasic code to an page. The simplest way is to separate the HTML and Xbasic on separate lines.

<%A5 if flag = .t. %>

<A HREF="http://www.alpha5.com">

<%A5 end if %>

Note that the if and end if form a logical unit, but are physically separated by a line of HTML.

The second approach creates a single Xbasic code block that incorporates any required HTML.

<%A5

if flag = .t.

? "<A HREF=\"http://www.alpha5.com\">"

end if

%>

Note that you have to:

  • Escape the HTML line by preceding it with a "?" symbol.

  • Treat the HTML line as  a string.

Consequently,

  • The HTML line must be surrounded by quote symbols

  • All quote symbols within the HTML line must be escaped with "\" symbols

Supported By

Alpha Five Version 6 and Above

Limitations

Web publishing applications only.
