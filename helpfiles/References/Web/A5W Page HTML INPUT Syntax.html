A5W Page HTML INPUT Syntax

<INPUT TYPE= Directive NAME= Variable VALUE= Value >

Argument

Type

Description

Directive

C

Defines the type of INPUT control. Values can be:

  • "SUBMIT" = Generates a button with a Value that is used as a label and that is placed into Variable.

  • "HIDDEN" = Generates a Value that is placed into Variable.

There are many other types of input controls. A good reference can be found at http://www.w3.org/TR/REC-html40/interact/forms.html#h-17.4.1.

Variable

C

The variable that will contain the Value.

Value

C

A literal value or an Xbasic expression that generates a value.

Description

A form is basically a collection of INPUT controls with NAME and VALUE attributes. These controls accept data from the user. When you click the SUBMIT button, the browser collects the stored values, associates them with their names, and (usually) sends the resulting data to the server. The METHOD attribute defines how to transmit the data to the server. The ACTION attribute specifies how the data is processed at the server.

Example

<FORM ACTION="<%A5 ? Request.script_name %>">

<INPUT TYPE="SUBMIT" NAME="a_dlg_button" VALUE="First">

<INPUT TYPE="SUBMIT" NAME="a_dlg_button" VALUE="Previous">

<INPUT TYPE="SUBMIT" NAME="a_dlg_button" VALUE="Next">

<INPUT TYPE="SUBMIT" NAME="a_dlg_button" VALUE="Last">

<INPUT TYPE="HIDDEN" NAME="this_recno" VALUE="<%A5 ? tbl.recno()%>">

</FORM>
