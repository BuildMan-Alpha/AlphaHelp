ServerSetting System Variable

The ServerSetting system variable provides information about the Application Server. All variables are read only.

Property

 

Description

.A5W_PageExtension

C

Defines the page extension that indicates an HTML page containing Xbasic code to be processed by the Application Server.

.Access_Log

C

The path to and name of the access log. Enabled by the .Logging property.

.AllowDenyIP

C

Indicates whether restricted IP addresses will be blocked or allowed. See also the .EnableIPRestriction and .IPList properties. The options are:

  • "Deny"

  • "Allow"

.AllowDirBrowse

N

Indicates whether directory browsing is enabled. The options are:

  • 1 (True)

  • 0 (False)

.AutoStart

N

Indicates whether the Application Server will start when Alpha Anywhere starts. The options are:

  • 1 (True)

  • 0 (False)

.DefaultPage

C

A priority list of default pages to open if the URL does not specify a page.

.Document_Root

C

The path of the document root folder.

.EnableIPRestriction

N

Indicates whether specific IP addresses will be blocked or enabled. See also the .AllowDenyIP and .IPList properties. The options are:

  • 1 (True)

  • 0 (False)

.Error_401

.Error_403

.Error_408

.Error_505

C

For each, the path and name of the error page to send.

.Error_Log

C

The path to and name of the error log. Enabled by the .Logging property.

.IPList

C

A list of IP addresses to allow or deny.

.Logging

N

Indicates whether server logging has been enabled. The options are:

  • 1 (True)

  • 0 (False)

.Max_Spare_Threads

N

See Maximum_spare_threads.

.Max_Threads

N

See Maximum_threads.

.Min_Spare_Threads

N

See Minimum_spare_threads.

.Prompt_on_Start

N

Determines whether to display the confirmation dialog when starting the server. The options are:

  • 1 (True)

  • 0 (False)

.Prompt_on_Stop

N

Determines whether to display the confirmation dialog when stopping the server. The options are:

  • 1 (True)

  • 0 (False)

.Prompt_with_URL

N

Determines whether to display the dialog shown from the sy startup that explains how to access the server. The options are:

  • 1 (True)

  • 0 (False)

.Raw_Log

C

The path and the name of the raw request log. Enabled by the .Raw_Logging property.

.Raw_Logging

N

Indicates whether raw request logging has been enabled. The options are:

  • 1 (True)

  • 0 (False)

.Server_Name

C

The name of the server.

.Server_Port

N

The port that the server monitors.

.Server_Software

C

The build of the server software.

.Session_Cookie_Name

N

The name of the session cookie.

.Session_Timeout

N

The number of seconds before a session times out.

.SessionCookiesEnabled

N

The options are:

  • 1 (True)

  • 0 (False)

.SSL_Enabled

L

Indicates whether SSL is enabled. The options are:

  • True

  • False

.Start_Threads

N

See Start_threads.

.CheckSecurity

N

Indicates whether the security framework has been enabled. If TRUE, the server will enforce page level security. The options are:

  • 1 (True)

  • 0 (False)

Limitations

Web applications only.
