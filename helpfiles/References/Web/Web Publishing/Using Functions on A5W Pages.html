Using Functions on A5W Pages

A5W web pages may contain calls to your custom built Xbasic functions. There are several approaches:

  • Put the function you need in a section of <%a5 ... %> code in your a5w page BEFORE your component is loaded. You can this by explicitly pasting the code into the page or including it from another file using A5W_INCLUDE().

  • Put the function into an .AEX library and use A5W_LOAD_AEX() to make it available to your web page.
