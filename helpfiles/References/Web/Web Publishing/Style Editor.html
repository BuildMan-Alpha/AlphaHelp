Style Editor

The Style Editor allows you to edit all the HTML attributes of an element on an A5W page.

Sets the point size of the font. The dimension options are:

  • "px" - pixels

  • "pt" - points (1 point = 1/72 inches)

  • "pc" - picas (1 pica = 12 points)

  • "cm" - centimeters

  • "ex" - x heights (the height of the letter "x")

  • "%" - percentage

Position Settings

Option

Description

Position: Mode

Sets the mode of calculating the position of text in its bounding box. The options are:

  • ""

  • " static"

  • "relative"

  • "absolute"

  • "fixed"

  • "inherit"

Position: Top

Sets the offset of the text from the top of its bounding box. The dimension options are:

  • "px" - pixels

  • "pt" - points (1 point = 1/72 inches)

  • "pc" - picas (1 pica = 12 points)

  • "mm" - millimeters

  • "cm" - centimeters

  • "em" - em spaces (the height of the element's font)

  • "ex" - x heights (the height of the letter "x")

  • "%" - percentage

Position: Left

Sets the offset of the text from the left side of its bounding box. The dimension options are:

  • "px" - pixels

  • "pt" - points (1 point = 1/72 inches)

  • "pc" - picas (1 pica = 12 points)

  • "mm" - millimeters

  • "cm" - centimeters

  • "em" - em spaces (the height of the element's font)

  • "ex" - x heights (the height of the letter "x")

  • "%" - percentage

Z-Index

The Z-Index property only applies to objects that have the Position Mode property set to "relative" or "absolute". Positive z-index values are positioned above a negative (or lesser value) z-index. Two objects with the same z-index are stacked according to source order. A positive value positions the element above text that has no defined z-index, and a negative value positions it below.

Width

Sets the horizontal width of the text element.

Height

Sets the height of the text element.
