Packing a Table from an A5W Page

Packing an Alpha Five table from an A5W page needs to accommodate the possibility that the pack will not succeed if the user does not have exclusive access to the table. Whether successful or not, the operation needs to inform the user of the resulting status. Here is a script by Peter Wayne that solves the problem.

Place this script at the top of the page that will pack the table named mytable then redirect to a page named myhomepage.a5w.

<%a5
on error goto no_pack
t=table.open("PathAlias.adb_path\mytable",file_rw_exclusive)
t.pack()
t.close()
on error goto 0

 

' success page

html=<<%txt%
< html >
< head >< /head>
< body >
< P >Pack completed!< /P >
< form method=post action="myhomepage.a5w" >
< input type=submit value=OK >
< /form >
< /body >
< /html >
%txt%

 

? html
end

no_pack:

 

' failure page

html=<<%txt%
< html >
< head >< /head >
< body >
< P >Pack unsuccessful, try later!< /P >
< form method=post action="myhomepage.a5w" >
< input type=submit value=OK >
< /form >
< /body >
< /html >
%txt%

? html
end
%>

< html >

< head >

< meta name="generator" content="Alpha Five HTML Editor" >

< title >< ;/title >

< /head >

< body >

< p >This is the calling page< /p >

< /body >

< /html >
