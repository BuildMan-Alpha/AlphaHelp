Creating Dialog Components

The Web Component Builder is a genie that you use to design the interactive database components of an A5W page (an A5W page is a hybrid HTML/Xbasic page that is processed by the Application Server ).

You can use the Web Component Builder to design a component and set all its properties, so that it is ready to use.

The Dialog Component allows you to create an HTML form that prompts the user for variables. Buttons on the form allow the user to submit the form. When the form is submitted, the values in the variables are validated. If all validation rules are satisfied, then Alpha Five runs code that you specify in the form's AfterValidate event.

The Web Component Builder has four tabs at the bottom of its window.

  • Preview - displays a WYSIWYG view of the component as you build it

  • HTML - a read-only display of the HTML code of the component as you build it

  • Xbasic - a read-only display of the HTML code of the component as you build it

  • Live Preview - allows you to run the component and test its features

Creating the Dialog Component

To create a dialog component.

  1. Start the Web Projects Control Panel.

  2. Click New on the toolbar.

  3. Select Web Component and click Next >.

  4. Select Dialog and click OK.

A dialog component may have as many controls of different types as you require for your application. The only requirement is that the controls have unique names. as you add controls Alpha Five will suggest unique names that you may change.

Adding Controls to the Dialog

To add a control:

  1. Display the Form > Controls menu page.

  2. Select the type of control you want in the Control Types list. and click to add it to the dialog.

  3. In the resulting New Control dialog, optionally change the Name of the control and Label associated with it.

  4. Select the control's data type, which may be:

    • Character

    • Numeric

    • Date

    • Logical

  5. To remove a control from the dialog component, select it in the Selected Controls list and click .

  6. Optionally, select entries in the Selected Controls list and use the buttons to reorder their sequence on the dialog box. Note : Components are displayed from left to right, top to bottom.

  7. Each control on a dialog component has a number of properties that you may want to change. Refer to Setting Dialog Control Properties.

Setting Dialog Properties

Display the Form > Properties menu page. The resulting dialog boxes set additional display and formatting properties of the component. Refer to Setting Dialog Component Properties.

Previewing Your Work

  1. Click Preview to update the view of your dialog design in the window at the bottom of the Web Component Builder.

  2. Click Browser to view your dialog in action in your web browser.

  3. Optionally, select Information. This tab provides fields to capture a Description of and Comments about the dialog component.

  4. Optionally, click Save Page to create a named .A5W page containing this component.

    1. Optionally, check Open page in HTML editor after saving.

    2. Click Save Page.

    3. Select the Web Project that will contain the page from the Web Project drop-down list box.

    4. Enter the page name in the File name field.

    5. Click Save.

  5. Click to preserve your work.
