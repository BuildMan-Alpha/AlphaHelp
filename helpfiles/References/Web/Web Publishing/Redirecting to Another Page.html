Redirecting to Another Page

The following code will display a button that displays "Click me to go to page 2" and then displays page2.a5w.

< body >

...

< form action="page2.a5w" >< input type="submit" value="Click me to go to page 2" >< /form >

...

< /body >

Alpha Five Version 6 and Above

Limitations

Web publishing applications only.
