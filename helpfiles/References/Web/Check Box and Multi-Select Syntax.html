Check Box and Multi-Select Syntax

<SELECT NAME= Array_Variable MULTIPLE >

Argument

Type

Description

Array_Variable

C

The array variable that will contain the selected values .

Description

When using check boxes or multi-select boxes, the form element name should be named as an array in order to contain the multiple values. Failure to follow this convention will cause only the last value passed in for this form element to be visible to the page being processed. Here is a snippet of HTML that would create a select box allowing multiple choices.

<SELECT NAME="foo[]" MULTIPLE>

    <OPTION VALUE="bar1">bar1</OPTION>

    <OPTION VALUE="bar2">bar2</OPTION>

    <OPTION VALUE="bar3">bar3</OPTION>

    <OPTION VALUE="bar4">bar4</OPTION>

</SELECT>

If all four options were selected when the form was submitted, the result will be the following array:

foo[1] = bar1

foo[2] = bar2

foo[3] = bar3

foo[4] = bar4

However if in the example above we used <SELECT NAME="foo" MULTIPLE>, foo would not be created as an array. Instead, foo would be a variable containing only "bar4", the last value stored by the form.
