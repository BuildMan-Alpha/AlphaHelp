Application Server FAQ

Can I install the same license on multiple computers?

Yes, you can install the same license on as many computers as you wish. However only one computer will be able to activate that license at a time.

What information do you collect when I activate a license?

When you activate a Application Server license, our server records your Alpha Anywhere license key, your Application Server license key, your computer's hostname and the IP address from which the activation request was made. No other information is sent by your computer during this process or collected by Alpha Software.

How can I activate a Application Server license if my computer is not connected to the Internet?

You cannot activate your license unless your computer has an active Internet connection at the time you activate. You must, at least temporarily, connect your computer to the Internet to complete the activation process. Once your license has been activated, your computer no longer requires use of the Internet connection for operation and you may disconnect it if you wish.

How can I move a Application Server license from one computer to another?

Simply deactivate the license on the first computer, then add it to the second computer and activate it there.

How can my users upload files to the Application Server?

You will have to write a short script to enable file uploads. See Uploading Files to the Application Server.

How can users download files from the Application Server?

If you want to make a file available to users of your web application, you simply place that file within your webroot and hyperlink to it. For example, if your webroot is the default c:\a5webroot, your server is www.myserver.com and you have a file called report.pdf that you want to make available as part of your application.

  1. Place report.pdf in c:\a5webroot.

  2. Provide a hyperlink to http://www.myserver.com/report.pdf.

That's it. Clicking the link will download the file for the user.
