Automatically Restarting the Application Server

Note : At least one user must be logged onto the Windows computer hosting the Application Server for it to run.

Windows XP Home

The following configuration will restart the Application Server as soon as the host computer is powered on and finished booting.

  1. You have to set Alpha Five to start the Application Server when Alpha Five starts. To do this:

    1. Start Alpha Anywhere.

    2. Select Tools > Application Server, to display the Application Server Control Panel.

    3. On the General tab check the box that indicates the Application Server should start when Alpha Anywhere starts.

    4. Close the Application Server Control Panel.

  2. You have to allow Windows to start up without a user logon.

    1. Select Start > Run.

    2. Type "control userpasswords2" and click ENTER to display the User Accounts dialog.

    3. Clear the check box that says "Users must enter a password to use this computer".

    4. Click Apply.

    5. When prompted enter the name and password of a default user for the computer.

    6. Click OK.

  3. Cause Alpha Anywhere to startup when Windows starts.

    1. Navigate to "c:\Documents and Settings\All Users\Start Menu\Programs\Startup\".

    2. Click File > New > Shortcut to display the Create Shortcut dialog.

    3. Enter the path to the Alpha Anywhere program file (e.g. "c:\program files\a5v6\alpha5.exe").

    4. Click Next > Finish.
