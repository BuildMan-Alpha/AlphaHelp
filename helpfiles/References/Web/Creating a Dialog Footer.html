Creating a Dialog Footer

This example shows a dialog footer.

A dialog footer is label that appears on the last row of your dialog box. To create a dialog footer:

  1. Display the Form > Properties page of the Dialog Component Builder.

  2. Set Dialog Footer Layout > Freeform layout to TRUE (checked).

  3. Click in the Dialog Footer Layout > Dialog footer template property to display the Freeform Column Layout dialog.

  4. Optionally, click HTML Editor to display the HTML Editor.

  1. Enter the HTML for the template in either dialog.

  2. Note that you can use the Freeform Column Layout dialog to place controls and buttons into the template. Position the cursor in the HTML and double click any field in the Available Fields list to insert it at the cursor location.

  3. Click Save in the HTML Editor to return to the Freeform Column Layout dialog.

  4. Click OK in the Freeform Column Layout dialog to return to the Dialog Component Builder.
