Formatting Toolbar

The Formatting toolbar contains:

Function

Control or Button

Description

Apply style

Applies a style to the currently selected text. Only available on the WYSIWYG tab.

Apply font

Applies a font to the currently selected text.

Apply point size

Applies a point size to the currently selected text.

Bold

Surrounds the currently selected text with and tags.

Italic

Surrounds the currently selected text with and tags.

Underline

Surrounds the currently selected text with and tags.

Larger

Increases the point size of the currently selected text.

Smaller

Decreases the point size of the currently selected text.

Apply Color

Displays the <span class=Screen>Color Picker</span>, allowing you to select a color to apply to a page object.

Left Align

Left aligns the currently selected paragraph.

Center Align

Center aligns the currently selected paragraph.

Right Align

Right aligns the currently selected paragraph.

Numbered List

Surrounds the currently selected text with

  1. and
tags.

Bulleted List

Surrounds the currently selected text with

  • and
tags.

Increase Indent

Adds another level of

    ... .
or
    ... .
 tags around the selected text.

Decrease Indent

Removes a level of

    ... .
or
    ... .
 tags from around the selected text.

Supported By

Alpha Five Version 8 Professional Edition and Above

Limitations

Web publishing applications only.
