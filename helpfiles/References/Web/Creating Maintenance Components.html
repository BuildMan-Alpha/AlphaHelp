Creating Maintenance Components

The Maintenance Builder creates components that will pack (remove deleted files) and re-index Alpha Five tables.

To create a maintenance component.

Define the Layout

  1. Display the Web Projects Control Panel.

  2. Select New > Alpha Five Web Component to display the Component Type dialog.

  3. Select Maintenance and click OK.

  4. Display the Form > Layout menu page.

  5. Make a choice in the Layout Options > Layout Type list. The options are:

    • "Tabular"

    • "Stacked Columns"

The Tabular and Stacked Columns layout options

  1. Optionally, click in the Layout Options > Style Name property to select a cascading style sheet.

  2. Optionally, select Layout Options > Use simple style to have the component use the style of its container page.

  3. Optionally, to create a freeform layout for the component:

    1. Select Layout Options > Freeform Layout.

    2. Click in the Layout Options > Freeform Template property to display the Freeform Layout dialog.

    3. You may create any legitimate HTML code between the

      ...
      tags. Place the {PackList}, {ReindexList}, {SubmitButton}, and {ResetButton} placeholders in any positions where you are allowed to place HTML text.

    4. Optionally, click HTML Editor to edit the layout in the <span class=Screen>HTML Editor</span>.

    5. Optionally, click Find Missing/Used Fields to identify any placeholders that are not in the layout.

    6. Optionally, click Revert to Default Layout to discard any edits to a new freeform layout.

    7. If you have saved the layout, clicking the Revert to Saved Freeform button will discard any changes made in the current editing session.

Define Options

  1. Display the Form > Options menu page.

  2. Optionally, for each item in the Option Types list.

    1. Set the Option Properties > Show Option property if you wish to display it.

    2. Set the number of rows to display in the Option Properties > Height property.

    3. Set the Option Properties > Show Select All Tables property if you wish to display this entry in the list.

    4. Edit the text of the Option Properties > Select All Tables Text property.

    5. Click in the Option Properties > In-line Style property to create an override style for the drop down list control with the <span class=Screen>Style Editor</span>.

    6. Edit the label used for the function in the Row Properties > Row Label property.

    7. Click in the Row Properties > Row Label In-line Style property to create an override style for the label.

    8. Enter bubble help text in the Row Properties > Label Bubble Help Text property.

    9. Click in the Row Properties > Cell In-line Style property to create an override style for the label's cell.

Define Button Properties

  1. Optionally, to hide the Submit button, clear the Submit/Cancel Buttons > Has Submit Button property.

  2. Optionally, change the Submit button label in the Submit/Cancel Buttons > Submit Button Label property.

  3. Optionally, click in the Submit/Cancel Buttons > Submit Button In-line Style property to create an override style for the button.

  4. Optionally, enter bubble help text in the Submit/Cancel Buttons > Submit Button Bubble Help Text property.

  5. Optionally, to hide the Cancel button, clear the Submit/Cancel Buttons > Has Cancel Button property.

  6. Optionally, change the Submit button label in the Submit/Cancel Buttons > Cancel Button Label property.

  7. Optionally, click in the Submit/Cancel Buttons > Cancel Button In-line Style property to create an override style for the button.

  8. Optionally, enter bubble help text in the Submit/Cancel Buttons > Cancel Button Bubble Help Text property.

  9. Optionally, select an alignment for the buttons in the Submit/Cancel Buttons > Button Alignment property. The options are"

    • "Center"

    • "Left"

    • "Right"

  10. Optionally, select an order for the buttons in the Submit/Cancel Buttons > Button Order property. The options are"

    • "Submit Button First"

    • "Cancel Button First"

Maintenance Complete Page

The maintenance complete page is a temporary page (automatically created for you) that displays while either a pack or re-index operation is processing. This page displays a "Completed Button".

  1. Optionally, change the Completed button label in the Completed Button > Completed Button Label property.

  2. Optionally, click in the Completed Button > Completed Button In-line Style property to create an override style for the button.

  3. Optionally, enter bubble help text in the Completed Button > Completed Button Bubble Help Text property.

Modify Messages

You may modify any of the messages that appear.

  1. The Messages > Table Not Selected message appears when the operator clicks the Submit button without first selecting a table.

  2. The Messages > Processing message appears on the maintenance complete page.

  3. The Messages > Completed message appears when the selected operation completes successfully.

  4. The Messages > Maintenance Failed message appears if an error occurs.

  5. The Messages > Table Skipped message appears if an operation could not be completed for one or more selected tables.

  6. The Messages > Unknown Error message appears if any other type of error occurs.

Supported By

Alpha Five Version 8 Professional Edition and Above
