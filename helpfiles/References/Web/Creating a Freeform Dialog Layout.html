Creating a Freeform Dialog Layout

This example shows a freeform dialog layout. A freeform layout allows you to position controls and buttons at any location on the dialog form.

To create a freeform dialog layout:

  1. Display the Form > Properties page of the Dialog Component Builder.

  2. Set Layout Options > Use Free-form layout to TRUE (checked).

  3. Click in the Layout Options > Free-Form Layout Definition property to display the Freeform Dialog Layout dialog.

  4. Optionally, click HTML Editor to display the HTML Editor.

  5. Edit the HTML of the layout.

  6. Note that you can use the Freeform Dialog Layout dialog to place controls into the template. Position the cursor in the HTML and double click any field in the Available Fields list to insert it at the cursor location.

  7. Click Save in the HTML Editor to return to the Freeform Column Layout dialog.

  8. Click OK in the Freeform Column Layout dialog to return to the Dialog Component Builder.

Limitations

Web publishing applications only; not for Ajax dialogs.
