A5 Properties

Property

Description

.window_title

The text that appears on the title bar of the window. Default = "Alpha Anywhere".

.left

The horizontal pixel coordinate of the Alpha Anywhere window.

.top

The vertical pixel coordinate of the Alpha Anywhere window.

.width

The width of the Alpha Anywhere window in pixels.

.height

The height of the Alpha Anywhere window in pixels.
