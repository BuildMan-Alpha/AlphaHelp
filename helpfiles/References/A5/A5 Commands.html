A5 Commands

Note : A5 Commands are not guaranteed to remain constant or available in future Alpha Five releases.

Syntax

A5.COMMAND( [ Command as C ] )

Argument

Description

Command

The command to send to Alpha Five.

Command

Description

DATABASE_CLOSE

Performs the File > Close Database procedure.

DATABASE_COMPACT

Performs the File > Database Compact procedure.

DATABASE_NEW

Performs the File > New Database procedure.

DATABASE_OPEN

Performs the File > Open Database procedure.

FILE_EXIT

Performs the File > Exit procedure.

FILE_PRINT_SETUP

Performs the File > Print Setup... procedure to display the Print Setup dialog.

HELP_ABOUT

Performs the Help > About Alpha Five... procedure.

HELP_CONTENTS

Performs the Help > Contents... procedure.

VIEW_CODEEDITOR

Performs the View > Code Editor procedure.

VIEW_DEBUGGER

Performs the View > Debugger procedure.

VIEW_RECORDER

Performs the View > Script Recorder Window procedure.

VIEW_RECORDER_END

Performs the Close Recorder procedure when the Script Recorder dialog is open.

VIEW_SETTINGS

Performs the View > Settings procedure.

VIEW_STATUSBAR

Performs the View > Status Bar procedure.

VIEW_TOOLBARS

Displays the Toolbar Launcher dialog box.

VIEW_TRACE

Performs the View > Trace Window procedure.

WINDOW_ARRANGE

Performs the Window > Arrange Icons procedure.

WINDOW_CASCADE

Performs the Window > Cascade procedure.

WINDOW_TILE_HORZ

Performs the Window > Tile Horizontally procedure.

WINDOW_TILE_VERT

Performs the Window > Tile Vertically procedure.
