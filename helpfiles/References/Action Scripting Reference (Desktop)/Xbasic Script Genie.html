Xbasic Script Genie

The Xbasic Script Genie generates the Xbasic code associated with the library of Action Scripting actions. The result of the process is code that is ready to be copied and pasted into your script or function. To use the Xbasic Script Genie :

  1. Display the Code Editor ( View > Code Editor ).

  2. Optionally, open an existing script.

  3. If you want to paste the Xbasic code that you are going to generate, position the cursor at the insertion point.

  4. Right click on the white space of the Code Editor and select Genies... > Xbasic Script Genie... .

  5. Select an item from the Category list.

  6. Select an action from the Action list.

  7. Click OK.

  8. Enter the information required by the action you selected.

  9. Click Finish to display the Xbasic Code dialog box.

  10. Optionally, click Copy Code to the Clipboard in order to paste it later.

  11. Optionally, click Insert Code in Code Editor to paste it directly at the current cursor location.
