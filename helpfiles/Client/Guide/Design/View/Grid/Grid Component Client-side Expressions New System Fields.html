Grid Component Client-side Expressions New System Fields

Client-side expressions (used in calculated fields, show/hide expressions, enable expressions, conditional style and conditional image expressions) can now reference two new system fields:
  • grid.rowIsChecked - a logical field that is true if the current row is checked
  • grid.checkedRowCount - a numeric field with the count of the number of checked rows
