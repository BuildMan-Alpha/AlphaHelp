{dialog.object}.accordionControl Method

Syntax

{dialog.object}.accordionControl(id,action);

Description

Either opens or closes all panes in an accordion object. action is either 'open' or 'close'. id is the Id of the accordion (i.e. Tab) object.
Note: The 'open' action will only open all accordion panes if the accordion has been configured to allow multiple open panes. 

{dialog.object}.accordionControl('TAB_1','open');
