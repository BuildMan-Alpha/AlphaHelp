Send a Table or Set

Overview

The Send a Table or Set action lets you select tables or sets to zip up and send by email. You can specify a specific table or set name in the genie, or prompt the user for the name at run-time.

Procedure

  1. Click the Set or Table radio button to specify what type of file you are sending.

  2. Select a table or set from the resulting Set Name or Table Name list.

  3. If you want the user to make the decisions in steps 4, 5, and 6, check the Display the Table/Set Send Options dialog box check box and skip to step 7. Otherwise, continue with step 4.

  4. Optionally, check Compress the table or set before sending. If you choose this option, Alpha Five will compress the table or set before sending the email. Compression reduces the size of the attached files.

  5. Optionally, check Include the table or set's data dictionary. The Data Dictionary stores any component object created by the table or set such as forms, browses, reports, query definitions, and scripts: the data dictionary for a table also stores the field rules. Specify whether or not to transmit this information with the table or set.

  6. Optionally, check Include indexes if you want to include the table or set's indexes in the email.

  7. Enter the email address of the recipient. You can use multiple addresses as long they are separated by commas.

  8. In the Send using list select either:

    • "Built-in Alpha Five email client"

    • "Default email client (e.g. Eudora, Outlook, etc.)"

If you select "Built-in Alpha Five email client", the user has the opportunity to make edits before sending the. Otherwise, the email is sent automatically without further user interaction.

  1. Click Next >.

  2. Optionally modify the comment that describes the action.

  3. Click Finish.
