Specifying the Toolbar for a Layout

To specify the toolbar for a form, browse, report, letter, or label:

  1. Optionally design custom toolbar(s) and save them. See Creating Custom Toolbars.

  2. Open the form or browse in design mode and press F12 to access its Properties dialog.

  3. Show the Menus/Toolbars tab.

  4. Select your customized toolbar(s) from the appropriate drop-down list box, and click OK.  

  1. Specify "" to turn off a toolbar, or specify "" to use the system default toolbar.
