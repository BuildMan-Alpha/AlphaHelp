Selecting the Data Source

This topic continues to describe the process of creating a grid component.

  1. Select Grid > Data Source.

  2. Select the source of the data. The options are:

    • "Alpha Five .dbf Tables"

    • "Remote Database accessed via ADO" - for Microsoft Access and other ADO/ODBC compatible databases

    • "Remote Database accessed via AlphaDAO - for SQL back-end databases

Note : AlphaDAO is preferred over ADO for accessing back-end SQL databases.
