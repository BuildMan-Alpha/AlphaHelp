Running a Netmailer Job

You can run Netmailer jobs by using the Netmailer Job Genie.

Method One

  1. Start Alpha Five and open a database.

  2. Right-click on a table or set, and select Addins > Netmailer > Run Job. The Netmailer Job Genie appears.

  1. Select the job you want to run, and click Next >.

  2. Optionally, add additional record selection criteria.

  1. Click Next > Finish.
