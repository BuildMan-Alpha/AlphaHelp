STOP

Syntax

STOP

Description

STOP suspends the execution of a script.

Note : If you want to cause a run-time error that can be trapped by an ON ERROR GOTO statement, use the ERROR_GENERATE() command.

Example

The script stops when x = 3.

trace.writeln("Start")

for x = 1 TO 10

    if x = 3 then

        stop

    end if

    trace.writeln(STR(x))

next x

trace.writeln("End")

end
