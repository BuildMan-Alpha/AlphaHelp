Publishing AlphaMovies

Note : The Publishing AlphaMovies topic describes a technique available to Alpha Five database users.

Although AlphaMovies has several tables and a set, the following example shows how easy it is to take one of them and publish it for use over the Internet.

Note : You should have already created the AlphaMovies database in the Creating a Database lesson of Getting Started with Alpha Five.

  1. Display the Web Projects Control Panel.

  2. Click New > OK. Select Grid and click OK to display the Component Type page of the Grid Builder.

  3. Select "Columnar" from the Grid layout list.

  4. Chick the Updateable radio button.

  5. Check Grid contains a 'Search' part.

  6. Display the Query (DBF) page.

  7. Select "customers" from the Table name field.

  8. Click to the right of the Order field.

  9. Select "Last_name" in the top list and "First_name" in the middle list. Click OK to continue.

  10. Display the Fields page.

  11. Select all the fields in the Available Fields list, with the exception of "" and "" and click to add them to the Selected Fields list.

  12. For each field in the Selected Fields list that has an underscore (_) in its name, change its Row Properties > Row Label (see picture below).

  1. Display the Properties page.

  2. Enter 3 into the Layout Options > Repeating Columns field. This means that the page will display three records horizontally.

  3. Enter 2 into the Layout Options > Rows of Data field. This means that the screen will show two data records and a third blank record for adding new customers.

  4. Select "Smaller" in the Layout Options > Font Size field. This reduces the default size of the type being used and reduces the size of the resulting screen.

  5. Display the Update Settings page.

  6. Enter 1 into the Permissions > Number of new record rows field.

  7. Display the Search Fields page.

  8. Select "First_Name", "Last_Name", "Company_Name", and "Phone_Number" and click to add them to the Fields to Search In list.

  9. Display the Search Properties page.

  10. Change the Search Options > Title when hidden field to "Find Customer".

  11. Save the component as "Alpha_Movies_Customers".

  12. Click Close to return to the Web Projects Control Panel.

  13. Select the Web Components page, select the new Alpha_Movies_Customers component, and click Publish > Publish.

  14. Click New > Alpha Five Web Page and click OK.

  15. Select File > Save As and name the new page "Alpha_Movies_Customers".

  16. Click Insert Component > Select. Pick the "Alpha_Movies_Customers" component and click OK > OK.

  17. Insert the following text above the grid component: "Alpha Movies Customer List".

  18. Select the new text and set it to the "Tahoma" font, size 3 (12 pt), and bold.

  19. Click and Close.

  20. Display the A5W Pages page.

  21. Right click the "Alpha_Movies_Customers" page and select Publish (Local Webroot) and open. Your new page will display as you see it above.

  22. Click Find Customer and the Search form appears.

  1. Enter "Kimberly" (without the quotes) into the First Name field and click Search. You should see something like the following screen.

This is a fully functional window into the AlphaMovies database that will let the remote Internet user add, edit, or delete records. Of course, you can limit any of those capabilities, making it a view-only form if you wish.
