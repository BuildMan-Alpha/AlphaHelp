Select Fields Dialog - Generic

The <span class=Screen>List View Genie</span> and <span class=Screen>ADO Import Genie</span> use the Specify Fields dialog to create SQL SELECT statements.

  1. Use the to add the fields you want to display to the Selected Fields list.

  2. Use the to set the fields display order.

  3. Select each field in the Selected Fields list:

    1. Enter the column title into the Properties > Column Title field.

    2. Make a selection in the Properties > Column Sizing list. The options are:

      • "Do not auto-size"

      • "Auto-size to data"

      • "Auto-size to column header"

      • "Specify width"

    3. If you selected "Specify width", enter the column width in inches into the Properties > Column Width field.

    4. Make a selection in the Properties > Justification list. The options are:

      • "Left"

      • "Center"

      • "Right"

  4. Click OK to continue.
