Record Navigator Designer

The Record Navigator Designer allows you to customize and create new record navigation button sets.

To define the record navigation controls on your web component.

  1. Open the selected grid component from the <span class=Screen>Web Projects Control Panel</span>.

  2. Display the Grid > Properties page.

  3. Make sure that the Record Navigator > Record navigation position value is not set to "No navigation bar".

  4. Click or in the Record Navigator > Record navigation layout field to open the Record Navigation Designer.

  1. Optionally, select a format in the Pre-Defined Format region.

  2. Optionally, select Custom layout design in the Pre-Defined Format region.

    1. In the Customize Links region place checkmarks to the left of the link buttons that you want to display.

    2. For each link button you display, optionally specify the text label to display.

  3. Optionally, make a selection from the Page numbers and links region. The options are:

    • No page numbers:

    • Current page only:

    • Current page and target page links:

  4. Optionally, change the Number of links to display. The maximum is 10.

  5. Optionally, check Show page count. Page counts are visible in two of the three examples above.

  6. Optionally, edit the Template for page number. Leave the {PageNumber} and {PageCount} fields unchanged.

  7. Optionally, select a different alignment of the navigator buttons from the Justify Record Navigator list. The options are:

    • "Left"

    • "Center"

    • "Right"

  8. Optionally, check Use images for links.

  9. Optionally, check Show disabled controls.

  10. Optionally, select a different style sheet from the Style drop-down list.

  11. Click OK to save your changes or Cancel to discard your changes.
