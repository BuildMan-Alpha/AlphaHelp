Stylesheet Properties

The following control properties can be set by a stylesheet.

Description

Border

The color of the border.

True or False determines which sides of the border are displayed.

Width

The width of the border.

Style

Select from the different border styles. The style attribute supercedes other elements for the border style definition. If Style is set to "No border", then Alpha will ignore the other style elements for Border.

Code

Onflyover

The Code category holds any Fly-over effects implemented for an object. Onflyover and Onflyoverleave contain Xbasic code which control the effect.

The Style property supercedes Shadow properties. If Style is set to "none", Alpha ignores Shadow properties.

Color

Color of the drop-shadow.

Shadow appearance

Browse

Applies to Browse objects only.

Readonly

True or False determines whether or not the Browse object is read-only (can't be edited).

Gridline_Horizontal

True displays the horizontal gridlines, False hides them.

Gridlline_Vertical

True displays the vertical gridlines, False hides them
