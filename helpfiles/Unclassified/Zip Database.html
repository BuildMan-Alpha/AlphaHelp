Zip Database

A Zip file is a standard file format that is used to store files in a compressed form. You can use products such as WinZip? to view the contents of a zip file. If you don't have WinZip, you can download a shareware version from www.shareware.com.

You may zip all or parts of a database into a compressed zip file.

Note : You cannot zip part of a password protected database.

The Zip Database function is an excellent way to move a database to a new location.

To zip a database:

  1. Select Tools > Zip Database... .

  2. Enter the path and name of the new zip file or click to specify the path and name of a zip file to replace.

  3. Optionally, check Compact Database before zipping? to potentially reduce its size.

  4. Optionally, check Include absolute path names in zip file? if you want the recipient to be able to replicate your directory structure.

  5. Optionally, check Include Web Project files. (Alpha Five version 6 and above.)

  6. Optionally, to make the list easier to read:

    1. Click Sort by Extension to group like files together.

    2. Click Sort by Name to alphabetize the list.

    3. Click Sort by Natural to sort the list by creation date.

  7. Optionally, check Hide support files? to show only database, table, and set files.

  8. Place checkmarks to the left of the files that you wish to send in the Files to Zip list.

    1. Optionally, click Mark All to place checkmarks next to all files.

    2. Optionally, click Unmark All to remove checkmarks from all files.

    3. Optionally, click Mark Files... to display the Mark Files dialog box.

    4. Click OK to place checkmarks to the left of all files.

    5. Click Data Files Only and OK to place checkmarks to the left of data files only.

    6. Click Application Files Only and OK to place checkmarks to the left of application files only.

    7. Optionally, click Unmark Files... to display the Un-mark Files dialog box.

    8. Click OK to remove checkmarks from the left of all files.

    9. Click Data Files Only and OK to remove checkmarks from the left of data files only.

    10. Click Application Files Only and OK to remove checkmarks from the left of application files only.
