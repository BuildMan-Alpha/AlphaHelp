UNDECLARE

Syntax

UNDECLARE Function_Name

Argument

Description

Function_Name

The name of the function.

Description

UNDECLARE removes an externally defined function (for example a Windows API function).

Example

Declare some structures to use - a rectangle and a 1024 character buffer.

declare user32 GetWindowRect LL(rect)

declare user32 gwt@GetWindowText LL(text) as L

undeclare GetWindowRect

undeclare gwt@GetWindowText
