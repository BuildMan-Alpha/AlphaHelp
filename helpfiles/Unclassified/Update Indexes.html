Update Indexes

Overview

The Update Indexes action updates the indexes for specified tables.

Procedure

  1. To the right of Update indexes for which tables select either:

    • "All tables in the Database". This will update the indexes for all tables EXCEPT the ones you select in the box.

    • "Selected tables". This will update ONLY the tables you select.

  2. Check boxes to the left of selected tables.

  3. Click Next >.

  4. Optionally modify the default comment which describes the action.

  5. Click Finish.
