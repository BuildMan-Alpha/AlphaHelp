Working with Rich Text Objects

A Rich Text Object is a formatted text control that you can display on a layout.

Adding Rich Text Objects to a Layout

To create a Rich Text Object, you can either:

  • Use the Rich Text Tool on the Toolbox.

  • Select Object > New... > Rich Text.

Once the text has been placed, you can set the following properties with the Properties Toolbar, or by right-clicking on the object to open the Properties dialog box.

  • Frame background color

  • Frame size and position

  • Border style, thickness, and color

  • Text alignment, font, size, color, character attributes (bold, italic, underline), and style
