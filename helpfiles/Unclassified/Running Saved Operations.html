Running Saved Operations

Note : The Running Saved Operations topic describes a technique available to Alpha Five database users.

You cannot run a saved operation directly from an A5W page or web component. However, you can copy the Xbasic code from an operation and place it into a script in an A5W page.

  1. Right click on the operation in the Control Panel and select view Xbasic.

  2. Copy this Xbasic and paste it into a page script.

  3. Remove any code that would display a user interface (e.g. a message box), since that will cause the server to hang.
