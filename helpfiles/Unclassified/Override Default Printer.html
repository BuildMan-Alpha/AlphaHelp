Override Default Printer

Overview

The Override Default Printer action allows you to set any available printer to be the default printer.

Note : This action will override all subsequent print settings, even if a specific printer is requested.

Procedure

  1. Select the new default printer from the Printer name list. Specify the printer name exactly as it appears in the list when you select the Start > Settings > Printers command in Windows.

  2. Select a printer port from the Specify the printer port list if the port is different from the port specified by the printer entry.

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
