Specify Order Dialog

The <span class=Screen>List View Genie</span> and <span class=Screen>ADO Import Genie</span> use the Specify Order dialog to create SQL SELECT statements.

  1. Enter an expression in the Order window.

  2. To insert a field name into the expression, position the cursor in the order expression, select the field in the Available Fields list, and click .

  3. Click OK to continue.
