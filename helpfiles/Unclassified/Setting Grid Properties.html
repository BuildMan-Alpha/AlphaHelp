Setting Grid Properties

This topic continues to describe the process of creating a grid component.

  1. Display the Grid Properties page. The resulting dialog boxes set additional display and formatting properties of the component.

  2. Click Layout Options > Style name > > Select style and select a Cascading Style Sheet to apply to the grid.

Note : CSS style sheets delivered with Alpha Five Version 6 may be used with Alpha Five Version 8 grid and dialog components, but not with Navigation System Components.

  1. Optionally, click Layout Options > Style name > > Edit Style to modify the style sheet in the Webform Style Builder.

  2. Optionally, click Save Page to create a named .A5W page containing this component.

    1. Optionally, select Automatically inherit page background from the component style.

    2. Optionally, check Open page in HTML editor after saving.

    3. Click Save Page.

    4. Select the Web Project that will contain the page from the Web Project drop-down list box.

    5. Enter the page name in the File name field.

    6. Click Save.

  1. Click to preserve your work.

  2. If you do not need to define Search or Detail View properties, click Browser to see the grid running in your browser.

Quick Search

The Quick Search (V7) feature is an alternative to putting a Search Part on a Grid. It takes up less real estate than the Search Part, but does not have all of its options. There are some features, however that exceed the capabilities of a Search Part. In particular, you can auto-submit the search when a value in a dropdownbox, radio button or check box control changes. For example, you might have a Quick Search on a State field. You would have a dropdownbox showing all states. When the user makes a new selection in this dropdownbox, a new search runs immediately, without the user having to click a Search button.

Refer to Defining a Quick Search for instructions.

NOTE : If you include a Search Part in addition to a Quick Search, then the Search Part will perform searches within the subset of records selected by the Quick Search.

Search style

V7

Specifies how the data should be searched. The options are:

  • "Wildcard" - search for an exact match, ignoring case and length (wildcard characters * and ? are allowed).

  • "Exact" - search for an exact match, ignoring case and length.

  • "Contains" - search for value anywhere in field. There is no need to use wildcards in the search string.

  • "Starts with" - search for the value at the beginning of field. There is no need to use wildcards in the search string.
