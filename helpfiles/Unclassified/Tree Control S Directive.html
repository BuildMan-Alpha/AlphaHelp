Tree Control S Directive

Syntax

%S= [ B ][ L ][ R ][ X ] %

Argument

Type

Description

B

C

Optional. Indicates that open/close boxes should be displayed.

L

C

Optional. Lines should be drawn on the tree control.

R

C

Optional. Open/close boxes should be drawn at the root level.

X

C

Optional. Causes the dialog to open with the tree control fully expanded.

Description

The formatting string %S=BLR% is used to specify that a tree control should be displayed with the familiar boxes and lines.

dim data10 as C

data1 = "a.1.A"

data2 = "a.1.B"

data3 = "a.2.A"

data4 = "a.2.B"

data5 = "b.1.A"

data6 = "b.1.B"

data7 = "b.2.A"

data8 = "b.2.B"

result=ui_dlg_box("Simple Tree",<<%dlg%

{ysize=2}

[%S=BLR%.32,10node^

;

.32node

%dlg% )
