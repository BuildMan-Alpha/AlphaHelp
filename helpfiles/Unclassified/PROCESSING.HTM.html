PROCESSING.HTM

Purpose

The PROCESSING.HTM is a frameset used to show a header while a page within the frameset is processing.

Description

PROCESSING.HTM is a frameset than contains the PROCESSHEADER.A5W page in the topFrame and CHECKOUTCALC.A5W pages in the mainFrame.

Note : Do not open this page in the WYSIWYG tab of the HTML Editor and then save it. This will result in the loss of Xbasic code, which the underlying Microsoft ActiveX control does not properly support.

Links

The CHECKOUTCOMP.A5W page calls the PROCESSING.HTM page.
