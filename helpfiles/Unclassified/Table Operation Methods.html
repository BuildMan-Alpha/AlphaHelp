Table Operation Methods

Prefix the following methods with the pointer. Use the TABLE.OPEN(), TABLE.CREATE_END(), and <OBJECT>.TABLE_GET() methods to return the pointer.

Function

Description

.APPEND()

Appends the records of to another table.

.COPY()

Copies all or some of the fields from or set to a new table.

.CROSSTAB()

Cross-tabulates the records of a table or set, creating a new table.

.EXPORT()

Exports the records of a table or set to a file of a different format.

.IMPORT()

Import the records of a file of a different format into a new table file.

.INTERSECT()

Creates a new table by merging the records from with matching records from a Secondary table. The new table contains only those records from that have a matching record in the Secondary table

.JOIN()

Create a new table by merging the records from with matching records from a Secondary table. The new table contains every record from , whether or not a matching record is found in the Secondary table.

.POST()

Replaces, adds, or subtracts field values in using field values from a Transaction table.

.SUBTRACT()

Creates a new table by merging the records from with records from a Secondary table. The new table contains one record for each record in that has no matching record in the Secondary table.

.SUMMARIZE()

Summarizes the records of , creating a new table to store the result.

.UPDATE()
