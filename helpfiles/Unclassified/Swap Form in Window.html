Swap Form in Window

Overview

The Swap Form in Window action opens a different form for the current record in a new window. You can specify the target form in advance or prompt the user at run-time.

A swap action differs from an open action in that the target form must be from the same table or set. The target form displays the focused record and closes the source form.

Procedure

  1. Select whether to specify the target form in advance or prompt the user at run-time. The options are:

    • "Prompt user for name of target Form at run-time"

    • "Specify target Form now"

  2. If you selected "Specify target Form now", select the form in the Form Name field.

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
