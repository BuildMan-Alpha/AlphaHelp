Zip Set

To collect one or more components of a table into a ZIP file:

  1. Display the Table/Sets tab of the Control Panel.

  2. Select the table of interest, right click, and select Zip... to display the Zip Set dialog.

  3. On the Zip File Name tab click to navigate to the folder that will contain the zip file. Enter the name of the zip file and click Open.

  4. Display the Options tab.

  5. Optionally, clear Include Data Dictionary if you do not want to include Field Rules, Layouts, and Operations defined for the tables in the set.

  6. Optionally, clear Include Indexes, if you do not want to include indexes. Indexes can make a zip file significantly larger. The indexes can be recreated without loss of information later.

  7. Optionally, clear Compress set before zipping. Compression removes deleted records from the database, resulting in a smaller zip file.

  8. Optionally, display the Files tab to see which set components will be included in the zip file.

  9. Click Create Zip File to continue or click Cancel to quit without taking any action.
