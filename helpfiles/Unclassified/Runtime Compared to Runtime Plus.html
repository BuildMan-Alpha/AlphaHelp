Runtime Compared to Runtime+

The Alpha Five Runtime and Alpha Five Runtime+ products both allow you to run your Alpha Five applications. Runtime+ allows you to add ad-hoc reporting to your applications. The following table summarizes the differences between the two versions of the Runtime.

Feature

Runtime

Runtime+

User can create new Reports

No

Yes

User can create new Letters

No

Yes

User can create new Labels

No

Yes

User can edit existing Reports

No

Yes

User can edit existing Letters

No

Yes

User can edit existing Labels

No

Yes

With Runtime+ you (i.e. the developer) can control which of your Reports, Letters and Labels can be edited by the users of your application. You can also control which of the Tables and Sets in your application can be used when users create new Reports, Letters and Labels.
