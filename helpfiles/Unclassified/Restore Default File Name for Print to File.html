Restore Default File Name for Print to File

Overview

The Restore Default File Name for Print to File action restores the default file name for print to file for your computer. You should not need to use it unless you previously used the Override Default File Name for Print to File action.

Procedure

  1. Optionally modify the comment that describes the action.

  2. Click Finish.
