V5 to V6 Application Conversion Issues

Using Parentheses in Array References

Alpha Fiver version 6 no longer supports using ( ) in an array reference.

Example

dim names[4] as C

names[1] = "George"

names[2] = "Washington"

ui_msg_box("my name ", names(2)) ' incorrect

ui_msg_box("my name ", names[2]) ' correct
