Enabling Allow Shrinking

This feature allows the control to be reduced in size when printing the form.

  1. Select the control, choose Object > Properties and display the Setup tab.

  2. Check Allow Shrinking.

Enabling Always Modeless

This feature allows the user to input and edit values without putting the form into edit mode.

  1. Select the control, choose Object > Properties and display the Setup tab.

  2. Check Allow Modeless.

Enabling Always on Top

This feature keeps the control on top of other controls. This feature is useful when controls may be layered.

  1. Select the control, choose Object > Properties and display the Setup tab.

  2. Check Always on Top.

Setting the Display Font

This feature sets the various characteristics of the font that the control uses to display its data. The Sample area at the bottom of the Field Properties dialog previews your selection.

  1. Select the control, choose Object > Properties and display the Font tab.

  2. Optionally, select a font from the Face list.

  3. Optionally, select or enter an integer value for the Size of the font.

  4. Optionally, check the Bold, Italic, Underline, and Strikeout style check boxes.

  5. Optionally, select a color from the Color list.

Setting Text Alignment

By default the text in a data field is top and left aligned.

  1. Select the control, choose Object > Properties and display the Format tab.

  2. Set the Horizontal Alignment by clicking Left, Center, or Right.

  3. Set the Vertical Alignment by clicking Top, Center, or Bottom.

  4. Optionally, check Word Wrap to allow text to use more than one line. You must resize the control to make the extra lines visible.

Setting the Display Format

The number of characters displayed on the form is not necessarily the same as the length of a field.

  1. Select the control, choose Object > Properties and display the Format tab.

  2. Enter or select a value in the Width field to set the number of display characters. This value includes the decimal point and numbers to the right of the decimal point.

  3. In the case of numeric fields, enter or select a value from the Decimal Places field.

  4. Optionally, select a Format Style. Refer to Display Formats.

Setting the Position and Size

Before setting a control's dimensions, you may want to change your unit of measurement settings. You may also set the control's position and size interactively. Refer to Sizing Objects.

  1. Select the control, choose Object > Properties and display the Dimensions tab.

  2. Enter or select a value in the Left field to set the horizontal location of the control on the form.

  3. Enter or select a value in the Top field to set the vertical location of the control on the form.

  4. Enter or select a value in the Width field to set the width of the control on the form.

  5. Enter or select a value in the Height field to set the height of the control on the form.

Enabling Dynamic Control Resizing

Alpha Five Version 6 allows you to stretch and move form controls as  the form is resized. The key benefit of this feature is that you can create forms that will look well designed regardless of the screen resolution (e.g. 800 x 600 or 1024 x 768 pixels).

Note : The form's Window properties must have the following two settings in the size area.

  • Width = Use Form's Width

  • Height = Use Form's Height

  1. Click the horizontal and vertical lines of the anchor to turn them on and off. There must always be one vertical (top or bottom) and one horizontal (left or right) anchor point turned on.

Changing the Field Description

  1. Select the control, choose Object > Properties and display the Setup tab.

  2. Optionally, change the Object Name. By default, the object name is the same as the table data field name. It is often useful to change this name to distinguish it from the underlying data field.

  3. Optionally, change the data Field that is bound to the control.

    1. Click to display the Field Select dialog box.

    2. Select the table from the Table list.

    3. Select the field from the Available Fields list.

    4. Click OK to continue or Cancel to discard your inputs.

  4. Optionally, change the variable that is bound to the control.

    1. Click to display the Field Select dialog box.

    2. To use an existing variable, select "Var" from the Table list.

    3. Optionally, click Add Variable to display the Add Variable dialog box. Specify the Scope, Data Type, and Name of the new variable and click OK.

    4. Select the variable from the Available Fields list.

    5. Click OK to continue or Cancel to discard your inputs.

  5. Optionally, change the calculated field that is bound to the control.

    1. Click to display the Field Select dialog box.

    2. To use an existing variable, select "Calc" from the Table list.

    3. Select the calculated field from the Available Fields list.

    4. Click OK to continue or Cancel to discard your inputs.

  6. Optionally, change the system field that is bound to the control.

    1. Click to display the Field Select dialog box.

    2. To use an existing variable, select "System" from the Table list.

    3. Select the system field from the Available Fields list.

    4. Click OK to continue or Cancel to discard your inputs.

  7. Optionally, change the Field-Type. Refer to Selecting a Field Data Control to understand the characteristics of the different options.

    1. Click to display the different control formats.

    2. Depending on the type of control selected, display the Choices tab and enter required information.

Enabling and Disabling Miscellaneous Properties

  1. Select the control, choose Object > Properties and display the Setup tab.

  2. Optionally, check Read Only if you do not want the user to be able to change the value of the field.

  3. Optionally, clear Tab Stop if you do not want the user to be able to tab to the field.

  4. Optionally, check Always on Top if you want the field to always be on top of any other layered controls.

  5. Optionally, check Always Modeless if you want the user to be able to edit the field without putting it in change mode.

  6. Optionally, check Hide if you want to make the field invisible.

  7. Optionally, check Disable if you do not want the user to be able to edit or select the data in the field.

  8. Optionally, check Allow Shrinking if the field can be made smaller during printing.

  9. Optionally, check Don't Spell Check if you do not want the field to be eligible for spell checking.

Specifying the Right-Click Menu

  1. Select the control, choose Object > Properties and display the Menus/Toolbars tab.

  2. Click to select an existing toolbar from the Right-Click Menu list.

  3. Optionally, click Set All Fields to apply this toolbar to all fields on the form.

Specifying a Help Topic

When the user presses F1, you have the option of displaying a help topic from an Alpha Five help file or a compiled help file in the .CHM or .HLP formats.

  1. Choose Object > Properties and display the Help Entry tab.

  2. Check Use Custom Help.

  3. Make a selection from the Help Source File list. The options are:

    • "Standard Windows Help File (.hlp file)"
    • "Compiled Windows Help File (.chm file)"
    • "Alpha Five Help File"

  4. If you selected either "Standard Windows Help File (.hlp file)" or "Compiled Windows Help File (.chm file)", refer to Linking to a External Help File Topic.

  5. If you selected "Alpha Five Help File", refer to Creating Alpha Five Help Text.

Providing Bubble Help

Alpha Five displays Bubble Help when the cursor is over an object on a layout. 

Bubble Help shows the object name and object type. With field objects, it shows the name of the field that the object is bound to, the display size of the object, and the size of the field. You can toggle Bubble Help on or off with the View > Object Bubble Help command.

  1. Choose Object > Properties and display the Help Entry tab.

  2. Optionally, check Free form XDialog if you would like to add images or other Xdialog controls to the bubble help.

  3. If you checked Free form XDialog :

    1. Enter Xdialog code into the  Bubble Help Text field.

    2. Click Preview to see how the bubble help will appear.

  4. If you did not check Free form XDialog :

    1. Optionally, enter the help text into the Bubble Help Text field.

    2. Optionally, click Edit... to display the Bubble Help dialog box.

    1. Enter the help text into the Bubble Help Text field.

    2. Optionally, change the Font, Font Size,  Bold,  Italic, and  Underline field settings.

    3. Optionally, enable word wrap by checking Word Wrap Text?.

    4. Optionally, change the default line length by entering or selecting a new value in the Wrap Size field.

    5. Click Preview to see how the bubble help will appear.

  1. Optionally, click Apply to all to set this as the bubble help for all fields on the form.

  2. Click OK to save your changes or Cancel to exit.

Creating Flyover Effects

You can temporarily change many of a control's properties when the mouse moves over it. The control's properties revert to their normal state when the mouse moves away from it.

  1. Choose Object > Properties and display the Setup tab.

  2. Check Fly-over Effects and click Set Fly-over Appearance to display the Fly-Over Properties dialog.

  1. Expand the properties list as necessary and select the property you want to change.

  2. Select the fly-over value from the Property value list at the bottom of the dialog.

  3. Click OK to apply your changes or Cancel to quit without making any changes.
