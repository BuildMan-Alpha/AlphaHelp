Prompt for a Filename

Overview

The Prompt for a Filename action will prompt the user for a filename and store it in a variable.

Procedure

  1. Enter the letters of the filters(s) which you want the user to see when selecting a filename.

  2. Enter the name of the variable that will receive the value that the user enters in the Variable Name field.

  3. Optionally, enter the title of the dialog box in the Dialog Title field. Optionally, click the button to open the <span class=Screen>Expression Builder</span>.

  4. Optionally, enter a default value in the Default Filename field.

  5. Make a choice in the File selected list:

    • Must Exist

    • Must Not Exist

    • Either

  6. Click the Test Dialog button to see how it will appear.

  7. Click Next >.

  8. Optionally modify the comment that describes the action.

  9. Click Finish.
