Run a Script

Overview

The Run a Script action runs an Xbasic script.

Procedure

  1. Select the script to run from the Name of script to run list.

  2. Select one of the Run Options. The script run mode options are:

    • "Standard"

    • "Local"

    • "Spawn"

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
