Updating Selected Records in a Table

When viewing a form or browse, you may update selected records in the current table using an Update Operation or when viewing a table through a form or browse.

To update records:

  1. Select Records > Update Selected Records... .

  1. Optionally, click to clear the fields of the dialog box.

  2. Select the field to update from the Fieldname list.

  3. Enter an expression that defines the field's new value in the Update Expression field. Optionally, click to use the <span class=Screen>Expression Builder</span> to help you define the expression.

  4. Repeat steps 3 and 4 up to four more times.

  5. Optionally, click to view the Xbasic code generated by this procedure.

  6. Click OK to proceed with the update or Cancel to quit without updating.
