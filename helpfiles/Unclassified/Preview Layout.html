Preview Layout

Overview

The Preview a Layout action allows you to print, preview, email, save, or fax a layout in a variety of formats.

Procedure

  1. On the Settings tab make a selection from the Select Layout Type list. The options are:

    • "Report"

    • "Label"

    • "Letter"

    • "Form"

    • "Browse"

  2. Optionally, if the action is running from a button on a layout, check Only show Layouts for current Table/Set to filter the contents of the Layout Name list.

  3. Optionally, check Display 'Print/Preview' prompt at runtime? check box if you would like the user to be able to choose between previewing and printing the layout. This option displays the Print Method dialog box.

  4. Select a layout from the Layout Name list.

  5. If you select "" form the Layout Name list, make a selection from When prompting, show layouts from list. The options are:

    • "Another Table/Set"

    • "Current Table/Set"

  6. If you selected "Another Table/Set", select a table or set from the Select Table/Set Name list.

  7. Click the Options tab.

  8. Make a selection from the Print Method list. The options are:

    • "Preview"

    • "Print"

    • "Email as an attachment"

    • "Save as a file"

    • "Archive"

Previewing

If you selected "Preview":

  1. Make a selection in the Window Style list. The options are:

    • "Modeless Window". Allows you perform other actions while the preview window is open.

    • "Modal Window". Requires you to close the preview window before accessing other windows.

  2. Continue with ((|#Conclude|Conclude)).

Printing

If you selected "Print":

  1. Optionally check the Override Printer Settings? check box.

  2. Click Set Printer Override Settings to display the Override Printer Settings dialog box and change your printer settings.

  3. Continue with ((|#Conclude|Conclude)).

Emailing

If you selected ""Email as an attachment":

  1. Make a selection from the File Format list. The options are:

    • "Adobe Acrobat PDF File". Can be displayed with Adobe Acrobat Reader.

    • "Dynamic HTML File". Can be displayed with a browser.

    • "Rich Text Format File". Can be displayed with Microsoft Word or Microsoft Wordpad.

    • "Plain Text File". Can be displayed with Microsoft Notepad and many other text editors.

  2. Optionally select Override default options for Email attachments and click Options to display the <span class=Screen>Print Options Dialog</span>.

  3. Optionally, enter a text string or an expression in the Subject, CC, Bcc, and Message fields. Use the button to open the Expression Builder if you want to use build content containing field values.

  4. Optionally, click the Edit button to enter a longer text message.

  5. Select the email program to use in the Email client list.

  6. Select your email client from the Email client list.

  7. Continue with ((|#Conclude|Conclude)).

Saving to File

If you selected ""Save as file":

  1. Make a selection from the File Format list. The options are:

    • "Adobe Acrobat PDF File". Can be displayed with Adobe Acrobat Reader.

    • "Dynamic HTML File". Can be displayed with a browser.

    • "Rich Text Format File". Can be displayed with Microsoft Word or Microsoft Wordpad.

    • "Plain Text File". Can be displayed with Microsoft Notepad and many other text editors.

  2. Optionally select Override defaults for Save options and click Options to display the <span class=Screen>Print Options Dialog</span>.

  3. Optionally, enter a name into the Filename field. If you leave the filename blank, Alpha Five will create a file with the same name as the layout in the same folder as the current database.

  4. Optionally, check the Open file with associated application? check box. This specifies whether Alpha Five should open the file with its associated application once it is created.

  5. Continue with ((|#Conclude|Conclude)).

Archiving

If you selected "Archive", continue with ((|#Conclude|Conclude)).

Conclude

  1. Click Next >.

  2. Optionally, define record selection criteria.

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
