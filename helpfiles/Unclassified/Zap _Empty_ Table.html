Zap (Empty) Table

Overview

The Zap (Empty) Table action empties all records from a table.

Note : Because a table must be closed before being emptied, you cannot use the Zap (Empty) Table action to delete the records from the current form's table or set.

Procedure

  1. In the Select table that you wish to zap field choose a table from your database.

  2. Optionally, check Prompt for confirmation before Zapping table?  Checking this will ask the user whether or not they want to zap the table.

  3. Click Next >.

  4. Optionally modify the default comment which describes the action.

  5. Click Finish.
