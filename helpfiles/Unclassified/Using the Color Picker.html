Using the Color Picker

The Color Picker allows you to select very subtle colors and to create colors with RGB (Red, Green, Blue) or HLS (Hue, Saturation, Luminance) definitions.

Using the HLS Tab

  1. Optionally, select a color from the System palette.

  2. Optionally, select a color from the Custom palette.

  1. Optionally, specify a new color:

    1. Enter a number from 0 to 255 in each of the H, L, S fields. The Hex field shows you the value that will be inserted into the HTML file.

    2. Enter a number from 0 to 255 in each of the R, G, B fields. The Hex field shows you the value that will be inserted into the HTML file.

    3. Optionally, click Add To Custom palette. This will add the new color to the Custom Color list.

  2. Click OK to apply the color or Cancel to discard your inputs.

Using the Relative Tab

The Relative tab is new with Version 8.

  1. Optionally, select a color from the System palette.

  2. Optionally, select a color from the Custom palette.

  1. Optionally, specify a new color. There are several techniques available in the Input Type list.

    • "One Color Hue Adjustment"

    • "Two Color Blend"

    • "Custom Palette"

  2. If you selected "One Color Hue Adjustment" in step 3:

    1. Click the single color square at the top of the array to select another color.

    1. Adjust the color array by selecting an entry from the Relative list. The options are:

      • "Luminance"

      • "Saturation"

      • "Cooler"

      • "Warmer"

    2. Select a color in the color array and click Add.

  1. If you selected "Two Color Blend" in step 3:

    1. Click the two color squares at the top of the color array to select two colors.

    2. Adjust the color array by selecting an entry from the Relative list. The options are:

      • "Luminance"

      • "Saturation"

      • "Cooler"

      • "Warmer"

    1. Select a color in the color array and click Add.

  2. If you selected "Custom Palette" in step 3:

    1. Click any combination of color squares at the top of the color array to select colors.

    2. Adjust the color array by selecting an entry from the Relative list. The options are:

      • "Luminance"

      • "Saturation"

      • "Cooler"

      • "Warmer"

    1. Select a color in the color array and click Add.
