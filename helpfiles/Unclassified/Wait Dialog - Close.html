Wait Dialog - Close

Overview

The Wait Dialog - Close action closes a wait dialog started by a Wait Dialog - Open action.

Procedure

  1. Enter a name for the wait dialog created in the Wait Dialog - Open action in the Identifier field.

  2. Click Next >.

  3. Optionally, enter a comment that describes the action.

  4. Click Finish.
