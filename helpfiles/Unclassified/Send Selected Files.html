Send Selected Files

Overview

The Send Selected Files action lets you select files to send by email as attachments. You can specify specific files in the Genie, or prompt the user at run-time.
Send Selected Files options.

Procedure

  1. In the first list box select either:

    • "Prompt for file names at runtime". Allows the user to specify the files to send using the Select Attachments Dialog Box.

    • "Specify the file names now". Allows you to specify the files to send using the Select Attachments Dialog Box.

  2. Optionally, enter the message subject into the Subject field. You can also use the Expression Builder to create an expression that generates one or more email addresses.

  3. Enter the recipient's email address or a comma delimited list of email addresses. You can also use the Expression Builder to create an expression that generates one or more email addresses.

  4. Optionally, fill in the CC and Bcc fields. Email addresses in the CC field will receive a copy of the email. Their addresses will appear in the email header. Recipients in the Bcc field will also receive a copy, but their addresses will not appear in the header. You can also use the Expression Builder to create an expression that generates one or more email addresses.

  5. In the Send using field select either:

    • "Built-in Alpha Five email client"

    • "Default email client (e.g. Eudora, Outlook, etc.)"

If you select "Built-in Alpha Five email client", the user has the opportunity to make edits before sending the. Otherwise, the email is sent automatically without further user interaction.

  1. If you selected "Built-in Alpha Five email client", you have the option to further specify the behavior of the action. You can select the type of interaction that results from the user initiating the action.

    • "Open Alpha Five email client as a modal dialog box"

    • "Open Alpha Five email client as a modeless dialog box"

    • "Don't open the Alpha Five email client - just send message then display confirmation"

    • "Don't open the Alpha Five email client - just send message without confirmation dialog"

If you selected "Built-in Alpha Five email client", the user has the opportunity to make edits before sending the. Otherwise, the email is sent automatically without further user interaction.

  1. Click Next >.

  2. Optionally modify the comment that describes the action.

  3. Click Finish.
