Run One-Step Backup

Overview

The Run One-Step Backup action runs a previously saved One-Step backup operation, created with the Backup Database dialog.

Procedure

  1. If you wish Alpha Five to automatically close all open windows to ensure that all files are backed up, check the Close all open Windows before displaying the Backup Dialog box check box.

  2. Click Next >.

  3. Optionally modify the default comment which describes the action.

  4. Click Finish.
