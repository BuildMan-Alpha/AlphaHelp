Spell-Check Field(s)

Overview

The Spell-Check Field(s) action spell checks specified fields and records.

Procedure

  1. Make a selection in the Set the insertion point in a field in which window? field. This field will be grayed out unless this action is based on a Form or Browse. The choices are:

    • "Current Form or Browse Window".

    • "Another Form or Browse Window".

  2. In the Window Name field select either:

    • "Forms"

    • "Browses"

  3. Select one of the listed forms or browses.

  4. In the Object Name field select a text field on the form or browse. Click the Value icon to select the form or browse that contains the field.

  5. Select a value in the Spell check which field(s) and records?. The possible values are:

    • "Current field"

    • "A field you specify now"

    • "All fields in the current record"

    • "All records in the form or browse window"

  6. If you select "All fields in the current record", click the Value icon to select the field from the Select the field that you want to spell check field.

  7. Click Next >.

  8. Optionally modify the comment that describes the action.

  9. Click Finish.
