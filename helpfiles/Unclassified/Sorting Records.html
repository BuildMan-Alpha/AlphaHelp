Sorting Records

There are two ways to specify the records to print and the order in which they print. You can either save the criteria in the label layout, or prompt the user at print time. These options are not mutually exclusive, so you can choose to save order and selection criteria, and then make changes to the criteria just before the Labels print.

To save order and selection criteria in a Label layout:

  1. Select Label > Properties.

Selecting Records to Print

  1. When the Label Properties dialog box appears, click the Records tab. In the Order Expression box, enter an order expression. In the Filter Expression box, enter a filter expression.

  2. The Label Properties dialog box also lets you choose only marked or unmarked records, by clicking the Marked or Unmarked radio buttons.

  3. For the Additional Record Order and Filter Criteria, you can choose whether to base record selection on the current selection of records, or prompt the user at print time.

The Use filter/order from Query button lets you select filter/order criteria from a previously defined query.
