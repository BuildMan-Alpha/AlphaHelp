Unhiding the Alpha Five Application

Every now and then someone accidentally hides Alpha Five with A5.HIDE()in their AUTOEXEC script. To recover from this problem, follow these steps.

  1. Create a new empty database.

  2. Go to the Alpha Five Control Panel.

  3. Select File > Database Properties.

  4. Display the Libraries tab.

  5. Click Add and select the library (.ALB file) from the problem database. The scripts/functions from the problem database will now be shown in the code tab of the current database.

  6. Edit the AUTOEXEC file of the problem database.

  7. Unattach the library the library from the empty database.

Supported By

Alpha Five Version 5 and Above
