Refresh Passive Link Table(s)

Overview

The Refresh Passive Link Table(s) action displays the causes Alpha Five to erase the contents of the table, then retrieve the table's data from its SQL data source.

Procedure

  1. In the Refresh list select either:

    • "All Passive Link Tables"

    • "Selected Passive Link Tables"

  2. Optionally, clear Run silently to have Alpha Five warn you before refreshing a table.

  3. If you chose "Selected Passive Link Tables" in step 1, place checks next to the names of the tables you want to refresh.

  4. Click Next >.

  5. Optionally modify the default comment which describes the action.

  6. Click Finish.
