Set Modeless Data Entry Style

Overview

The Set Modeless Data Entry Style action changes data entry style of a specified Form/Browse to Modeless.

If the data entry style is set to Modal, the user must first explicitly select the Change Record or Enter New Record command before making any changes to a record or entering new records.

By contrast, if the data entry style is Modeless, then the user can edit any field on the form, or enter new records without first selecting Change or Enter mode.

While Modeless data entry can be convenient, it allows for accidental editing of fields, and so you may prefer to set the data entry mode to modal.

Note : You can set the default data entry style of a form or browse window by editing the layout's properties.

Procedure

  1. Make a selection in the Set the Data Entry style to Modal in which Form/Browse field. This field will be grayed out unless this action is based on a Form or Browse. The choices are:

    • "Current Form or Browse Window".

    • "Another Form or Browse Window".

  2. If you selected "Another Form or Browse Window", in the Window Name field select either:

    • "Forms"

    • "Browses"

  3. Select one of the listed forms or browses.

  4. Click Next >.

  5. Optionally modify the comment that describes the action.

  6. Click Finish.
