Switch to Another (non-Alpha Five) Window

Overview

The Switch to Another (non-Alpha Five) Window action sets focus to another non-Alpha Five window. If you want to send keystrokes to the target window after it gets focus, use the Send Keys action.

Procedure

  1. Specify how to select the target window. The options are:

    • "Specify Window name now"

    • "Show list of windows at runtime"

  2. If you selected "Specify Window name now", pick the window from the Window Name field.

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
