Using the Desktop Server : A5DesktopServer.exe

Setting configuration options:

The configuration is downloaded with the executables and then read from A5DesktopServer.cfg when A5DesktopServer.exe starts.

A sample of the file contents is:

Port=16136

webroot= ..\WebSite

ApplicationName=Alpha Sample Application

ApplicationDescription=Alpha Five Sample Applications for Oracle Mobile Server

AddToStartMenu

CreateDesktopIcon

where recognized entries are:

Entry

Description

Port

The port to use on the machine to listen to requests for the web application.

WebRoot

The directory to use for the web pages that make up the application.

ApplicationName

The name to be displayed on the desktop when creating a shortcut.

ApplicationDescription

The description to be used in tool tips for the desktop shortcut and the system tray.

AddToStartMenu

The entry required to add A5DesktopServer to the Windows start menu. Reboots will restart the server.

CreateDesktopIcon

The entry required to have a shortcut created on the desktop to start the application. This shortcut will start Internet Explorer and navigate to the appropriate page.

Supported By

Alpha Five Version 8 Enterprise Edition

Limitations

Desktop applications only.
