Restore Database Dialog Box

Overview

The Restore Database Dialog Box action displays the Restore Database dialog box, which asks the user to specify a folder and to select a backup zip file from those that the folder contains.

Procedure

  1. If you wish Alpha Five to automatically close all open windows to ensure that all files are restored, check the Close all open Windows before displaying the Backup Dialog box check box.

  2. Click Next >.

  3. Optionally modify the default comment which describes the action.

  4. Click Finish.
