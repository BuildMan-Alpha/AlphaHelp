Set the Values of One or More Variables

Overview

The Set the Values of One or More Variables action creates one or more variables. It also sets the value of multiple new or existing variables.

Note : Use ((Set Variable_s_ to Field Values _Fields on a Form_|Set Variable(s) to Field Values (Fields on a Form))) to set a variable to the a value on a form.

Procedure

  1. Click either Add New Variable or Set Value or Existing Variable.

  2. If you clicked Add New Variable :

    1. Enter the Variable Name.

    2. Select the Scope of the variable. The options are:

      • "Shared" - the variable is available to scripts based on the underlying form or browse

      • "Local" - the variable is available only to the script and the underlying form or browse

      • "Global" - the variable is available to other scripts

    3. In the Value field enter the data or expression that defines the variable's value. Optionally, click the button to open the <span class=Screen>Expression Builder</span>.

    4. Click OK to continue or Cancel to discard your input.

  3. If you clicked Set Value or Existing Variable :

    1. Select the variable from the Set Value of Variable dialog box.

    2. In the Value field enter the data or expression that defines the variable's value. Optionally, click the button to open the <span class=Screen>Expression Builder</span>.

    3. Click OK to continue or Cancel to discard your input.

  4. Click Next >.

  5. Optionally modify the comment that describes the action.

  6. Click Finish.
