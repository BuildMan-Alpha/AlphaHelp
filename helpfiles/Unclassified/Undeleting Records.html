Undeleting Records

Note : You may want to unmark all records in the table before using this procedure. This will allow you later to identify the records that you have undeleted.

To restore records marked as Deleted:

  1. Display the Tables/Sets tab of the Control Panel.

  2. Select a table, then click Table > Utilities > Undelete Records... to display the Undelete Records dialog box.

  3. Optionally, click Range of records, if you know the starting and ending record numbers of the records you would like to restore.

    1. Enter the Start Record number.

    2. Enter the End Record number.

  4. Optionally, check Mark records that are deleted? This allows you to identify the records that you have restored.

  5. Click OK to undelete records or Cancel to quit without taking any action.
