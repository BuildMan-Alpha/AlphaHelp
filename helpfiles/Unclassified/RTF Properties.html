Properties

The table below summarizes the RTF object properties. Use of the following properties requires the pointer returned by the RTF.CREATE() method.

Property

Description

.BINARY_TEXT

Gets the binary value of the RTF object.

.PLAIN_TEXT

Gets the plain text in the RTF object.

.RTF_TEXT

Gets the ASCII encoded text in the RTF object.

.STYLE

Gets a pointer to the array containing the style names.
