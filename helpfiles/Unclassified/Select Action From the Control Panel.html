Select Action From the Control Panel

Overview

The Select Action From the Control Panel action allows an object to run one of the actions listed in the Control Panel 's menu.

Procedure

  1. Click Select Action... to display the Control Panel menus.

  2. Navigate the menus and select a specific menu function.

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
