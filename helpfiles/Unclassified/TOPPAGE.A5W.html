TOPPAGE.A5W

Purpose

The TOPPAGE.A5W page contains the page header.

Description

TOPPAGE.A5W contains the HTML for the header. Other pages include TOPPAGE.A5W. We saved this page as .A5W instead of .HTM to allow the code section for comments, and also allow for later dynamic content build by xbasic. It could be saved as .HTM with the <%a5...%> section removed. Since this page is always included, it does not need any <html> or <body> tags.

Note : Do not open this page in the WYSIWYG tab of the HTML Editor and then save it. This will result in the loss of Xbasic code, which the underlying Microsoft ActiveX control does not properly support.
