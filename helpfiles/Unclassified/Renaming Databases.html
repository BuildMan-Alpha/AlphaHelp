Renaming Databases

A database consists of several files. They have the same prefix as the database, and have the extensions: .ALB, .ALX, .ALM, and .ADB

To rename a database

  1. Use the Windows Explorer to rename or delete the database's files. For example, if you wanted to change a database's name from NEW_SALES to OLD_SALES, you would rename the files as follows:

    • "NEW_SALES.ALB" to "OLD_SALES.ALB"

    • "NEW_SALES.ALX" to "OLD_SALES.ALX"

    • "NEW_SALES.ALM" to "OLD_SALES.ALM"

    • "NEW_SALES.ADB" to "OLD_SALES.ADB"

  2. If there are any libraries associated with the old database name.

    1. Select File > Database Properties.

    2. Display the Libraries tab.

    3. Select each library associated with the old database name and click Remove.
