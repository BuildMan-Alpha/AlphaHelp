Wait Dialog - Set Percentage

Overview

The Wait Dialog - Set Percentage action is used with a Wait Dialog - Open action of style "Percent" to indicate the progress of a process.

Procedure

  1. Enter a name for the wait dialog created in the Wait Dialog - Open action in the Identifier field.

  2. Enter the Step number.

  3. Enter the Total number of steps.

  4. Optionally, enter the Title of the dialog.

  5. Optionally, enter a message to the user in the Top Message field.

  6. Optionally, if the number of message lines is greater than 1, enter a message to the user in the Bottom Message field.

  7. Click Next >.

  8. Optionally, enter a comment that describes the action.

  9. Click Finish.
