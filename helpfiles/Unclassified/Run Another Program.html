Run Another Program

Overview

The Run Another Program action executes another program outside of Alpha Five. You can specify the program to run now or prompt the user at run-time.

Procedure

  1. In the Program Name field enter the path to the program executable. Optionally, click the Folder icon to browse for the program executable.

  2. Optionally, enter command(s) you wish to pass to the program in the Command Line field.

  3. Select the appearance of the window the program opens in the Window Mode field. The options are:

    • "Normal"

    • "Minimized"

    • "Maximized"

  4. Optionally, select Pause script till program has been closed. This causes the action script to pause until the external program has finished running.

  5. Click Next >.

  6. Optionally modify the comment that describes the action.

  7. Click Finish.
