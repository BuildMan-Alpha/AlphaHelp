Using InstallMaker to Build Installation Packages

NOTE: InstallMaker is not available in V12 and above.

The InstallMaker utility creates a package that will install your Alpha Five Runtime application at your customer's site. To use this utility:

  1. Run your new application with Alpha Five Runtime.

  2. Exit from the Alpha Five Runtime application.

  3. Run your new application with the Alpha Five development software.

  4. From the Control Panel, select Tools > Utilities > Create Install Package.

  5. The first screen of the InstallMaker will identify the location of your Alpha Five Runtime application.

  6. Click OK to continue.

Open an Existing Project

Optionally, click Open Project to display the Open InstallMaker Project dialog. Select the project from the list and click OK to continue.

Step 1: Method

  1. Make a selection from the Install list. The options are:

    • "Application Files only" - appropriate for an update of an existing application

    • "Both Alpha Five Runtime and application files" - appropriate for a new installation

  2. Define the name of the installation program, which is by default " c:\setup.exe ".

  3. Click Next >>.

Step 2: Registry

  1. In the Options list, place checkmarks next to the registry keys that you want to set upon installation. To place a checkmark, select the desired key and select the Include Key check box.

  2. Click Next >>.

Step 3: Customizations

  1. If you selected Both Alpha Five Runtime and application files" in step 7, the Other files to include list will be enabled.

  2. Place checkmarks next to the files to include.

  1. If there are additional files to include (such as images):

    1. Click Install Additional Files to display the Select Additional Files dialog box.

    2. Click to navigate to and select a file to add. This file must be in the application's folder or in a sub-folder.

    3. To remove a file, select it in the Additional Files list and click .

    4. Click OK to continue.

  2. Click Next >>.

Step 4: Install Locations

  1. Make a selection in the Install Location list. The options are:

    • "Install the Runtime and Application files in the same folder"

    • "Install the Runtime and Application files in different folders"

  2. Enter the location of the application files in the Default Application Folder field.

  3. If you selected "Install the Runtime and Application files in different folders" in step 14, enter the installation location of the Alpha Five Runtime software in the Runtime Folder field.

  4. Click Next >>.

Step 5: Display

  1. Optionally, click in the Icon File field to select the icon image that you want to associate with your application.

  2. Optionally, enter text to display in the background during installation into the Background text field.

  3. If you entered background text, select the where the text should be displayed from the Text Location list. The options are:

    • "Upper Left"

    • "Upper Right"

    • "Lower Left"

    • "Lower Right"

    • "Top Center"

    • "Bottom Center"

  4. Optionally, click in the Background BMP field to select an image to display in the background during installation.

  5. Enter the text to display in the title bar of the installer into the Title Bar Text field.

  6. Click Next >>.

Save the Project

Optionally, click Save Project to display the Save InstallMaker Project dialog. Enter a name for the project and click OK.

Step 6: Shortcut

  1. Click Modify Shortcut.

  2. On the General tab enter the name that will be attached to the shortcut in the Shortcut name field.

  3. Select the location for the shortcut. The options are:

    • "None"

    • "Custom Group"

    • "Desktop"

    • "Start Menu"

    • "Startup Folder"

  4. If the target computer is running Windows NT, Windows 2000, or Windows XP, in the Owner field select whether the program will be available to the Current user or All users.

  1. On the Appearance tab select whether the main Alpha Five window will have a custom title. If so, enter the title in the Use custom title field.

  2. When loading, select whether Alpha Five will use:

    • its Default splash screen

    • No splash screen

    • a Custom bitmap

  3. If you selected "Custom bitmap", click in the Custom bitmap field to select an image to display when Alpha Five is loading.

  4. When loading, select whether Alpha Five will use:

    • its Default icon

    • Custom icon

  1. If you selected "Custom icon", click in the Custom icon field to select an icon to display when Alpha Five is loading.

  1. On the Window tab select the size of the main Alpha Five window. The options are:

    • Default - previous size of main Alpha Five window

    • Maximized - maximize the main Alpha Five window

    • Minimized - minimize the main Alpha Five window

  2. Select whether to disable the ability to minimize or maximize the main Alpha Five program window.

  3. Select whether to open Alpha Five without a system menu.

  4. On the Advanced tab, enter Xbasic commands to run when Alpha Five is started.

  5. Click OK to continue.

  1. Click Create Install. InstallMaker will create a self extracting ZIP file named (by default) setup.exe. This name was defined on the first page of the InstallMaker utility.
