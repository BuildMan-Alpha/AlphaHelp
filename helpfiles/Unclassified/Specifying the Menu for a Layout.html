Specifying the Menu for a Layout

To specify the toolbar for a form, browse, report, letter, or label:

  1. Optionally design custom menu(s) and save them. See How to Design a Custom Menu.

  2. Open the layout in design mode and press F12 to access its Properties dialog.

  3. Display the Menus/Toolbars tab.

  4. Clickto open the Menus View list and/or the Menus View right click list and select your newly designed menu.
