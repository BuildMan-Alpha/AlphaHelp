PROCESSHEADER.A5W

Purpose

The PROCESSHEADER.A5W page displays content that indicates that the application is working.

Description

The PROCESSHEADER.A5W page is an ordinary HTML page without Xbasic coding. It only appears in the top section of a frame set.

Since this page contains no Xbasic, we could have saved it as a .HTM file instead of .A5W. If the page were saved as .HTM, the references to the page in the frame sets would have to change. We originally saved the page as .A5W to allow the potential to use Xbasic to add dynamic content.

Links

The ASWPROCESSING.A5W and PROCESSING.HTM pages call the PROCESSHEADER.A5W page.
