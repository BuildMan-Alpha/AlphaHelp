Send an Email Message

Overview

The Send an Email Message action allows the user to send email using either the built-in Alpha Five email client, or the user's default email client (e.g. Eudora, Outlook etc.). This is an extremely versatile function that lets you send an email message that includes data from the current record.

Procedure

  1. Select the email client you wish to use. The options are:

    • "Built-in Alpha Five email client"

    • "Default email client (e.g. Eudora, Outlook, etc.)"

  2. Optionally, check the Load message header check box. This causes three fields to appear.

  3. In the Template source field select the library file that contains the template.

  4. In the Prompt for template when? list, select either:

    • "Specify template name now"

    • "Prompt for template name at run time"

  5. If you selected ""Specify template name now", select the template from the Template name field. You may click the button to launch the Expression Builder to define the template with an expression.

  6. Optionally, if you did not check the Load message header check box, enter a character string or an expression that evaluates to a character string in the Subject field.

  7. Enter one or more email address(es) or an expression that evaluates to one or more email address(es) in the Send to field.

  8. Optionally, enter email address(es) into the CC and Bcc fields.

  9. Optionally, click Attachments to display the Select Files dialog box or enter an expression to select attachments.

  10. Optionally, check the Put attachments into a Zip file? check box. If you select multiple files to attach, Alpha Five automatically zips the files for you.

  11. Click Next >.

  12. Optionally, check the Load message body from a saved email template check box. If you checked it, continue with step 13. Otherwise, continue with step 17.

  13. Optionally check the Use same template as for message header check box. If you did not check the Use same template as for message header check box, continue with step 14. Otherwise, continue with step 17.

  14. Select the library that contains the template from the Template source field.

  15. In the Prompt for template when? list, select either:

    • "Specify template name now"

    • "Prompt for template name at run time"

  16. If you selected ""Specify template name now", select the template from the Template name field. You may click the x-y button to launch the Expression Builder to define the template with an expression.

  17. Optionally, enter a text message in the Message window, or use the Expression Builder. The window contents appear as the default message in the email. Unless you specify that the email should not open for the user, the user will be able to edit this message at run-time.

  18. Optionally, select the message type. The options are:

    • "HTML"

  19. If you choose to use the Alpha Five email client in step 1, you have the option to further specify the behavior of the action. You can select the type of interaction that results from the user initiating the action. The following options are available:

    • "Open Alpha Five email client as a modal dialog box"

    • "Open Alpha Five email client as a modeless dialog box"

    • "Don't open the Alpha Five email client - just send message then display confirmation"

    • "Don't open the Alpha Five email client - just send message without confirmation dialog"

  20. Choosing not to display the Alpha Five email client means that the user can't edit the message before it's sent, so choose the option that best corresponds with the action's needs.

  21. Click Next >.

  22. Optionally modify the comment that describes the action.

  23. Click Finish.
