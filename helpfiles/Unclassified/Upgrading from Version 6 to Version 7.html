Upgrading from Version 6 to Version 7

The Version 7 Application Server is able to support applications that read Version 5, 6, and 7 tables. There were no changes to the table formats.

The web components changed from Version 6 to Version 7. While you should be able to run your V6 components on the V7 server, there is a small possibility of something getting broken. The fix would be as simple as updating the component to V7. You would not have to touch any of the underlying data or application files.
