PropertyGrid

Description

The PropertyGrid object is useful when you want to display many prompts to the user and accept many inputs.

Method

Type

Description

.ExpandItem()

V

Expands or contracts a named category.

.IsExpanded()

L

Returns TRUE (.T.) if a category is expanded.

.SetFocus()

V

Sets focus to an item, expanding its container category if necessary.

Limitations

Desktop applications only.
