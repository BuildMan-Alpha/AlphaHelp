Write-Only Grid Component

The following lesson describes how to create a web component that allows users to enter records, but not to read, modify, or delete records. This procedure covers only the critical steps. There are many potential customizations that you may optionally apply to the resulting grid component.

  1. Create a new tabular grid based on the AlphaSports customer table.

  2. Display the Component Type menu.

  3. Select Updateable.

  4. Select "Columnar" in the Grid layout list.

  5. Display the Grid > Fields menu.

  6. Move the "Firstname", "Lastname", "Company", and "Phone" fields from the Available Fields list to the Selected Fields list.

  7. Display the Grid > Properties menu.

  8. Set the Layout Options > Repeating columns field to 1.

  9. Check Layout Options > Add records only.

  10. Display the Grid > Update Settings menu.

  11. Set the Permissions > Number of new record rows field to 1.

  12. Set the Customization > Target page field to the URL of the page that will confirm the user's input.

  13. Click Browser to see how the grid will appear.

  1. Place the component into an A5W page and publish it.
