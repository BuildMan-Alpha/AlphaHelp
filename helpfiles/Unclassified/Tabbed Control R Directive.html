Tabbed Control R Directive

Syntax

%R= Rows %

Argument

Type

Description

Rows

N

The number of tab rows.

Description

The %R% directive specifies the number of rows of tabs for the {TAB} function.

Example

This example creates 5 tabs arranged in 2 rows.

ui_dlg_box("Tabs",<<%dlg%

{tab=%R=2%page}

{pane=Pane 1}

[%m%.30,10]

{pane=Pane 2}

{pane=Pane 3}

{pane=Pane 4}

{pane=Pane 5}

{endtab}

%dlg%)
