DotNet::Services.CreateObject Method

CreateObject() creates an instance of an arbitrary .NET class without registering the assembly or class in the Xbasic type system. Once the instance is destroyed, the assembly is still loaded in the Common Language Runtime (CLR), but Xbasic will retain no knowledge of the type.

    BYREF Instance as P,
    ClassName as C,
    OPTIONAL Assembly as DotNet::AssemblyReference = null_value(),
