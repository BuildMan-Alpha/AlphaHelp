Dump_Csv Method

Syntax

Dump_Csv as c([hasFieldNames as l[,restrictColumns as c]])

Argument

Description

hasFieldNames

If true, Emit field names for the first row.

restrictColumns

Comma separated list of field names to emit, allowing column ordering and filtering of which columns are used.

Description

Initailize a property array with from a passed in CSV string.
