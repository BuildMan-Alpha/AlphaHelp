Array Functions and Methods

Alpha Anywhere provides the following methods for manipulating arrays. Prefix the following methods with the <ARRAY> pointer or the name of the array variable.

Method

Description

.APPEND_ARRAY()

Appends Array2 to Array1 to produce New_Array.

.APPEND_ARRAYS_FILTERED()

Appends selected elements from Array2 to Array1 to produce New_Array.

.CLEAR()

Clears the contents of the array.

.COPY_FROM()

Copies elements from a source array into a target array, overwriting existing elements and resizing the target array as necessary.

.DELETE()

Deletes Count entries in an array starting at Position.

.DUMP()

Creates a CR-LF delimited string of the array's contents.

.DUMP_PROPERTIES()

Creates a CR-LF delimited string of a property array's contents.

.FIND()

Finds an element in an array and returns its Index number.

.FIRST_EMPTY()

Returns the index of the first NULL element in an array.

.GetDynamic()

Returns the dynamic sizing state of an array

.INITIALIZE()

Populates an array with values contained in a character Data_String.

.INITIALIZE_FROM_TABLE()

Loads field values from a table into a property array.

.INITIALIZE_PROPERTIES()

Loads field values from a data string into a property array.

.INSERT()

Inserts Count entries in an array starting at Position.

.MATCHING()

Returns a CR-LF delimited list of array elements matching a filter.

.MOVE()

Moves an array element From the specified position specified To the specified position.

.PROPERTY_ORDER()

Creates a list containing the sequence of reordered array elements.

.REORDER()

Reorders an array based on an order list of the numbers of array elements.

.RESIZE()

Increases the size of an array, without losing any of the existing data.

.SetDynamic()

Controls dynamic sizing of an array

.SIZE()

Returns size of an array.

.SORT()

Sorts an array.

The following functions and methods are useful for manipulating arrays.

Function or Method

Description

*ARRAY_APPEND()

Re-dimensions an array and assigns a value to the new array element.

*ARRAY_RANGE_ASSIGN()

Copies elements from one array to another.

*ARRAY_RANGED REFERENCE_TO_RANGE()

Takes a list of array variable definitions and returns a new list that can be used to properly DIM them.

A5_ADO_TO_ARRAY()

Retrieves records from an ADO compliant database and populates an array.

ArrayType()

Returns the data type of an array.

ISARRAY()

Indicates whether an object is an array.

SORT_ARRAY()

Sorts the specified character, numeric, or date array.

UI_GET_LIST_ARRAY()

Displays a dialog box with a list box filled with the contents of a character array.

<TBL>.POPULATE_GRID()

Populates arrays quickly with data from a table.
