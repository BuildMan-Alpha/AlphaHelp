RTF Functions and Methods

The following methods require the pointer returned by the RTF.CREATE() method. These methods are intended for use with RTF text on an Alpha Anywhere layout.

Method

Description

RTF.CREATE()

Creates an RTF object and returns the pointer to it.

.BINARY_TEXT

The property that returns the binary representation of the RTF text.

.CENTER_JUSTIFY()

Center justifies the RTF text.

.CLASS()

Returns the class of the object referenced by .

.COPY()

Copies the currently selected text and puts it on the clipboard.

.CUT()

Cuts the currently selected text and puts it on the clipboard.

.DELETE()

Deletes text from an RTF object.

.FULL_JUSTIFY()

Full justifies the RTF text.

.GET_CURSOR()

Returns the cursor location.

.GET_FONT()

Gets the font at the position specified by position.

.GET_FONT_COLOR()

Gets the font color at the position specified by position.

.GET_JUSTIFICATION()

Returns the justification mode of the RTF text.

.INSERT_ELSE()

Inserts an ELSE command.

.INSERT_ELSEIF()

Inserts an ELSEIF command.

.INSERT_ENDIF()

Inserts an ENDIF command.

.INSERT_IF()

Inserts an IF and ENDIF command.

.INSERT_IF_UNPAIRED()

Inserts an IF command.

.INSERT_IFELSE()

Inserts an IFELSE and ENDIF command.

.INSERT_IFELSE_UNPAIRED()

Inserts an IFELSE command.

.INSERT_PAGEBREAK()

Inserts a page break at the cursor location.

.INSERT_TEXT()

Inserts text at the cursor location.

.IS_VALID()

Reports whether the RTF Editor is open.

.LEFT_JUSTIFY()

Left justifies the RTF text.

.PASTE()

Pastes from the clipboard into the RTF object.

.PLAIN_TEXT

The property that returns the plain text representation of the RTF text.

.PROPERTIES()

Displays the Properties dialog.

.REPAINT()

Redisplays the text buffer.

.REPLACE_TEXT()

Replaces the currently selected text.

.RESET()

Erases the text buffer.

.RIGHT_JUSTIFY()

Right justifies the RTF text.

.RTF_TEXT

The property that returns the ASCII encoded representation of the RTF text.

.SELECT()

Selects text in the RTF control.

.SET_CURSOR()

Sets the cursor's location.

.SET_FONT()

Sets the font name and color for the next insertion into the RTF object.

.SET_STYLE()

Sets the style for the next insertion into the RTF object.

.SHOW_CARET()

Displays the cursor.

.SHOW_CODES()

Toggles the display of formatting codes.

.STYLE

Description of the dot variable that contains style properties.

.TEXTREPLACE()

Places text into an RTF object.

.TEXTSELECT()

Selects text in an RTF object.

Alpha Five provides the following Rich Text functions.

Function

Description

*BIN_TO_HTML()

Converts RTF binary (memo) format to HTML text.

*BIN_TO_RTF()

Converts RTF binary (memo) format to ASCII encoded RTF text.

*HTML_TO_BIN()

Converts HTML text to RTF binary (memo) format.

*HTML_TO_RTF()

Converts HTML text to ASCII encoded RTF text.

*PLAIN_TO_BIN()

Converts unencoded text to ASCII encoded RTF text.

*PLAIN_TO_RTF()

Converts plain ASCII text to RTF text.

*RTF_TO_BIN()

Converts ASCII encoded RTF text to RTF binary (memo) format.

*RTF_TO_HTML()

Converts ASCII encoded RTF text to HTML text.

*RTF_TO_PLAIN()

Converts ASCII encoded RTF text to plain ASCII text.

A5_RTF_EDITOR2()

Displays the RTF Editor and returns a dot variable with edited RTF text.

INSERT_TEXT _ IN_RTF_MEMO()

Inserts formatted text into a rich text field.

RTF_TO_TEXT()

Converts an RTF string, referenced by the variable Blob_RTF, to a text string.

RTF_FIELD_TO_TEXT()

Returns a text string with the data in a RTF memo field.
