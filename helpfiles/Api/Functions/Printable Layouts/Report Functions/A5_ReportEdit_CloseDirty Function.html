A5_ReportEdit_CloseDirty()

Syntax

Result as C = A5_ReportEdit_CloseDirty()

Argument

Description

Result

 

Description

The A5_ReportEdit_CloseDirty() function allows the developer to override the built-in dialog that is displayed when the user closes a report, label or letter that has been edited, but not yet saved. The function must return a character value of Yes, No, or Cancel.

Limitations

Desktop applications only.
