ImageName Object

Description

The ImageName object is useful when you want to create a logical image name and then map it to the physical image names defined in Alpha Anywhere. If you use logical image names, then you don't have to change your code if you want to change the image that it uses, you just have to change the image map.

Methods

Create an ImageName object with the <ImageResource>.GetBitmapNameMap() method. The ImageName object has these methods:

Method

Type

Description

<ImageName>.Clear()

P

Remove all keys from the string dictionary.

<ImageName>.Exists()

P

Determines if an image name exists in the string dictionary.

<ImageName>.Get()

P

Returns an image name from the string dictionary.

<ImageName>.Initialize()

P

Adds entries to the string dictionary.

<ImageName>.Remove()

P

Removes an entry from the string dictionary.

<ImageName>.Set()

P

Sets the value of an entry in the string dictionary.
