.BINARY_TEXT

Syntax

RTF_Data as B = .BINARY_TEXT

Description

The .BINARY_TEXT property returns the binary value of the RTF text.

Example

dim ptr as P

dim bdata as B

ptr = rtf.create("Alpha!")

bdata = ptr.binary_text

bdata ->  0000 : 00 00 00 00 00 02 00 00 00 00 00 00 00 00 00 00 ....
