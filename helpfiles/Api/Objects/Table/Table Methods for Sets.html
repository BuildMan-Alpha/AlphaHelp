Table Methods for Sets

Alpha Five provides the following Set related functions and methods.

Method

Description

<TBL>.CHILD_GET()

Returns the name of a child table.

<TBL>.LINK_EXPR_GET()

Returns the expression that links two tables in a set.

<TBL>.LINK_FILTER_EXPR_GET()

Returns the filter expression used when linking two tables in a set.

<TBL>.LINK_INDEX_FLAGS_GET()

Returns the options used when linking two tables in a set.

<TBL>.PARENT_GET()

Returns the alias of a table's parent in a set.

<TBL>.PARENT_LINK_EXPR_GET()

Returns the link expression of the parent table of a child table in a set.

<TBL>.RELATION_ADD()

Establishes a relationship, or link, between two open tables.

<TBL>.RELATION_CHANGE()

Changes the definition of an existing relationship between two open tables.

<TBL>.RELATION_DROP()

Removes the link between parent and a child table.

<TBL>.RELATION_GET()

Returns the type of relationship between and it's parent table.

<TBL>.SET_CREATE()

Creates a set definition file from the relational links established by the <TBL>.RELATION_ADD() method.
