Active-Link and Passive-Link Tables - Convert to Native Tables

You can now right-click on an active-link or passive-link table and convert the table to a native Alpha Five table. In the case of an active-link table, this will import the data from the remote table and store it in a native table. In the case of a passive-link table, the data is already stored as a native .dbf table, but the link to the remote SQL database is severed, so the table can no longer be 'refreshed'.
