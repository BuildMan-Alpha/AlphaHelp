Ping() Function

Allows you to ping a network address.
Syntax
L Result = Ping(C URL [, P status])

 

Examples:

dim np as p

?ping("www.alphasoftware.com",np)
= .T.

?np
= Bytes = 32
StatusCode = 0
StatusText = "Success"
Text = "www.alphasoftware.com (72.32.123.61) is responding"
Time = 75
TTL = 116

 

'ping a computer on your LAN

?ping("mymachinename")
