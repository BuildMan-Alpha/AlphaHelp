Read a bitmap from the Clipboard

You can now copy a bitmap to the clipboard and then use Xbasic to read the clipboard data. The following Xbasic script demonstrates how you can use this feature:

 

'assume you have used an image capture program to copy an image to the clipboard

dim b as b
b=clipboard.Get_Data(2)
 


?b.size()
= 1426874

 

'convert to jpg (because .bmp images are very large).

dim j as b
j =bitmap_to_jpeg(b,25)
file.From_blob("c:\j.jpg",j)

 

'display the image in an Xdialog
ui_dlg_box("","{image_opaque=c:\j.jpg}")
