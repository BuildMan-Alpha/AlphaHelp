*gzip_blob() and *gunzip_blob() Functions - New Arguments

The *gzip_blob() and *gzip_unblob() functions now take an optional second parameter, that allow these functions to be used in your own Xbasic code. Previously, the *gzip_blob() function changed the header of the file to a format that was necessary in order for the GZip encoding feature in the Application Server. This resulted in an object that could not be unzipped using *gunzip_blob(). Now, if you set the optional 'NativeFormat' flag to .t., the object created by *gzip_blob() can be unzipped using *gunzip_blob().

Example:

fn = "C:\Program Files (x86)\a5V10\a5v10generic.jpg"
b = file.to_blob(fn)
?b.size()
= 13235
b2 = *gzip_blob(b,.t.)
?b2.size()
= 12123
b3 = *gunzip_blob(b2,.t.)
?b3.size()
= 13235
