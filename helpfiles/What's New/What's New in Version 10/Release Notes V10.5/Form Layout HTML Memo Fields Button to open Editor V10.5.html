TIP: Form Layout - HTML Memo Fields - Button to open Editor

To edit an HTML memo on a Form Layout, you click the small pencil icon that is enabled when the field has focus. Some users would like to create their own button on the form to open the HTML Memo editor. There are two advantages to doing this:
  • The button can be made much larger than the small pencil icon that Alpha Five uses by default
  • The button is always visible on the form and can be clicked as soon as the form is opened. In contrast, the pencil icon is only shown when the field has focus.

This code shows how you can do this:

'Assume that the Form has a control called 'mymemo' that
'is bound to the HTML memo field

dim p as p
p = mymemo.this
a5_form_field_html_editor(p)
