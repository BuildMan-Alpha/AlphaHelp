Grid Component - Font Scaling

When you create a Grid component, you can set the font size option to 'Smallest', 'Smaller', 'Larger', 'Largest' etc. When you set this property, it scales the font size definition in the Grid Style classes, but it also scales the font in the following tags: BODY, TABLE, TH, TR, TD, UL, OL, LI, P, A, BLOCKQUOTE

Now, you can set a new property in Override Settings to limit Alpha Five to scaling the font in ONLY the CSS classes that are associated with the Grid Style.

To set the property, add this code to the OnGridExecute event:

e.tmpl.fontScalingOption = "GridStyleClassesOnly"
