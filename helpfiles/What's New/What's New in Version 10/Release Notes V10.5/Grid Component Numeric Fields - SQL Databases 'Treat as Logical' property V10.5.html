Grid Component - Numeric Fields - SQL Databases - 'Treat as Logical' property

Typically, logical fields in SQL databases are defined as 'bit' or 'logical' fields. But in some cases they may be defined as integer fields that contain 0 and 1 for the false and true value. A new property in the Grid builder allows you to treat a numeric field as logical field. This is important if you want to use a checkbox control for the field. If you check the property and you specify that the control is a checkbox, Alpha Five will treat the control as a 'logical checkbox'.
