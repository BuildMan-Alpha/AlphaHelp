TIP: File Upload Feature Pack - Uploading to a Binary Field - Refresh Row After Upload

Some users have asked how to refresh the current Grid row after a file has been uploaded into a binary field in the Grid.

First, add this function in the Javascript Function Declarations:
function refreshRow() {
    var rowNum = {grid.object}._selectedRow;
    {grid.object}.refreshRow(rowNum);
}

Then, in the Action Javascript that defines the File Upload button, set the After Upload client-side function to:
refreshRow
