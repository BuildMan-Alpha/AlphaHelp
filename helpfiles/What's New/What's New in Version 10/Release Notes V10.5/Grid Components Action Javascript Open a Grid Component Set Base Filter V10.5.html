Grid Components - Action Javascript - Open a Grid Component - Set Base Filter

When you use Action Javascript to create an action that opens a child Grid, the child Grid can either be 'linked' or 'filtered'. If the case where you 'filter' the child Grid you can set the child Grid's 'base' filter or 'user' filter. The builder explains the difference between 'base' and 'user' filter. In the case where you set the 'base' filter, the 'base' filter that was set in the Action Javascript replaced the base filter (if any) that was set in the Grid component itself.

No facility existed for having the base filter set by the Action Javascript be 'additive' to the base filter that was defined in the Grid.

Now, by default, when you set a base filter in Action Javascript, it is additive to the base filter that is defined in the Grid. If you want to override this behavior and make the base filter replace the base filter defined in the Grid, you can set the new 'baseFilterSetMode' property in the Override Settings section in the Action Javascript builder.

For example:
baseFilterSetMode = "Add"
baseFilterSetMode = "Replace"

An example of the difference between 'Adding' to and 'Replacing' the child Grid's base filter follows.

Assume that the child Grid is based on a .dbf table and you set this filter in the Grid builder:
state = "MA"

Or, the child Grid is based on a SQL table and you defined this filter in the Grid builder:
state = 'MA'

Or, the child Grid is based on a SQL statement and you defined this statement:
SELECT * FROM CUSTOMERS WHERE state = 'MA'

In all three cases the child grid has a base filter of State = "MA" or State = 'MA'.

In your Action Javascript, say that you define this Base Filter:
OrderTotal < 1000

When you execute the action, the child Grid is opened and the filter on the Child grid will be:
state = "MA" .and. OrderTotal < 1000

or, in the case of a SQL grid:
state = 'MA' .and. OrderTotal < 1000

However, if in the Override Settings section in the Action Javascript builder, you entered this command:
baseFilterSetMode = "Replace"

When you execute the action, the child Grid will have this base filter:
OrderTotal < 1000
