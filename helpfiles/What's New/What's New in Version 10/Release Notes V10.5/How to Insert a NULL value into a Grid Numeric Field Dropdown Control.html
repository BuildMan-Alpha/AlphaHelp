How to Insert a NULL value into a Grid Numeric Field Dropdown Control

Say you have a numeric field in a Grid. The control type for this field is a Dropdownbox that has the following stored values that correspond to the available choices: -1, 1, 2, 3, 4 ec. The -1 choice corresponds to the "No Selection" option in the Dropdownbox. If the user submits the Grid without making a selection in this field, the value submitted will be -1. You would like to store a NULL in the field under this condition.

You will need to add code to the OnBeforeSQLCommandExecute event in Server-side Events.

dim args as sql::arguments 
args.xml = e.argsxml
dim id as n
'The argument name for the 'Response1' value is 'New_Response1'
id = args.ArgumentNumber("new_Response1")
if id &gt; 0 then 
    if args[id].data = -1 then 
        args[id].isNull = .t.
    end if 
    e.argsXMLModified = args.xml
end if
