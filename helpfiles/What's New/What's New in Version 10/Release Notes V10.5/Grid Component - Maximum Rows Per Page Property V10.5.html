Grid Component - Maximum Rows Per Page Property

If you set the Rows property to 0, the Grid will display all of the records in the Grid on a single page. There may be a very large number of records and displaying all of these records could cause a sever strain on your server. Therefore, a new property has been added. The Maximum rows per page property is shown when the Rows property is set to 0. This property allows you to specify a maximum number of records to show on a single page.
