Reports - PDF - 0 inch Margins

You can now set the margins on a report to 0 inches and when you print the report to a PDF file, the PDF file will now be correctly created with a 0 inch margin on all sides. Previously, even if you set the margins in the Report definition to 0 inches, the PDF report would insert a minimum margin.

Note: When you preview a report with 0 inch margins, the Preview window will show clipping. However, the printed PDF report will not be clipped.
