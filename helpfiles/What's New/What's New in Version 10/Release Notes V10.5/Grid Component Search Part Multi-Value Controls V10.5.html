Grid Component - Search Part - Multi-Value Controls

A checkbox control and a dropdown control (when configured appropriately) are multi-value controls. These controls allow you to select multiple values. For example, a checkbox that displays 'Red', 'Green' and 'Blue' as choices, will allow you to select 'Red' and 'Green'. When you use a multi-value control in the Search Part of the Grid, you can now control whether the multiple selections are combined with an AND or OR clause.

For example, assume that you have a Grid with a field called 'FavoriteColor'. The data in this field is a comma delimited list of choices. So, for example, in the first record, the field might contain 'Red,Blue'. The second record might contain 'Red,Green,Blue', the third record might contain 'Blue' and so on. In the Search Part, assume that the user checks 'Red' and 'Blue'. Previously, Alpha Five would search for records where the FavoriteColor was 'Red' OR 'Blue'.

Now, you can configure Alpha Five to search for records where the FavoriteColor is 'Red' AND 'Blue'. In order to configure the Grid to combine multiple values with an AND set the Search Style for the field to '21'. In order to configure the Gri to combine multiple values with an OR, set the Search Style to '20'. These values are documented in the Help section in the Property Grid in the Grid Builder (Tip: You will need to resize the Help window to see all of the help text).
