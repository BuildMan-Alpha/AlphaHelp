Grid Component - Custom Ajax Callbacks - e._set Object

In a custom Ajax callback, if you want to set the value in one or more controls in the Grid, you can set properties in the e._set object and Alpha Five will then automatically generate the correct Javascript to set the values in the controls.
e.g. You can put this code in your Xbasic function that handles the Ajax callback:

e._set.company.value = "Alpha Software"

However, if you had a Checkbox or multi-select DropdownBox, there was no way to use the e._set object to set multiple values in the control. (You could set a single value in the control). Now, a new syntax is supported to set multiple value in the control.

To set multiple values in a CheckBox or multi-select Dropdownbox control, set the .value property to a dot variable that contains an array of values. The array must be called 'value'.

Example:

Dim a as p 'a is an arbitrarily named dot variable
dim a.value[3] as c 'dim an array called 'value'
a.value[1] = "alpha"
a.value[2] = "beta"
a.value[3] = "gamma"
e._set.greekletters.value = a

This will select 3 options ('alpha', 'beta' and 'gamma') in the checkbox group for the 'greekletters' control.
