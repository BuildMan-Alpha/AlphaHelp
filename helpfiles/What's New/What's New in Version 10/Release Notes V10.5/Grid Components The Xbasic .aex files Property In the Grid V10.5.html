Clarification - AEX Files - Grid Components - The 'Xbasic .aex files' Property In the Grid

When you set this property, the .aex files that you specify can be automatically added to the aex files in the Web Project Properties. This is a convenience to save you from having to manually visit the Web Project Properties and add the .aex files there yourself.

Note that it is not strictly necessary to set the 'Xbasic .aex files' property in the Grid anymore (if you don't use Working Preview). If you have specified the .aex files that are needed in the Web Project Properties dialog, the Grid will run correctly in Live Preview mode and when run in the Browser. However, if you want to run the Grid in Working Preview mode, you WILL need to set the 'Xbasic .aex files' property because Working Preview does NOT load the settings in Web Project Properties.

The recommended technique is therefore to set the 'Xbasic .aex files' property in the Grid. That will then automatically update the Web Project Properties to add the .aex file that you specified in the Grid Settings.
