Ajax Callback - Linked Grids - 'e' Object - Linking Values

When a user-defined Ajax callback is made, the 'e' object that is passed in now includes these additional properties:
e._isLinkedGrid - .t. if the Grid is a linked Grid (i.e. it has a parent Grid)
e._linkValues = linking value(s).

The ._linkValues property contains a CRLF delimited list of linking values in this format: fieldname|type|linkingValue.
For example, say that the Grid was linked to its parent on the 'OrderId' field, and that its was opened for a parent record where OrderId was 234. The e._linkValues property would look like this:
e._linkValues = 'OrderId|N|234'
If the Grid was linked on 'Firstname' and 'Lastname', then an example value for this property would be:
e._linkValues = <<%txt%
Firstname|c|Fred
Lastname|c|Smith
%txt%
