Grid Components - SQL Tables - FIRST Clause

If a Grid is based on a SQL table, when Alpha Five constructs the SQL to execute to run the Grid it now automatically adds a FIRST clause to the SQL that is generated. (Note: FIRST is the Portable SQL syntax command for controlling how many records a query returns.)

For example, if you have set the Grid to show 20 records per page, and you are viewing page 2 of your data, the SQL that is generated follows this pattern:

SELECT FIRST 40 &lt;fieldlist&gt; FROM &lt;tablelist&gt; 
    WHERE &lt;filter&gt; ORDER BY &lt;sortlist&gt;

On Grids that are based on very large SQL tables, this can result in a significant performance improvement.
