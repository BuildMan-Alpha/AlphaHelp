Grid Components - Native SQL

Previously the Grid Component would only accept Portable SQL Syntax. Now, the Grid will work if you enter Native SQL Syntax. You may not be able to perform search and sort operations however if Alpha Five cannot parse the SQL statement that you execute.

TIP You can test to determine whether Alpha Five can parse your SQL statement by using this Xbasic in the Interactive window:

dim q as sql::query
?q.parse("Your SQL Statement")

If this returns .F., you can use this command to see why Alpha Five could not parse the SQL:

?q.callResult.text
