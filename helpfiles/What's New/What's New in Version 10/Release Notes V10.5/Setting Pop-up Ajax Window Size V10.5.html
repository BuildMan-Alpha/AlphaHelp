Setting Pop-up Ajax Window Size

A common requirement is to size the pop-up window relative to the parent window size. For example, you might want to set the pop-up window to:
  • a percentage of the parent window size
  • equal to the parent window size with a margin of x pixels on the width and y pixels on the height.
The window settings do not allow you to set a relative size. To solve the problem you must define Javascript code in the onShow Window event.

This example shows how you can set the pop-up Ajax window to be 100 pixels less than the size of the main window.

//get the size of the parent window (i.e. the 'View Port')
var cs = AUI.u.getVPSize();
//compute the size of the pop-up window.
cs.width = cs.width - 100;
cs.height = cs.height - 100;
//set the window size
this.setSize(cs.width+'px',cs.height+'px');
//position the window
this.setPosition('center');
