File.zip() Method - Zipping Files that Have Commas in the Name

This method takes a list of input files and creates a zip files. Previously the list of input files had to be comma delimited. However, that prevented you from zipping a file that had a comma in the filename. Now, you can include ',' in the list of optional flags passed into to the method (third argument) to specify that the list of filenames is CR-LF delimited.

files = <<%txt%
c:\data\names, addresses.dbf
c:\data\orders, credits.dbf
%txt%
file.zip("c:\data\zipfile.zip", files, ",")
