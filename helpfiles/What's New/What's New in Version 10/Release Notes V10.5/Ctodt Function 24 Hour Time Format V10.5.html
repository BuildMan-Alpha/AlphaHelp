Ctodt() Function - 24 Hour Time Format


If the 'am' or 'pm' flag is left out, Alpha Five now assumes that the user is specifying a time using a 24 hour format. Previously, a time value of '00:00:00' was considered to be invalid. Now this is a valid 24 hour format time value.

?ctodt("12/1/1972 00:00:00")
= 12/01/1972 12:00:00 00 am
?ctodt("12/1/1972 23:00:00")
= 12/01/1972 11:00:00 00 pm
