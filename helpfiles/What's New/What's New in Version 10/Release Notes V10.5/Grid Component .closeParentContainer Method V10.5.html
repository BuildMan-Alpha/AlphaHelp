Grid Component - .closeParentContainer() Method

In the case where a child Grid is opened in either a pop-up Ajax window, or a tabbed UI Pane, you can call this method of the Grid object to close the pop-up Ajax window, or Tabbed UI pane that contains the Grid.
For example, say you have a Grid that displays in a pop-up Ajax window, or a Tabbed UI pane. After the user submits data (either an update, insert or delete), if there were no errors, you want to close the parent container for the Grid (i.e. you want to close the pop-up Ajax window or the Tabbed UI Pane). Here is code you could put into the onRowSubmit and onGridSubmit client-side Javascript events:
if(e.hasErrors == false) {
        {grid.object}.closeParentContainer();
}
