Grid Methods - New {grid.Object}.filterCheckedRows() Method

Allows you to search for the records in the Grid that were checked. For example, say the user checks 3 rows in in the Grid. After executing this method, a search will be performed and just the 3 checked records will be selected. If you then run a report, or export to Excel (for example) in which you specify the report or export should operate on the current selection of records, you will in effect be operating on the records that were checked by the user.
