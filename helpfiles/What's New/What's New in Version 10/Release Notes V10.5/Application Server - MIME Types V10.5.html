Application Server - MIME Types

The Application Server now recognizes Annodex, Flac, and Ogg Vorbis files and sets the MIME type appropriately. Specifically, the following MIME types are now supported:

.ANX=application.annodex
.AXA=audio/annodex
.AXV=video/annodex
.FLAC=audio/flac
.OGA=audio/ogg
.OGG=audio/ogg
.OGV=video/ogg
.OGX=application/ogg
.SPX=audio/ogg
