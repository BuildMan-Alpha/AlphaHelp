a5_runImageMagicCommand() Function

Transforms an image by running an ImageMagic command. Only commands supported by the ImageMagic 'Convert.exe' program are supported.

Syntax:

P pResult = a5_runImageMagicCommand(C fileIn ,C fileOut , C command )

pResult contains these properties:
  • .hasError - .t. or .f.
  • .errorText - error text if there was an error

Example ImageMagic Commands
-resize 20% Resize the image to 20% of its original size.
-resize 500                   Resize the image width to 500 pixels. Height is adjusted so that the aspect ratio is unchanged.
-resize x300          Resize the image height to 300 pixels. Width is adjusted so that the aspect ratio is unchanged.
-resize 800x300! Resize the image and force the size to 800x300 regardless of the original aspect ratio.
