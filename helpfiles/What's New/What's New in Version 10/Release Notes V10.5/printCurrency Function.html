printCurrency()

Syntax

P printCurrency(N number ,C currencySymbol ,N decimal_places [,C thousands_separator [,C decimal_separator [,L TrailingCurrencySymbol [,L blankIfZero [,C cssStyle ]]]]])

Argument Description
number The number value representing the amount
currencySymbol The currency symbol string
decimal_places The number of decimal places to print
thousands_separator The character to use as a thousands separator ("," in the U.S.)
decimal_separator The character to use as a decimal separator ("." in the U.S.)
TrailingCurrencySymbol Set to .T. if the currency symbol should trail the value rather than lead the value
blankIfZero Set to .T. if a blank should display when the value is 0
cssStyle Any CSS style override settings to use for the currency string

Description

Used in a report to print an international currency symbol. A builder will help you select the currencySymbol.

If you print to PDF be sure to set PDF options to allow 'Multilingual support'.

Limitations

Examples

For example, to print a 'price' field using the Nigerian Naira symbol, you would define the following calculated field in the report:

calcfield1 = printCurrency(price,"NGN",2)
