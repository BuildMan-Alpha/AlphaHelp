How to Place the Detail View above the Grid Part in the Grid Component Master Template

The recommended technique for doing this, as documented in the Master Template editor, has been changed.

Change the placeholders in the Master Template to this:

&lt;div id="DTARGET{grid.componentname}"&gt;&lt;/div&gt;
{grid.Grid_Part}
&lt;div id="D{grid.componentName}"&gt;{grid.DetailView_Part}&lt;/div&gt;


Add this to the Javascript tab:

var Dele = $('D{grid.componentName}');
var DTargetEle = $('DTARGET{grid.componentName}');
DTargetEle.appendChild(Dele);
