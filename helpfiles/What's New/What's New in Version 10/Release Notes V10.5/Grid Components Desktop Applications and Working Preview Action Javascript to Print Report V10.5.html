Grid Components - Desktop Applications and Working Preview - Action Javascript to Print Report

Previously, if you added a button to a Grid component to print a Report, the report would only render if the Grid was running in the Browser or in Live Preview. It did not render if the Grid was running on the Desktop, or in Working Preview. Now, using the new 'client-side xbasic' feature, the report will also render when the Grid is running on the Desktop.
