Grid Components - Initial Value Property - Page Variables

For new records in a Grid, you can specify an Initial Value that shows in the new record as a default value. The Initial Value property could previously be a constant (e.g. Fred), an expression (e.g. =date() ), or a session variable (e.g. session.firstname). Now, you can also use page variables (e.g. PageVariable.firstname).

A page variable is specified in the query string. For example:
mypage.a5w?firstname=Fred&lastname=Johnson
