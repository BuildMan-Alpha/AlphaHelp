Grid Component System Javascript Events V10.5

TIP: A typical use of the onGridRender event is to use a Grid method to set a value in the Grid. In some cases your Javascript, while correct, will not give the desired results because of a timing problem. The solution is to wrap the event code in the setTimeout() function so that the Grid can fully render before the event code is fired. For example, this code sets the value of the Firstname field in the first new record rows to 'James':

setTimeout(function() {
{Grid.object}.setValue('G','FIRSTNAME',-1,'James');
}, 10);
