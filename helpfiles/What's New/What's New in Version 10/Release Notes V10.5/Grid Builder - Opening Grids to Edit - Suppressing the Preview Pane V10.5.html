Grid Builder - Opening Grids to Edit - Suppressing the Preview Pane

Some users have Grids where the data displayed in a memo field in the Grid contains Javascript code wrapped in <script> </script> tags. While this is a somewhat unusual use case, it does allow a particular class of problems to be solved. One issue with this situation is that every time you open the Grid to edit it, the Preview pane shows a preview of the Grid and it executes the Javascript in the <script> tags. This is undesirable and a method was requested to suppress the display of the Grid in the Preview pane when the Grid was loaded for editing. You can now do this by going to the 'Advanced' property of the Grid and adding this code to the 'Advanced, Other Properties' property:

tmpl.showPreviewHTML = .f.
