Event Bubbling - $e.stopEvent(event)

Tip: Assume you have defined an OnClick event for a row in the Grid. You have also placed a button in the Grid row and defined an OnClick event for the button. At run-time when you click the button, the button's OnClick event will fire, and then the row's OnClick event will fire. You may not want this to happen, even though it is standard behavior in HTML for events to 'bubble' up to their parent containers.

In the OnClick event for the button, you would include the command:

$e.stopEvent(event)

This will stop the event from bubbling up to any parent containers.
