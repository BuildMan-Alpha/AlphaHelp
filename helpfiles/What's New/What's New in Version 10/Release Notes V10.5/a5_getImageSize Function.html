a5_getImageSize() Function

Returns the size of an image in inches and pixels.

Syntax:

P pResult = a5_getImageSize( c filename)
pResult contains these properties:
  • .hasError - .t. or .f.
  • .errorText - error text if there was an error
  • .width - in inches
  • .widthPixels
  • .height - in inches
  • .heightPixels
