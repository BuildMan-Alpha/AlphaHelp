Runtime - Grid Components - CSS V10.5

If you are using the Grid component in a Desktop application that you deploy using the Runtime, you will need to ensure that your Runtime install includes the CSS folder. The standard Runtime install does install the CSS folder now, but did not when V10 was first released. That means if you installed the Runtime some time ago, and have been applying updates to the original install, you will not have the CSS folder. In that case, simply copy the CSS folder from your full version to your Runtime.
