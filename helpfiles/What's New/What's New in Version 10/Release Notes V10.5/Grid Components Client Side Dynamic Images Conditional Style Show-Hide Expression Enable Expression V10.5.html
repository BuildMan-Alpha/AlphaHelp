Grid Components - Client Side Dynamic Images, Conditional Style, Show/Hide Expression, Enable Expression

You can now use the special data.rowNumber field to reference the row number of the current row in the expression.
For example, say you only want to show a button in a Grid on row 1 of the Grid. You would define this expression for the Show/Hide expression:

data.rowNumber = 1

Previously, you would have had to use a System Javascript event to achieve this effect.
