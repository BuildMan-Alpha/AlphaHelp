Grid Component - Validation Rules

In build 3387-3602 a new property was added to Field Validation Rules - 'Invalid data type error message'. This is the error message that is displayed if the user types in an invalid value into a Numeric, Date, Time, or Shorttime field. The default value for this property was 'Invalid value', but this has now been changed to a blank at the request of some users. Also, the Invalid Data Type Rule was been evaluated for blank values. Now, the rule is only evaluated for non-blank values.
