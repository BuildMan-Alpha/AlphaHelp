Grid Component - Edit-Combo and Auto-Suggest Controls - AfterSelect Javascript

These two controls now allow you to specify any Javascript to execute after the user has made a selection from the list. For example, the following Javascript shows how you can automatically set focus to the 'Company' field in the Grid after the user has made a selection from the list.

var rowNumber = {grid.object}._selectedRow;
var ele = $('{grid.componentname}.V.R' + rowNumber + '.COMPANY');
ele.focus();
