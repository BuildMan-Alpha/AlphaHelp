Set Editor 10.5

The Set Editor has been rewritten in Xbasic. It is now substantially faster to open a Set for editing, especially when editing Sets that are based on active-link tables.

Note

If you have created any Sets using any of the previous Release Candidates and your Set included tables that have spaces in their names, you will not be able to edit those sets. You will need to re-create those sets using the release version of V10.5. This comment does not apply to Sets created with the old set edit in all versions of Alpha Five prior to V10.5.
