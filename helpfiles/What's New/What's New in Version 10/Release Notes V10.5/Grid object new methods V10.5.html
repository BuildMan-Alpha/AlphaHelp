Grid object new methods

In the release notes for build 2641-3512 on December 17 2009, we described how it was necessary to define Linked Resources if a Grid was being opened as a child Grid. This is no longer necessary, except for locally defined CSS. I.e. if the child Grid specifies locally defined Javascript, linked Javascript, or linked CSS files, these resources no longer need to be linked by the parent Grid. This new functionality relies on the following new methods of the Grid Object:

{grid.object}.loadJavascript();
{grid.object}.loadJavascriptFile();
{grid.object).loadCSSFile();
