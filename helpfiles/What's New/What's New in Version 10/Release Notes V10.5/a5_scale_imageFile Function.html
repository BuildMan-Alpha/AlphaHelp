a5_scale_imageFile()

Syntax

V a5_scale_imageFile(C filenameIn ,C filenameOut ,N scale [,N maxSize [,C units [,C appliesto [,C scaleMethod [,N width [,N height ]]]]]])

Argument Description
filenameIn The filename of the image you want to scale. The supported types are .png, .jpg, .jpeg, .bmp.
filenameOut The filename of the image file you want to create. This is the scaled image. Supported types are same as for filenameIn.
scale The ratio of the scaled image to the input image. For example if scale is set to .5, then the scaled image is 50% of the input image. If scale is set to 1.5, then the scaled image is 50% bigger than the input image. If the scale parameter is not zero then the maxSize parameter is ignored.
maxSize (This parameter is ignored if scale is not zero). Specifies the dimension of the scaled image's width or height, depending on the setting of the appliesTo argument.
units 'Inches' or 'Pixels'
appliesto 'Width', 'Height', or 'Max'. If appliesTo is set to 'Max', the dimension applies to the image height, if that is its largest dimension, or its width, if that is its largest dimension. For example, if appliesTo is set to 'Max", then if the input image size has a width of 4 inches and a height of 2 inches, the maximum dimension on the input image is 4 inches. If maxSize is set to 3, then the image will be scaled to have a width of 3 and a height of 2 * (3/4) (i.e. 1.5 inches)
scaleMethod 'Proportional' or 'Independent' - specifies if the image is scaled so that its width and height remain in the same proportions, or if the width and height are scaled independently. Default is 'Proportional'
width If scaleMethod is 'Independent', the width of the image.
height If scaleMethod is 'Independent', the height of the image.

Description

Scales an image file. Input and output supported filetypes: .png, .jpg, .jpeg, .bmp. Scale is a decimal number. e.g. .5 = 50% of original size, 1.5 is 150% of original size. If scale is set to 0 you can specify maxSize - the dimension in inches or pixels of the height, width, or maximum dimension of the image. Units is 'inches' or 'pixels'. AppliesTo is 'Max', 'Width' or 'Height'.)
