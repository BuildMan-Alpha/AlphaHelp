Grid Components - Client Side Expressions - System Fields

Client side expressions can be defined for calculated fields, show/hide expressions, enable expressions, conditional images and conditional styles. Now, when you define your client side expression, in addition to referencing fields values from the current row, you can also reference the following system fields:
  • grid.rowState
  • grid.isRowDirty
  • grid.isDirty
  • grid.pageNumber
  • grid.pagesInGrid
  • grid.logicalRecordNumber
  • grid.hasDetailView
  • grid.isOnFirstPage
  • grid.isOnLastPage
  • grid.hasNextPage
  • grid.hasPrevPage
  • grid.isNewRecordRow
  • detailView.isDirty
  • detailView.rowState
  • detailView.isOnFirstRecord
  • detailView.isOnLastRecord
  • detailView.hasNextRecord
  • detailView.hasPrevRecord

Note: These fields are only available on the client side. Do not try to use them in Xbasic expressions or server-side events.
