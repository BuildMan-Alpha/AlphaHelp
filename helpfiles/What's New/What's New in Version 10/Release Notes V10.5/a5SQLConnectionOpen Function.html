a5SQLConnectionOpen() Function

This new function can be used as a replacement for the sql::connection.open() method. The advantage of using this function rather than opening the connection on the connection object directly is that if you have set a session username and password, the username and password that you have set will be used when the connection is opened.

Example:

Here is some code that opens a connection directly:

dim cn as sql::connection
flag = cn.open("::name::myconnectionString")
if flag = .f. then 
    ?"Could not connect to database. Error reported was: " + cn.callresult.text
end if

Here is the equivalent code using a5SQLConnectionOpen():

dim cn as sql::connection
'pass in a pointer to the session variables as the third argument!
flag = a5SQLConnectionOpen(cn,"::name::myconnectionString", session)
if flag = .f. then
    ?"Could not connect to database. Error reported was: " + cn.callresult.text
end if
