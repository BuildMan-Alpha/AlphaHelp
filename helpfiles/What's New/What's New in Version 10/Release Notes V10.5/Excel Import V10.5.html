Excel Import

Alpha Five offers 3 different ways to import Excel files:
  • Classic
  • AlphaDAO
  • ADO
The 'Classic' importer was previously limited to Excel V4.0 and older file formats. Now, you can use the 'Classic' importer for Version 2003 and earlier. For the most flexibility, you should continue use the AlphaDAO option. However, in some cases the 'Classic' method will be faster.
