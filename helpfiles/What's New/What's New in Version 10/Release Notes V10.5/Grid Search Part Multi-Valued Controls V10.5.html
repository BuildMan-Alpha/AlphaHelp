Grid - Search Part - Multi-Valued Controls

When using multi-valued controls (checkboxes and dropdown boxes configured to show multiple lines and allow multiple selections) in a Search Part, there are now some new Search Styles that you can set. These are:
  • 201 - Combine selections with 'OR', searching for exact matches
  • 202 - Combine selections with 'OR', searching for the search values anywhere in the field (same as style 20)
  • 203 - Combine selections with 'OR', searching for values that start with the search value.
