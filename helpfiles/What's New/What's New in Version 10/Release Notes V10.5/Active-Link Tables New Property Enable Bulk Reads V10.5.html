Active-Link Tables - New Property 'Enable Bulk Reads'

If you are not sharing access to the SQL table with other users, checking this property will improve performance of active-link tables on large SQL tables in certain cases. This property only affects ODBC based connections (this would exclude Oracle, MySQL and DB2, but include SQL Server and Access). The downside risk of checking this property is that another user might encounter a read lock when trying to use the SQL table.
