Grid Component - Textarea Control - HTML Editor - Click hyperlink to Edit

In the Grid part, when you configure a textarea control to use an HTML editor, at run-time when the user clicks on the control, the HTML editor pops up, allowing the user to start editing the field. In the Detail View part, you have the option of either displaying the HTML editor in-line, or as a pop-up. (One exception is if the Detail View position is set to 'InGrid' the HTML editors in the Detail View are always set to 'pop-up' style.)

Now, you can set a new property for HTML editors to pop them up for editing mode only when the user clicks on an 'Edit' hyperlink. If this option is selected, clicking on the control itself will no longer pop up the editor.

The reason for this change is that the new optional File Upload feature pack allows you to upload files while you are editing in the HTML editor. If you upload a file (say a .pdf file, or an Excel file), a hyperlink to the file is inserted into the field. In order to download the file, you need to click on the hyperlink. You therefore do not want the action of clicking on the control to open the pop-up editor.
