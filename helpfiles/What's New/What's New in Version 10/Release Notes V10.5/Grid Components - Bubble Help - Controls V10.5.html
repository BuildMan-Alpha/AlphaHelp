Grid Components - Bubble Help - Controls

The Grid builder already allowed you to define bubble help for each column in the Grid. However, it did not allow you to define bubble help for all of the different control types.

Now, you can define bubble help for textbox, textarea and link controls. The bubble help you define for these controls can include fieldname placeholders (e.g. {lastname}). The control bubble help property is distinct from the column bubble help property. If bubble help is defined at both the control level and the column level, the help defined at the control level will be shown.
