Reports - Sets - Active Link Tables

When you create a new Report, Label or Letter that is based on a Set that joins Active-Link tables, Alpha Five analyzes the Set structure and the definition of each of the Active-Link tables to see if the Report could instead be based on a SQL Query. If so, it will put up a dialog box suggesting that it is more efficient to base a report on a SQL query than a Set. If you can base the report on a SQL Query (that joins tables on the Database Server, rather than lets Alpha Five join the tables), your reports will render much more quickly.
