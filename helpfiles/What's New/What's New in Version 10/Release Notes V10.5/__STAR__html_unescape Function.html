*html_unescape()

Syntax

C Result = *HTML_UNESCAPE(C text [,C flags])

Argument Description
C result Un-escaped text
C text Escaped text
C flags (optional) 'A' converts ascii-unicode into ascii text

Description

Unescape HTML characters (i.e. back to plaintext, where it can be encoded).

Supported by

V10.5 build 4031-3638 and later
