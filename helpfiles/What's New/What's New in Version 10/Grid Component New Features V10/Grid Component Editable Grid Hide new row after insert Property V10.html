Grid Component - Editable Grid - 'Hide new row after insert' Property

A new property has been added to the Grid. In the case of a Grid that allows inserts, if the new record row(s) are set to 'ShowOnDemand' a new property, 'Hide new row after insert' can be checked to hide the new record row(s) after a record is inserted. Previously, once you had clicked the 'New record' hyperlink to show the new record rows(s), they remained visible until they were explicitly hidden.
