How to Refresh the Grid

In some applications you might be adding or editing data in the table that a Grid is based in another Grid or in some custom Xbasic code. You may want to refresh the Grid that is currently being displayed so that it shows the changes and/or new records that were added in the external process. The Grid object supports three different refresh methods:

  • .refreshRow(rowNum)  - refreshes the current Grid row
  • .refresh() - refreshes current page in the grid.
  • .detailViewRefresh() - refreshes the data shown in the Detail View

For example:

GRID1_GridObj.refreshRow(1)

Where 'GRID1' is the Grid component alias. It must be upper case.
