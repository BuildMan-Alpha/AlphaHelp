Deprecated

Response.Add_Cookie() is deprecated and has been superseded by Response.AddCookie Method?. It will continue to work in Version 10 but will be removed in a future version. Any new code should use the replacement Response.AddCookie Method? and it is recommended that existing code be updated when convenient.
