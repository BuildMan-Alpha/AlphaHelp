Number Functions

$u.n.toStr(NUMBER as number, DECIMALPLACE as number , FORMAT as string, PREFIX as string, SUFFIX as string)

Convert a number into a string. The second parameter allows you to specify decimal places to round to. The optional third parameter specifies format flags. Format flags are:

  • "B" - return a black string if the number equals "0".
  • "," - place commas in the number to separate thousands (comma string used is stored in $u.comma, decimal string is stored in $u.decimal - you can change these values if you wish to use some other separators).
  • "-" - use a "-" before the number if it is negative.
  • "(" - bracket the number in parentheses if it is negative.

The fourth and fifth parameters allow you to specify an optional prefix and suffix.

Example

$u.n.toStr(11222.152,2,',','$',' (US)')

/* result is: '$11,222.15 (US)'*/

$u.n.toStr(-11222.152,2,'(,','$',' (US)')

/* result is: '($11,222.15 (US))'*/

$u.n.toPlaceStr(NUMBER as number)

Convert a number into a place string (i.e. "1st", "2nd", "3rd", "4th", "105th"...)

Example

$u.n.toPlaceStr(23)

/* result is: '23rd'*/

$u.n.round(NUMBER as number, DECIMALPLACE as number, DIRECTION as string)

Round a number to the given decimal place. The direction optional argument allows you to specify the direction you want to round. The "direction" flag has three possible values:

  • "u" - round the number up always.
  • "d" - round the number down always.
  • "normal" - if the value of the digit after the rounding decimal point is less then 5 round down, otherwise round up. This is the default behavior.

Example

$u.n.round(23.12142342342342,1)

/* result is: 23.1*/

$u.n.round(23.12,1,'u')

/* result is: 23.2*/

$u.n.round(23.18,1,'d')

/* result is: 23.1*/
