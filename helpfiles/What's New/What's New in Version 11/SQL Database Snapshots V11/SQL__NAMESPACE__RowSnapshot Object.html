SQL::RowSnapshot

A SQL::RowSnapshot loads and stores the data for a single table row.

Properties

Name Type Description
Name C The unique name of the snapshot to be used if the snapshot is included in a collection of snapshots.
Data SQL::ColumnValue[] A collection of SQL::ColumnValue objects that each contains the data for a specific column in the row.
XML C The persisted XML string that describes the snapshot

Functions

AddData

Add a column to the collection of columns.

L AddData (Column as SQL::ColumnValue)

Adds a snapshot of an individual column to the row snapshot. Returns true if a column was added and false if there was an error.

Clone

Make a complete copy of the object and its contents.

SQL::RowSnapshot Clone ()

Returns a copy of the snapshot

DataNumber

Looks up the index number of the column name provided.

N DataNumber (Name as C)

Returns the index number of the column if found and:1 if the name does not match a column in the collection of values.

DeleteData

Removes the SQL::ColumnValue object at a specific index in the collection of column values.

L DeleteData (Index as N)

Returns true if the column value was deleted and false if there was an error.

InsertData

Inserts a column value into the data collection just prior to the row specified by the argument named InsertBefore.

L InsertData (Column as SQL::ColumnValue [, InsertBefore as N = 1 ] )

Returns true if the column was inserted and false if there was an error.

MoveData

Move a block of one or more column value items within the data collection.

L MoveData( MoveTo as N, MoveFrom as N [, ItemsToCopy as N = 1])

Removes a block of ItemsToCopy column values beginning at the column indexed by MoveFrom and places them before the column indexed by MoveTo. Returns true if the columns were moved successfully and false if an error occurred.

NewInstance

Make a new and empty RowSnapshot.

P NewInstance ()

Returns a new instance of RowSnapshot
