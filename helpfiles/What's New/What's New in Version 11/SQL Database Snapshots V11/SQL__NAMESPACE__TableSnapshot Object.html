SQL::TableSnapshot

A SQL::TableSnapshot loads and stores the definition and data for a single table.

Properties

Name Type Description
CallResult SQL::CallResult An object with properties that describe the result of each call to a function of SQL::TableSnapshot. When the function returns false, you can get more complete information concerning the results using the CallResult property.
Name C The unique name of the snapshot to be used if the snapshot is included in a collection of snapshots.
Row SQL:: RowSnapshot[] A collection of SQL::RowSnapshot objects that each contains the data for a specific row.
TableInfo SQL::TableInfo An object that manages the table definition in the snapshot. The SQL::TableInfo object can be used to generate the scripts to drop and create and update the table.
XML C The persisted XML string that describes the snapshot

Functions

AddRow

Add a row to the collection of rows.

L AddRow (Row as SQL::RowSnapshot)

Adds a snapshot of an individual row to the table snapshot. Returns true if a row was added and false if there was an error.

Clear

Removes all data and definitions from the snapshot.

V Clear ()

Clone

Make a complete copy of the object and its contents.

P Clone ()

Returns a copy of the snapshot

DeleteRow

Removes the RowSnapshot at a specific index in the collection of row snapshots.

L DeleteRow (Index as N)

Returns true if the row was deleted and false if there was an error.

InsertRow

Inserts a RowSnapshot into the collection of row snapshots just prior to the row specified by the argument named InsertBefore.

L InsertRow (Row as SQL::RowSnapshot [, InsertBefore as N = 1 ] )

Returns true if the row was inserted and false if there was an error.

Load

Load definitions and data from a table in an open database connection.

L Load (Connection as SQL::Connection [, TableInfo as SQL::TableInfo [, EventScript as C] ])

Returns a true if the load succeeded and false if an error occurred.

LoadDBF

Load definitions and data from a table in the currently open Alpha Five DBF database.

L LoadDBF( TableName as C, [EventScript as C] )

Returns true if the definitions and data were loaded successfully and false if an error occurred.

MoveRow

Move a block of one or more row items within the collection of rows.

L MoveRow( MoveTo as N, MoveFrom as N [, ItemsToCopy as N = 1])

Removes a block of ItemsToCopy rows beginning at the row indexed by MoveFrom and places them before the row indexed by MoveTo. Returns true if the rows were moved successfully and false if an error occurred.

NewInstance

Make a new and empty TableSnapshot.

P NewInstance ()

Returns a new instance of TableSnapshot

RowNumber

Looks up the index number of the row name provided.

N RowNumber (Name as C)

Returns the index number of the row if found and -1 if the name does not match a row in the collection of rows.

Store

Creates a table and stores data contained in the snapshot into the database referenced by the open connection provided. If the parameter CreateTable is set to false, the table is assumed to exist.

L Store (Connection as SQL::Connection [, CreateTable as L = .t. [, EventScript as C ] ] )

Returns true if the function executed successfully and false if an error occurred.

StoreDBF

Creates a DBF table and populates it from the definitions and data contained in the snapshot. The table is created in the directory of the currently open database and added to the database.

L StoreDBF( [EventScript as C] )

Returns true if the function executed successfully and false if an error occurred.
