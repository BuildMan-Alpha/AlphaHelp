SQL::DatabaseSnapshot

A SQL::DatabaseSnapshot loads and stores the definition and data for a collection of tables.

Properties

Name Type Description
CallResult SQL::CallResult An object with properties that describe the result of each call to a function of SQL::DatabaseSnapshot. When the function returns false, you can get more complete information concerning the results using the CallResult property.
Name C The unique name of the snapshot to be used if the snapshot is included in a collection of snapshots.
Schema SQL::Schema An object that manages the table definitions in the snapshot and the relationships between them. The schema is able to generate the scripts to drop and create tables and to be sure that tables are dropped and added in the right order, accounting for dependencies between them.
Table SQL::TableSnapshot[] A collection of SQL::TableSnapshot objects that each contains the definition and data for a specific table.
XML C The persisted XML string that describes the snapshot


Functions

AddTable

Add a SQL::TableSnapshot object to the collection of tables.

L AddTable (Table as SQL::TableSnapshot)

Adds a snapshot of an individual table to the database snapshot. Returns true if a table was added and false if there was an error.

Clear

Removes all data and definitions from the snapshot.

V Clear ()

Clone

Make a complete copy of the object and its contents.

P Clone ()

Returns a copy of the DatabaseSnapshot

DeleteTable

Removes the TableSnapshot at a specific index in the collection of table snapshots.

L DeleteTable (Index as N)

Returns true if the table was deleted and false if there was an error.

Load

Load definitions and data from an open database connection..

L Load (Connection as SQL::Connection [, Schema as SQL::Schema [, EventScript as C] ])

Returns true if the load succeeded and false if an error occurred.

LoadDBF

Load definitions and data from the currently open Alpha Five DBF database.

L LoadDBF( [EventScript as C] )

Returns true if the definitions and data were loaded successfully and false if an error occurred.

NewInstance

Make a new and empty DatabaseSnapshot.

P NewInstance ()

Returns a new instance of DatabaseSnapshot

Store

Creates tables and stores data contained in the snapshot in the database referenced by the open connection provided. If the parameter CreateTables is set to false, the tables are assumed to exist.

L Store ( Connection as SQL::Connection [, CreateTables as L = .t. [, EventScript as C ] ] )

Returns true if the function executed successfully and false if an error occurred.

StoreDBF

Creates DBF tables and populates them from the definitions and data contained in the snapshot. The tables are created in the directory of the currently open database and added to the database.

L StoreDBF( [EventScript as C] )

Returns true if the function executed successfully and false if an error occurred.

TableNumber

Looks up the index number of the table name provided.

N TableNumber ( Name as C )

Returns the index number of the table if found and -1 if the name does not match a table in the collection of tables.
