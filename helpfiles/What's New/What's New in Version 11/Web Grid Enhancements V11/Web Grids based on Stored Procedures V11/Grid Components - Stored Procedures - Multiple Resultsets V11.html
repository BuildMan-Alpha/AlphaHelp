Grid Components - Stored Procedures - Multiple Resultsets

In some databases, it is possible to create a stored procedure that returns multiple resultsets. If you are basing your Grid on a stored procedure that returns multiple resultsets, you need to specify which resultset to use. You do this by appending a directive to the end of the SQL command to execute the stored procedure. For example:

exec myMultiResultSetStoredProcedure @param1 = '1234'   {ResultsetNumber:2}
