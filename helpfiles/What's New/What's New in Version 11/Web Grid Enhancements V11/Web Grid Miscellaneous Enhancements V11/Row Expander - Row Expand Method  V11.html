Row Expander - Row Expand Method

By default, an icon is shown to the left of the row for the user to click on to expand or contract the row. Now, you set the new 'Row expand method' property to 'Click Icon', 'Click Row', or 'DoubleClick Row'. When the property is set to 'Click Row' or 'DoubleClick Row', then the row expander icon is not shown.

When the row expand method is set to click or double click row, you might want to make your own icon to indicate if the row is open or closed. Here is how you can add a dynamic image control to the Grid to show the state of the row expander.

First, define the following Javascript function in Javascript Function Declarations with this function:

function isRowExpanded(rowNum) {

     var trEle = $( '{Grid.ComponentName}.R' + rowNum + '.__ROWEXPAND_TR');

     if(trEle.style.display == '') return true;

     else return false;

}


Then define a dynamic image control. Set the type to 'client-side' and define these expressions:

Expression Image Name
isRowExpanded(data.rowNumber ) = false $$mini.navigate.next
isRowExpanded(data.rowNumber ) = true $$mini.navigate.down
