Row Expander and Linked Content Section

Linking Fields

Previously, when defining a linked Grid, you could specify up to five pairs of matching fields. Now, you can specify up to 15 pairs of matching fields.

Content Type

Previously you could only display linked Grids in a row expander, or Linked Content section. Now you can display any type of content in a Row Expander or Linked Content Section. This includes .A5W pages, custom Ajax callbacks, Dialog, Page Layout, Tabbed UI and Custom components.

Security

You can now assign security groups to each object shown in a row expander or Linked Content section. For example, when you expand the row expander, three different Grids might be shown in a tab object to users who are in the Administrator Group, but only a single Grid might be shown to users in the Sales Group.

Display Additional Content Types

Previously, the Row Expander could only display nested Grid components. Now, the Row Expander can display the following nested components:
  • Grids
  • Dialogs
  • PageLayout Components
  • Custom Components
  • Any User Defined Components
  • A5W Pages

Security

When you defined the content to be shown in a Row Expander or Linked Content Section, you can now specify security groups for each object. For example, when someone in the Administrator group expands a row, she might see Grid1 and Grid2, but someone in the Sales group might only see Grid2.
