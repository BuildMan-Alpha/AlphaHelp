Grid Component - Search Part - Query By Form Operator Exceptions

A new property has been added to the Grid to allow you to specify exceptions to the list of QBF Operators. The QBF operators are:
  • ..
  • =
  • ,
  • >=
  • <=
  • >
  • <
If you have turned on Query by Form in the search part, and you enter a search value of 'Alpha..Gamma', this would normally be interpreted as a search for values greater or equal to 'Alpha' and less than or equal to 'Gamma'.

However, if you specify that the two dots operator (..) is an exception, then the search will look for records with a value of 'Alpha..Gamma'.

However, other QBF operators will still be honored. So, for example, if you enter 'Alpha..Gamma,Zeta' as your search criterion, this would be interpreted as a search for records where the value is 'Alpha..Gamma' or 'Zeta'.
