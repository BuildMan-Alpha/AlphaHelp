Grid and Dialog Components - 'Smart' Fields 'Container Inline Style' Property

'Smart' fields on the Grid and Dialog (i.e. Edit-combo, Lookup, Date/time pickers, etc), now have a 'container inline style' as well as the 'inline style' property.

The 'container inline' style property applies to the container element that surrounds both the <input> control and the associated icon. The 'inline' style property applies to the actual <input> control. If you want to set the width of the control AND the associated icon, you would set the 'container inline' style property. If you set a width property in both the 'container inline style' and 'inline style' property, the width property on the 'inline style' is thrown out, and only the width propert of the 'container' is honored.
