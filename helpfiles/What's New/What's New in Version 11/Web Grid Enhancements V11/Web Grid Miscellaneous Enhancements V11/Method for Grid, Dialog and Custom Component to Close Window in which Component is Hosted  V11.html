Method for Grid, Dialog and Custom Component to Close Window in which Component is Hosted

Using JavaScript, it is easy to create an action that opens a Grid, Dialog or Custom Component in a pop-up window. This new method allows you to create an action in the component to close the window in which the component is hosted.

For example:

{grid.object}.closeContainerWindow(ele);

where 'ele' is the ID of an element in the window. Typically, this action will be on a button, and 'ele' will be set to 'this'.

For example:

{grid.object}.closeContainerWindow(this);
