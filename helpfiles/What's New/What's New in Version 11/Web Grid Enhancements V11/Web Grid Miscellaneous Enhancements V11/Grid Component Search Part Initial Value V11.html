Grid Component - Search Part - Initial Value

The textbox controls in the Search Part now have a new property - Initial Value. This allows you to set default values in textbox fields when the Grid is initially displayed.

You can enter expressions by entering a leading = sign. For example, the 'Date' field might have this default value:

=date()


If the Search Field is configured as a Range Search, enter two values, separated by a double pipe. For example:

=date()||=date()+5


This would fill in today's date in the 'From' field and the date 5 days from today in the 'To' field.
