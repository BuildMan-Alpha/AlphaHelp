Grid and Dialog Control - .activateControlContainer() Method

If a control is inside a tab or accordion, this method will give focus to the tab/accordion pane in which the control is located. The method works regardless of how many levels down the control is nested. For example, you might have a Dialog with a tab control inside another tab control.

Syntax for Grid:

{grid.object}.activateControlContainer(colName,part,rowNum)

Where part is 'G' or 'D' (Grid part, DetailView part)

Example:

{grid.object}.activateControlContainer('LASTNAME','G',5)

Syntax for Dialog:

{grid.object}.activateControlContainer(colName)
