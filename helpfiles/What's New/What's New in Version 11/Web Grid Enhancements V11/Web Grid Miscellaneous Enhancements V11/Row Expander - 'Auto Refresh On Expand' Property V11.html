Row Expander - 'Auto Refresh On Expand' Property

By default, when you open the row expander for a given row, an Ajax callback is made to get the row expander contents. But if you then close that row and then re-open it, a callback is not made - the cached row expander contents is shown. Now, you can set a new property to force an Ajax callback every time the row expander for a given row is opened.
