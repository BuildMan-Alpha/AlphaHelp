Grid - Search Part - Free Form Search Layout

When the Search Part is embedded in the Grid Part, there is an option for free-form layout of the Search Part. If you choose this option, there are now additional placeholders that you can use when you construct the free-form layout. These are:
  • {<SortOrderListBox>}
  • {<SortDirectionListBox>}
  • {<RecordsPerPageListBox>}
  • {<SaveSearchButton>}
  • {<LoadSearchButton>}
  • {<SearchHelpButton>}
