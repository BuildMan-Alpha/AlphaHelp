Grid Component - Client-side Events - 'canNewRowsShow' event

This new event fires when the user tries to show the new record rows. (If the Grid is set to 'show new record rows on demand'). The method has a single parameter - 'rowsInEditMode' - the number of existing record rows that are in edit mode. This method can be used to prevent the user from displaying the new record rows if any of the existing records are in edit mode. For example:

if(e.rowsInEditMode == 0) return true;
else return false;
