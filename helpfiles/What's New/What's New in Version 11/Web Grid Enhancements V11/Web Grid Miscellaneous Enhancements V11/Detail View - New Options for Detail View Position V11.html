Detail View - New Options for Detail View Position

There are new options for the position of the Detail View. These options are:

  • SplitView_Left - puts the Detail View to the left of the Grid part
  • SplitView-Right - puts the Detail View to the right of the Grid part
  • DIV - puts the Detail View in an arbitrary DIV (this div must NOT be placed in a freeform edit region that surrounds the Grid, but can be placed in the Master Template - if you design a custom Master Template)
