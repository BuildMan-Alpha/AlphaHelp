PDF Printing - Amyuni Printer Driver - New Version - Version 4.5

We have updated the version of the Amyuni PDF Printer Driver that is bundled with Alpha Five. We now bundle V4.5 of the Amyuni Printer Driver. Amyuni's web site claims that this version performs better on 64 bit operating systems. It also removes the limitation that watermarks were not working when using a 64 bit operating system. After you install the update, you will notice that the Printers folder in the Windows Control Panel now has a new printer called 'AlphaFivePrinterV4p5'. The original printer installed by V11, 'AlphaFivePrinterV4' can be left there. This printer will be used by V10 if you have V10 and V11 installed on the same machine.
