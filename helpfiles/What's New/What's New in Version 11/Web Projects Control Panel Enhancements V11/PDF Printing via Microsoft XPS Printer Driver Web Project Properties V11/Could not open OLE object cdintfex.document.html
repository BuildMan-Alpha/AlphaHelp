Could not open OLE object 'cdintfex.document.4'

If you get this error, your Amyuni ActiveX controls are not properly registered.
You can fix this issue by opening a Command Window as an administrator and then running a batch file that Alpha Five installs (Registerfiles.bat, or RegisterFilesVistaWin7.bat).

Here is how you do this:
  • Click the Windows Start button
  • Type 'cmd' in the Search Bar. It should find 'cmd.exe'
  • Right click on 'cmd.exe' and select 'Run as administrator'.
  • Navigate to the folder where Alpha Five, or the Alpha Five Application Server is installed
  • Type 'RegisterFilesVistaWin7' and press Enter (if you are using XP, type 'RegisterFiles')
