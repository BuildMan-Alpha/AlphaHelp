Office::Excel2003Document Class

The Office::Excel2003Document class is the parent class for creation and editing of Excel 2003 documents.

Properties

All properties in this class are read-only
  • ActiveSheetIndex as N
  • CallResult as CallResult
  • ErrorMessage as C
  • FileName as C
  • FontCount as N
  • FormatCount as N
  • SheetCount as N
