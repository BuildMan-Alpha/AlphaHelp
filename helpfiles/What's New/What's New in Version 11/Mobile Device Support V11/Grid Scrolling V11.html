Grid Scrolling

To support a common design pattern used in mobile applications (e.g. Gmail on the iPad or iPhone), the Grid now has a new option for scrolling the Grid body section while the title and footer remain fixed in place.
Note
When you select this option, it is important to also select the Fixed Column Widths property.
When this option is checked, you can then specify a property to control the height of the scrollable section. If there are more records that can be displayed, the Grid gets a vertical scrollbar (on a Desktop browser). On a mobile browser, you can use a gesture to scroll the records in the Grid.
