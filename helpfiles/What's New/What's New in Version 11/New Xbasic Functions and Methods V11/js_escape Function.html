js_escape()

Syntax

C Result = JS_ESCAPE(text as c [, flags as C])

Argument Description
Result as c Escaped string
text as c Original string
flags as c U - encode as Unicode, UTF8 - encode as Unicode, assume that input is UTF8

Description

Escape a string to embed in emitted javascript.

Supported by

Version 11 and later
