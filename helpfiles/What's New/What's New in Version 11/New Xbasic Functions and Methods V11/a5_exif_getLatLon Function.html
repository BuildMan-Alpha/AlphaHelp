a5_exif_getLatLon()

Syntax

P Result = a5_exif_getLatLon(C fileName )

Argument Description
P Result A dot variable holding the Lat, Lon, and other properties
C fileName The filename of an image that may contain EXIF data


Result properties
Result Description
C lat Latitude
C lon Longitude
C json Lat, Lon formatted for Google Maps
L haserror .T. if there was an error
C errorText Error text, if any


Description

Reads a jpeg file and extracts GPS Latitude and Longitude, formatted for use with Google Maps, if the information is present.

Limitations

Examples

dim latlon as p
latlon = a5_exif_getLatLon("c:\images\photo.jpg")
? latlon
