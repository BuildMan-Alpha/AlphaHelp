argVal()

Syntax

A value = argVal(SQL::Arguments args ,A NameOrIndex )

Argument Description
A value argument value
SQL::Arguments args Arguments object
A NameOrIndex Argument name or index

Description

Returns a value from a SQL::Arguments object by name or item index. It is a convenience function that is basically a shortcut for this formal pattern: args[args.argumentNumber(ArgumentName)].data

Limitations

Examples

dim args as sql::arguments
args.add("name","Fred")
name = argVal(args,"name")
