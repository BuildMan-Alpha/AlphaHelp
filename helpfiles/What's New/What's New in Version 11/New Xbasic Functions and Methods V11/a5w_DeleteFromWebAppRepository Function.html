a5w_DeleteFromWebAppRepository()

Syntax

L flagError = a5w_DeleteFromWebAppRepository(C key [,* pResult ])

Argument Description
L flagError Error flag
C key Unique key of repository item
P pResult (optional) Pointer for results if desired;after the function executes, pResult.hasError is .t. or .f. and pResult.errorText is the error text if there was an error.

Description

Used in a Web application to delete data from the Web Application Repository.

Supported by

Version 11 and later
