a5ws_RevalidatePassword()

Syntax

L Result = a5ws_RevalidatePassword(C Password [,P request ])

Argument Description
C Password Password to validate for the current user
P request (optional) Request object

Description

Web Security - Verify the currently logged in user's password ONLY.

The function tests if the supplied password matches the password for the currently logged in user only. It returns only true or false, request is required, and it only runs in a published application. If no one is logged in, or it is run from the local project, it returns false.

Limitations

This can only be run from the Application Server
