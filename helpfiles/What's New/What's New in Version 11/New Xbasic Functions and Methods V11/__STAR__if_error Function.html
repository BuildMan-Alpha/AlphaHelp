*IF_ERROR()

Syntax

A Result = *IF_ERROR(A expr, A errorValue)

Argument Description
A expr expression to evaluate and return if no error
A errorValue error value to return

Description

Adds error checking around an expression - if there is an error in the expression, the errorValue is returned.

Limitations

Examples

goodvar = 3
? *IF_ERROR(goodvar,2)
= 3
? *IF_ERROR(badvar,2)
= 2
