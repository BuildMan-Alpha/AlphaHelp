find_duplicate_lines()

Syntax

C Result = find_duplicate_lines(C list )

Argument Description
C list crlf-delimited list of values.

Description

Finds duplicate entries in a crlf-delimited list of values.

Supported by

Limitations

Examples

list = comma_to_crlf("alpha,beta,gamma,alpha")
?find_duplicate_lines(list)
= "alpha"
