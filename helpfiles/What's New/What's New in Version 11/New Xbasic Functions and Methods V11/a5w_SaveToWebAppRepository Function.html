a5w_SaveToWebAppRepository()

Syntax

L Result = a5w_SaveToWebAppRepository(C key ,C Data [,C objectType [,C componentGuid [,C userId ]]])

Argument Description
C key Key value of the data you are saving. This is how data are retrieved from the repository. Must be a unique value. Limit 40 characters.
C Data The data to save in the repository. Must be character data. No length limit.
C objectType (optional) A description of the type of data; the description can be determined by the user
C componentGuid (optional) The GUID of the component from where the data was saved
C userId (optional) The userId of the user who saved the data in the repository

Description

Used in a Web application to save data to the Web Application Repository. Returns .t. if the data was saved, or .f. if it was not saved.
