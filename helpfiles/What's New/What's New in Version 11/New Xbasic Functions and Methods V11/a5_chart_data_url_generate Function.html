a5_chart_data_url_generate()

Syntax

C Result = a5_chart_data_url_generate(P objSettings ,P customChart ,N width ,N height )

Argument Description
C Result Generated REST URI string to pass to the chart provider, typically the Charting::Chart Class
P objSettings A Reporting::ReportChart object
P customChart A Charting::ChartDataDefinition object or, often, a function that returns a Charting::ChartDataDefinition
N width The desired width of the chart in pixels
N height The desired height of the chart in pixels

Description

Generates the URI string that describes a chart for an embedded chart object. This string can be used by the .NET Framework System.Web.UI.DataVisualization.Charting.Chart class via the Alpha Five Charting::Chart Class discussed as part of the Chart API V11.

Limitations

Examples

dim objSettings as Reporting::ReportChart
objSettings.chartType = "Pie"
dim uri as c = a5_chart_data_url_generate(objSettings,PieChart(),500,300)
