a5_html_label()

Syntax

C Result = a5_html_label(value as a[,searchHighlights as c])

Argument Description
Result as C HTML text output
value as a value to convert
searchHighlights as c (optional) A CSS class name followed by = and the pattern(s) to match for highlighting

Description

Convert any data type to html text, optional search highlighting feature.

Supported by

Version 11 and later

Limitations

Examples

? a5_html_label(now())
= "01/16/2012 03:41:30 27 pm"

? a5_html_label(1)
= "1"

? a5_html_label("hello world","highlight=world")
= hello <span class="highlight" A5HL="Y" >world</span>

? a5_html_label("hello world","highlight=world"+crlf()+"hel")
= <span class="highlight" A5HL="Y" >hel</span>lo <span class="highlight" A5HL="Y" >world</span>
