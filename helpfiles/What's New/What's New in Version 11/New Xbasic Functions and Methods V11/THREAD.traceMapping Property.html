THREAD.traceMapping Property

Syntax

A cr-lf delimited list that contains pairs of <pane>=<filename>. It also supports wildcards that allow redirection of all remaining panes (or all panes if this is the only mapping defined) to folders with filenames generated to include the trace panel name.

t.traceMapping = "*=c:\dev\trace\trace_*.log"

Description

The purpose of this property to allow redirection of trace output to a file, based on the name of the pane in the trace window to which the output would otherwise go.

Supported by

Version 11 and above

Limitations

Examples

In an interactive session

' Get a pointer to the current thread
t = thread.get(thread_get())

' define redirection for user and sql panes of trace panel - put
output in file instead (only in scope of the current thread).
t.traceMapping = <<%str%
#user=c:\dev\traceLog.txt
#sql=c:\dev\traceLog_Sql.txt
#%str%

trace.WriteLn("This goes to default trace")
trace.WriteLn("This goes to SQL trace","SQL")

? file.to_string("c:\dev\traceLog.txt")
= This goes to default trace

? file.to_string("c:\dev\traceLog_Sql.txt")
= This goes to SQL trace


In an A5W page

<%a5
t = thread.get(thread_get())

t.traceMapping = <<%str%
user=c:\dev\wtraceLog.txt
sql=c:\dev\wtraceLog_Sql.txt
%str%

trace.WriteLn("This goes to default trace<br>")
trace.WriteLn("This also goes to default trace<br>")
trace.WriteLn("This goes to SQL trace<br>","SQL")

? file.to_string("c:\dev\wtraceLog.txt")
? "Above: Default<br><hr>Below: SQL<br>"
? file.to_string("c:\dev\wtraceLog_Sql.txt")

%>
