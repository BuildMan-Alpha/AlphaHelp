a5_exif_getIphoneFormattedData()

Syntax

P Result = a5_exif_getIphoneFormattedData(C fileName )

Argument Description
P Result A dot variable holding the Lat, Lon, photo parameters and other properties
C fileName The filename of an iPhone photo


Result properties

Result Description
C html HTML formatted results
C results A dot variable holding all found properties
C json All found properties formatted for JavaScript
L haserror .T. if there was an error
C errorText Error text, if any

Description

Extracts html and JSON formatted exif data from an iPhone photo.

Supported by

Version 11 and later

Limitations

Examples

dim formatted_exif as p
formatted_exif = a5_exif_getIphoneFormattedData("c:\images\photo.jpg")
? formatted_exif
