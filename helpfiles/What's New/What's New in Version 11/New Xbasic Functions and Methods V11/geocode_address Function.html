geocode_address()

Syntax

P Result = geocode_address(C address [,C method ])

Argument Description
P Result Properties from geocoding: .lat,.lon, .error_code and .error_text
C address Address to be geocoded
C method (optional) "Nominatim" or "Google" (default)

Description

Geocode an address using a Web Service, either OpenStreetMap (Nominatim) or Google. Returns .lat and .lon. Also returns .error_code and .error_text. Address must be entered with commas between each part. e.g. 123 Main St., Boston, MA, 02116, USA

Nominatim does not impose any limits on the number of requests made per day. Google has a query limit of 2,500 geolocation requests per day and a rate limit to prevent abuse. Google often provides better results than Nominatim, however.

Google Premier has much higher limits, but this function does not currently support Google Premier. If you need Google Premier support, please contact us.

Supported by

Version 11 and later

Limitations

Must have a working Internet connection. See the Description section for the limitations of each geocoding provider.

Examples

dim address as C = "70 Blanchard Rd, Burlington, MA"
?geocode_address(address, "Nominatim")
= error_code = 0
error_text = ""
lat = "42.478836"
lon = "-71.2017918888889"
 
?geocode_address(address, "Google")
= error_code = 0
error_text = ""
lat = "42.4785700"
lon = "-71.2033580"
