Tabbed UI and Page Layout Components in Xdialog Windows

You can now create Xdialog windows that host TabbedUI and PageLayout web components. There are two ways in which your can do this:
  • Right click on a Tabbed UI or Page Layout in the Web Control Panel and select 'Xbasic Script for Desktop Windows', or
  • create a new Action Script. Action Scripting has a new category, 'Web Components' that allows you to create Xdialog windows that host your web components.
The ability to use TabbedUI and PageLayout components in Xdialogs (i.e. in applications designed to be run on the desktop) is very significant because, combined with the existing ability to host Grid components in Xdialogs, it means that you can create an entire desktop application using Web components. The advantage of taking this approach to creating a desktop application is that when you are ready to move your application to the web, you don't have to do any additional work - all of the components that you desktop application can be used 'as is' on the web.
