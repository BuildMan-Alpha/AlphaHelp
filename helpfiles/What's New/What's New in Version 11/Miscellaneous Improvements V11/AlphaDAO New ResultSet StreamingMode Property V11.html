New AlphaDAO <ResultSet>.StreamingMode Property

<resultset>.StreamingMode affects the behavior of <resultset>.ToString(), <resultset>.ToDBF() and <resultset>.NextRow()

A common pattern in Xbasic after you execute a query that returns a resultset is to use the .NextRow() method on the resultset to see if the resultset contains any data. If the .NextRow() method returns .t., then you know that the resultset contains data. Using the .NextRow() method in this manner (before you actually try to read any data from the resultset) works well in the pattern shown below because the first call to .NextRow(), if successful, positions the record pointer on the first record, not the second record, as you might suspect. For example:

dim cn as sql::connection 
?cn.open("::Name::sqlserver_alphasports")
= .T.
?cn.Execute("select lastname, bill_state_region from [customer with space] where bill_state_region = 'ca'")
= .T.
rs = cn.ResultSet
?rs.NextRow()
= .t. 
?rs.data("Lastname")
= "Winka"   'this is the first record in the query, not the second as you might suspect
rs.NextRow()
?rs.data("Lastname")
= "Harrington" 'this is the second record in the query

The technique of using <resultset>.NextRow() did not work correctly if you then called the .ToString(), .ToDBF() or other data streaming methods on the resultset.

For example:

dim cn as sql::connection 
cn.open("::Name::sqlserver_alphasports")
cn.Execute("select lastname, bill_state_region from [customer with space] where bill_state_region = 'ca'")
rs = cn.ResultSet
?rs.NextRow()
= .T.
?cn.ResultSet.ToString()
ERROR: Database API specific error while converting ResultSet to String
[Microsoft][SQL Server Native Client 10.0]Invalid Descriptor Index
SQL State is: 07009

However, you can now set the new .StreamingMode property on the resultset, so that it is still possible to use the .NextRow() method to test if the resultset has data. For example:

dim cn as sql::connection 
cn.open("::Name::sqlserver_alphasports")
cn.Execute("select lastname, bill_state_region from [customer with space] where bill_state_region = 'ca'")
rs = cn.ResultSet
cn.ResultSet.StreamingMode = .t.
?rs.NextRow()
= .T.
?cn.ResultSet.ToString()
= Winka CA 
Harrington CA 
McAndrews CA 
Burtonski CA 
Harrison CA 
Mahoney CA
