Tabbed UI - Home Page - Auto-refresh Property

You can now specify that the Home Page on the Tabbed UI should be refreshed every time the user returns to the home page. This is useful if your home page contains 'dashboard' type information. If the home page is a .a5w page, it might be necessary to include the following two meta tags in the <head> section of the .a5w page to force the browser not to cache the page contents:

&lt;META HTTP-EQUIV="Cache-Control" CONTENT="no-cache" /&gt;
&lt;META http-equiv="expires" content="0" /&gt;
