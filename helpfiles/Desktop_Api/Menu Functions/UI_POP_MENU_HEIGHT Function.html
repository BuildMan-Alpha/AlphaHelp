UI_POP_MENU_HEIGHT()

Syntax

Height as N = UI_POP_MENU_HEIGHT( Menu_Name as C )

Argument

Description

Height

The height of the menu in inches.

Menu_Name

The name of a menu.

Description

The UI_POPUP_MENU_HEIGHT() function returns the height of a menu.

Limitations

Desktop applications only.

Example

? ui_popup_menu_height("menu1")

= 0.322917
