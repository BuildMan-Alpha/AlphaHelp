PropertyGrid.IsExpanded()

Syntax

Open_Status as L = PropertyGrid.IsExpanded( Name as C )

Argument

Description

Open_Status

.T. = Category is open (expanded)

.F. = Category is closed

Name

The name of a category in a property grid.

Description

The PropertyGrid.IsExpanded() method reports whether a named category within a property grid is open (expanded) or closed.

Note : Only the {PROPGRID} command can create the PropertyGrid object.

Example

Refer to {PROPGRID} for an example.
