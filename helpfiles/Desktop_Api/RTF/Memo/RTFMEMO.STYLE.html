.STYLE

Style Properties

Element

Type

Description

Style.First_Left_Indent

N

Indent of first line of paragraph.

Style.Left_Indent

N

Left indent of paragraph.

Style.Right_Indent

N

Right indent of paragraph.

Style.Name

C

Name of style.

Style.Tab.Align

C

Alignment of tab.

Style.Tab.Pos

N

Position of tab.

Description

The RTF Memo Editor supports up to 8 different styles.

Limitations

Desktop applications only.

Examples

dim obj as P

obj = :Customer_Information:rtfmemofield.this

obj.rtf.insert_text("This is RTF text")

obj.rtf.style.name = "My Style"

obj.rtf.style.left_indent = 1

obj.rtf.style.right_indent = 1

? obj.rtf.style.

= First_Left_Indent = 0

Left_Indent = 1

Name = "My Style"

Right_Indent = 1

+Tab.
