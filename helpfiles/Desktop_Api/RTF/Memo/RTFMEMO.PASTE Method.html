.PASTE()

Syntax

.PASTE()

Description

The .PASTE() method pastes from the clipboard into the RTF object at the cursor location.

Example

The following example pastes text into an RTF field in an open form.

dim obj as P

obj = :Customer_Information:rtfmemofield.this

obj.rtf.insert_text("This is RTF text")

obj.rtf.select(1,10)

obj.rtf.cut()

obj.rtf.paste()
