.INSERT()

Syntax

.INSERT( Text_to_Insert as C )

Argument

Description

Text_to_Insert

The text to insert at the current cursor position

Description

The .INSERT() method inserts text at the current cursor position in an RTF Memo field.

Limitations

Desktop applications only.

Examples

The following example inserts text into an RTF field on an open form.

dim obj as P

obj = :Customer_Information:rtfmemofield.this

obj.rtf.set_cursor(1)

obj.rtf.insert_text("This is RTF text")
