                    <title>Setting up the Development Server</title>
                    <steps>
                        <step>
                            <description>On the Web Projects Control Panel, open the Development Server Settings. They can be accessed under the Web menu. Select "Development Server".</description>
                        </step>
                        <step>
                            <description>Change the Server Port to another port number. Port 80 is typically in use by other software on a computer. Pick a high-number port, such as 8080, for your Server Port setting.</description>
                        </step>
                        <step>
                            <description>Click *[ui:Save]* to save the changes to the Develpoment Server. If the Development Server was already running, click the *[ui:Restart Server]* button for the changes to take effect. If the Development Server is not running, click *[ui:Start Server]* to start the server.</description>
                        </step>
                        <step>
                            <description>Click *[ui:Close]* to close the Development Server Settings and return to the Web Projects Control Panel.</description>
                        </step>
