Starting the Web Application Server

To start the Web Application Server :

  1. In the main toolbar, click Web Application Server and click Yes. The button will change to Web Application Server.

  2. Alternatively, click Tools > Web Application Server > Start Server.

To display the Web Application Server Control Panel, click Tools > Web Application Server.

In the resulting dialog, make sure that you have properly configured the Web Application Server. For details, refer to Configuring and Running the Server.

 

 

The Server Status icon will change from red to green to show that the server is running. If you wish to stop or restart the server, you may use the Stop Server and Restart Server buttons. Note the Document Root setting. This is where you want to place the HTML pages that the Web Application Server will server up to client applications. You can now minimize the Web Application Server Control Panel to get it out of your way.

Starting the Web Application Server Automatically

The Web Application Server does not currently run as a Windows service. In order to have it start up automatically check the Automatically start the Web Application Server check box on the General tab of the Web Application Server Control Panel.
