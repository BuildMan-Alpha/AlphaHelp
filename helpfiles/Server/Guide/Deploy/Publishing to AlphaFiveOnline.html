Publishing to AlphaFiveOnline

Here is what you need to do to configure your Alpha Five applications so that they will work with your account.

To configure Alpha Five to publish to your account:

  1. Open the Alpha Five Control Panel.

  2. Select Tools > Web Projects Control Panel to display the Web Projects Control Panel.

  3. Click once on the Web Projects Control Panel so that it becomes the active window.

  4. Select Web > Project Settings to display the Project Settings dialog.

  1. Select "" from the Profile list.

  2. Replace the General > Profile Name with a profile name of your choice.

  1. Select "Publish via FTP" in the General > Publishing Type list.

  2. Click in the General > Aliases property to display the Define Path and Connection Aliases dialog.

  3. Modify the first entry to the following: PathAlias.ADB_Path =

  1. Click OK to return to the Project Settings dialog.

  6. Check Publish Database Files > Publish data files.

  7. When you know what database files you need to publish, itemize them under Publish Database Files > Files to publish.

  8. Change Publish Database Files > Folder property to "DATA ".
