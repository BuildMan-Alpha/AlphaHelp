Install a hotfix

Alpha Anywhere has a utility called A5HotFix that patches the System.aex file with hotfix aex files that are installed in the hotfix folder.

The directions to install the hotfix in your build are

  1. Copy the hotfix file into the hotfix folder, this will often be "c:\Program Files (x86)\A5v12\hotfix".
  2. Make sure there are no instances of "alpha5.exe" or "A5ApplicationServer.exe" running - you can use task manager to determine this (In windows 10, make sure to go to the Details tab and sort on the 'Name' column).
  3. Open a command window in Administrator mode (you can do this by right clicking on the command menu item and selected 'Run as Admininstrator')
  4. Go to the EXE folder (i.e. cd "c:\Program Files (x86)\A5v12")
  5. Type A5HotFix, which should provide feedback.
  6. If there were no problems (i.e. a hotfix generally valid only for a specific range of releases of alpha five) the fix will be installed.
  7. If you do run into problems do to the installed release being to old, you may not have the latest available build, install the latest prerelease of Alpha Anywhere before applying the hotfix.
