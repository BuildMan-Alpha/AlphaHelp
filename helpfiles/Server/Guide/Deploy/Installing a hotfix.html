Adding a Hotfix

A hotfix can be added to the Alpha Anywhere full development program and / or the application server program.

The [program folder] is the program folder for the program executable that will get the hotfix.

The default folders would be:

alpha5.exe - C:\Program Files (x86)\a5V12

A5ApplicationServer.exe - C:\Program Files (x86)\a5V12 ApplicationServer

  1. Stop all running instances of any Alpha program. That would include any instance of

    Alpha5.exe, or A5ApplicationServer.exe.You may need to use the Windows Task Manager to confirm.

  2. Open Windows Explorer and navigate to the [program folder].
  3. Create a new sub-folder under the [program folder] called “hotfix”.  If the folder already exists, remove any existing files in the folder.
  4. Unzip or copy the hotfix file into this new folder.
  5. In Windows Explorer find the “A5Hotfix.exe” executable file in the list of files (in the executable). DO NOT double click on this entry. Instead, right click and select to “Run as administrator”. This will install the hotfix.

  6. Open the [program folder] and find the file “System.aex”.

    The time stamp on the file should be the current time after the hotfix installed.

  7. Restart the program.
