Select Image Dialog Box


 

 

 

 

Add an image to a project:

  1. Click Add Image to Project.

  2. In the resulting Add Image to Project dialog box:

    1. Click to navigate to and select an image file in the Source filename field.

    2. Optionally, modify the Target filename.

    3. Click OK.

To place an image on a HTML page:

  1. In the HTML Editor select Insert > Image.

  2. Optionally, select an image from the list of images in the project.

  3. Optionally, click Select Image Outside Project.

    1. Optionally, make a selection from the Files of type list. The options are:

      • "BMP - Windows Bitmap (*.bmp)"

      • "GIF - CompuServe GIF (*.gif)"

      • "JPEG/JPG - JPEG Images (*.jpeg;*.jpg)"

      • "PNG - Portable Network Graphics (*.png)"

      • "All Files (*.*)"

    2. Navigate to, select the image, and click Open.

  4. Optionally, change the display size of the image.

    1. Click Get Image Size.

    2. Set the Height image size and units.

    3. Set the Width image size and units.

  5. Click OK to insert the image.
