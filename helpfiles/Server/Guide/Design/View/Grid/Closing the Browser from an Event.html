Closing the Browser from an Event

An event script cannot directly close a browser, because its code is executed on the Application Server and does not manipulate the browser.

However, you can redirect the browser to page that contains the appropriate JavaScript to close the browser. The code for close.html is illustrated below.

Note : By default, Mozilla will not let you close a parent page using JavaScript.

The Event Code

...

CurrentForm.RedirectTarget = "close.html"

...

The Code for Close.HTML

<html>

<body onLoad= "window.close();" ></body>

</html>
