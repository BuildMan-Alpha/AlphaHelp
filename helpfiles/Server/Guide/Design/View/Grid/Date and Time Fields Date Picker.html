Date and Time Fields - Date Picker

You can now turn off the Date picker for Date and Time fields. By default, for every date and time field, the Grid shows an icon that you can click to open the date picker. Turning off the date picker is useful if you want to use a 3rd party date picker, such as a JQuery date picker. The ability to use 3rd party controls in the Grid is another new V11 feature.

You can also configure the method used to open the date picker. In V10 the only method was to display an icon which the user clicks to open the data picker. In V11, you can set the open method to either 'ClickIcon' or 'Focus'. If you select 'Focus' then the icon is not shown and the date picker is opened automatically as soon as the user clicks in the date field.
