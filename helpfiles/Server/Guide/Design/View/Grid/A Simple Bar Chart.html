A Simple Bar Chart

 

This example shows how to use the value of a field in a freeform layout to modify the width of a graphic, thereby producing a simple bar chart. This example will use the Alphasports database .dbf Tables for its data. The database should be located in the Samples folder under the Alpha Anywhere installation folder. If you are not familiar with the Grid Builder, go first to the Web Projects Control Panel by clicking the Web Projects button on the Alpha Anywhere toolbar.

Once you are in the Web Projects Control Panel, click the 'Create a new grid component' button near the top left corner of Web Projects Control Panel toolbar. This will load the 'Select Grid Component Template' Dialog, that gives you the option to build your grid from existing grid designs. Select 'Start with a blank Grid Component' and click OK. The Grid Builder dialog will open. We will now create a new tabular grid based on the AlphaSports product table.

 

Creating the Grid

  1. Make sure the Design tab of the Grid Builder is open. Click on Component Type at the top of the tree menu.

  2. With the Component Type page open, go to the Grid Layout menu in the middle of the page and select Tabular.

  3. Leave the Grid is: Read only radio button on the Component Type page selected.

  4. On the design tree menu select the Grid > Data Source page.

  5. On the Data Source page select the .dbf Tables radio button.

  6. Go back to the design tree menu and click on Grid > Query (DBF).

  7. Select the 'Display data from: Table' radio button at the top of the Grid > Query(DBF) page.

  8. Next to the Table name: setting click the  to open a drop down of all the tables in Alphasports. Select the "product" table.

  9. Click Calculated Fields to display the Define Calculated Fields dialog.

  10. Enter the following expression and click OK. This creates a new calculated field named "Markup".

 

markup = (Retail - Cost)/Cost * 100

 

Selecting the Fields to Display

  1. In the design tree menu click on  the Grid > Fields page.

  2. Move the "Description", "Cost", "Retail", and "Markup" fields from the Available Fields list to the Selected Fields list using the single right pointing arrow between the two lists .

  3. You will need to set the currency type for your "Cost" and "Retail" fields. From the Selected list click on one of the two to start with.

  4. Now look at the Field Properties menu on the right-hand side of the screen. Go to the Display Settings list and on the Display format option click to open the Display Format dialog.

  5. Click on the International Currency Format button. From the Country: drop-down list at the top of the International Currency Format Dialog choose a country and click OK. The formatCurrency function should appear in the Display Format Dialog. Click OK again and repeat the procedure for your second field.
     

 

Note: If you have many fields that you would like to set to the same field property use the paste tool to. This will let you paste the current property value to multiple controls at once.

  1. From the Selected list click on the "Markup" field. In the Field Properties menu scroll down to Label Properties. Click the to open the Style Editor. In the Style Editor scroll down to the Text property menu. Find the Alignment property, click the dropdown button, and select 'left'. Click OK.


    2. Select Alignment for a Field
     

  2. With the "Markup" field still selected, scroll down the Field Properties  to Column Properties. Click on the Freeform layout check box to check it.

  3. Check to make sure there is no HTML code in the "Markup" field's  Display Settings > Display format setting.

  4. Click the   button in the Column Properties > Freeform template property setting to display the Free-form Column Layout dialog. A freeform field may contain any legitimate HTML code. In this case it will contain an <div> tag. Enter the following HTML code into the dialog box and click OK. When you publish the  page later on, the value of the "Markup" field will replace the {Markup} place holder. The background color will be the color of your bar chart.

 

<div style="display: inline-block; width: {data.Markup}px; height: 10px; background: green;"></div> {Markup}%

 

Setting the Grid Properties

  1. On the design menu click on the Grid >Properties page. Go to Layout options and click the Style name property. This will load the Select Style Dialog. Create a style sheet to your liking. Here I went to System under the Location menu and then chose ModBlue.

  2. Scroll down the Grid > Properties menu until you get to the Shading and Dividers section. Clear the Alternate row shading check box.

  3. Just below the Alternate row shading setting check the Shading and Dividers > Row separator check box.

  4. Save the component as "ProductMarkup" and exit from the Grid Component Builder.

Displaying the Page

  1. In the Web Project Control Panel toolbar click New. When the New File Dialog loads click A5W Page and Next >.

  2. Still in the New File Dialog, choose 'Blank Page' from the options presented and click Next > again.

  3. The HTML Editor will open. Click File > Save As, enter "ProductMarkupBarChart" in the File name field, and click Save.

  4. From the toolbar at the top of the screen click on Insert Component.


    5.  

  5. When the Insert Component Dialog loads click on the Select button. Pick "ProductMarkup", from the Select Component Dialog that loads, and click OK.

  6. Click OK the close the Insert Component Dialog.

  7. Save the A5W page again then close the HTML Editor.

  8. After returning to the Web Projects Control Panel, Right click the "ProductMarkupBarChart" A5W page you created.  select Publish (Local Webroot) open.
