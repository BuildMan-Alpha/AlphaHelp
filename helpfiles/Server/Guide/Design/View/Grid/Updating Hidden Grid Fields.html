Updating Hidden Grid Fields

Alpha Five version 7 introduces the Hidden control type. With this control type Alpha Five displays a grid column (or row), but does not display any data. Grids also support the Hide property, which hides the grid column (or row).

In this example we would like to achieve three goals:

  • set the value of customer_ID with a script in the CanInsertRecord event

  • display the value of customer_ID

  • prevent the user from editing customer_ID (in other words, display it

This example starts with three table fields: customer_ID, firstname, and lastname, plus cust_ID, which is a calculated field derived from customer_ID.

  1. Display the Component Type page.

  2. Select the Updateable radio button.

  3. Display the Grid > Query page.

  4. Click Calculated fields.

  5. Enter the following calculated field definition and click OK. This step gives you a copy of the "customer_ID" field that can be displayed but placed in a non-editable label control.

cust_id = customer_id

  1. Display the Grid > Fields page.

  2. Move "customer_ID", "firstname", "lastname", and "cust_ID" fields from the Available Fields to the Selected Fields list.

  3. Select the "customer_Id" field in the Selected Fields list.

    1. Change the Display Settings > Control type to "Hidden".

    2. Change the Column Properties > Hide column to "True".

  4. Select the "firstname" field in the Selected Fields list.

    1. Change the Column Properties > Column heading to "First Name".

    2. Change the Column Properties > Heading in-line style to "text-align: left;".

    3. Change the Column Properties > Freeform layout to "True".

    4. Click in the Column Properties > Freeform template field to display the Freeform Column Layout dialog.

    5. Enter the following field definition and click OK. This step is important. This setp places the editable "customer_ID" field on the grid, so that the event script can modify it. However, since, the "customer_ID" control is hidden, only the "firstname" field shows.

{Customer_Id}{Firstname}

  1. Select the "lastname" field in the Selected Fields list.

    1. Change the Column Properties > Column heading to "First Name".

    2. Change the Column Properties > Heading in-line style to "text-align: left;".

  2. Select the "cust_ID" field in the Selected Fields list.

    1. Change the Column Properties > Column heading to "Customer ID".

    2. Change the Column Properties > Heading in-line style to "text-align: left;".

    3. Change the Display Settings > Control type to "Label".

  3. Display the Grid > Properties page.

  4. Set the Layout Options > Rows of data to 5.

  5. Display the Grid > Update Settings page.

  6. Set the Permissions > Number of new record rows to 1.

  7. Save the grid. The results look like the following.

Now, you can place a script in the CanInsertRecord event that will update the customer_ID field.
