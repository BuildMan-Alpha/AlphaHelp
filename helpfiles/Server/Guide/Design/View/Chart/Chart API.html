Chart API

Alpha Five Version 11 includes a powerful built-in charting engine that uses the charting capabilities of Microsoft .NET Framework 4. You can program the charting engine directly using the API described on this page, or create charts in a builder, described at Chart Control V11. Using the API gives you the maximum amount of flexibility, but using the builder you can produce most commonly-used charts quickly and easily.

a5_chart_data_url_generate() Function

Generates the REST URI string that describes the chart to be drawn by the .NET Framework System.Web.UI.DataVisualization.Charting.Chart class (via the Alpha Five Charting::Chart Class discussed below).

Example:

dim objSettings as Reporting::ReportChart
objSettings.chartType = "Pie"
dim uri as c = a5_chart_data_url_generate(objSettings,PieChart(),500,300)

See a5_chart_data_url_generate Function for the full description of the parameters. See Using Charts in Xdialogs V11 and the V11 Charting section of the Learning Xdialog Alpha Five sample project for the PieChart() sample code and other chart generation functions.

Charting::Chart Class

Interpreter that reads a REST URI that specifies a chart and in turn uses the .NET Framework System.Web.UI.DataVisualization.Charting.Chart class to render a chart.

Properties

  • dataTemplate as Charting::ChartEventTemplate
  • legendTemplate as Charting::ChartEventTemplate

Methods

  • Chart Constructor
Create a new Chart object

Example:

dim aChart as Charting::Chart

  • Dispose()
Release all the resources of a Chart object. Should be called automatically.
  • EnableMapping(L enable)
Use HTML image mapping
  • Populate(C parameterString)
Populates the chart data from the REST URI using the .NET Framework System.Web.UI.DataVisualization.Charting.Chart class

Example:

aChart.Populate(uri)

  • SaveImage(C filename)
Save an JPEG image of the current chart to a file in the specified location.
  • B bmpImage = SaveImageToBmp()
Save a BMP image of the current chart to a byte stream (blob)
  • B jpegImage = SaveImageToJpeg()
Save a JPEG image of the current chart to a byte stream (blob)
  • B pngImage = SaveImageToPng()
Save a PNG image of the current chart to a byte stream (blob)
