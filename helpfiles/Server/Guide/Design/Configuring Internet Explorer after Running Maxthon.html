Configuring Internet Explorer after Running Maxthon

After Running MYIE2 (Maxthon)

This condition has occurred when running on a LAN with a proxy server after switching from MYIE2 (Maxthon) to Internet Explorer. A dialog box may pop up requesting that you select a language.

  1. Select Tools > Internet Options... to display the Internet Options dialog.

  2. Display the Connections tab and click LAN Settings... .

  3. In the Proxy server section check Use a proxy server for your LAN.

  4. Check Bypass proxy server for local addresses.

  5. Click OK and OK.
