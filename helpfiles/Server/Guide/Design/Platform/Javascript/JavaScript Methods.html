  • A5.u.element.toggle(<element>,<animationSettings>);
Toggle between show and hide depending on the current state.
<element> The element to show or hide
<animationSettings> A JSON object of animation settings.


The animation settings object can either be exactly the same as show/hide, or can nest two different objects with settings specific to show and hide:

 
A5.u.element.toggle(ele,{type: 'fade', duration: 1000{);
 
A5.u.element.toggle(ele,{
          show: {type: 'fade', duration: 1000{,
          hide: {type: 'fade', duration: 500{
{);
