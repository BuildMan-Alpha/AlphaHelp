Viewing Xbasic Code Generated by a Procedure

  1. Optionally, click Save to Clipboard so you can later paste the code into the Code Editor.

  2. Optionally, click Save as Script, enter a Script name.

  3. Optionally, check Put script on Favorites list?

  4. Optionally, add a Description.

  5. Optionally, click the Image button and select an image from the Insert Image Dialog Box.

  6. Click OK to continue or Cancel to discard your inputs.
