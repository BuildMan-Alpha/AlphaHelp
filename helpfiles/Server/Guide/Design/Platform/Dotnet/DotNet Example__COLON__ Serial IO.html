DotNet Example: Serial IO

This is an interactive session that assumes there is a modem on port COM1: and dials the modem.

dim sp as system::io::Ports::SerialPort
?sp.PortName
= "COM1"
 
?sp.BaudRate
= 9600
 
?sp.Parity.value__
= 0
 
sp.Open()
 
sp.WriteLine("atdt9785551212")
 
sp.Close()
