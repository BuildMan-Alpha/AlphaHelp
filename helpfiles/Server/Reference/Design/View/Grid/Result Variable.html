Result Variable

The Result variable is available for use in the CanUpdateRecord, CanDeleteRecord, and CanInsertRecord grid event scripts.

Property

Description

Result.Cancel

Default = .F. Change to .T. to prevent the event from completing.

Result.ErrorHTML

The message to display to the user when Result.Cancel has been set to .T. .
