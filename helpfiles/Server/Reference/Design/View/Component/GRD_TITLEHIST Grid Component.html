GRD_TITLEHIST Grid Component

Purpose

The GRD_TITLEHIST grid component is a device that allowed the developer to place unrelated tables into the TGRDL_HIST tabbed gridlinker.

Description

The GRD_TITLEHIST component is a read-only tabular grid based on the dummy table. This is a table with no records. The grid has a single calculated field. This field provides a constant common value that can link to the child tables in the tabbed gridlinker. We removed the code that displays GRD_TITLEHIST from the TGRDL_HIST tabbed gridlinker component. You will never see the GRD_TITLEHIST component.

link = "1"

ASWSALESHIST.A5W, TGRDL_HIST
