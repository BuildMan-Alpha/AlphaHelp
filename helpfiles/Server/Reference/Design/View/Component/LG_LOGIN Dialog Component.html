LG_LOGIN Dialog Component

Purpose

The LG_LOGIN login component provides a method to login to the application. The security configuration allows password change and recovery, so the component shows views for those options.

The LG_LOGIN component Login view

The LG_LOGIN component Change Password view

The LG_LOGIN component Lost Info view

Description

The LG_LOGIN dialog component contains three views to show during login. Each view can have its own layout and stylesheet. The Login view is a freeform layout based on a tabular layout. The other views are also tabular.

The options and views shown in the builder are determined by the security settings. Some options are not shown because the settings do not permit them.
