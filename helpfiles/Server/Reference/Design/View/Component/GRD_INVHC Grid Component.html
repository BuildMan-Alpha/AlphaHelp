GRD_INVHC Grid Component

Purpose

The GRD_INVHC grid component provides a list of invoices placed by all customers. It is part of the ASWCUSTOMERS.A5W page and is intended for use by the application administrator.

The GRD_INVHC Component in the Browser

Description

The GRD_INVHC component is a read-only tabular grid based on the invoice_header table. Clicking the invoice number (Inv#) link opens the ASWINVOICES.A5W page with full details of an order.

Containers

ASWCUSTOMERS.A5W, TGRDL_CUST
