DLG_MAINT Maintenance Component

Purpose

Application administrators use the DLG_MAINT component to pack or reindex tables on the ASWPACKREINDEX.A5W page..

The DLG_MAINT component in the Browser

The DLG_MAINT component in the Browser if operation fails.

Description

The DLG_MAINT component shows a list of DBF tables that can be packed or reindexed. If a selected operation can not be completed because a table is open, a view will display showing a list of all tables that could not be updated.

Containers
