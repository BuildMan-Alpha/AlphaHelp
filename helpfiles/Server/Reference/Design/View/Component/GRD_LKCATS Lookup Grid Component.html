GRD_LKCATS Lookup Grid Component

Purpose

The GRD_LKCATS grid component provides a lookup of category names and IDs for the GRD_INTPROD grid component. This is done in the context of maintaining product records on the ASWPRODUCTS.A5W page.

Description

The GRD_LKCATS component is a read-only tabular grid based on the product_category table. The grid's records are ordered by category.
