GRD_LKVEN Lookup Grid Component

Purpose

The GRD_LKVEN grid component provides a lookup of vendor names and IDs for the GRD_INTPROD grid component. This is done in the context of maintaining product records on the ASWPRODUCTS.A5W page.

Description

The GRD_LKVEN component is a read-only tabular grid based on the vendor table.

The GRD_LKVEN component has a calculated field.

button = ""
