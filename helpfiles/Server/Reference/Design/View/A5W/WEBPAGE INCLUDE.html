WEBPAGE INCLUDE

Syntax

WEBPAGE INCLUDE A5W_Page as C

Argument

Description

A5W_Page

An expression that evaluates to the path and filename of a page containing HTML text and/or Xbasic code.

Description

The WEBPAGE INCLUDE language element loads the file named in expression (if it is not already in the current session packages) preprocesses it as an A5W page, parses and executes it, and adds it to the packages for the current script. It requires the Server.Include variable be defined in the current variable name space.

Server.Include contains a search path (like DOS) with the following syntax: [ directory path ; directory path ].

For example:

c:\webroot\mystuff ;C:\webroot

Examples

WEBPAGE INCLUDE "testinclude.a5w"
