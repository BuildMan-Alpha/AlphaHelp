LOGIN.A5W

Purpose

LOGIN.A5W is the main login page for all users.

Description

LOGIN.A5W contains the NAV_MAIN navigation and LG_LOGIN login components.

Links

The AWSCUSTDET.A5W, ASWCUSTOMERS.A5W, ASWCUSTONE.A5W, ASWINDEX.A5W, ASWINVOICES.A5W, ASWMAINT.A5W, ASWPRINTRPT.A5W, ASWPRODONE.A5W, ASWPRODUCTS.A5W, ASWSALESHIST.A5W, ASWSEARCH.A5W, ASWSENDEMAIL.A5W, ASWSPECLIST.A5W, ASWUSERLIST.A5W, ASWUSERONE.A5W, ASWVENDORS.A5W, CHECKOUTUSER.A5W, LOGINTEST.A5W, TEMPLATEINT.A5W, and USERINFO.A5W pages, and the NAV_INT and NAV_MAIN navigation components call the LOGIN.A5W page.

LOGIN.A5W in the WYSIWYG tab of the HTML Editor

LOGIN.A5W in the Browser

Page Source

The page code begins logging out any current users by calling the function A5WS_LogoutUser().

a5ws_logoutuser(session,request,response)

The location = "" statement removes the highlighting from any of the menu's entries. The page script saves the navigation style, to use it later.

with tmpl_NAV_MAIN

    location = ""

    pgstyle = style_name

    componentName = "NAV_MAIN"

end with

The page header section loads a standard Alpha Five JavaScript file that is used on a link to open a popup page.

<script type="text/javascript" src="javascript/a5_html_popup_generic.js"></script>

The A5W_INCLUDE() statement is in the top row of the table. It loads TOPPAGE.A5W and displays a standard page header. The style saved earlier is applied to the body of the page.

<body class="<%a5 ?pgstyle %>PageBODY">

<p>

<table cellSpacing=0 cellPadding=0 width="800" border=0>

  <tr>

    <td colspan="2"> <%a5 a5w_include("toppage.a5w") %> </td>

  </tr>

This code uses the style of the dialog to style the prompt to enter user ID and password.

<tr>

    <td align="middle"><!-- Text above login --><span class = " <%a5 ? pgstyle %> FormHeaderFont"><font color=#000080 size=3><strong>Login to the System</strong></font></span><br>

    </td>

</tr>

A link is added at the bottom of the page to open the LOGINLIST.A5W page as a popup. This is added for testing only to show a list of current registered users in the security system. On a real system, the following line would be removed.

<p><a href="#" onclick="a5_popup_open('','loginlist.a5w','480','550','no','no')">Show List of Users</a></p>
