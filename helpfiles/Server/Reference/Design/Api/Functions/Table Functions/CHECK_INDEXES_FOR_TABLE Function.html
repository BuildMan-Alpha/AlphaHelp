CHECK_INDEXES_FOR_TABLE()

Syntax

Result_Flag as L = CHECK_INDEXES_FOR_TABLE( Table_Name as C )

Argument

Description

Table_Name

The name of the table owning the indexes you wish to check.

Result_Flag

  • .T. = Index is valid.

  • .F. = Index is corrupt.

Description

The CHECK_INDEXES_FOR_TABLE() function checks the integrity of the indexes of a table.

Example

? check_indexes_for_table("customer")

= .T.
