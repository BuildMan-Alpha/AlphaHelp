*WORD_REMOVE()

Syntax

New_String as C = *WORD_REMOVE( Old_String as C, Start_Word as N, Separator as C [, Word_Count as N ] )

Argument

Description

New_String

The modified text with word(s) removed.

Old_String

The original unmodified text.

Start_Word

The number of the first word to remove.

Separator

The word separator character(s).

Word_Count

Optional. Default = 1. The number of words to remove.

Description

The *WORD_REMOVE() function removes one or more words from a list.

Example

? *word_remove("one two three four", 3, " ")

= "one two four"
