*WORD_REPLACE()

Syntax

New_String as C = *WORD_REPLACE( Old_String as C, Insert_Text as C, Start_Word as N, Separator as C [, Word_Count as N ] )

Argument

Description

New_String

The string containing the replacement(s).

Old_String

The starting string.

Insert_Text

The text to insert.

Start_Word

The number of the first word to replace.

Separator

The word separator character(s).

Word_Count

Optional. Default = 1. The number of words to replace.

Description

The *WORD_REPLACE() function replaces one or more words in a list.

Examples

? *word_replace("one two three four","THREE!!",3," ")

= "one two THREE!! four"

? *word_replace("one two three four","TWO AND THREE",2," ",2)

= "one TWO AND THREE four"
