*AVERAGE()

Syntax

Average as N = *AVERAGE( List_of_Values as C )

Argument

Description

Average

Numerical average of values.

List_of_Values

Delimited list of values.

Description

The *AVERAGE() function returns the numerical average of a delimited list of numbers.

Examples

dim list as C = "1,3,5,10"

? *average(list)

= 4.75

dim list2 as C

list2 = <<%a%

1

2

5

10

%a%

? *average(list2)

= 4.5
