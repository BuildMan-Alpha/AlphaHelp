Help Functions

Alpha Anywhere provides the following Help functions.

Function

Description

A5_DISPLAYHELPTOPIC()

Displays Alpha Anywhere Help topics from the local AlphaFiveHelp.chm file or the webhelp.

HELP_FILE_EDIT()

Allows you to edit or create a topic in an Alpha Anywhere help system help file.

HELP_FILE_VIEW()

Displays a topic from the Alpha Anywhere help system.

UI_DLG_HELP()

Displays a help topic for the current dialog.

WINHELP_SHOW()

Displays a topic in a standard Windows help (.hlp) file.
