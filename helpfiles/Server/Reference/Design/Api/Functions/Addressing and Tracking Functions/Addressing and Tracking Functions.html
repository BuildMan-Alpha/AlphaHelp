FEDEX_Delivery_Status()
 


The FEDEX_Delivery_Status() function contacts the FedEx web site and looks up the status of a shipment sent through FedEx. It then returns the status of a shipment sent via FEDEX. Note : Use of this function requires an active Internet connection.
 

Syntax: Status as C = FEDEX_Delivery_Status( Tracking_Number as C )

Status
The status of a shipment sent through FEDEX.
Tracking_Number
The tracking number of a shipment sent through FEDEX.

Example:

 


 ? fedex_delivery_status("9612014730443905198600")
= "Delivered"
? fedex_delivery_status("9612478019051986000000") 'invalid number
= ""

USPS_City_From_ZIP()
 


The USPS_City_From_ZIP() function contacts the USPS web site and looks up the city containing the specified ZIP code. It the returns the city that contains the specified ZIP code.  Note : Use of this function requires an active Internet connection. ZIP codes are used by the US Postal Service (USPS) for addressing mail in the United States.
 

Syntax: City as C = USPS_City_From_ZIP( Zip5 as C )

City    The name of the city containing the specified ZIP code.
Zip5    A 5 or 9 digit ZIP code. The function truncates 9 digit codes to 5 digits.

Example:

 


 ? usps_city_from_zip("01803")
= "BURLINGTON"
? usps_city_from_zip("12345")
= "SCHENECTADY"
 

 

 

 

USPS_CityState_From_ZIP()
 


The USPS_CityState_From_ZIP() contacts the USPS web site and looks up the city and state containing the specified ZIP code. It then returns the state and city that contains the specified ZIP code.  Note : Use of this function requires an active Internet connection. ZIP codes are used by the US Postal Service (USPS) for addressing mail in the United States.
 

Syntax: CityState as P = USPS_CityState_From_ZIP( Zip5 as C )

CityState    A pointer variable with two properties City and State.
City    The name of the city containing the specified ZIP code.
State    The name of the state containing the specified ZIP code.


Zip5    A 5 or 9 digit ZIP code. The function truncates 9 digit codes to 5 digits.
 

Example:


? usps_citystate_from_zip("01803")
= city = "BURLINGTON"
state = "MA"

? usps_citystate_from_zip("12345")
= city = "SCHENECTADY"
state = "NY"
 

 

 

 

USPS_Delivery_Status()


The USPS_Delivery_Status() function contacts the USPS web site and looks up the status of a shipment sent through the United States Postal Service (USPS). It then returns the status of the shipment.
Note : Use of this function requires an active Internet connection.
 

 

Syntax: Status as C = USPS_Delivery_Status( Tracking_Number as C )

Status    The status of a shipment sent through the United States Postal Service (USPS).
Tracking_Number    The tracking number of a shipment sent through the United States Postal Service (USPS).
 

Examples:


? usps_delivery_status("01038454000555749770")
= "Delivered"
? usps_delivery_status("12345678900000000")
= "There is no record of this item."
? usps_delivery_status("123456789")
= "Delivery status information is not available for your item via this web site."

USPS_ZipCode_Lookup()


The USPS_ZipCode_Lookup() function contacts the USPS web site and looks up the ZIP code for the specified address. The function returns "" (NULL) if the ZIP code is not found. It returns the first ZIP code if multiple are available for the specified address. Note : Use of this function requires an active Internet connection. ZIP codes are used by the US Postal Service (USPS) for addressing mail in the United States.
 

Syntax:  ZipCode as C = USPS_ZipCode_Lookup( Address1 as C, City as C, State as C [, Address2 as C [, Urbanization as C [, Zip5 as C ]]] )

ZipCode    A 9 digit ZIP code
Address1    The street portion of an address.
City    The city portion of an address.


State    The state portion of an address.
Address2    Optional. Default = "". A second street address.
Urbanization    Optional. Default = "". Urbanization denotes an area, sector, or development within a geographic area. In addition to being a descriptive word, it precedes the name of the area. The URB descriptor, commonly used in Puerto Rican urban areas, is an important part of the addressing format of Puerto Rico, as it describes the location of a given street.


Zip5    Optional. Default = "". A 5 or 9 digit ZIP code. The function truncates 9 digit codes to 5 digits.
 

Example:


? usps_zipcode_lookup("70 blanchard road", "burlington", "ma")
= "01803-5100"

? usps_zipcode_lookup("91 Middlesex Turnpike", "burlington", "ma")
= "01803-4922"

? usps_zipcode_lookup("1600 Pennsylvania Avenue NW", "Washington", "DC")
= "20500-0003"
