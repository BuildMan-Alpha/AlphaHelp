A5_SetAppVersionNumber()

Syntax

Result_Flag as L = A5_SetAppVersionNumber( Version_Number as N )

Argument

Description

Result_Flag

Indicates whether the option was successful.

Version_Number

The version number to set.

Description

The A5_SetAppVersionNumber() function sets an application's version number. This is the same version number that can be set on the Network Optimization dialog.

Limitations

Desktop applications only.

Examples

? A5_SetAppVersionNumber(1.2)

= .T.

You can also set the version number using the A5_SAVE_SETTINGS()function.

dim ptr as P

a5_load_settings("NetworkOptimize", ptr)

? ptr

= check_portion = 1

outofdate_flag = 2

version_number = 1.2

ptr.version_number = 1.3

a5_save_settings("NetworkOptimize",ptr)

a5_load_settings("NetworkOptimize",ptr)

? ptr

= check_portion = 1

outofdate_flag = 2

version_number = 1.3
