*PLAIN_TO_BIN()

Syntax

RTF_Binary_Text as B = *PLAIN_TO_BIN( Text as C )

Argument

Description

RTF_Binary_Text

Binary formatted RTF text as it exists in a RTF memo field.

Text

Unencoded ASCII text.

Description

The *PLAIN_TO_BIN() function returns the RTF encoded version of plain text.
