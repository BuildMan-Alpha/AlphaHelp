A5_Show_URL()

Syntax

V a5_show_URL(C target_url [,C title [,C WidthInInches [,C HeightInInches [,L resizeable ]]]])

Argument Description
target_url Internet address of page to display
title (Optional) Title for display window
WidthInInches (Optional) Width of display window
HeightInInches (Optional) Height of display window
resizeable (Optional) Is display window resizable?

Description

Opens a live browser window showing the contents of a URL.

Limitations

Examples

a5_show_url("http://www.google.com","Search")
