A5_COUNT_WEBSESSIONS()

Obsolete
This method is obsolete and has been removed from the software as of Version 11.


Syntax

Active_Session_Count as N = A5_COUNT_WEBSESSIONS()

Argument

Description

Active_Session_Count

The number of active Application Server sessions at the time of the function call.

Description

The A5_COUNT_WEBSESSIONS() function returns the number of active Application Server sessions at the time of the function call..

Deprecated in Alpha Five Version 10
Removed in Alpha Five Version 11

Example

This approach allows you to read the values that are set.

dim nSessions as N

nSessions = a5_count_websessions()

? nSessions

= 15
