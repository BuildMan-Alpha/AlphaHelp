Return a list of security groups defined for the current project. It has no input parameters and should be used in place of the a5ws_Get_Groups in the V11 Dialog component.

a5ws_GetGroupsDialog()

Syntax

C Result = a5ws_GetGroupsDialog()

Argument Description
C Result List of security groups

Description

Returns a list of security groups defined for the current web project. Used in the v11 dialog server-side events. It has no input parameters and should be used in place of the a5ws_Get_Groups Function in the V11 Dialog component.
