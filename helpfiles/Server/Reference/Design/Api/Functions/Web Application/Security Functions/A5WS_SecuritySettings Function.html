Displays the Security Settings dialog

A5WS_SecuritySettings()

Syntax

A5WS_SecuritySettings( [ Project as C ] )

Argument

Description

Project

Optional. The name of the project that will use these security settings. If not specified, Alpha Anywhere will use the current project.

Description

The A5WS_SecuritySettings() function displays the <span class=Screen>Security Settings</span> dialog box.

Limitations

Web publishing applications only.
