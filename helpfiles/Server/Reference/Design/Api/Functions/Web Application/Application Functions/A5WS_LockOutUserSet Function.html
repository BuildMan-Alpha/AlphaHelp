Locks out a user value for a specified amount of time.

A5WS_LockOutUserSet()

Syntax

Result_Flag as L = A5WS_LockOutUserSet( User_Value as C [, Until_Time as T ] )

Argument

Description

Result_Flag

.T. = The lock out was set.

.F. = The lock out was not set.

User_Value

A user value to be temporarily disabled.

Until_Time

Optional. Default = user is locked out indefinitely. A time value in the future, when the user ID will be enabled.

Description

The A5WS_LockOutUserSet() function locks out a user value for a specified amount of time.

Limitations

Web publishing applications only.

Example

This example locks out the user with the user id of "joe@email.com" until Jan 1, 2007 at 10:00 pm.

dim ut as T

ut = ctodt("01/01/2007 10:00:00 00 pm")

a5ws_LockOutUserSet("joe@email.com", ut)
