a5w_enumerateWebAppRepository()

Syntax

C Result = a5w_enumerateWebAppRepository([C objectType [,C componentGuid [,C userId ]]])

Argument Description
C Result List of Key values meeting the filter, one value per line
C objectType (optional) The object type description set when the value was stored
C componentGuid (optional) The GUID of the component from where the data was saved
C userId (optional) The userId of the user who saved the data in the repository

Description

Enumerates the key values of the objects stores in the repository. You can filter the objects by objectType, componentGuid, or userId.
