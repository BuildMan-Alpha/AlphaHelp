UI_DLG_EVAL_VALID()

Syntax

Result_Flag as L = UI_DLG_EVAL_VALID( Dlg_Title as C, Expression as C )

Argument

Description

Result_Flag

  • .T. indicates that the variable exists.

  • .F. indicates that the variable does not exist.

Dlg_Title

The title of a Xdialog.

Expression

An expression that can be interpreted as the name of a variable.

Description

The UI_DLG_EVAL_VALID() function tests for the existence of a variable defined inside another Xdialog.

Limitations

Desktop applications only.
