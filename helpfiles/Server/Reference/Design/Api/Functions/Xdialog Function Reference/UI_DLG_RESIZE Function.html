UI_DLG_RESIZE()

Syntax

UI_DLG_RESIZE( Title as C, Width as C, Height as C )

Argument

Description

Title

The name of the dialog box.

Width

The new width of the dialog box.

Height

The new height of the dialog box.

Description

The UI_DLG_RESIZE() function resizes a modeless dialog box.

Limitations

Desktop applications only.

Example

ui_modeless_dlg_box("Visible Xdialog", <<%dlg%

<20,3Click Here>

%dlg%)

ui_dlg_resize("Visible Xdialog",10,10)
