A5_PRINT_CURRENT_RECORD()

Syntax

A5_PRINT_CURRENT_RECORD( [ Print_Method as C [, Layout_Type as C ]] )

Argument

Description

Print_Method

Optional. Default = "". The options are:

  • "print"

  • "preview"

  • "save"

  • "fax"

  • "email"

Layout_Type

Optional. Default = "". The options are:

  • "letter"

  • "label"

  • "form"

  • "browse"

  • "report"

Description

The A5_PRINT_CURRENT_RECORD() function displays the Print Current Record dialog box. This function allows you to select a Print Option :

  • Print

  • Preview

  • Email

  • Save to File

  • Fax

and Layout type :

  • Browse

  • Form

  • Label

  • Letter

  • Report

and then choose from any available layouts of the specified type to print.

Limitations

Desktop applications only.

Examples

a5_print_current_record("print", "report")
