Clipboard Methods

Alpha Anywhere provides the following CLIPBOARD methods. Prefix each method with the CLIPBOARD keyword.

Function

 

Description

.CLEAR_DATA()

V5

Clears the Windows Clipboard.

.GET_DATA()

V5

Retrieves data from the clipboard and stores it in Variable.

.HAS_DATA()

V5

Returns .T. or .F. depending on whether the clipboard has data of the type specified by Clipboard Format.

.REGISTER_TYPE()

V5

Allows the user to specify a name for the type of data stored on the clipboard.

.SET_DATA()

V5

Puts the data in the variable Data on the clipboard.
