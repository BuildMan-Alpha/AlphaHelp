UI_HOTSPOT_FROM_MAP()

Syntax

C Result = UI_HOTSPOT_FROM_MAP(imageMap as c [,attribute as c])

Argument Description
imageMap as c Image map to use to generate UI hotspots
attribute as c (optional) Attribute to apply to generated hotspots, e.g. "href"

Description

Converts an HTML image map to a ui_hotspots list.

Limitations

Examples

This is an interactive Xbasic session.
