WEEK()

Syntax

Week_Number as N = WEEK( Date_Value as D )

Argument

Description

Week_Number

An integer equal to the week number (1-53) for the specified Date_Value.

Date_Value

A variable containing a date value.

Description

The WEEK() function returns the integer equal to the week number (1-53) for the specified Date_Value. A week includes any number of days in the year. For example, if January 1 is on a Saturday, it is in week 1. January 2 would be in week 2.

Examples

Assuming start contains the value {1/1/2005}.

? week(start)

= 1

? week"12/27/04"?

= 53
