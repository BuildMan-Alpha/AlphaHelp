WEEK_ISO_FULL()

Syntax

ISO_Week as C = WEEK_ISO_FULL( Date as D )

Argument

Description

ISO_Week

The fully qualified ISO week.

Date

The date to be analyzed.

Description

The WEEK_ISO() function returns the fully qualified ISO week number in the format YYYY-M0#-D.

Examples

? week_iso_full({1/3/1999})
= "1998-W53-7"

 

? week_iso_full({1/4/1999})
= "1999-W01-1"

 

? week_iso_full({12/29/2003})
= "2004-W01-1"

 

? week_iso_full({12/28/2003})
= "2003-W52-7"

 

? week_iso_full(date() )
= "2005-W35-3"
