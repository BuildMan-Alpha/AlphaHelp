DTOC Function Formatted_Date as C = DTOC(D date[,C date_format[,L localized]]) or Formatted_Date as C = DTOC( Date_Time as T [, Format as C ] ) Formatted_Date The Date_Value formatted into a character string. date A variable containing a date value or a character string containing a legitimate representation of a date value ( ISDATE (Date_Value) = .T.). Date_Time A variable containing a date time value. date_format Optional. Default = "MM/DD/YYYY". Refer to Predefined Date Formats. localized Logical Converts a date into a character value in the form 'MM/DD/YYYY'. The first argument to the time function does the formatting. The optional second argument sets the value. If the second argument is omitted, now() is called to get the current date and time time function Date and Time Format Elements
