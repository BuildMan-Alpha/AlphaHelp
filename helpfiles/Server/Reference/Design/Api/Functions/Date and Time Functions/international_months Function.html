international_months()

Syntax

C Result = international_months([C options ])

Argument Description
C Result Names of months, one per line
C options (optional) Comma-delimited list of parameter-value expressions, such as: language=cultureCode, short, shortest

Description

Returns name for all months, using current culture settings

Limitations

Examples

? international_months()
= January
February
...
 
' spanish
? international_months("language=es")
= enero
febrero
...
