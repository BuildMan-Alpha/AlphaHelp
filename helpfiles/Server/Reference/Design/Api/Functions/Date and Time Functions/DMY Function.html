DMY()

Syntax

Formatted_Date as C = DMY( Date as D )

Formatted_Date as C = DMY( Date_Time as T )

Argument

Description

Date

A variable containing a date value or a character string containing a legitimate representation of a date value ( ISDATE (Date_Value) = .T.).

Date_Time

A variable containing a date time value.

Description

DMY() returns a formatted character string in the "DD Month YYYY" format for the specified Date or Date_Time value.

Example

dmy({12-18-92}) ->  "18 December 1992"
