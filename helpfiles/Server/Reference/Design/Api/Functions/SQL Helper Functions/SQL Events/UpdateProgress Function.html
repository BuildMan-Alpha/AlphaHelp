UpdateProgress()

Syntax

UpdateProgress( Context as P, RowsUpdated as N, RowsExpected as N, [ BYREF Cancel as L ] )

Argument

Description

Context The object on which the function is called (generally the connection or a result set).
RowsUpdated The number of rows successfully updated.
RowsExpected The number of rows to be updated. Note: The number of rows in an applied result set is not always known. In this case, RowsExpected will be -1.
Cancel Optional. Default = .F. To cancel the execution of the event, set Cancel = .T. .

Description

AlphaDAO calls UpdateProgress()repeatedly as rows are processed.
