SQL_TableImportDialog()

Syntax

Result_Flag as L = SQL_TableImportDialog()

Argument

Description

Result_Flag

.T. = The import was successful.

.F. = The import was unsuccessful or user cancelled.

Description

The SQL_TableImportDialog() function displays the <span class=Screen>Import Table from SQL</span> dialog, which allows you to import a table from a SQL back-end database to an Alpha Five .DBF table.

Limitations

Desktop applications only.
