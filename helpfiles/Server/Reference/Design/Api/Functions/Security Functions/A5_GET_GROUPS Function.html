A5_GET_GROUPS()

Syntax

Group_List as C = A5_GET_GROUPS( [ Master_Password as C ])

Argument

Description

Group_List

A CR-LF delimited list of defined groups.

Master_Password

Optional. If you are not logged in as the administrator, you must supply the administrator password.

Description

If you have set up Users and Groups for a Database through the Tools > Security commands, the A5_GET_GROUPS() function returns names of all the groups that have been defined.

Limitations

Desktop applications only.

Example

? a5_get_groups()
