Database Functions

Alpha Anywhere provides the following Database functions and methods.

Function or Method

Description

A5.CLOSE()

Closes all open tables and shuts down Alpha Anywhere.

A5.DATABASE_CLOSE()

Closes the current database (.adb file), without closing Alpha Anywhere itself.

A5_DATABASECOMPACT()

Compacts a database and rebuilds indexes.

A5.GET_NAME()

Returns the filename of the current database.

A5.LIBRARY_ENUM()

Returns a list of library files.

A5.LOAD()

Closes the current database and opens the specified Database_Name.

A5_BACKUP_LOGFILE_EXIST()

Determines whether a logfile was created by a backup operation.

A5_BACKUP_SETTINGS_EXIST()

Determines whether backup settings have been saved.

A5_IS_DATABASE_OPEN()

Returns TRUE (.T.) if any database is open.

BACKUP_DATABASE()

Displays the Backup Database dialog, allowing you to make a backup copy of the current database.

CREATE_SHADOW()

Creates a Network Optimized (shadow) database.

CREATE_SHADOW2()

Creates a Network Optimized (shadow) database that supports updates by multiple concurrent users.

EXPORT_TO_QUICKBOOKS()

Exports an Alpha table to QuickBooks.

FILE_ADD_TO_DB()

Adds the .dbf table to the database so that the Table_Filename appears in the Control Panel.

FILE_DROP_FROM_DB()

Drops the .dbf table from the database so that the Table_Filename no longer appears in the Control Panel.

IMPORT_FROM_QUICKBOOKS()

Imports a table from QuickBooks and creates an Alpha table.

LIST_FILES_IN_DATABASE()

Creates a CR-LF delimited string that contains the names of all of the files in an .adb (database) file.

LOAD_ADB_DEF()

Creates a string that contains the names of all of the files in an .adb (database) file.

MOST_RECENTLY_

USED_DATABASE_ENUM()

Creates a CR-LF delimited list containing the fully qualified filenames of the eight different most recently used databases.

ODBC_IMPORT_TABLE()

Imports a data from an external source and creates an Alpha table.

ONESTEP_BACKUP()

Creates a backup of the database.

ON_APPLICATION_EXIT()

Executes the Code_String when Alpha Anywhere closes.

ON_DATABASE_EXIT()

Executes code when the current database is closed.

ON_DATABASE_INIT()

Executes the Code_String when Alpha Anywhere opens the database.

REFRESH_SHADOW()

Refreshes a Network Optimized database.

RESTORE_DATABASE()

Displays the Restore Database dialog box.

SAVE_ADB_DEF()

Creates an .ADB (database) file.

SYSTEM_EVENT_DATA_GET()

Returns the events that have been registered by the CAN_DATABASE_EXIT() and ON_DATABASE_EXIT() functions.

SYSTEM_EVENT_DROP()

Removes an event registered by CAN_DATABASE_EXIT() or ON_DATABASE_EXIT().
