TIME()

Syntax

Formatted_Date_Time as C = TIME( [ Format_String as C [, Date_Time_Value as T ]] )

Argument

Description

Format_String

Optional. Default = "0h:0m:0s". A time format created according to the rules of Date and Time Format Elements.

Date_Time_Value

Optional. Default = NOW(). A time value to format.

Description

TIME() returns a Date_Time_Value containing a formatted Date Time value. If Format_String is not specified, the time is returned in the format HH:MM:SS. The system time is obtained from the system clock (which can be set from DOS).

For information on performing time arithmetic, see the TOSECONDS()and TOTIME()functions.

The optional Format_String parameter allows you to create custom date and time formats.

Examples

time("0h:0m:0s:2") ->  "02:02:30:22"

time("h:0m:0s:2") ->  "2:02:30:22"

time("0h:0m:0s:2") ->  "14:02:30:22"

time("0h:0m:s:2 AM") ->  "02:02:30:22 AM"

time("Weekday Month x, yyyy") ->  "Friday January 29th, 1999"

time("dd-Mon-yy") ->  "29-Jan-99"

Converting a Time value to a Character value:

time("", now() ) -> returns the current date time without any special formatting (e.g. "02/04/199910:37:37 63 am")

time("0h-0m-0s.2", t1) -> returns the value in t1, formatted as specified (e.g. "10-42-16.63")
