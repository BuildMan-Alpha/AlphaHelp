CYEAR()

Syntax

Year_String as C = CYEAR( Date as D )

Year_String as C = CYEAR( Date_Time as T )

Argument

Description

Date

A variable containing a date value or a character string containing a legitimate representation of a date value ( ISDATE (Date_Value) = .T.).

Date_Time

A variable containing a date time value.

Description

CYEAR() returns a character string (in the format "YYYY") containing the year specified in Date or Date_Time.

You can combine the CYEAR() , CMONTH() , CDOW() , and DAY() functions to form custom date formats. For examples, see the DAY() function.

Examples

cyear(START) -> "1995", if START contains 03/12/95

cyear( ctod( "05/19/94") ) -> "1994"
