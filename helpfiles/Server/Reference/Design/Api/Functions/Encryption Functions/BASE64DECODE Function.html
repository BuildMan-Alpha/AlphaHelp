BASE64DECODE()

Syntax

DecodedData as B = BASE64DECODE( Base64EncodedString as C )

Argument

Description

Base64EncodedString

Non-null character string encoded with the BASE64ENCODE()function.

Description

BASE64DECODE() returns binary data decoded from a base-64 encoded character string of arbitrary length.

Example

dim blob1 as B

dim blob2 as B

dim blob3 as B

blob1 = "This is the time"

blob2 = base64encode(blob1)

blob2 -> "VGhpcyBpcyB0aGUgdGltZQA = "

blob3 = base64decode(blob2)

blob3 -> "This is the time"
