A5_SYSTEM_TOOLBAR_SHOW()

Syntax

Toolbar_Title as C = A5_SYSTEM_TOOLBAR_SHOW( Vars as P, Toolbar as P [, Relative_Position as C ] )

Argument

Description

Toolbar_Title

The title of the toolbar.

Vars

A pointer to the variable frame.

Toolbar

A pointer to the toolbar.

Relative_Position

Not used.

Description

The A5_SYSTEM_TOOLBAR_SHOW() function displays a previously loaded toolbar.

Limitations

Desktop applications only.

Examples

dim vl as P

dim tbar as P

vl = local_variables()

tbar = a5_toolbar_def_load("Invoice")

a5_system_toolbar_show(vl,tbar)
