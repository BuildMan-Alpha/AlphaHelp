EMAIL_POP_RETR_UI()

Syntax

Message_Count as N = EMAIL_POP_RETR_UI( [ Profile as C [, Inbox as C [, Display as L [, UI as L [, Filter as C [, Status_Window as L [, Status_Bar as L [, Parent_Dlg_Title as C [, Silent as L ]]]]]]]]] )

Argument

Description

Message_Count

The number of messages on the email server that match you filter.

Profile

Optional. Default = active email profile (can be retrieved using EMAIL_GET_ACTIVE()). The name of the email profile to use.

Inbox

Optional. Can be used to override the inbox specified in the Profile above.

Display

Optional. Default = .T.

  • .T. = displays the Email Reader after downloading mail.

  • .F. = does not display Email Reader.

UI

Optional. Default = .T.

  • .T. = displays a dialog allowing the user to download selected mail, delete messages, filter, etc.

  • .F. = does not display dialog.

Filter

Optional. Default = "" (all messages). An email filter that can be built with A5_EMAIL_FILTER_BUILDER(). Allows you to download only messages matching certain criteria.

Status_Window

Optional. Default = .T.

  • .T. = displays a window with the current download progress

  • .F. = does not display progress

Status_Bar

Optional. Default = .F.

  • .T. = displays current progress on the Status Bar.

  • .F. = does not display progress

Parent_Dlg_Title

Internal use only

Silent

Optional. Default = .F.

  • .T. = suppresses all errors.

  • .F. = displays all errors.

Description

The EMAIL_POP_RETR_UI() function displays the Download Email dialog box and returns a count of messages on your email server that match your filter specifications.

Limitations

Desktop applications only.

Examples

? email_pop_retr_ui()

= 1
