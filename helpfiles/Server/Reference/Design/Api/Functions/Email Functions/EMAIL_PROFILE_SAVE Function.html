EMAIL_PROFILE_SAVE()

Syntax

Result_Flag as L = EMAIL_PROFILE_SAVE( Profile_Name as C, Profile as P )

Argument

Description

Result_Flag

Indicates whether the operation was successful.

Profile_Name

The name of an email profile. See EMAIL_GET_ACTIVE().

Profile

A dot variable containing email profile settings. See EMAIL_PROFILE_LOAD().

Description

The EMAIL_PROFILE_SAVE() function saves an email profile to the Windows registry.

Limitations

Desktop applications only.

Example

dim p as P

? email_profile_save(email_get_active() , p)

= .T.
