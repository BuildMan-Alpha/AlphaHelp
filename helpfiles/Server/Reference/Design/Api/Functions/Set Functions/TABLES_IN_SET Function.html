TABLES_IN_SET()

Syntax

Output_String as C = TABLES_IN_SET( Set_Name as C [, Fully_Qualified_Filenames_Flag as L ] )

Argument

Description

Set_Name

The name of a set. The extension ".set" is assumed.

Fully_Qualified_Filenames_Flag

Optional. Default = .F.

  • .T. (TRUE) to display fully qualified filenames.

  • .F. (FALSE) to show only filenames.

Description

TABLES_IN_SET() returns a CR-LF delimited list of files composing the set defined by Set_Name.

Limitations

Desktop applications only.

Examples

tables_in_set("code",.f.)   ->   c:\databases\functions\FUNCTIONS.DBF

c:\databases\functions\METHODS.DBF

tables_in_set("code",.f.)   ->   FUNCTIONS.DBF

METHODS.DBF

tables_in_set("code")   ->   FUNCTIONS.DBF

METHODS.DBF
