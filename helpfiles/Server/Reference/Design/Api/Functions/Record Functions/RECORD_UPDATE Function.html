RECORD_UPDATE()

Syntax

Result_Flag as L = RECORD_UPDATE( Table_Name as C, Record_Filter as C, Data_List as C [, Style as N )

Argument

Description

Result_Flag

.T. = the record was modified. 
.F. = the record was not modified.

Table_Name

The name of a table

Record_Filter

A character filter expression that evaluates to a logical value and selects a record from the table.

Data_List

A CR-LF delimited list of the field values to write into the selected record. The data can be presented in two different formats as defined by the Style argument.

Style

An optional style selector. The default value is 1.

Style = 1

"Fieldname1 = Value1" ↩
"Fieldname2 = Value2" ↩
"FieldnameN = ValueN"

Style = 2

"Value1" ↩
"Value2" ↩
"ValueN

When entering data in an auto-increment field, use Style 1 and leave the value of the field blank as in the following line:

"Fieldname1 = "

Description

RECORD_UPDATE() modifies a record in table Table_Name. The record is selected using Record_Filter and updated with the field values specified by Data_List.

Note : If more than one record matches the filter, the function only modifies the first record found in record number order.

Example

The following example changes the values of the firstname and lastname fields in the first record of the salespeople table.

dim str as C

str = <<%dat%

firstname = george

lastname = white

%dat%

? record_update("salespeople", ".t.", str)

= .T.
