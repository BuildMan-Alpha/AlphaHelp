PNG_TO_BITMAP()

Syntax

BMP as B = PNG_TO_BITMAP( PNG as B [, Flag as C ] )

Argument

Description

BMP

The output BMP image.

PNG

The input PNG image.

Flags

Optional. Default = "C".

  • "C" = Combined

  • "R" = RGB only

  • "A" = Alpha blend only

Description

The PNG_TO_BITMAP() function converts a PNG formatted image in memory to a Windows Bitmap (.BMP) formatted image.

Examples

dim img as B

dim img2 as B

dim fptr as P

dim fptr2 as P

dim length as N

fptr = file.open("C:\NextYear.png",FILE_RW_EXCLUSIVE)

length = fptr.bytes_get()

img = fptr.readb(length)

img2 = png_to_bitmap(img)

fptr2 = file.create("c:\NextYear.bmp",FILE_RW_EXCLUSIVE)

fptr2.writeb(img2)

fptr2.close()

fptr.close()
