JS_LIST_TO_ARRAY Function

Syntax

js_list_to_array(list as c [, flagListisCommaDelimited as l])

Argument

Description

list

list string parameter to populate array from

flagListisCommaDelimited

If true, delimit with commas instead of cr-lfs.

Description

Takes a list of values and generates a string in Javascript array syntax.

Example

?js_list_to_array("alpha,beta,gamma",.t.)

= "alpha,beta,gamma"

?js_list_to_array("alpha,beta,gamma",.t.)

= "alpha,bet\a,gamma"

dim list as c 

list = <<%txt%

#alpha

#beta

#gamma

#%txt%

?js_list_to_array(list)

= "alpha,beta,gamma"
