INCHESTOCM()

Syntax

Centimeters as N = INCHESTOCM( Inches as N )

Argument

Description

Centimeters

The centimeter equivalent of an inch value.

Inches

The number of inches to convert.

Description

The INCHESTOCM() function converts an inch value to a centimeter value.

Examples

? inchestocm(10)

= 25.4
