MIN()

Syntax

Smallest_Number as N = MIN( Number1 as N, Number2 as N )

Argument

Description

Number1

A number.

Number2

A number.

Description

MIN() returns the lesser of two numbers, Number1 or Number2.

Example

min(MATH, VERBAL) -> 400, if MATH contains 600 and VERBAL contains 400
