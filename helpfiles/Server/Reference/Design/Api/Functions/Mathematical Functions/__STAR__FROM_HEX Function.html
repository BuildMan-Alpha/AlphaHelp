*FROM_HEX()

Syntax

Binary_Data as B = *FROM_HEX( Hex_Value as C )

Argument

Description

Hex_Value

A hexadecimal number value.

Description

The *FROM_HEX() function converts an hexadecimal string to a BLOB value.

Example

? *FROM_HEX("41") ? A
