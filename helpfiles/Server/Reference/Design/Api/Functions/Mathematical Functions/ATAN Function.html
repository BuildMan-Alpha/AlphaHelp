ATAN()

Syntax

Arc_Tangent as N = ATAN( Radian_Angle as N )

Argument

Description

Radian_Angle

A number between - p and + p.

Description

ATAN() returns the arc tangent of a Radian_Angle. For angles measured in degrees, use ATAN(DTOR( Degree_Angle )).

Examples

atan(0) -> 0

atan(1) -> .78
