Character Replacement Functions

Alpha Anywhere provides the following character replacement functions.

Function

Description

*LINE_REMOVE()

Removes line(s) of text in a CR-LF delimited string.

*LINE_REPLACE()

Replaces line(s) of text in a CR-LF delimited string.

*REVERSE()

Creates a new character string by reversing the sequence of characters in the Input_String.

*TRANSPOSE()

Creates a character string by reversing the characters in the Input_String at positions First_Position and Second_Position.

*WORD_REMOVE()

Removes one or more words from a string.

*WORD_REPLACE()

Replaces one or more words in string.

CHRTRAN()

Replaces characters in Input_String that match those in Search_Pattern with characters in the same position in Replace_Pattern. Case sensitive.

COMMA_TO_CRLF()

Converts a comma delimited list to a CR-LF delimited list.

CompileStringTemplate()

Generates a compiled object that replaces placeholders in a string with variable values.

CRLF_TO_COMMA()

Converts a CR-LF delimited list to a comma delimited list.

EVALUATE_STRING()

Replaces placeholders in a string with variable values.

LINE_INSERT()

Inserts a line into a CR-LF delimited list.

MASK()

Inserts characters from Unformatted_String into Template_String to replace all of the spaces in Template_String.

REGEX_MERGE()

Searches Text for one or more matches to Search_Expresssion and conditionally performs replacement operations.

REMSPECIAL()

Strips all non-alphabetic and non-numeric characters, including spaces, from a string.

STRIP_TRAILING_CHAR()

Removes the specified character from the end of the Input_String.

STRITRAN()

Replaces every occurrence of Find_String in Input_String with a Replace_String. Case insensitive.

STRITRAN_MULTI()

Replaces every occurrence of multiple Find_Strings in Input_String with appropriate Replace_String. Case insensitive. Case insensitive.

STRITRAN_SMATCH()

Replaces strings with variable content and optionally reorders the content as the replacements are being made. Case insensitive.

STRITRAN_SPECIAL()

Replaces occurrence(s) of a variable length string. Case insensitive.

STRTRAN()

Replaces every occurrence of Find_String in Input_String with a Replace_String. Case sensitive.

STRTRAN_MULTI()

Replaces every occurrence of multiple Find_Strings in Input_String with appropriate Replace_String. Case sensitive.

STRTRAN_SMATCH()

Replaces strings with variable content and optionally reorders the content as the replacements are being made. Case sensitive.

STRTRAN_SPECIAL()

Replaces occurrence(s) of a variable length string. Case sensitive.

STRWORDTRAN()

Replaces every occurrence of Find_Word in Input_String with a Replacement_Word. If Replacement_Word is omitted, Find_Word is replaced with NULL characters. A value N for the optional Start_Word begins the replacement at the N th occurrence of Word.   Case sensitive.

STRWORDITRAN()

Replaces every occurrence of Find_Word in Input_String with a Replacement_Word. If Replacement_Word is omitted, Find_Word is replaced with NULL characters. A value N for the optional Start_Word begins the replacement at the N th occurrence of Word.   Case insensitive.

STUFF()

Replaces the specified Number_Of_Characters in Input_String, beginning at the specified Starting_Position, with the Insert_String.

TOKEN_SEARCH_REPLACE()

Replaces multiple search string with multiple replace strings.

WORD_MOVE()

Moves a word or entry from one position to another in a CR-LF delimited list.

WORD_REMOVE()

Removes the word indicated by position from a CR-LF delimited list.

WORD_REPLACE()

Replaces a word or entry in a CR-LF delimited list.

WORD_SUBTRACT()

Removes all of the "words" in Excluded_Words from the list of words or entries in string. Case insensitive.

WORD_SUBTRACTC()

Removes all of the "words" in Excluded_Words from the list of words or entries in string. Case sensitive.

<StringScanner>.Replace()

Inserts and optionally replaces text in the StringScanner buffer.

<StringScanner>.ReplaceSmatch()

Replaces the first instance of Find_Text in the StringScanner buffer with Replace_Text. Supports the use of wildcard characters in Find_Text.

<StringScanner>.ReplaceSmatchI()

Replaces the first instance of Find_Text (case insensitive) in the StringScanner buffer with Replace_Text. Supports the use of wildcard characters in Find_Text.

<StringScanner>.ReplaceString()

Replaces the first instance of Find_Text in the StringScanner buffer with Replace_Text.

<StringScanner>.ReplaceStringI()

Replaces the first instance of Find_Text (case insensitive) in the StringScanner buffer with Replace_Text.
