TEXTSEARCH()

Syntax

Result_Flag as L = TEXTSEARCH( Search_In_String as C, Find_String as C )

Argument

Description

Find_String

The character string to find.

Search_In_String

The character string to examine for occurrences of Find_String.

Description

Returns .T. (TRUE) if Find_String string is found in Search_In_String. Find_String string can contain the reserved Boolean words "and" and "or". After a query is run using the TEXTSEARCH() function, Alpha Anywhere highlights the words that were found when the data is viewed using a form. The search is case insensitive and may contain wildcard characters (* and ?).

Examples

textsearch("now is the time for all good men", "time AND good") -> .T.

textsearch("now is the time for all good men", "time AND bad") -> .F.

textsearch("now is the time for all good men", "im") -> .F.

textsearch("now is the time for all good men", "*im*") -> .T.
