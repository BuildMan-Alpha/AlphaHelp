TOKEN_SEARCH_REPLACE()

Syntax

New_Expn as C = TOKEN_SEARCH_REPLACE( Expn as C, Search1 as C, Replace1 as C [, Search2 as C, Replace2 as C [, SearchN as C, ReplaceN as C ]] )

Argument

Description

New_Expn

The translated expression.

Expn

The starting expression that contains zero or more instances of the various search arguments.

Search1 ... SearchN

The character strings that you are searching for.

Replace1 ... ReplaceN

The replacement character strings, paired with the search strings.

Description

The TOKEN_SEARCH_REPLACE() function replaces multiple search strings with multiple replace strings.

Example

? token_search_replace("if(sf1>sf2, rs1=rs2+1, rs2=rs1-1)", "sf1", "a", "sf2", "b", "rs1", "c", "rs2", "d")

= "if(a>b,c=d+1,d=c-1)"
