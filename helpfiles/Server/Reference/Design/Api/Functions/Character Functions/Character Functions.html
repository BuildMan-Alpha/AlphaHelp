Character Functions

Alpha Anywhere provides the following character functions.

Function

Description

Character Case Functions

Functions to change the case character strings.

Functions to find character strings and analyze the results.

Character Conversion Functions

Functions for converting data to and from character format.

Character Counting Functions

Functions for finding and counting character strings.

Character Extraction Functions

Functions for returning strings from other strings.

Character Formatting Functions

Functions for formatting character strings.

Character Replacement Functions

Functions for finding and replacing character strings.

Character Search Functions

Functions for searching through character strings.

Character Sort Functions

Functions for sorting character strings.

List Processing Functions

Functions useful for processing CR-LF delimited lists.

Regex Functions

Search and replacement functions for strings and file contents.

Type Checking Functions

Functions that determine the type of variables.
