LT()

Syntax

Output_String as C = LT( Input_String as C )

Argument

Description

Output_String

A lower case version of the Input_String.

Input_String

The text to process.

Description

The LT() function changes all upper case changes in Input_String to lower case.

Examples

? lt("This IS the Time")

= "this is the time"
