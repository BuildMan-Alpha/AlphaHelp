SMATCH()

Syntax

Match_Flag as L = SMATCH( Target_String as C, Find_String as C )

Argument

Description

Target_String

The character string to compare to Find_String.

Find_String

A character string.

Description

Returns .T. (TRUE) if Find_String matches Target_String.

Either parameter can be a constant, character variable, or an expression that returns a character value. SMATCH() returns TRUE only if the entire Find_String matches the Target_String.

SMATCH() is useful for selecting records using query selection criteria, filter expressions, or conditional expressions.

Note : Either character string parameter can include a Memo field. However, only the first 1024 characters of the Memo field are searched. Use SMATCH_MEMO() to search an entire Memo field.

Examples

? smatch("alpha ", "alpha*")

= .T.

? smatch("alpha ", "alpha")

= .F.

? smatch("Beta", "beta")

= .T.

? smatch("alpha ", "alpha  ")

= .F.
