CHR_MULTI()

Syntax

Result_String as C = CHR_MULTI( ASCII_Value as N, Count as N )

Argument

Description

Result_String

A string with Count repetitions of the character specified by ASCII_Value.

ASCII_Value

The ASCII value of a character.

Count 

The number of times to repeat the character.

Description

The CHR_MULTI() function produces a string with Count repetitions of the character specified by ASCII_Value.

Example

? chr_multi(78, 12)

= "NNNNNNNNNNNN"
