ANSI_TO_OEM()

Syntax

Result_String as C = ANSI_TO_OEM( Source_Buffer as C, Number_of_Characters as N )

Argument

Description

Source_Buffer

A string variable containing the data to be translated.

Number_of_Characters

The number of characters to translate.

Description

The ANSI_TO_OEM() converts the specified Number_of_Characters in the Source_Buffer from the ANSI (Windows) character set to the OEM (DOS) character set, and returns the result in Result_String.
