A5_BROWSE_CREATE()

Syntax

 as P = A5_BROWSE_CREATE( Table_Name as C )

Argument

Description

A pointer to the newly created browse.

Table_Name

The name of a table. The .DBF extension is assumed.

Description

The A5_BROWSE_CREATE() function displays the New Browse dialog, which allows you to select the fields that will appear in the new browse.

Example

dim ptr as P

ptr = a5_browse_create("travel time")
