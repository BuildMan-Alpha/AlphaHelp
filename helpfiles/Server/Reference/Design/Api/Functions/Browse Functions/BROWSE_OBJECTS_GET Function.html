BROWSE_OBJECTS_GET()

Syntax

Object_List as C = BROWSE_OBJECTS_GET( Browse_Name as C [, Library as C [, Prefix as C [, class=Prefix as C ]] )

Argument

Description

Browse_Name

The name of a saved browse layout.

Library

 

Prefix

 

class=Prefix

 

Description

The BROWSE_OBJECTS_GET() function returns a list of the columns on a browse layout.

Example

? browse_objects_get("CustomerBrowse")

= Customer Id

Last Name

First Name

Company

Phone Number

Fax Number
