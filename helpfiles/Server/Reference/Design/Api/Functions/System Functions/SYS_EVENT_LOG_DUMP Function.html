SYS_EVENT_LOG_DUMP()

Syntax

OutputString as C = SYS_EVENT_LOG_DUMP( Application as C, Format as C )

Argument

Description

Application

The name of your application.

Format

The information to return. The format can include:

  • "S" = string

  • "C" = computer

  • "N" = name

  • "T" = time

  • "W" = time written

  • "K" = kind of event

  • "E" = event ID

  • "X" = subcategory

Description

The SYS_EVENT_LOG_DUMP() function dumps events from the system event log.

Limitations

Desktop applications only.

Example

? sys_event_log_dump("Alpha Anywhere", "NTW" + CRLF() )
