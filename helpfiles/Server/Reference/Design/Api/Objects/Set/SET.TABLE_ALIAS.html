.TABLE_ALIAS

Syntax

 as P = . Table_Alias as C

Argument

Description

A pointer to a component table of a set.

Table_Alias

The name alias of a table in the set.

Description

The .TABLE_ALIAS method returns a pointer to a table in a set.

Example

dim st as P

dim tb as P

st = set.open("invoice")

tb = st.customer

? tb.name_get()

= "Customer"
