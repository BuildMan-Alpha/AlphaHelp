.IS_VALID()

Syntax

Result_Flag as L = .IS_VALID()

Argument

Description

Result_Flag

.T. if the file pointer is still valid.

.F. if the file pointer is not valid.

Description

The .IS_VALID() method tells you if the file pointer is valid. If you had opened a file across a network link and the network failed, you could get an invalid result.

Example

dim fp as P

fp = file.open("C:\Combobox.Txt", FILE_RO_EXCLUSIVE)

? fp.is_valid()

= .T.
