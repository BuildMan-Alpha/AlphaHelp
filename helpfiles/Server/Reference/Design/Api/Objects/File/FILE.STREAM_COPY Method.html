FILE.STREAM_COPY()

Syntax

FILE.STREAM_COPY( Stream_From as P, Stream_To as P, Control as P [, Chunksize as N [, Max_Length as N ]] )

Argument

Description

Stream_From

A file pointer or socket pointer.

Stream_To

A file pointer or socket pointer.

Control

A pointer with two elements.

Argument

Description

pControl.bytes

a running count of how many bytes have been streamed so far. It does not matter what you set this to, it gets reset to 0 at the beginning of the stream operation and it automatically updated as the stream progresses. This can be used to implement a progress bar.

pControl.exit

A logical flag. If it is set to .t., the stream operation will abort. This can be used to implement a cancel button

Chunksize

Optional. Indicates the size of the chunks in which the stream is transferred.

Max_Length

Optional. Indicates the maximum bytes that should be streamed. The operation will stop if it reaches this size, even if there is more data available.

Description

Streaming is a faster, optimized way of copying. The FILE.STREAM_COPY() method copies data from a file or socket to a file or socket.

Examples

dim pControl as P

dim pControl.bytes as N = 0

dim pControl.exit as L = .f.

f1 = file.open("c:\somefile.ext",FILE_RO_SHARED)

f2 = file.create("c:\newfile.ext",FILE_RW_EXCLUSIVE) file.stream_copy(f1, f2, pControl)
