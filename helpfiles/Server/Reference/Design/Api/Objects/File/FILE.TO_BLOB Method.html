FILE.TO_BLOB()

Syntax

Data as B = FILE.TO_BLOB( Filename as C )

Argument

Description

Filename

The full path specification of the file.

Description

The FILE.TO_BLOB() method reads a binary file from disk into a blob variable, Data. The data can be in either 8 bit Windows Bitmap (.BMP) or JPEG (.JPG) formats.

Example

String = file.to_blob("c:\image.bmp")
