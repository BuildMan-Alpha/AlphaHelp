<ImageResource>.GetBitmapNameMap()

Syntax

Image as <ImageName> = <ImageResource>.GetBitmapNameMap()

Argument

Description

Image

An ImageName object.

<ImageResource>

An ImageResource object created by the ImageResource.Get() method.

Description

The <ImageResource>.GetBitmapNameMap() method returns an ImageName object.

Examples

dim ir as P

dim nm as P

ir = ImageResource.get()
nm = ir.GetBitmapNameMap()

 

nm.Initialize(<<%str%
a5_control:browse=$a5_browse
%str%)

 

ui_dlg_box("","{image=a5_control:browse}")
