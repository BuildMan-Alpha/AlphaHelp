A5.LETTER_ENUM()

Syntax

Object_List as C = :A5.LETTER_ENUM( [ Flag as N ] )

Next_Object_Name as C = :A5.LETTER_ENUM( Object_Name as C )

Argument

Description

Object_List

A list of letters in various formats, depending on the Flag variable.

Next_Object_Name

The next letter in the list of letters.

Flag

Optional. Default = Null.

  • Null = Object name, full path, and filename.

  • 2 = Object name

  • 4 = Object type, object name, full path, and filename

  • 8 = Quoted object name, full path, and filename.

  • 12 = Object type, quoted object name, full path, and filename formatted for tree control.

Object_Name

The quoted name of a letter.

Description

The A5.LETTER_ENUM() method returns a CR-LF delimited list of all saved letter layouts.

Limitations

Desktop applications only.

Example

Refer to A5.FORM_ENUM().
