A5.LOAD()

Syntax

:A5.LOAD( Database_Name as C )

Argument

Description

Database_Name

The fully qualified path and name of an Alpha Anywhere database. The extension .adb is assumed.

Description

The A5.LOAD() method closes the current database and opens the specified Database_Name.

Examples

The following script opens the Orders database.

:a5.load("c:\data\orders.adb")

The A5.LOAD() method does not check to see if the new database exists before closing the current database. This script tests to see if database cDB exists first.

dim cDB as C

dim msg as C

dim title as C

if file.exists(cDB)

    :a5.load(cDB)

else

    title = "Unable to load requested database"

    msg = "Invalid alias or filename"

    ui_msg_box(title, msg, UI_INFORMATION_SYMBOL)

end if
