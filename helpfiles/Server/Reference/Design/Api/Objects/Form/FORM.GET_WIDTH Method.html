.GET_WIDTH()

Syntax

Width as N = .GET_WIDTH( Format as C [, Style as P ] )

Argument

Description

Format

A data format, as selected from the Select Format dialog box.

Style

Internal use only.

Description

The .GET_WIDTH() method returns the Width of a data format in inches, given a display style.

Limitations

Desktop applications only.
