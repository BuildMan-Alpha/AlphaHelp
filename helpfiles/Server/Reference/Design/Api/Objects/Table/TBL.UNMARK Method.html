<TBL>.UNMARK()

Syntax

<TBL>.UNMARK()

Description

The <TBL>.UNMARK() method unmarks the current record.

Note : To unmark all records use <TBL>.UNMARK_RANGE().

Example

dim tb as P

tb = table.open("customer")

tb.fetch_first()

tb.change_begin(.t.)

tb.mark()

? tb.is_marked()

= .T.

tb.unmark()

? tb.is_marked()

= .F.
