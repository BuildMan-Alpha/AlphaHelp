<TBL>.ADD_BLANK_RECORDS()

Syntax

Result_Flag as L = <TBL>.ADD_BLANK_RECORDS( Num_Records as N )

Argument

Description

Result_Flag

  • .T. = Records were added.

  • .F. = Records were not added.

Num_Records

The integer number of records to add.

Description

The <TBL>.ADD_BLANK_RECORDS() method adds Num_Records of blank records to a table.

Example

This script adds 1,000 blank records to a table.

dim t as P

t = table.open("test")

t.add_blank_records(1000)
