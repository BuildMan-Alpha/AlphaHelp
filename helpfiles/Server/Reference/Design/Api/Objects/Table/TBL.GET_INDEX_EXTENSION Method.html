<TBL>.GET_INDEX_EXTENSION()

Syntax

Extension as C = <TBL>.GET_INDEX_EXTENSION( Table_Filename_Extension as C )

Argument

Description

Extension

The filename extension that would be used for the table's index file.

Table_Filename_Extension

The three character file type extension associated with any Alpha Anywhere table.

Description

The <TBL>.GET_INDEX_EXTENSION() method returns the filename extension that would be used for the table's index file.

Example

? ptr.get_index_extension("dbf")

= ".CDX"

? ptr.get_index_extension("set")

= ".SEX"

? ptr.get_index_extension("ddd")

= ".DDX"
