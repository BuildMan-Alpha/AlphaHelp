.PERSIST()

Syntax

.PERSIST( Preserve_Flag as L )

Argument

Description

Preserve_Flag

.T. = Preserve the query.
.F. = Discard the query if necessary.

Description

The .PERSIST() method sets a flag to indicate that a query that is used as an index should be preserved.

Example

dim tbl as P

dim indx as P

tbl = table.open("customer")

indx = tbl.index_primary_put("Lastname")

indx.persist(.t.)
