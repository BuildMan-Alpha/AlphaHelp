OLE.RemoveLicense()

Syntax

Result_Flag as L = OLE.RemoveLicense( ProgID as C )

Argument

Description

Result_Flag

Returns TRUE if the programmatic identifier had a license key.

ProgID

The software's programmatic identifier. See OLE Automation Object Browser.

Description

The OLE.RemoveLicense() method removes a programmatic identifier and its associated license key from the global cache.

Examples

? ole.addlicense("a", "lic1")

= .T.

? ole.getlicense("a")

= "lic1"

? ole.loaddeveloperlicense("b")

ERROR: OLE: No license found for 'b'

? ole.removelicense("a")

= .T.

? ole.getlicense("a")

ERROR: OLE: No license found for 'a'
