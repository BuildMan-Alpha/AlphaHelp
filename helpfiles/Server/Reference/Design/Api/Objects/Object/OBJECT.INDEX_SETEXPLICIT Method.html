<OBJECT>.INDEX_SETEXPLICIT()

Syntax

<OBJECT>.INDEX_SETEXPLICIT( Index_Name as C )

Argument

Description

Index_Name

The tag name of an index.

Description

The <OBJECT>.INDEX_SETEXPLICIT() method applies to:

  • Browses (for <OBJECT> use either the <BROWSE> pointer or the name of the browse)

  • Forms (for <OBJECT> use either the <FORM> pointer or the name of the form)

The <OBJECT>.INDEX_SETEXPLICIT() method sets the primary index for the layout. It displays error message if Index_Name is not found.

Limitations

Desktop applications only.

Example

dim ptr as P

ptr = form.load("Customer Information")

ptr.Index_SetExplicit("Lastname1")

? ptr.Index_Name()

= "Lastname1"

ptr.Index_SetExplicit("Lastname2")

? ptr.Index_Name()

= "Lastname2"
