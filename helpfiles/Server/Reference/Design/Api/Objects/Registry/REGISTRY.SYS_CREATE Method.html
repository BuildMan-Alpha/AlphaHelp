REGISTRY.SYS_CREATE()

Syntax

REGISTRY.SYS_CREATE( Entry as C, Data as C )

Argument

Description

Entry

The registry entry name and full path.

Data

The value to write into the Registry.

Description

The REGISTRY.SYS_CREATE() method writes a zero terminated string value (type REG_SZ) into the Registry. This method is not limited to the Alpha Five section, as is REGISTRY.SAVE_SETTINGS(). The power of this function makes it dangerous to use. Make sure that you back up your Registry before making any changes.
