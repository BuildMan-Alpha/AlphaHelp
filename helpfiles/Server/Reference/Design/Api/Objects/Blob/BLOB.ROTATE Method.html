.ROTATE()

Syntax

.ROTATE( Bits as N )

Argument

Description

Bits

The number of bits to rotate the variable's data.

Description

The .ROTATE() method rotates the bits of a binary variable.

Example

dim b as B

b = "alpha software"

? b.dump("a")

= "alpha software. "

b.rotate(8)

? b.dump("a")

= ".alpha software "

b.rotate(8)

? b.dump("a")

= "e.alpha softwar "

dim b as B

b = "alpha software"

? b.dump("a")

= "alpha software. "

b.rotate(-8)

? b.dump("a")

= "lpha software.a "
