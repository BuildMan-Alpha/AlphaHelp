.NAME()

Syntax

Filename as C = .NAME()

Description

The .NAME() method returns the Filename of the current file in the list of files referenced by the object pointer.

Example

dim ptr as P

ptr = filefind.first("c:\*.txt")

? ptr.creation_time()  ->02/27/2003 10:05:50 65 am

? ptr.name()  -> "c:\Button.Txt"
