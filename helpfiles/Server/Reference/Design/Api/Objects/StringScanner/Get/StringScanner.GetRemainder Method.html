<StringScanner>.GetRemainder()

Syntax

Text as C = <StringScanner>.GetRemainder()

Description

The <StringScanner>.GetRemainder() method returns the text from the current offset to the end of the buffer.

Example

dim scanner as StringScanner

dim txt as C

txt = <<%text%

This is line one

and this is line two

and finally line three

%text%

scanner = stringscanner.Create(txt)

? scanner.ScanOver(12)

= "This is line"

? scanner.GetRemainder()

= one

and this is line two

and finally line three
