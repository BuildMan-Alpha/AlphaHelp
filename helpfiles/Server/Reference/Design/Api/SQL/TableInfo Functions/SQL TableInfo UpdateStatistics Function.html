SQL::TableInfo::UpdateStatistics()

Syntax

Result_Flag as L = .UpdateStatistics(  as SQL::Connection )

Argument

Description

Result_Flag

Indicates whether the operation was successful.

A SQL::TableInfo object.

A SQL::Connection object.

Description

The UpdateStatistics() method gathers information on the table for the specified connection.

A successful call to UpdateStatistics()will set the LastKnownRowCount property and the MaximumActualLength (for character and binary columns) property for each DataTypeInfo(Column).

See Also

SQL::TableInfo Object

Limitations

Desktop applications only.
