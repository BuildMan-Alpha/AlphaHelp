<ARRAY>.APPEND_ARRAYS_FILTERED()

Syntax

New_Array as P = <ARRAY>.APPEND_ARRAYS_FILTERED( Array1 as P, Filter1 as C, Array2 as P, Filter2 as C )

Argument

Description

New_Array

The new array produced by appending Array2 to Array1.

Array1

The first array.

Filter1

A character filter expression that selects elements from Array1.

Array2

The second array.

Filter2

A character filter expression that selects elements from Array2.

Description

The <ARRAY>.APPEND_ARRAYS_FILTERED() method selects elements from Array2 and Array1 and merges them to produce New_Array. Both Array1 and Array2 must have a single dimension and have the same structure.

Example

dim cars[10] as P

cars.initialize_properties("manufacturer|model",<<%str%

Ford|Taurus

Ford|Focus

GM|Malibu

Dodge|Status

%str%)

dim trucks[10] as P

trucks.initialize_properties("manufacturer|model",<<%str%

Ford|F-150

Ford|Ranger

Dodge|Dakota

%str%)

dim vehicles[2] as P

vehicles.append_arrays_filtered(cars,"manufacturer='Ford'",trucks,"manufacturer='Ford'")

values = vehicles.dump_properties("manufacturer model")

ui_dlg_box("show_array",<<%dlg%

[.50,10index^||values]

%dlg%)
