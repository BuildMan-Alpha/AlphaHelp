- (Concatenation with Trim Operator)

Syntax

Result as C = Operand1 as C - Operand2 as C

Argument

Description

Operand1

A character, date, or numeric value.

Operand2

A character value or NULL ("").

Description

The Concatentation with Trim operator removes trailing spaces from Operand1 and then appends Operand2 to Operand1.

Examples

dim cc as C

cc = "word "

? cc - ""

= "word"

? cc - "abc"

= "wordabc"
