Logical .NOT.

Syntax

Result as L = .NOT. Expression

Argument

Description

Expression

A value or an Xbasic expression that evaluates to a value.

Description

Logical NOT (.NOT.) reverses the logical state of a single expression.

Example

For example, if Sales is 900 and State is "NY":

? .NOT. (3 > 2)

= FALSE

? .NOT. (3 > 4)

= TRUE

? .NOT. (SALES > 800)

= FALSE

? .NOT. (SALES > 1000 .AND. STATE = "CA") .OR. (STATE = "MA")

= TRUE
