How to hide the tab band that appears at the top of the Tab UI Component

A user recently asked how to hide the tabs that appear at the top of the Tabbed UI Component. At first, this seemed like a strange request. How would you change from one open pane to another? But of course, you can navigate panes by simply clicking on the buttons on the left hand side that open the panes in the first place. And if your screen is space constrained, then it might make sense to hide the tabs.

Here is how you can do it.

A new property has been added to the Tabbed UI Component called 'Javascript to run at startup'. In this property, add this code
e = $('tbi.TABBAND');
e.style.display = 'none';

Next, in the onClick event for each of the buttons on the Tabbed UI you will see code like this:

action_{alias}_04214afe_6bfe_41ab_beef_9a8585fb8678();
