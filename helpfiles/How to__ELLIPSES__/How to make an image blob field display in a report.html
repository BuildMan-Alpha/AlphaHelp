How to make an image blob field display in a report

If you have images in a BLOB field of a SQL table and try to display them in a report, you might see the field name display on the report instead of the images. You can fix this by overriding the field definition when you define the SQL data source for the report.

In the Specify Data Source dialog, click on Advanced Options near the bottom of the dialog. On the Advanced Options dialog check Override field definitions of selected fields and press the Field Definitions button. In the Field Definitions dialog, pick the field you want to override and press the Edit field definition button to make the field convert to the desired type in the temporary data table created for the report.
