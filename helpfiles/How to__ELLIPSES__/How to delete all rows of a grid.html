How to delete all rows of a grid

Say you have an updateable grid. To delete a record, you click the checkbox for 'delete' on each row. What if you want to check all of the rows with a single click?

To do this, just go to the JavaScript function declarations and define this function:
function deleteAllRows() {

       var x = {grid.object}._rowsInGrid;

       //capture the state of the rowRefrshOnDirty property so that it can be 
       //restored later

       var flag = {Grid.object}.rowRefreshOnDirty;

       //set the rowRefreshOnDirty property to false so that no ajax callbacks 
       //are made each time a row is set to be dirty

       {Grid.object}.rowRefreshOnDirty = false;
       for(var i = 0; i <= x; i++) {
           ele = $('{grid.componentName}.V.R'+i+'.__DELETE');
           $svs(ele,true);
           $e.execute(ele,'change');
       }

       {Grid.object}.rowRefreshOnDirty = flag;

}

Then put a button on your grid that calls deleteAllRows(). For example, if you have a free-form section below the grid you would write:
<button onclick="deleteAllRows()">Delete</button>
