How to debug an Alpha Anywhere installation

  1. Locate the installation directory, typically C:\Program Files (x86)\a5V<version>, e.g. C:\Program Files (x86)\a5V11, on a 64-bit system, and C:\Program Files\a5V<version>, e.g. C:\Program Files\a5V11, on a 32-bit system. For a server installation, add " ApplicationServer" to the path, e.g. C:\Program Files (x86)\a5V11 ApplicationServer.
  2. Look for A5INSTALL.LOG
  3. Supply this file to Alpha Software along with your bug report. The bug reporting alias for Version 11 is a5v11bugs at alphasoftware.com
  4. If instructed, you may run the Alpha Anywhere installer from the command line with the /M switch and watch for more information.
