How to call a SQL Server stored procedure with parameters from Xbasic

A SQL Server stored procedure MyStoredProc is defined as follows:

USE [Northwind]
GO
SET ANSI_NULLS ON
GO
SET QUOTED_IDENTIFIER ON
GO
ALTER procedure [dbo].[MyStoredProc] 
    @whatcity varchar(50)
AS
begin
    SELECT CustomerID, ContactName, City from [dbo].[customers] where City = @whatcity
end

As you can see, in this simple case, there is only one parameter.

Here is an interactive window session to call this stored procedure:

dim cn as sql::connection
'Note: the following is really all on one line
cn.open("{A5API='SQLServer', A5DBVersion=2008, Server='(local)', A5ANSINullPadWarn=Default, Trusted_connection=yes, Database='Northwind'}")

dim args as sql::arguments
args.add("cityName","Paris")
?cn.Execute("exec MyStoredProc @whatcity = :cityName",args)
= .T.

'If you are passing in parameters in the same order in which they
' are defined, you don't need to explicitly name each argument.
' So this also works:
?cn.Execute("exec MyStoredProc :cityName",args)
= .T.
