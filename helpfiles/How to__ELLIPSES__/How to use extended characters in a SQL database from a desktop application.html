How to use extended characters in a SQL database from a desktop application

In Alpha Anywhere web applications, characters are displayed as UTF8 by default. In desktop applications, however, characters are displayed in Windows ANSI Code Pages (ACP). Therefore, you need to explicitly convert your data to UTF8 in order to enter high order characters.

cn2.open("Your connection string")
