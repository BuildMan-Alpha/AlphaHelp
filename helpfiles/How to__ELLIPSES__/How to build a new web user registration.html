How to build a new web user registration

See Adding users with a web component or the newer, more complete Registering Web Users in Alpha Five Version 10 for a tutorial.

A dialog is very flexible as it allows the developer to link to other tables and create a number of variations.

The Application Server demo application supplied with Alpha Five includes a couple of variations. Each includes comments in the various event code sections explaining what the code is doing. The dialogs named "DialogWebUsers" (this is the same as the dialog in the tutorial) and "Cust_Security" are good starting points.
