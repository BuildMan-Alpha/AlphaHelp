Overview of Subtract Operations

A Subtract operation takes records from the master table which do not have matching records in the transaction table and copies them to the result table. The result table can contain fields from either the master table or the transaction table, but in all cases the transaction table fields are blank.

Performing an Intersect, Join or Subtract Operation

  1. Click the Operations tab on the Control Panel.

  2. Click the New button. The Create New Operation dialog box appears.

  3. Select "Intersect/Join/Subtract tables" as the operation.

  4. Select the Master table from the Select Table/Set list.

  5. Click the Create button to display the Join Builder or check the More check box and click Create Using Genie to display the Intersect, Join, or Subtract Genies (see Using the Intersect Genie ).

  1. Select the "intersect", "join", or "subtract" from the Operation list.

  2. Select the Transaction and Result tables. In both cases you can click on the Drive button to use the File dialog box.

Specifying Linking Criteria

The linking keys are the fields or expression you use to indicate the fields the master and transaction tables have in common.

  1. Select the Linking key expressions for the Master and Transaction tables.

Specifying the Fields to Include in the Result Table

At the bottom of the Join Builder are three list boxes which you use to determine the fields to include in the result table. To add a field to the result table, click and drag it from the Master Fields list box or the Transaction Fields list box and drop it in the Result Fields list box. If you want to include every field in the master or transaction table, click the Select All button at the bottom of either the Master Fields or Transaction Fields list box.

You can remove unwanted fields from the Result Fields list box by highlighting them and pressing Delete. To highlight multiple fields, hold down SHIFT and click on the fields you want to remove.

  1. In the Field lists indicate the fields from the Master and Transaction tables that should be included in the Result table.

Specifying Filter and Order Criteria

If you want to filter-out records from the master table or the transaction table so they are not included in the Intersect operation, you can enter filter expressions in the appropriate Filter boxes. For a record to be selected, it must cause the filter expression to evaluate to TRUE.

  1. Optionally, specify Filter expressions for the Master and Transaction tables.

  2. Optionally, click to specify additional filter or order expressions for the Master and Transaction tables.

  3. Click the button to run the operation.

Using the Intersect, Join, and Subtract Genies

  1. When the genie starts, click Next > to continue.

  2. Enter the name of the transaction table.

  3. Click Next > to continue.

  1. Enter the name of the result table.

  1. Optionally, click the Advanced Options button to display the Advanced Options dialog box.

  2. Choose whether to warn before overwriting an existing table.

  3. Click OK to continue or Cancel to discard your input.

  4. Click Next > to continue.

  1. Specify the linking fields or expressions for the Master table and Transaction table fields.

  1. Optionally, click ... to display the Advanced Option dialog box to specify options for using field values.

  2. Click OK to continue or Cancel to discard your input.

  3. Click Next > to continue.

  1. Select the fields that you wish to include in the Result Table.

  2. Click Next > to continue.

  1. Optionally, specify additional filtering (a logical expression ) and ordering (a character expression ) criteria for the Master table.

  2. Click Next > to continue.

  1. Optionally, specify additional filter or order criteria for the Transaction table.

  2. Click Next > to continue.

  1. Indicate whether you want to run the operation now, save the operation and then run it, or view the settings of the Intersect/Join/Subtract Card.

  2. Click Finish.

  1. If you selected Run the Intersect/Join/Subtract operation now, (but first save the definition for future use), the Save Intersect/Join/Subtract settings as... dialog box will appear.

  2. Enter the name of the operation into the Name field and click OK.

  3. If you selected Don't run the Intersect/Join/Subtract operation now, just show me how the genie has filled in the Intersect/Join/Subtract card, the Intersect/Join/Subtract Card will appear.

  4. Optionally, click Save to display the Save intersect/join/subtract settings as... dialog box.

  5. Enter the name of the operation into the Name field and click OK.

  6. Optionally, click the Run button on the toolbar to run the operation.
