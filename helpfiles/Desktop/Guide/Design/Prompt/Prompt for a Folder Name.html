Prompt for a Folder Name

Overview

The Prompt for a Folder Name action will prompt the user for a folder name and store it in a variable.

Procedure

  1. Enter the letters of the drive(s) which you want the user to see when selecting a folder.

  2. Enter the name of the variable that will receive the value that the user enters in the Variable Name field.

  3. Optionally, enter the title of the dialog box in the Dialog Title field.

  4. Optionally, enter a default value in the Default Folder field.

  5. Click the Test Dialog button to see how it will appear.

  6. Click Next >.

  7. Optionally modify the comment that describes the action.

  8. Click Finish.
