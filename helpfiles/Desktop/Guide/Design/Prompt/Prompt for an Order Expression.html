Prompt for an Order Expression

Overview

The Prompt for an Order Expression action will prompt the user for an order expression and store it in a variable.

Procedure

  1. Optionally, enter the title of the dialog box in the Dialog Title field. Optionally, click the button to open the Expression Builder.

  2. Optionally, enter a prompt that will appear next to the text box in the Prompt field. Optionally, click the button to open the Expression Builder.

  3. Enter the name of the variable that will receive the value that the user enters in the Variable Name field.

  4. Optionally, enter a default value in the Default Value field. Optionally, click the button to open the Expression Builder.

  5. Select the table or set that will provide values for the Expression Builder.

  6. Click the Test Dialog button to see how it will appear.

  7. Click Next >.

  8. Optionally modify the comment that describes the action.

  9. Click Finish.
