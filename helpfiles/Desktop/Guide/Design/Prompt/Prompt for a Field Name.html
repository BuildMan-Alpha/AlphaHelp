Prompt for a Field Name

Overview

The Prompt for a Field Name action will prompt the user for a field name and store it in a variable.

Procedure

  1. Optionally, enter the title of the dialog box in the Dialog Title field. Optionally, click the button to open the <span class=Screen>Expression Builder</span>.

  2. Enter the name of the variable that will receive the value that the user enters in the Variable Name field.

  3. Optionally, enter a default value in the Default Value field. Click the Test Dialog button to see the field values in the selected table. Optionally, click the button to open the <span class=Screen>Expression Builder</span>.

  4. Select the table or set that will provide values for the Expression Builder.

  5. Click the Test Dialog button to see how it will appear.

  6. Click Next >.

  7. Optionally modify the comment that describes the action.

  8. Click Finish.
