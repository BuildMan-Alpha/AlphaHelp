Creating the Vendors Set

Procedure: Creating the Vendors Set

Follow these instructions to create a set similar to the Vendors set.

  1. Display Tables/Sets tab of the Control Panel.

  2. Click New > New Set.

  3. Select "Vendor" in the Select a table list.

  1. Click Add link....

  2. Select "Product" in the Child Table list.

  3. Select "Vendor_id" in the Parent key list.

  4. Select "Vendor" in the Child key list.

  1. Accept the default values for the Link Type, Referential integrity, and Include record from Parent table controls.

  2. Click OK.
