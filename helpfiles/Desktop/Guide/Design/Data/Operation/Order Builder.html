Order Builder

The Order Builder lets you create powerful sort operations by navigating through an easy-to-understand dialog box.

Procedure

  1. Optionally, select a field from the first Sort by list to be the basis of the secondary sort.

  2. Optionally, click A...Z to sort in descending order.

  3. Optionally, select a field from the first then,sort by list to be the basis of the secondary sort.

  4. Optionally, click A...Z to sort in descending order.

  5. Optionally, select a field from the second then,sort by list to be the basis of the tertiary sort.

  6. Optionally, click A...Z to sort in descending order.

  1. Click OK to continue.

  2. Click Cancel to discard your inputs and exit from the Filter Builder.

  3. Click to display the Order Expression dialog box, which shows the Xbasic filter expression you created in the Order Builder. This is an excellent way to learn the syntax for creating filter expressions.

  4. Click Expression Builder to switch to the <span class=Screen>Expression Builder</span>.
