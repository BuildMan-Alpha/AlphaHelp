Sorting and Selecting Overview

A key function of a database management system is the ability to select specific records and put them in a particular order. Ordering records puts them in a specified sequence, like ordering last names alphabetically. Ordering is also referred to as sorting.

Alpha Anywhere provides a variety of tools for finding, selecting, and ordering records. You can perform powerful and complex searches to group your data into lists that match your specific criteria.

  • Ascending Sort - quickly perform a single field sort

  • Descending Sort - quickly perform a single field sort

  • Multi-Field Sort - quickly perform a multi-field sort

  • Quick Filter - quickly display the set of records that match a field value

  • Quick Filter (Exclude) - quickly display the set of records that do not match a field value

  • Quick Filter For... - quickly display the set of records that do match any value

  • Multi Field Quick Filter - quickly display the set of records that match several field values

  • Find by Key - quickly display the first record that matches an index value

  • Find Text - search for text in one or more fields

  • Find by Form - quickly display the set of records that match several field values
