Resize Fields to Fit Data

To resize field lengths in a table to fit its data:

  1. Display the Tables/Sets tab of the Control Panel.

  2. Select the table.

  3. Right click and select Utilities > Size Fields to Fit... .

  4. Specify the number of characters to add above minimum requirements to pad character fields.

  5. Specify the number of positions to add above minimum requirements to pad numeric fields.

  6. Specify the number of positions to add above minimum requirements to pad decimal places in numeric fields.

  7. Optionally, uncheck Create a backup copy of the original table. Not recommended.

  8. Click OK.
