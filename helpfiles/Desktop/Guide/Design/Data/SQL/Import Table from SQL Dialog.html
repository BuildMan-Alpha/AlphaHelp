Import Table from SQL Dialog

The Import Table from SQL dialog helps you import a SQL back-end table to an Alpha Anywhere .DBF table.

Procedure

  1. Select a back-end SQL database type from the SQL Connection list.

  2. Click Connect to display the Connect to Database dialog.

    1. Navigate to and select a database in the File Name control.

    2. Enter the other values required to access the database.

    3. Click Test Connection.

    4. Click OK if the connection succeeded.

  3. Select the SQL table to import from the SQL Table control.

  4. Select the local Alpha Anywhere .DBF table to create from the Local File list or click to navigate to, name, or select a table to overwrite.

  5. Optionally, click Advanced to display ...

  6. Click Import.

  7. Click Close to exit.

Limitations

Desktop applications only.
