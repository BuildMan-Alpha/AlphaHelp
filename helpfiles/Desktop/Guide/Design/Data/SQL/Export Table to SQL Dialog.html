Export Table to SQL Dialog

The Export Table to SQL dialog helps you export a local Alpha Anywhere table to a back-end SQL database.

Procedure

  1. Select the local Alpha Anywhere .DBF table to export from the Local File list or click to navigate to and select the table.

  2. Select the SQL database type from the SQL Connection list.

  3. Click Connect to display the Connect to Database dialog.

    1. Navigate to and select a database in the File Name control.

    2. Enter the other values required to access the database.

    3. Click Test Connection.

    4. Click OK if the connection succeeded.

  4. Enter the name of the SQL table to create or select a table to overwrite in the SQL Table control.

  5. Click to display the Primary Key Columns dialog.

    1. select one or more fields from the Table Columns list and click to move them (in order) to the Primary Key Columns list.

    2. Click OK to continue.

  6. Optionally, click Advanced to display the Export Table to SQL - Advanced Settings dialog.

  7. Click Export.

  8. Close Close to exit.

Limitations

Desktop applications only.
