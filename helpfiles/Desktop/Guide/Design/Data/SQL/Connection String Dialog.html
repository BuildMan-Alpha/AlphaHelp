Connection String Dialog

The Connection String dialog allows you to define a connection with a SQL back-end database. You can open the Connection String dialog from the Database Explorer.

  1. Optionally, to use an existing connection string:

    1. Select Use a named connection string.

    2. Select the connection string from the drop-down list.

    3. Click OK to continue.

  2. Optionally, to create a new connection string:

    1. Select Specify a connection string.

    2. Optionally, enter the string into the text box.

    3. Optionally, click Build to display the <span class=Screen>Create SQL Connection String</span> dialog.

    4. Optionally, click Save as named connection string to create a reusable connection.

  3. Click OK to continue.

Limitations

Desktop applications only.
