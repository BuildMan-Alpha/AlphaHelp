Emptying a Table

To remove records from the tables in a set:

  1. Display the Tables/Sets tab of the Control Panel.

  2. Select the table.

  3. Right click and select Utilities > Empty the table .

  4. Optionally, check Pack table after emptying. This will remove all deleted records.

  5. Click OK.
