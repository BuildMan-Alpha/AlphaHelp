Applying a Range to a Table

To apply a range to a table while viewing a form or browse:

  1. Select Records > Advanced > Apply Range... .

  2. Select an index or query from the Index/Query list.

  3. Select Ascending or Descending order.

  4. Click to select the first record from the Index Key Values dialog. Note, the dialog only shows the records selected by the current filter.

  1. Click to select the last record from the Index Key Values dialog.

  2. Optionally, add additional selection criteria in the Filter field. Optionally, click to use the Expression Builder to help you define the expression.

  3. Optionally, select whether you want to show Marked, Not Marked, or Both (all) records.

  4. Optionally, check Unique Records Only to avoid showing duplicates.

  5. Click Use to apply the range and update the form or browse.

  6. Click OK to apply the range without updating the display.

  7. Click Close to exit without applying the range.
