Using the Alpha Anywhere Internal Email Client

Button6 sends an email to the person identified by the firstname and lastname fields at the address identified by email field. If the email address exists, the Action Script opens the Alpha Anywhere email client, as illustrated below.

If the email field is blank, the Action Script displays an error message that tells the user to first input an email address.

Action Script Run by the Button6 OnPush Event

The text of the Action Script is as follows:

Display a message Box(ID, 'noemail')

Send an email message

An Explanation of the Action Script

The first line of the Action Script displays a message box ( Category : Dialog Boxes, Action : Display a Message Box). The action is a conditional event that runs if the email field is blank.

Display a message Box(ID, 'noemail')

The second line of the Action Script displays sends an email message ( Category : Email, Action : Send an email message). The action is a conditional event that runs if the email field is blank is non-blank.

Send an email message

The email order_confirm template defines the variable contents of the email message. For information about creating an email template, see Creating an Email Template.

Procedure: Creating the Action Script

  1. Sketch a new button onto the form.

  2. Enter its caption into the Label field.

  3. Click the Next > button to display the Code Editor.

  4. Click Add New Action to start the Script Genie.

  5. Select "Dialog Boxes" in the Category list.

  6. Select "Display a Message Box" in the Action list.

  7. Click OK.

  8. Click Next > to accept the default button arrangement (below).

  9. Click the radio button to the left of the error icon.

  10. Enter the word "Error" for the message box title.

  11. Enter the following message text: "You have not specified an email address for this customer. Please click the Edit icon next to the Billing Address and enter an email address."

  12. Click Next >.

  13. Enter a unique name for the dialog.

  14. In the then... field select "Do Nothing - just set a Flag Variable to .T.".

  15. Click Next > .

  16. Optionally enter a description in the Comment field.

  17. Click Finish to exit the Script Genie.

  18. In the Code Editor click If Conditions.

  19. Check the Action is conditional check box.

  20. In the Execute action if field select the "Condition expression is True".

  21. Enter the Expression Customer->Email = "".

  22. Click Add New Action.

  23. Select "Email" in the Category list.

  24. Select "Send an email message" in the Action list.

  25. Click OK.

  26. In the Open which email client list select "Built-in Alpha Anywhere Email client".

  27. Select "order_confirm" from the Template name list.

  28. Click Next >.  

  29. Check Load message body from a saved email template.

  30. Check Use same template as for message header.

  31. Select "Open Alpha Anywhere email client as a modal dialog box".

  32. Click Finish.

  33. Check Action is conditional.

  34. In the Execute action if list select "Condition expression is True".

  35. In the Expression field enter Customer->Email <> "".

  36. Click Next >.
