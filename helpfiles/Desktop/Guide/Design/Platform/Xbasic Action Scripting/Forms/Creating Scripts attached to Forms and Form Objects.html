Creating Scripts attached to Forms and Form Objects

To create a new script:

  1. From the menu bar, select Scripts > New Script, and choose Use Action Script Editor or Use Xbasic Code Editor.

  2. If you are going to create a script:

    1. Select either the Script Genie or Xbasic radio buttons.

    2. Enter the name of the script in the Script Name field.

    3. Optionally, check Put script on Favorites list. This will make the script appear in the Scripts menu for the form or browse.

    4. Optionally, enter a comment in the Script Description field. Your Favorites list will show the description in place of the script name.

    5. Optionally, check Associate image with script to display the Image Name field.

    6. Click to the Insert Image dialog box to select an image.

  3. If you are going to create a function:

    1. Display the Functions tab.

    2. Enter the Function Name.

    3. Select the Return Type.

    4. Optionally, if you know what arguments the function will take, check Specify optional arguments.

    5. Enter the names and types of the arguments.

  4. Click OK to continue or Cancel to discard your inputs.

    1. If you selected Script Genie in step 2a, the Action Script Editor opens.

    2. If you selected Xbasic in step 2a or decided to create a function, the Code Editor opens.

When you are done, save the script and it will appear in the Scripts menu for the form or browse.
