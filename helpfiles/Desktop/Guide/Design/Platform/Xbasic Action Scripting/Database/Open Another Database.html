Open Another Database

Overview

The Open Another Database action closes the current database and opens another. You can specify the database to open ahead of time, or prompt the user at run-time.

Procedure

  1. On the first page of the Script Genie select either:

    • Specify name of database to open now

    • Prompt for name of database to open at runtime

  2. If you selected "Specify name of database to open now", click the folder icon on the Database Name field, navigate to and select the database you want to open. Click Open.

  3. Click Next >.

  4. Optionally modify the default comment which describes the action.

  5. Click Finish.
