Duplicate Record

Overview

The Duplicate Record action creates a duplicate copy of the current record. When the operation is completed, focus is given to the duplicate.

This action is useful for repetitive or redundant data entry, as when entering records for several people at the same company, or preserving an older copy of a record. The current record is copied to a new record, ready for editing.

Note : You must be in Enter mode before copy data. For this, you may use the Enter New Record action.

Procedure

  1. Make a selection in the Duplicate the current record in which Form/Browse window? field. This field will be grayed out unless this action is based on a Form or Browse. The choices are:

    • "Current Form or Browse Window".

    • "Another Form or Browse Window".

  2. If you selected "Another Form or Browse Window", in the Window Name field select either:

    • "Forms"

    • "Browses"

  3. Select one of the listed forms or browses.

  4. If you selected a form in step 3, optionally select whether you are working with the form ( Parent Level ) or with an Embedded Layout.

  5. If you selected "Embedded Layout" in step 4, click the button to select an embedded form or browse.

  6. Make a selection from the Option list. The options are:

    • "Display a confirmation dialog box"

    • "Do not display a confirmation dialog box"

  7. Click Next >.

  8. Optionally modify the comment that describes the action.

  9. Click Finish.
