Display Script Favorites Pick-list

Overview

The Display Script Favorites Pick-list action displays script favorites in the Select Script to Run dialog box, toolbar, or pop-up menu.

Procedure

  1. Select a style from the Pick List Style list. The options are:

    • "Dialog box"

    • "Pop-up Menu"

    • "Toolbar"

  2. Click Next >.

  3. Optionally modify the comment that describes the action.

  4. Click Finish.
