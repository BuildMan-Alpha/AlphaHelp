Set Field Values

 

Overview

The Set Field Value action enters values into one or more fields on a form or browse layout.

Procedure

  1. Make a selection in the Set the value of fields in which window? field. This field will be grayed out unless this action is based on a Form or Browse. The choices are:

    • "Current Form or Browse Window".

    • "Another Form or Browse Window".

  2. In the Window Name field select one of the open forms or browses.

  3. Click the button to add fields to the Set the value of the following fields list using the Select a Control dialog box, or click to remove them.

  4. For each field, select a value in How will you specify the field's value?. There are three options:

    • "Specify the value now"

    • "Specify a variable the contains the value"

    • "Specify an expression that evaluates to the value"

  5. If you selected "Specify the value now", enter a value in the Value field.

  6. If you selected "Specify a variable the contains the value", click the Value icon to select an existing variable in the Variable field.

  7. If you selected "Specify an expression that evaluates to the value", enter an expression into the Expression field. Optionally, click to use the Expression Builder.

  8. Click Next >.

  9. Optionally modify the comment that describes the action.

  10. Click Finish.
