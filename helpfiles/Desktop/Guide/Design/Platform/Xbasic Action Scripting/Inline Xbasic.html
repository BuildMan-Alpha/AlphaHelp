Inline Xbasic

Overview

The Inline Xbasic action inserts Xbasic into a script. This should only be used if you know how to use Xbasic. The window that pops up is a Code Editor window with standard features such as bubble help, code completion, and syntax checking.

Procedure

  1. Enter your Xbasic code.

  2. Click F4 to validate your Xbasic code.

  3. Click Zoom to Resizable Window if you want to work with a larger window. Picture

  1. Optionally modify the comment that describes the action.

  2. Click Finish.
