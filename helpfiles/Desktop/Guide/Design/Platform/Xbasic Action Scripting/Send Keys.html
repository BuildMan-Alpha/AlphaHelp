Send Keys

Overview

The Send Keys action specifies keystrokes to send to Alpha Anywhere or another application. Immediately before the Send Keys action to set focus? to the window and/or object to which you want to send keystrokes.

Procedure

  1. Optionally, check the Record keystrokes check box. Enter the keystrokes that you wish to send into the text window. Preface shifted keys with the plus "+" character. Preface control keys with the caret "^" character. Preface ALT keys with the caret "%" character.

  2. Click Clear to erase the contents of the text window.

  3. Select where you are going to send the keystrokes. The options are:

    • "Send keystrokes to an Alpha Anywhere window"

    • "Send keystrokes to another program"

  4. If you selected "Send keystrokes to another program", select the program from the External program window name field.

  5. Click Next >.

  6. Optionally modify the comment that describes the action.

  7. Click Finish.
