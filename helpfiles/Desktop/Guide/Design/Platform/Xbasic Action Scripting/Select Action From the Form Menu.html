Select Action From the Form Menu

Overview

The Select Action From the Form Menu action allows an object to run one of the actions listed in the form's menu.

Procedure

  1. Click Select Action... to display the form menus.

  2. Navigate the menus and select a specific menu function.

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
