Trapping Tab Control Events

You can generate an event when the user changes the tab page. The syntax is:

{tab=variable!event_name}

Whenever the user changes to a different pane on the tab control, the event_name is fired. This allows you to create event handling scripts that check to see if the user is allowed to visit that pane.
