The Main Menu Form

The Menu Menu form illustrates two types of menuing techniques and the recommended architectural approach for attaching scripts to buttons.

  • Starting the AlphaSports Application shows what AlphaSports does as it starts up.

  • Ending the AlphaSports Application shows what AlphaSports does to qualify the user's request to exit and to clean up afterwards.

  • Displaying the Main Menu shows how to display a form on Alpha Anywhere startup.

  • Displaying the Navigation Toolbar shows how to decide when to display the Navigation Toolbar.

  • Creating the Navigation Toolbar shows how to create the Navigation Toolbar.
