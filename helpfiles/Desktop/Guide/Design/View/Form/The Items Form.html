The Items Form

The Items form allows the operator to enter a new item record. However, the operator should only use the form when the number of the parent invoice is known.

  • Using Xbasic to Control Access to a Form illustrates a method for implementing this restriction.

  • Linking to a External Help File Topic shows how to connect a topic in a compiled HTML help file (.CHM) or Windows help file (.HLP) to a form or button.
