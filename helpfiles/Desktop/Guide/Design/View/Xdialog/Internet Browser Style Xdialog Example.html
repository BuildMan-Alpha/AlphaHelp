Internet Browser Style Xdialog Example

Within the Action Scripting Editor, select "Dialog Boxes" and "Display an Xdialog Box".

  1. Step one is to specify the variable associated with the browser and the control's dimensions.

  1. Step two is to specify how the URL will be specified. You can optionally enable ActiveX events (for which you will have to write custom code).

  1. Next, you set the name and scope of the return variable from the underlying UI_DLG_BOX() function.

  1. Next, you set the dialog title and other text that might appear on the dialog.
