Line Breaks

In the Putting a Button on the Dialog example, it would be nice if the button were on a new line. This is accomplished by using the line break symbol - a semi-colon ( ; ).

Instead of using this code, which puts the text and buttons on the same line:

result=ui_dlg_box("Title",<<%dlg%

Hello World

%dlg% )

the following script puts the buttons on a new line:

result = ui_dlg_box("Title", <<%dlg%

Hello World ;

%dlg% )
