Browse Banding

You can specify the background color and style of alternate groups of browse rows. This feature, named Green Banding is illustrated below. Each group can be any number of rows and have any fill style.

This browse has alternate bands of light green and white.

This browse has two bands of gradient color and one band of white. Picture

To apply banding to a browse.

  1. Display the Browse tab of the Control Panel.

  2. Select the browse to modify and click Design.

  3. Click Browse > Browse Properties... .

  4. Display the Browse tab of the Properties dialog.

  5. Click Row Color Properties to display the Row Color Properties dialog.

  1. Define color group one.

    1. Select "Color 1" and check Enabled.

    2. Define its fill style and color.

    3. Select and entry from the Number of Rows list.

  2. Define color group two.

    1. Select "Color 2" and check Enabled.

    2. Define its fill style and color.

    3. Select and entry from the Number of Rows list.

  3. Optionally, define color group three.

  4. Optionally, define color group four.

  5. Click OK.

Limitations

Desktop applications only.
