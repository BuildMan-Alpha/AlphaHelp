Placing Dynamic Content into Browse Columns

You may place dynamic (computed) content into any column of a browse. The column may contain any combination of the following content:

  • static text

  • images

  • dynamic text (computed from data in the current row)

To create dynamic content for a browse column:

  1. Open a browse in design mode.

  2. Select Column > Properties.

  3. Display the Other tab of the Column Properties dialog.

  4. Click Edit Display Expression to display the Conditional Expression Builder.

  5. Click OK to save your edits and exit from the Column Properties dialog.
