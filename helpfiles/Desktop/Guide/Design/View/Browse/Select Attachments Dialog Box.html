Select Attachments Dialog Box

Use the Select Attachments dialog box to add attachments to an email message.

  1. Click Select File to browse for files on your computer to add to the list. Press OK to attach the file(s).

  2. Click the button to use the Expression Builder or prefix the entries with "=" to use expressions.

  3. Check the Put attachment into a Zip file check box if you want Alpha Five to zip up your selections.

  4. Click OK to continue or Cancel to discard your selections.
