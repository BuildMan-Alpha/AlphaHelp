Working with Named DataSources

Create a Named DataSource

To create a named data source:

  1. Create an ad hoc data source.

  2. In the Specify DataSource dialog click Save as named datasource.

  3. Enter a name in the Save DataSource dialog and click OK.

Use a Named DataSource

If you have already created one or more named data sources:

  1. Select the Use named DataSource radio button.

  2. Click OK.

  3. Continue with Design-Time Properties.

Edit a Named DataSource

To modify a named data source:

  1. Display the Reports tab of the Control Panel.

  2. Click New to display the New Report dialog.

  3. Select "" and click OK to display the Specify DataSource dialog.

  4. Choose the data source from the Select Named DataSource list and click Edit.

Rename a Named DataSource

To modify a named data source:

  1. Display the Reports tab of the Control Panel.

  2. Click New to display the New Report dialog.

  3. Select "" and click OK to display the Specify DataSource dialog.

  4. Choose the data source from the Select Named DataSource list and click Rename.

  5. Enter the new name and click OK.

Copy a Named DataSource

To modify a named data source:

  1. Display the Reports tab of the Control Panel.

  2. Click New to display the New Report dialog.

  3. Select "" and click OK to display the Specify DataSource dialog.

  4. Choose the data source from the Select Named DataSource list and click Save As.

  5. Enter the new name and click OK.
