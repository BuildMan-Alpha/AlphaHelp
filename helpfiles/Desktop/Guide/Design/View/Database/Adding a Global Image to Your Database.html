Adding a Global Image to Your Workspace

You can store bitmap images in the workspace (e.g., in the ALB file). The bitmaps are shown on the Code tab in the Control Panel. The advantage of storing the bitmaps in the workspace is that the bitmap files will be copied when you copy the workspace.

You can insert icon bitmaps (ICO files) into your workspace from the Windows Explorer by dragging and dropping the ICO file into the Alpha Anywhere Control Panel.

Adding a Global image to your Workspace

  1. Display the Code tab of the Control Panel.

  2. Click New > Bitmap to display the Create Global Bitmap dialog box.

  3. Enter the image's name in the Bitmap Object Name field.

  4. Optionally, click Import Bitmap from File.

    1. Navigate to and select the image file.

    2. Click Open.

  5. Optionally, if you have previously copied an image to the clipboard, click Paste Bitmap from Clipboard.

  6. Optionally, if you want to store the image in a different workspace.

    1. Click to navigate to and select a different workspace library (.ALB) file.

    2. Click Open.

  7. Click OK to add your new global image or Cancel to quit without action.

You will have to exit from your workspace and reload it before you can use a stored global bitmap. To use a stored global bitmap.

  1. Display the Insert Image Dialog Box.

  2. Select Current Workspace Images.

  3. Select the image from the list that appears.

  4. Click Insert.
