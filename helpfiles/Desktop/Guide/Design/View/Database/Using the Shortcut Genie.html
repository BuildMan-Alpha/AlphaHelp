Using the Shortcut Genie

The Create Shortcut Genie creates a shortcut to load Alpha Five and start a particular database. You can place the shortcut on the Start menu, in a particular group, or on the desktop. The Create Shortcut Genie lets you specify your own splash screen, title, and icon for the Alpha Five program window, and also to specify special flags, such as whether the Alpha Five window should be opened minimized or maximized and whether the user should be able to resize the main Alpha Five window.

To open the Create Shortcut Genie, select Tools > Create Shortcut.

To create a shortcut:

  1. Enter a name into the Shortcut name field.

  2. Make a selection for the location of the shortcut. The options are:

    • None (the entry may still be in the Start Menu)

    • On desktop

    • In specified folder

  3. If you selected "In specified folder" in step 2, click to navigate to and select the destination folder.

  4. Make a selection for the location of the Start Menu entry. The options are:

    • None

    • On main Start menu

    • In Start menu Program group

  5. If you selected "In Start menu Program group" in step 4, enter the name of the program group or click to select one from the list.

  6. Optionally, change the shortcut's owner to Current User. This option is available for Windows NT/2000/XP users only.

  7. Display the Appearance tab.

  8. Optionally, click Use custom title and specify the title of the main Alpha Five window.

  9. Optionally, change the Alpha Five splash screen. The options are:

    • No splash screen

    • Custom bitmap

  10. If you selected "Custom bitmap" in step 9, click to navigate to and select a bitmap.

  11. Optionally, change the Alpha Five icon. The options are:

    • No splash screen

    • Custom icon

  12. If you selected "Custom icon" in step 11, click to navigate to and select as icon.

  13. Display the Window tab.

  14. Optionally, change the size of the main Alpha Five window. The options are:

    • Default - previous size of main Alpha Five window

    • Maximized

    • Minimized

  15. If you want to restrict the user's ability to manipulate the size of the main Alpha Five window, check Disable ability to minimize or maximize.

  16. If you want to restrict the user's ability to close the main Alpha Five window, check Open Alpha Five without a system menu.

  17. Display the Advanced tab.

  18. Enter any Xbasic commands that should be executed when Alpha Five is started from the shortcut. This code executes after the autoexec script, if you have one.

To display an initial or startup form, refer to Creating and Assigning a Startup Form.
