Restore a Workspace Backup


In Alpha Anywhere the Restore Workspace utility supports dates in MM/DD/YY formats.

 

Restore a workspace:

 

  1. From the Control Panel select Tools > Backup/Restore Workspace > Restore Workspace.

  1. Optionally, select a backup file from the list and click Restore.

  2. Optionally, click Files and select the specific workspace backup to restore, or select to back up and restore individual files.

  1. Alternatively, you can use the Alpha Anywhere Welcome screen Workspace & Tasks tab to select a workspace to restore.
