Users and Groups

DESKTOP

Define Users and Groups

If you define individual user accounts, you specify the user name, password, and an optional startup script for when that user logs on. This also lets you define different startup menus for each application user. You can also specify what group each user belongs to.

To define users and groups, select Tools > Security > Define Users and Groups... to display the Users and Groups dialog box.

Adding a New User

To add a new user:

  1. Click Add User to display the Add New User dialog box.

  2. Enter the User Name.

  3. Enter the Password.

  4. Re-enter the password in the Confirm Password field.

  5. Optionally, select a script from the Startup Script list.

  6. Click OK to add the new user or Cancel to quit.

To modify a user's settings, select a user from the Users list and click Edit User. This allows you to change the User Name, Password, or Startup Script fields.

Deleting a User

To delete a user, select it from the Users list and click Delete User.

Creating a Group

To create a group:

  1. Display the Groups tab and click Add Group to display the Add New Group dialog box.

  2. Enter the Group Name and click OK.

Editing a Group

To edit a group, select the group in the Groups list and click Edit Group. This allows you to change the Group Name field.

Deleting a Group

To delete a group, select it from the Groups list and click Delete Group. Click OK to confirm the deletion.

Adding a User to a Group

To add a user to a group:

  1. Display the Users tab.

  2. Select the user in the Users list.

  3. Place a check mark next to all groups that include the user.

Exporting User and Group Information

To export user and group information to an Alpha Anywhere database:

  1. Click Import/Export.

  2. Click Export User/Group Information.

  3. Click OK.

  4. The default name of the file will be User_group_info for .Adb.

  5. Select where to store the file.

  6. Click OK.

Importing User and Group Information

To import user and group information from an Alpha Anywhere database:

  1. Click Import/Export.

  2. Click Import User/Group Information.

  3. Click OK.

  4. Navigate to and select the User Accounts file to import.

  5. Click OK.

Importing Windows User Information

To import Windows user information:

  1. Click Import/Export.

  2. Click Import Windows User Information.

  3. Click OK to display the Select Users dialog.

  4. In the Select users from list pick "Computer" or "Domain".

  5. If you selected "Domain" in step 4, select a domain from the Domain Name list.

  6. Click Get List.

  7. Place a checkmark next to each user name that you wish to import.

  8. Optionally, click Select All or Unselect All to speed your user selection.

  9. Click OK to import names or Cancel to quit without taking any action.

  10. Provide a default password for the new users.

  11. Click OK.

Importing Windows Group Information

To import Windows group information:

  1. Click Import/Export.

  2. Click Import Windows Group Information.

  3. Click OK to display the Select Groups dialog.

  4. In the Select users from list pick "Computer" or "Domain".

  5. If you selected "Domain" in step 4, select a domain from the Domain Name list.

  6. Click Get List.

  7. Place a checkmark next to each user group that you wish to import.

  8. Optionally, click Select All or Unselect All to speed your user selection.

  9. Click OK to import names or Cancel to quit without taking any action.

Logging in as a Different User

To log in as a different user:

  1. Select Tools > Security > Log in as a Different User... to display the Changed Logged on User dialog box.

  2. Click OK to confirm.

  3. Select your User Name.

  4. Enter your Password.

  5. Click Login.
