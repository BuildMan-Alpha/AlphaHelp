Restore Default Printer

Overview

The Restore Default Printer action restores the default printer for your computer. You should not need to reset it unless you previously used the Override Default Printer Action. To get to the Restore Default Printer option, open a workspace and then go to View > Code Editor. Once the Editor is open you can Select an Action by doing Right Click > Genies > XBasic Script Genie. Scroll down to Reports (Print, Preview, Send, etc.) under the category list and then choose Restore Default Printer from the list of actions that appear on the right of the Select Action dialog box.

Procedure

  1. Optionally modify the comment that describes the action.

  2. Click Finish.
