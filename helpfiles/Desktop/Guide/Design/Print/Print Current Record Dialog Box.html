Print Current Record Dialog Box

  1. Select one of the four radio buttons in the Print Option group.

  2. Select the Layout type. There are five options.

    • Report"
    • "Label"
    • "Letter"
    • "Form"
    • "Browse"

  3. Select a layout from the list.

  4. Optionally, click the button to view the Xbasic code generated by this action.

  5. Optionally, click the button to save these settings for the next time you run this action.

  1. If you selected "Print" in the Print Option group, click Print.

  2. If you selected "Preview" in the Print Option group, click Preview.

  3. If you selected "Email" in the Print Option group, optionally click Send Options.

  4. Select the type of data you are sending from the Send As list. There are four options:

    • "Adobe Acrobat PDF File". Can be displayed with Adobe Acrobat Reader.

    • "Dynamic HTML File". Can be displayed with a browser.

    • "Rich Text Format File". Can be displayed with Microsoft Word or Microsoft Wordpad.

    • "Plain Text File". Can be displayed with Microsoft Notepad and many other text editors.

  5. Select the email client you would like to use from the Sending Using list.

  6. Click OK to continue or Cancel to discard your selections.

  7. Click Send.

  1. If you selected "Save to File" in the Print Option group, optionally click Save Options.

  2. Select the type of data you are sending from the Send As list. There are four options:

    • "Adobe Acrobat PDF File". Can be displayed with Adobe Acrobat Reader.

    • "Dynamic HTML File". Can be displayed with a browser.

    • "Rich Text Format File". Can be displayed with Microsoft Word or Microsoft Wordpad.

    • "Plain Text File". Can be displayed with Microsoft Notepad and many other text editors.

  3. Optionally, check the Open file with associated application? check box. This specifies whether Alpha Anywhere should open the file with its associated application once it is created.

  4. Click OK to continue or Cancel to discard your selections.

  5. Click Save.
