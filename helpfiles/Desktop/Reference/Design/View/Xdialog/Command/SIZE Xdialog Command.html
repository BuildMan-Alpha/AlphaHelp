{SIZE}

Syntax

{SIZE= Width |remember= Name }

Argument

Type

Description

Width

N

The width of the dialog in the units of the measure (the default is inches).

Name

C

A reference name to use in storing and retrieving the size information.

Description

The {SIZE} command sets the width of the dialog, using the units specified in the {UNITS} command.

Limitations

Desktop applications only

Example

This example sets the width of a dialog.

ui_dlg_box("Xdialog Sample", <<%dlg%

{size=3};

{lf};

<15,2OK>;

%dlg%)

You can also retrieve and set the size previously stored with the {size=remember=name} command.

ui_dlg_box("Sticky",<<%dlg%

{position=remember=sticky}

{size=remember=sticky}

This is a sticky dialog | ;

{stretch=center}

.30,10dummy

%dlg%)

? ui_dlg_remembered_get("sticky")

= " ,0.78125,2.666667,2.510417,2.427083,0,0"
