Letting the User Modify a Browse's Row Height

This topic demonstrates a technique for allowing a user to modify and save the height of a browse row. In this case the browse is embedded on a form. There are 2 buttons to increase and decrease row height.

The form has the following code under its OnInit event. The script reads the saved value of the browse's row height from the registry. If the value has not been previously saved or it is zero (  ptr.i = 0 ), the script assigns a default value ( 20 ) to the row height, saves this value in the registry, and sets the row height of the browse (  browse1.row_height = ptr.i ).

dim ptr as P

dim ptr.i as N

registry.load_settings("Browse_Row_Height", ptr)

if (ptr.i = 0) then

    ptr.i = 20

    registry.save_settings("Browse_Row_Height", ptr)

end if

browse1.row_height = ptr.i

The button has the following code under its OnPush event. The script reads the current value of the browse's row height (  browse1.row_height ), increases it by 1, and saves the value in the registry.

dim i as N

dim ptr as P

dim ptr.i as N

ptr.i = browse1.row_height

browse1.row_height = ptr.i+1

registry.save_settings("Browse_Row_Height", ptr)

The button has the following code under its OnPush event. The script reads the current value of the browse's row height (  browse1.row_height ), decreases it by 1, and saves the value in the registry.

dim i as N

dim ptr as P

dim ptr.i as N

ptr.i = browse1.row_height

browse1.row_height = ptr.i-1

registry.save_settings("Browse_Row_Height", ptr)
