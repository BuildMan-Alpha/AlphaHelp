email_send_noprofile()

Syntax

P Result = email_send_noprofile(P pSmtpSettings ,P pMessageSettings ,C secureMode [,L autoWrap [,L flagUseDotNetLibrary ]])

Argument Description
P pSmtpSettings See below
P pMessageSettings See below
C secureMode blank, 'SSL', or 'TLS'
L autoWrap (optional) Word-wrapping mode; does not apply if flagUseDotNetLibrary is .t., and irrelevant for HTML mail; default is .T.
L flagUseDotNetLibrary (optional) By default, the e-mail is sent using Alpha Five's internal 'sockets' object. Since V11 has access to the full Microsoft .NET Framework, you can also send the e-mail using methods in the .NET Framework by setting this value to .T.; default is .F.

Description

Sends an e-mail message. All e-mail settings are passed in. Does not use an Alpha Five e-mail profile. To see what properties must be set in pSmtpSettings and pMessageSettings when coding, you can run the a5_ShowEmailSettings() function from the Interactive Window.

Returns an object with two properties:

pFlag.error either .t. or .f.
pFlag.errorText error text if there was an error


pSmtpSettings is an object with these properties:

psmtpSettings.smtp_server the address of the smtp server. e.g.'smtp.gmail.com'
psmtpSettings.smtp_username your email account user name, e.g. 'fred' or 'fred@mycompany.com'
psmtpSettings.smtp_password your email account password, e.g. 'mysecretpassword'
psmtpSettings.smtp_port the port e.g. 465. This depends on the server. For example with Gmail, the port is often 587


pMessageSettings is an object with these properties:
(You can omit certain of the properties, if not needed)

pmessage.message the plain text version of the message
pmessage.html_message the html version of the message
pmessage.send_to a comma delimited list of email addresses
pmessage.subject the message subject, in plain text
pmessage.attachments a comma delimited list of file attachements
pmessage.cc a comma delimited list of email addresses
pmessage.bcc a comma delimited list of email addresses
pmessage.lRelated .t. or .f.
pmessage.ReturnReceipt .t. or .f.
pmessage.from the email address of the sender
pmessage.from_alias friendly/display name of the sender
