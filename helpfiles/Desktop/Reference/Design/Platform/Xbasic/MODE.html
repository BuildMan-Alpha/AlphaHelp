MODE

Syntax

MODE Argument

Argument

Description

Argument

Defines the mode to use. The options are:

  • "GRAPHICS"

  • "TEXT"

Description

Alpha Five has a very powerful set of Xbasic commands that allow you to create bitmaps. You can experiment with these commands in the Interactive window by executing the MODE GRAPHICS command. This will turn the top half of the Interactive window into a bitmap where you can see the results of executing graphics commands.

Example

Try this in the Interactive window.

mode graphics

 

set_brush("blue")
fill_rect(0,0,2,2)
set_font("Arial,15,b","Red")
text("Hello World",.5,.5)
bitmap("$a5_open",.5,.7)
set_brush("Cyan")
fill_oval(.2,.9,.5,.5)

mode text
