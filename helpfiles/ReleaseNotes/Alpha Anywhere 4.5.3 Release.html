Alpha Anywhere 4.5.3 - Overview

(Released April 9, 2018) This release includes a number of new features in virtually every part of Alpha Anywhere, with special emphasis on features related to data integration and security.

Data Integration Features

Alpha Anywhere has made it easier to consume web services, and to create your own web services. The features include:

  • Improved Salesforce Integration - Alpha Anywhere has made it even easier to connect your applications to Salesforce, making Alpha an ideal way to build applications using Salesforce as the system of record.
  • Alpha TransForm Integration - Alpha Anywhere includes major new features for working with Alpha TransForm (a no-code solution for building mobile data collection applications). Features include full CRUD support for TransForm in List Controls in the UX - without the need to code. Also, an API explorer tool to make it easy to write code against the TransForm API.
  • REST API Builder - Easily create your own RESTful services that you write yourself or import from Swagger - greatly improving the interoperability of your apps.

Security Features

Alpha Anywhere continues to make improvements to keep your applications safe. Improvements in this version include:

  • Upgraded SSL/TLS Support - You now have fine grain control over TLS connection options when working with remote servers. This allows your application to comply with specific company standards.
  • Cipher Upgrades - The Alpha Anywhere default cipher list have been updated to comply with newer standards, and you have the ability to specify your own cipher lists.
  • Cookie Enforcement - Alpha Anywhere now enforces cookie use rules to eliminate the use of cookieless sessions, which could otherwise pose a session hijacking threat.
  • Better support for self-signed certificates - Testing applications in Chrome using the development server with SSL enabled has been improved.
  • OpenSSL Update - Alpha Anywhere uses the latest version of OpenSLL, version 1.0.2o.

Other Features

Aside from data integration and security improvements, many features have been added in other areas as well. Some of these features include:

  • Instant Update for Mobile Apps - This allows you to (instantly) update your mobile applications on users devices - without requiring you to submit your app to PhoneGap build, re-publish your application to the app stores, or requiring your users to look for and manually install updates.
  • PhoneGap Version Upgrades - Alpha Anywhere now support the latest version of PhoneGap including PhoneGap cli-7.1.0 and PhoneGap 8.0.0.
  • Significant IDE improvements - These include new tools, hotkeys, and new browser options when testing UX Components in Live Preview.
  • New Genies for SQL - Alpha Anywhere now includes new genies that write Xbasic code that performs CRUD operations against SQL databases.
  • New Application Templates - These include a UX for managing users and roles in the Alpha Anywhere security framework and a UX template for creating master detail applications using multiple components.
  • New Client-side Data Presentation Features - These include friendly dates (such as tomorrow, 3 days ago, etc.), new dynamic disclosures allowing your app to create disclosures on the fly using ControlBar controls (i.e. button, button-lists, etc.).
  • New Xbasic Functions - These include a new functions for working with JSON and Alpha TransForm.

For a full list of changes and features in this release, visit the Release Notes.
