+ (Date Addition)

Syntax

Result as D = Operand1 as D + Days as N

Argument

Description

Operand1

A date value.

Days

A number.

Description

Date addition produces a date value by adding or subtracting an integral or fractional number of Days to Operand1.

Examples

dim start as D

start = {12/1/02}

? start + 30

= {12/31/2002}

? CTOD("11/25/95") + 5

= {11/30/1995}

? {1/2/94} - 20

= {12/13/1993}
