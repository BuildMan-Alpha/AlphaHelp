Alpha Anywhere Feature Peek #27: Grid templates

 

Whenever you create a new grid component in Alpha Five Version 10, the Grid Templates Genie pops up, which lets you select one of several pre-defined templates or a template from your own library of templates.

Templates can be a great time saver because you can pre-set a multitude of properties whenever you create a new component. The video demonstrates how you can use existing templates, and how you can define your own templates.
