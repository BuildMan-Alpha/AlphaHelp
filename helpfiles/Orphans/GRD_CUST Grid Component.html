GRD_CUST Grid Component

Purpose

Application administrators use the GRD_CUST component to view and edit customer records on the ASWCUSTDET.A5W page, which displays customer information and customer sales histories.

The GRD_CUST component in the Browser

Description

The GRD_CUST component is an updateable grid that displays all the fields of a single record from the customer table in three tab panes. Users may edit records, but may not delete or add records. It may also show a link to open the ASWCUSTSEC.A5W page.

Containers

AWSCUSTDET.A5W, ASWCUSTONE.A5W, TGRDL_CUST
