GRD_CUSTN Grid Component

Purpose

Application administrators use the GRD_CUSTN component to enter new customer records on the ASWCUSTOMERS.A5W page, which displays customer information and customer sales histories.

The GRD_CUSTN Component in the Grid Builder

Description

The GRD_CUSTN component is an updateable grid that displays all the fields of a single record from the customer table in three tab panes. Users may only add or edit records.

Containers

ASWCUSTOMERS.A5W, TGRDL_CUST
