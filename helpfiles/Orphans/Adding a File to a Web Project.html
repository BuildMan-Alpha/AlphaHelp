Adding a File to a Web Project

In common practice, the files that you add to projects will be images and previously created HTML pages.

  1. Select the project in the Web Projects list to which you wish to add the file.

  2. Click   Add File.

  3. Optionally, select a file type from the Files of type drop-down list. The options are:

    • "A5 Web Components (*.a5wcmp)"

    • "A5W Pages (*.a5w)"

    • "HTML Files (*.html;*.htm)"

    • "CSS Style Sheets (*.css)"

    • "JPEG Images (.jpg)"

    • "Bitmaps (*.bmp)"

    • "PNG Images (*.png)"

    • "GIF Images (*.gif)"

    • "All Files (*.*)"

  4. Navigate to the file you want to add, select it, and click Open.

Limitations

Web publishing applications only.
