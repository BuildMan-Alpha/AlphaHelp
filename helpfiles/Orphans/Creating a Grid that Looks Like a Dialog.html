Creating a Grid that Looks Like a Dialog

Many users would like to create a dialog component that is bound to a table, and thereby provide record level record addition, deletion, and editing. If you are familiar with Xbasic, the task is fairly simple. However, if you would like a no-programming solution, the answer is create a grid component that looks like a dialog component.

  1. Open the Web Projects Control Panel.

  2. Click New > Alpha Five Web Component > OK > Grid > OK.

  3. On the Component Type page of the menus, select "Columnar" in the Grid layout list.

  4. Display the Grid > Data Source page and select the appropriate data source for your application (in this example we selected Alpha Five .dbf Tables ).

  5. Display the Grid > Query page and select the table or view that is providing the data (in this example we selected "Customer" from the Table Name list).

  6. Display the Grid > Fields page.

  7. Select and format the fields you wish to display.

  8. Display the Grid > Properties page.

  9. Change the value in the Layout Options > Rows of data property to 2.

  10. Display the Grid > Update Settings page.

  11. Change the value in the Permissions > Number of new record rows property to 1.

  12. Select the Permissions > Show new records on last page only check box.

  13. Display the Grid > Properties page.

  14. Change the value in the Layout Options > Rows of data property to 1.
