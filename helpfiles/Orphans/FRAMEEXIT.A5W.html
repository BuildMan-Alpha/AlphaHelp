FRAMEEXIT.A5W

Purpose

The AlphaSportsWeb application uses the FRAMEEXIT.A5W page to exit from an inner frame in a frame set. In AlphaSportsWeb frame sets display a header in the upper frame, while a processing page in the lower frame is doing some sort of computation.

Description

The processing page calls FRAMEEXIT.A5W, which then displays a page in the "_top" layer. This replaces the frame set. The call to FRAMEEXIT.A5W provides the name of the desired page as an argument named pg. For example: frameexit.a5w?pg=index.a5w.

Note : Do not open this page in the WYSIWYG tab of the HTML Editor and then save it. This will result in the loss of Xbasic code, which the underlying Microsoft ActiveX control does not properly support.

Links

The ASWMAINTOPT.A5W and CHECKOUTCALC.A5W pages call the FRAMEEXIT.A5W page.

Page Source

If FRAMEEXIT.A5W is called with a URL that is missing the pg parameter or the pg parameter is NULL, the default target page is INDEX.A5W.

if eval_valid("pg") = .F.

    pg = "index.a5w"

elseif pg = ""

    pg = "index.a5w"

end if

The page contains a form that uses pg as the value for its action. The JavaScript causes the page to submit to pg in the action="pg".

<form action=" <%a5 ? pg %> " method="post" name="nextpage" target="_top" id="nextpage">

</form>

<script language=javascript>

    document.nextpage.submit();

</script>
