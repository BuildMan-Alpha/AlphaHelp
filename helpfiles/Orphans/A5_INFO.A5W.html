A5_INFO.A5W

Purpose

Administrative users use the A5_INFO.A5W page to display information about the Application Server and current sessions.

Links

The Page Examples > A5_Info entry in the NAV_INT navigation component calls the A5_INFO.A5W page.

Description

The A5_INFO.A5W page executes the A5W_INFO() function, which creates and displays a page of Application Server and session information.
