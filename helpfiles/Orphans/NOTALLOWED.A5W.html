NOTALLOWED.A5W

Purpose

The NOTALLOWED.A5W page displays a message to indicate the user does not have permission to see a requested page.

Description

The NOTALLOWED.A5W page was originally created in another HTML editing program. The page uses Xbasic to populate a single variable used in the Return to Index link on the page. The link opens the ASWINDEX.A5W page for an internal user and INDEX.A5W for a customer.

NOTALLOWED.A5W is opened by the security system if a logged in user tries to open a page and they do not have permssion to see that page.

NOTALLOWED.A5W in the WYSIWYG tab of the HTML Editor

NOTALLOWED.A5W in the Browser

Page Source

The a5w_include("loggedinvalues.a5w") statement runs the code on the LOGGEDINVALUES.A5W page. This page creates and sets the values of session.protectedpUser.userid, session.protectedpUser.type, and session.protectedpUser.name. The users type value will determine which index page should be opened by the link.

a5w_include("loggedinvalues.a5w")

 returns values for pUser, session.protectedpUser

dim link as c

if session.protectedpUser.type = "I"  internal, show link to internal menue

    link = "aswindex.a5w"

else

    link = "index.a5w"

end if

The page source HTML includes styles for the page and uses an embedded <%a5 ... %> string to add the page to show in the link.
