+ (Concatenation Operator)

Syntax

Result as C = Operand1 as C + Operand2 as C

Result as C = Operand1 as D + Operand2 as C

Result as C = Operand1 as N + Operand2 as C

Argument

Description

Operand1

A character, date, or numeric value.

Operand2

A character value or NULL ("").

Description

The concatenation operator appends Operand2 (a character value) to Operand1. If Operand1 is not a character value, it is automatically converted to a character value before the concatenation operation.

Examples

dim dd as D

dim nn as N

dim cc as C

dim s as C

dd = {1/31/03}

s = dd + ""

? s

= "01/31/2003"

nn = 12

s = nn + ""

? s

= "12"

cc = "word1"

s = cc + "word2"

? s

= "word1word2"
