Home Page

Overview

The Home Page action assists you in building a startup form that can be used as a main menu for your application..

Procedure

  1. Click either Edit Choices using Home Page Genie or Edit Choices using Menu Builder .

  2. When you return to the action script genie, the screen will show your selections in a tree control, something like this:

  1. Click Next >.

  2. Optionally, click Header Text, Footer Text, Left Panel Text, or Right Panel Text to add static text to the top, bottom, left, or right sides of the dialog box. All of these buttons take you to the Static Text Genie .

  3. Click Next >.

  4. Optionally, modify the comment that describes the action.

  5. Click Finish.
