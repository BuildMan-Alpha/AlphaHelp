Execute UPDATE Query on a SQL Database

The Execute UPDATE Query on a SQL Database action allows you to modify one or more records in a remote SQL database. Alpha Five will use AlphaDAO (Alpha Five Data Access Objects) to connect to the remote database.

Note : It is not necessary to use the Connect to SQL Database action before using this action.

Procedure

  1. Enter the required connection string into the Options > Connection string property. If you do not have a ready-made connection string, you may click  to display the Connection String dialog.

    1. Optionally, select a named connection string from the drop-down list.

    2. Optionally, select Specify a connection string and click Build to display the  <span class=Screen>Create SQL Connection String</span> dialog.

    3. Click OK to return to the Script Genie.

  2. In the Options > Table to update property click  to display the Select Table dialog.

    1. Select a table or view from the list of entries.

    2. Select an entry and click OK

  3. In the Options > Columns to update property click  to display the Select Columns dialog.

  4. Select the fields you wish to update and click OK.

  5. In the Options > Set column values property click in the Options > Arguments property to display the <span class=Screen>Column Mapping</span> dialog.

  6. In the Options > WHERE clause property click  to display the <span class=Screen>WHERE Clause</span> dialog.

  7. Optionally, if you want to confirm record updates, select the Options > Prompt for confirmation property.

  8. Optionally, click in the Options > Arguments property to display the <span class=Screen>Define/Edit Arguments</span> dialog. Arguments provide values to use in filter and order expressions. Argument values may come from variables or from user selections at run time.

  9. Click Next >.

  10. Optionally, enter a comment that describes the action.

  11. Click Finish.

If you ran this action to generate sample Xbasic code, right click on the action and select View Xbasic.

Limitations

Desktop applications only.
