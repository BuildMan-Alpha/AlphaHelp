A5 V10 Feature Peek #5: Easy searching

 

Alpha Five Version 10 comes with a slew of new search features that you can add to grids, includingsearch part, quick search, searching on current sort column, query by example, incremental searching(QBE syntax), and alphabet button bars.
 

This video uses the customer grid we created in yesterday's Ajax grid Feature Peek, and shows you how easy it is to search.
