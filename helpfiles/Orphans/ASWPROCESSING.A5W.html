ASWPROCESSING.A5W

Purpose

The ASWPROCESSING.A5W page displays opens a frame set. The upper frame to show the system is "processing". The lower frame contains a page that is performing a maintenance operation.

Description

The upper frame ( topFrame ) displays the PROCESSHEADER.A5W page, which indicates that the system is "processing". ASWMAINTOPT.A5W opens in the lower frame ( mainFrame ).

Note : Do not open this page in the WYSIWYG tab of the HTML Editor and then save it. This will result in the loss of Xbasic code, which the underlying Microsoft ActiveX control does not properly support.

Links

The ASWMAINTOPT.A5W page calls the ASWPROCESSING.A5W page.
