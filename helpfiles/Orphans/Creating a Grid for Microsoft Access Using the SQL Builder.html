Creating a Grid for Microsoft Access Using the SQL Builder

  1. Create a new grid component.

  2. Display the Component Type page.

  3. Leave the Grid Layout selection as "Stacked Columnar". This produces a multi-record view that looks like multiple forms arrayed in a grid, with field labels above fields.

  4. Select the Updateable radio button.

Selecting the Data Source

  1. Display the Grid > Data Source page.

  2. Select the ADO/ODBC/Access radio button. The ADO/ODBC/Access button allows you to retrieve your data from Microsoft Access or a database that supports the ADO or ODBC interface standards.

Selecting the Records to Display

  1. Display the Grid > Query (ADO) page.

  2. Leave the Connection type selection as "Access Database".

  3. Click to select the "Northwind" database in the MDB Filename field.

  4. Click Connect. This opens the Access database so you can select the table or view that you want to use.

  5. Select the SQL Select Statement radio button.

  6. Click the Edit SQL button to display the <span class=Screen>SQL Builder</span> dialog box.

  7. Click Tables in Query to display the Define Join dialog box.

  8. Click Add Table to display the Add Table dialog box.

  9. Select the "Customers (Table)" entry in the Tables list and click OK.

  10. Click Add Table to display the Add Table dialog box.

  11. Select the "Orders (Table)" entry in the Tables list and click OK.

  12. Click Fields in View to display the Select Fields in View dialog box.

Note : Since this grid is updateable, you must include both the fields that you wish to display plus the primary keys of the tables that contain these fields, even if you do not wish to display them.

    1. Select the "Customers" table from the drop-down list.

    2. Select the "CustomerID", "CompanyName", "City", "Country" fields and click .

    3. Select the "Orders" table from the drop-down list.

    4. Select the "OrderDate" and "OrderID" fields and click .

    5. Click OK > OK to continue.

  1. Leave the Read only check box cleared.

  2. Click Select Primary Key(s) to display the Specify Primary Keys dialog box.

    1. Select the "Customers" table from the Tables list.

    2. Check the "CustomerID" field.

    3. Select the "Orders" table from the Tables list.

    4. Check the "OrderID" field.

    5. Click OK to continue.

Selecting the Fields to Display

  1. Display the Grid > Fields page.

  2. Leave the Grid is editable check box unchanged. Although a grid starts out as updateable, it can still be read only. This has the virtue of using text boxes instead of labels to represent the data in the grid, which in turn provides more reliable formatting alternatives, as well as the ability to copy data from the fields.

  3. In the Available Fields list select "Customerid" through "Orderdate" and click .

Formatting the Grid

  1. Display the Grid > Properties page.

  2. Click to display the Select style dialog.

  3. Select "TeaCup" and click OK. You can add new styles or modify existing styles.

  4. Change the Layout Options > Repeating columns field value to "4".

  5. Change the Layout Options > Snaking style field value to "LRTB".

Note : With the Stacked Columnar grid style, the Layout Options > Rows of data field actually specifies the number of forms to display. Since the Layout Options > Repeating columns field value is "4" and the Permissions > Number of new record rows on the Grid > Update Settings page will be set to "1", this field value will be set to "3" ( 4 - 1 = 3).

  1. Change the Layout Options > Rows of data field value to "3".

  2. Click the Layout Options menu to collapse it.

  3. Click the Shading and Dividers menu to collapse it.

  4. Click the Column Titles and Sorting Options menu to collapse it.

  5. Click the button in the Record Navigator > Record navigator layout field to display the Record Navigator Designer dialog box. You can modify the default navigator styles or create new navigator styles.

  6. Select the Pre-Defined Format and click OK.

  7. Change the Record Navigator > Record navigator position field to "Above and Below Grid".

  8. Click the Record Navigator menu to collapse it.

  9. Click the button in the Customization > 'Total records' message text field to display the HTML Editor.

    1. Display the Source tab.

    2. Erase all the HTML code and click Save.

  10. Click the Customization menu to collapse it.

  11. Click the button in the Freeform Edit Regions > Above Grid field to display the HTML Editor.

    1. Enter and format "Northwind Customer Orders".

    2. Click Save.

  12. Click Save.

Defining Update Settings

  1. Display the Grid > Update Settings page.

  2. Clear the Permissions > Allow delete check box.

  3. Change the Permissions > Number of new record rows to "1".

Previewing the Grid

  1. The first stage of previewing your work is to create a page onto which to place the new component.

    1. Click to preserve your work. Name it "Access_SQL_Grid".

    2. Click Close to return to the Web Projects Control Panel.

    3. Click on the toolbar.

    4. Select Alpha Five Web Page > OK to open the HTML Editor.

    5. Click File > Save As. Name the page "Access_SQL_Grid" and click OK.

  2. The next stage is to place the component on the page.

    1. Click Insert Component to display the Insert Component dialog.

    2. Click Select to display the Select Component dialog.

    3. Select "Access_SQL_Grid" and click OK.

    4. Change the component Alias to "ASG" and click OK. This makes the code that Alpha Five generates in the background more compact, but the main benefit is it allows you to transparently replace the component with another component with the same alias.

    5. Click and close the HTML Editor.

  3. Next, publish the files to the local webroot. The local webroot is presumably the location where you will test your website. In most cases you will create a second profile (in this case "AlphaSports") where you will publish your finished public website.

    1. Display the A5W Pages page of the Web Projects Control Panel.

    2. Right click "Access_SQL_Grid" and select Publish (local webroot) and open. You should see something like the following.
