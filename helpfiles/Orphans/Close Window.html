Close Window

Overview

The Close Window action closes a window. You can must have the window open to select it.

Procedure

  1. In the Specify the Window to close list select either:

    • "Specific Window"

    • "Current Window"

    • "Prompt at Runtime"

  2. If you selected "Specific Window", select the window from the Window name list.

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
