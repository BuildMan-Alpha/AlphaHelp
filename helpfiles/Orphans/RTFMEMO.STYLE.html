.STYLE

Style Properties

Element

Type

Description

StyleN.First_Left_Indent

N

Indent of first line of paragraph.

StyleN.Left_Indent

N

Left indent of paragraph.

StyleN.Right_Indent

N

Right indent of paragraph.

StyleN.Name

C

Name of style.

StyleN.Tab.Align

C

Alignment of tab.

StyleN.Tab.Pos

N

Position of tab.

Description

The RTF Memo Editor supports up to 8 different styles.

Supported By

Alpha Five Version 5 and Above

Limitations

Desktop applications only.

Examples

dim obj as P

obj = :Customer_Information:rtfmemofield.this

obj.rtf.insert_text("This is RTF text")

obj.rtf.style1.name = "My Style"

obj.rtf.style1.left_indent = 1

obj.rtf.style1.right_indent = 1

? obj.rtf.style.1

= First_Left_Indent = 0

Left_Indent = 1

Name = "My Style"

Right_Indent = 1

+Tab.
