GRD_HISTQUAN Grid Component

Purpose

Application administrators use the GRD_HISTQUAN grid component to review product sales.

Description

The GRD_HISTQUAN component is a read-only tabular grid based on a view that combines the sales_summary_quan, product, and product_category tables. The GRD_HISTQUAN component is a child grid in the TGRDL_HIST tabbed gridlinker, which is located on the ASWSALESHIST.A5W page.

The entries in the Description column are links to the ASWPRODONE.A5W table.

The GRD_HISTCUST component has one calculated field. The TGRDL_HIST tabbed gridlinker uses this field to link the GRD_CUSTLIST grid to GRD_HISTQUAN.

link = "1"

Containers

ASWSALESHIST.A5W, TGRDL_HIST
