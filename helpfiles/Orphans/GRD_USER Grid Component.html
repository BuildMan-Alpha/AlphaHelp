GRD_USER Grid Component

Purpose

The GRD_USER grid component allows a customer or administrator to modify his or her user name and password.

The GRD_USER grid component in the Grid Builder Picture

The GRD_USER grid component in the Browser

Description

The GRD_USER component is an editable tabular grid based on the user_names table.
