Displaying a Selected Record in a Dialog

A common application requirement is to select a record from a list and then to display the details of that record by itself in a form. This example shows how to build two grid components and two A5W pages that do this.

Creating a Grid with a Hyperlink

The following grid has a hyperlink next to each record. Clicking on the hyperlink selects that record for display in the second grid.

  1. Create a new grid component based on the AlphaSports "customer" table.

  2. Sort the table's records on the "Lastname" and "Firstname" fields.

  3. Display the Grid Fields page.

  4. Move the "Firstname", "Lastname", "Bill_City", "Bill_State_Region", "Bill_Country", and "" fields from the Available Fields list to the Selected Fields list.

  5. Edit the Column Properties > Column heading fields for the "Bill_City", "Bill_State_Region", and "Bill_Country" fields.

  6. Select "" in the Selected Fields list:

    1. Move it to the top of the Selected Fields list.

    2. Change its Display Settings > Control type to "Link".

    3. Erase the column heading in the Column Properties > Column heading field.

    4. Change its Link Properties > Display what in link field to "Static image".

    5. In the Link Properties > Static image field click to display the Select Image dialog.

    6. Select "RowMarkerOn.png" in the Images list and click OK.

    7. In the Link Properties > Link address type field select "Computed from fields in the grid".

    8. In the Link Properties > Link address field click to display the <span class=Screen>Link Address Builder</span>. Enter the following code into the Link Address field and click OK.

Selected_Customer_Record.a5w?Selected_Custom_Filter=firstname = "{Firstname}" .and. lastname = "{Lastname}"

Note : Remove the space characters from the finished URL.

This URL displays the Selected_Customer_Record.a5w page and sets its record filter ( Selected_Custom_Filter ) equal to firstname="{Firstname}".and.lastname="{Lastname}". There are several important points to understand about this expression.

  • The argument to the right of "?" starts with "Selected_Custom_Filter", which is the alias of the component. The filter expression must reference the component's alias, which is automatically trimmed to 15 characters.

  • To the right of "Selected_Custom_Filter=" are the arguments in the filter expression. Because you want to replace the field value with the value in the selected record, you must surround field names with quotes and curly braces "{}".

  • Since there are two arguments in the filter expression, they are combined with an Xbasic ".and.".

  • There are no spaces in the expression.

The Application Server substitutes the values of the current record's Firstname and Lastname fields for the {Firstname} and {Lastname} placeholders. See A5W Page Filter and Order Syntax for more information.

  1. On the Grid Properties page change L ayout Options > Rows of data to 6.

  2. In the Freeform Edit Regions > Below grid field click to display the HTML Editor. Enter "Click the triangle to see the selected record". Click Save.

  3. Click File > Save as.., name the grid component "Pick_a_Customer" and click Save. The grid should look something like this (if you are using the Sunset style).

  1. Click Close to return to the Web Projects Control Panel.

Creating the Pick_a_Customer Page

  1. Create a new A5W page.

  2. Click File > Save As, name the page "Pick_a_Customer" and click Save.

  3. Enter "Pick a Customer" on the page. Select the text and change it to "Verdana" and size "5 (18pt)".

  4. Click Insert Component to display the Insert Component dialog. Select "Pick_a_Customer" from the Component list and click OK.

  5. Center all the items on the page, save your results, and close the HTML Editor.

Creating the Selected_Customer_Record Grid Component

  1. Create a new grid component based on the AlphaSports "customer" table.

  2. On the Component Type page select "Columnar" in the Grid layout list.

  3. Select Updateable.

  4. Display the Grid Fields page.

  5. Move the "Customer_ID", "Firstname", "Lastname", "Company", "Phone", "Fax", and "Email" fields from the Available Fields list to the Selected Fields list.

  6. Display the Grid Properties page.

  7. Change the Layout Options > Rows of data field to 1.

  8. Display the Grid Update Settings page.

  9. Select "False" in the Permissions > Allow insert list (de-select the check box).

  10. Save your grid as  "Selected_Customer_Record" and preview it. Your grid should look something like this.

Creating the Selected_Customer_Record Page

  1. Create a new A5W page.

  2. Click File > Save As, name the page "Selected_Customer_Record" and click Save.

  3. Enter "Selected Customer Record" on the page. Select the text and change it to "Verdana" and size "5 (18pt)".

  4. Click Insert Component to display the Insert Component dialog. Select "Selected_Customer_Record" from the Component list.

Note : The alias of the component is truncated to 15 characters, which produces "Selected_Custom". This is the name of the component that must be used in Filter and Order expressions.

  1. Click OK.

  2. Center all the items on the page, save your results, and close the HTML Editor.

  3. Select the "Selected_Customer_Record.A5W" page, right click, and select Publish (Local Webroot) and open.

  4. Select the "Pick_a_Customer.A5W" page, right click, and select Publish (Local Webroot) and open.

  5. Click the blue triangle to the left of any record and it will be displayed in the Selected_Customer_Record page.
