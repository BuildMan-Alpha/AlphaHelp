NAV_MAIN Navigation Component

Purpose

The NAV_MAIN navigation component provides the mechanism for moving between the public pages of the application. It also provides access to login page and the administrative part of the application.

The NAV_MAIN navigation component

Description

The NAV_MAIN navigation component is a has a horizontal style and uses mouse over to open submenus. The "Contact Us" entry is coded with HTML and creates an email link to Alpha Software.
