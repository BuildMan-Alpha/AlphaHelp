Display 'Print Current Record' Dialog

Overview

The Display 'Print Current Record' Dialog action displays the Print Current Record dialog box, which allows the user to print, preview, email, or save the current record using any layout they choose. Appropriate options are also presented to the user for the different choices. These options can be saved to avoid having to re-select them each time.

Note : This action must be attached to a form to work.

Procedure

  1. Optionally modify the comment that describes the action.

  2. Click Finish.
