GRD_CUSTLIST Grid Component

Purpose

Application administrators use the GRD_CUSTLIST component to select customers on the ASWCUSTOMERS.A5W page, which displays customer information and customer sales histories.

The GRD_CUSTLIST component in the Grid Builder

The GRD_CUSTLIST component on the ASWCUSTOMERS.A5W page

Description

The GRD_CUSTLIST component is the parent grid in the TGRDL_CUST tabbed gridlinker. The child grid components ( GRD_CUST, GRD_CUSTN, and GRD_INVHC ) display customer information and sales histories. The component contains a Quick Search that uses the "wildcard" style of searching. This means that a search looks for an exact match, ignoring case and length. The wildcard characters "*" and "?" are supported.

Containers

ASWCUSTOMERS.A5W, TGRDL_CUST
