<> (Not Equals Operator)

Syntax

Result as L = Operand1 as A <> Operand2 as A

Argument

Description

Operand1

A value or expression of any type.

Operand2

A value or expression of any type.

Description

The Not Equals operator returns .T. (TRUE) if both Operand1 and Operand2 are of different types or have different values. The comparison between character strings is less demanding. The Equals operator removes trailing blanks and ignores case when making a comparison.

Examples

2 <> 3 -> .T.

ZIP <> "02139" -> .T., if ZIP equals "02138"
