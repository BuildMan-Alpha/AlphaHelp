Add Text to Memo Field

Overview

The Add Text to Memo Field action adds text at any location in a memo field. You can also insert new lines before or after the inserted text.

Procedure

  1. Make a selection in the Insert text into a memo in which window? field. This field will be grayed out unless this action is based on a Form or Browse. The choices are:

    • "Current Form or Browse Window".

    • "Another Form or Browse Window".

  2. In the Window Name field select either:

    • "Forms"

    • "Browses"

  3. Select the form or browse that you wish to use from the list.

  4. In the Object Name field select a memo field on the form or browse.

  5. In the Text to Insert field enter either a text string or click to use the Expression Builder to create an expression that evaluates to a text string.

  6. Click the Start new line before text check box to insert a blank line before the new text.

  7. Click the Insert new line after text check box to insert a blank line after the new text.

  8. Specify the insertion point for the new text.

  9. Click Next.

  10. Optionally modify the comment that describes the action.

  11. Click Finish.
