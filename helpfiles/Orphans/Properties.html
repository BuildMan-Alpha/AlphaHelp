Properties

Every object in Alpha Five has a collection of properties that describe the object. For example, a field object on a form has an "align" property that specifies how the text in the field is aligned.

Xbasic allows you to read the value of an object property, or to set the value of a property. By setting the value of an object property, you change the behavior or appearance of that object.

For example, assume you have a form called "Customer" with a field object called "City". To read the current value of this object's "align" property, you would use this Xbasic statement:

value = :Customer:city.align

To set the alignment of this object to "Center", you would use this Xbasic statement:

:Customer:city.align = "center"

In general, the syntax for reading the value of an object property is:

value = object address.property

The syntax for setting an object property is:

object address.property = value

Some object properties are "multi-level". For example, the "font" property has the following sub-properties: "bold, "color, "italic", "name", "size", "strikeout", and "underline".

For example, to set the font of the "City" field in the above example to "Arial, 12 point, bold, italic", you would use the following Xbasic statements:

:Customer:city.font.NAME = "Arial"

:Customer:city.font.SIZE = 12

:Customer:city.font.bold = .T.

:Customer:city.font.italic = .T.
