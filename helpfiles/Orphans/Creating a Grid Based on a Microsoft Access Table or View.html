Creating a Grid Based on a Microsoft Access Table or View

  1. Create a new grid component.

  2. Display the Component Type page.

  3. Leave the Grid Layout selection as "Tabular". This produces a multi-record view, similar to a conventional Alpha Five browse, with field labels at the tops or bottoms of columns.

  4. Leave the Read only radio button selected.

  5. Check the Grid contains a 'Search' part check box. This is an optional form that allows you to query your table and return matching records.

  6. Check the Grid contains a 'Detail View' part check box. The Detail View is the form used to add, delete, and edit records.

  7. Select the DetailView is Updateable radio button.

Selecting the Data Source

  1. Display the Grid > Data Source page.

  2. Select the ADO/ODBC/Access radio button. The ADO/ODBC/Access button allows you to retrieve your data from Microsoft Access or a database that supports the ADO or ODBC interface standards.

Selecting the Records to Display

  1. Display the Grid > Query (ADO) page.

  2. Leave the Connection type selection as "Access Database".

  3. In the MDB Filename field click to navigate to and to select the "AlphaSports" database (located in the MDBFiles folder under your program folder).

  4. Click Connect. This opens the Access database so you can select the table or view that you want to use.

  5. Leave Base on Table or View radio button selected.

  6. Click to select the "Customer (Table)" from the Table/View name list.

  7. In the Filter field click to display the Specify Filter dialog.

    1. Select "Company" in the Available Fields list and click .

    2. Complete the expression Lastname >= 'A'. Note that Access requires that you single quote character expressions.

    3. Click OK.

  8. Leave the Order by field unchanged.

  9. Leave the Read only check box cleared.

Selecting the Fields to Display

  1. Display the Grid > Fields page.

  2. Leave the Grid is editable check box unchanged. Although a grid starts out as  updateable, it can still be read only. This has the virtue of using text boxes instead of labels to represent the data in the grid, which in turn provides more reliable formatting alternatives, as well as the ability to copy data from the fields.

  3. In the Available Fields list select "Customer_id" through "Phone" and click .

Modifying Field Properties

  1. Select the "Customerid" field in the Selected Fields list.

    1. Change the Column Properties > Column heading field to "ID".

    2. Change the Column Properties > Heading in-line style field to "text-align: left;".

  2. Select the "Firstname" field in the Selected Fields list.

    1. Change the Label Properties > In-line style field to white-space: nowrap;.

    2. Change the Column Properties > Column heading field to "First Name".

    3. Change the Column Properties > Heading in-line style field to "text-align: left;".

  3. Select the "Lastname" field in the Selected Fields list.

    1. Change the Label Properties > In-line style field to white-space: nowrap;.

    2. Change the Column Properties > Column heading field to "Last Name".

    3. Change the Column Properties > Heading in-line style field to "text-align: left;".

  4. Select the "Company" field in the Selected Fields list.

    1. Change the Label Properties > In-line style field to white-space: nowrap;.

    2. Change the Column Properties > Heading in-line style field to "text-align: left;".

Formatting the Grid

  1. Display the Grid > Properties page.

  2. Click to display the Select style dialog.

  3. Select "Arizona" and click OK. You can add new styles or modify existing styles.

  4. Change the Layout Options > Font Size to "Smaller".

  5. Click the button in the Record Navigator > Record navigator layout field to display the Record Navigator Designer dialog box. You can modify the default navigator styles or create new navigator styles.

  6. Select the Pre-Defined Format and click OK.
