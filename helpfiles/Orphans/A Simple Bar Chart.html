A Simple Bar Chart

This example shows how to use the value of a field in a freeform layout to modify the width of a graphic, thereby producing a simple bar chart.

The "AlphaSports" database is located in the Samples folder under the Alpha Anywhere installation folder.

Creating the Grid

  1. Create a new tabular grid based on the AlphaSports product table.

  2. Display the Component Type page.

  3. Leave the Read only radio button selected.

  4. Display the Grid Data Source page.

  5. Select Alpha Anywhere .dbf Tables.

  6. Display the Grid Query (DBF) page.

  7. Click to select the "product" table.

  8. Click Calculated Fields to display the Define Calculated Fields dialog.

  9. Enter the following expression and click OK. This creates a new calculated field named "Markup".

markup = (Retail - Cost)/Cost * 100

Selecting the Fields to Display

  1. Display the Grid Fields page.

  2. Move "Description", "Cost", "Retail", and "Markup" from the Available Fields list to the Selected Fields list.

  3. For the "Cost" and "Retail" Display Settings > Display format fields click to select an appropriate currency format.

  4. Select the "Markup" field and select its Column Properties > Freeform layout check box.

  5. Erase any HTML code its its Display Settings > Display format field.

  6. Click in the Column Properties > Freeform template field to display the Freeform Column Layout dialog. A freeform field may contain any legitimate HTML code. In this case it will contain an <IMG> tag. Enter the following HTML code into the dialog box and click OK. When the page is published, the value of the "Markup" field will replace the {Markup} place holder.

<img src="greenstripe.gif" width="{Markup}" height="10" alt="{Markup}%"> {Markup}%

Note : This example presumes that you have a green GIF image about 10 pixels square. However, the specific image that you use (for demonstration purposes) is irrelevant. The only constraint is that the image should about about 10 pixels tall.

Setting the Grid Properties

  1. Click Edit Style to select the "Corporate" style sheet.

  2. Clear the Shading and Dividers > Alternate row shading check box.

  3. Set the Shading and Dividers > Row separator check box.

  4. Save the component as "ProductMarkup" and exit from the Grid Component Builder.

Displaying the Page

  1. Click New > OK to create a new A5W page.

  2. Click File > Save As, enter "ProductMarkupBarChart" in the File name field, and click Save.

  3. Select Insert Component > Select, pick "ProductMarkup", and click OK > OK.

  4. Save the page and close the HTML Editor.

  5. Right click the "ProductMarkupBarChart" page, select Publish (Local Webroot) and open. The result should look like this.
