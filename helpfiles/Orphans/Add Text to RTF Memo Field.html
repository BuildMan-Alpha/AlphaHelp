Add Text to RTF Memo Field

Overview

The Add Text to RTF Memo Field action adds text at any location in an RTF memo field. You may specify the font, style, size, and color. You can also insert new lines before or after the inserted text.

Procedure

  1. Make a selection in the Insert text into an RTF memo in which window? field. This field will be grayed out unless this action is based on a Form or Browse. The choices are:

    • "Current Form or Browse Window".

    • "Another Form or Browse Window".

  2. In the Window Name field select either:

    • "Forms"

    • "Browses"

  3. Select the form or browse that you wish to use from the list.

  4. In the Object Name field select an RTF field on the form or browse.

  5. In the Text to Insert field enter either a text string or use the Expression Builder to create an expression that evaluates to a text string.

  6. Click the Start new line before text check box to insert a blank line before the new text.

  7. Click the Insert new line after text check box to insert a blank line after the new text.

  8. Specify the insertion point for the new text.

  9. Optionally specify a new font or point size for the display. Either enter the font name and point size separated by a comma, or click the black triangle on the Font list to display the Select Font dialog box.

  10. Optionally check the Bold, Italics, and/or Underline check boxes to set these attributes.

  11. Optionally specify a new font color. Click the black triangle on the Font color list to select a color.

  12. Click Next >.

  13. Optionally modify the comment that describes the action.

  14. Click Finish.
