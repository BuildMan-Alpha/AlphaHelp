Editing a Component or Page

If you want to modify a previously created component or web page:

  1. Optionally, filter the list of files by selecting:

    • "A5W Pages"

    • "HTML Pages"

    • "CSS Style Sheets"

    • "Images"

    • "Web Components"

  2. Select a file from the Name list.

  3. Double click the filename or click Edit.

  4. Depending on what you selected, one of the following dialogs will appear:

    • Grid Builder

    • Dialog Builder

    • GridLinker Builder

    • Tabbed GridLinker Builder

    • Navigation System Builder

    • Login Builder

    • HTML Editor

    • CSS Editor

The applications that open in other cases will be those defined in Windows as the default editor for the particular file type.

Limitations

Web publishing applications only.
