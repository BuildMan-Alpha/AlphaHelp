Hiding Controls That are Filled by a Lookup

All fields that you want to modify must be on a grid, even if you want to populate these fields with a lookup. But what if you wish to hide the fields that the user does not directly edit or input? The following is a solution to this application problem.

This example is based on the customer table of the AlphaSports sample database.

  1. Create a new read-only tabular grid based on the AlphaSports customer table.

  2. Place and format the lastname, firstname, phone, fax, and email fields.

  3. Save this grid with the name "cust_look_to". It should look something like this.

  1. Create an editable tabular grid based on the customer table of the AlphaSports sample database.

  2. Place and format the customer_ID, lastname, firstname, phone, fax, email, and company fields.

  3. Select the customer_ID field in the Selected fields list and change its Display Settings > Control type property to "Label".

  4. Select the firstname field in the Selected fields list:

    1. Change its Lookup > Lookup property to TRUE (checked).

    2. Click in its Lookup Properties > Lookup definition property to display the Define Lookup Window dialog.

    3. Select "cust_look_to" in the Lookup Window > Grid name list.

    4. Click in its Field Mapping > Field map property to display the Define Field Mapping dialog.

    5. Add entries that map the lastname, firstname, phone, fax, email, and company source fields to target fields.

    6. Click OK > OK to continue.

  5. Select the company field in the Selected fields list.

    1. Change its Display Settings > Control type property to "Label".

    2. Change its Column Properties > Freeform layout property to TRUE (checked).

    3. Click in its Column Properties > Freeform template property to display the Freeform Column Layout dialog.

    4. Define the layout as  {Company}{Phone}{Fax}{Email} and click OK.

  6. Select the phone field in the Selected fields list:

    1. Change its Display Settings > Control type property to "Hidden".

    2. Change its Column Properties > Hide column property to TRUE (checked).

  7. Select the fax field in the Selected fields list:

    1. Change its Display Settings > Control type property to "Hidden".

    2. Change its Column Properties > Hide column property to TRUE (checked).

  8. Select the email field in the Selected fields list:

    1. Change its Display Settings > Control type property to "Hidden".

    2. Change its Column Properties > Hide column property to TRUE (checked).

  9. Save this grid with the name "cust_look_from". It should look something like this.

  1. Place the "cust_look_from" grid on a page. Publish both grids and the page. When you look up a record, you will see all 5 fields in the "cust_look_to" grid, but only 2 of those fields on the "cust_look_from" grid. The other 3 will be hidden.
