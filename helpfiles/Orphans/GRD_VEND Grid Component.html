GRD_VEND Grid Component

Purpose

The application administrator uses the GRD_VEND grid component (on the ASWVENDONE.A5W and ASWVENDORS.A5W pages) to input, edit, and delete vendor records.

Description

The GRD_VEND component is an editable columnar grid with a search part. It is based on the vendor table.

Containers

ASWVENDONE.A5W, ASWVENDORS.A5W
