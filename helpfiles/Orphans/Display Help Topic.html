Display Help Topic

Overview

The Display Help Topic action opens a help file (Windows .HLP or .CHM formats or an Alpha 5 help file) to display a topic to the user.

Procedure

  1. In the Display help topic from which type of help file list choose either a standard Windows help file or an Alpha 5 help file. The options are:

    • "Standard Windows Help File (.hlp or .chm)"

    • "Alpha Five Help File"

  2. Enter the name of the help file into the Help Filename field. Optionally, click the Folder icon to search for the file.

  3. If you selected "Alpha Five Help File", optionally enter the chapter name into the Help Chapter field. Leave the field blank to display the open help file with no topic selected.

  4. If you selected "Standard Windows Help File (.hlp or .chm)", enter the topic name into the Help Topic field.

  5. If you selected "Alpha Five Help File", optionally enter the topic name into the Help Topic field. Leave the field blank to display the open help file with no topic selected.

  6. Click Next >.

  7. Optionally modify the comment that describes the action.

  8. Click Finish.
