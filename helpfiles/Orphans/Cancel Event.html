Cancel Event

Overview

The Cancel Event action cancels a form event if a condition is not met.

Procedure

  1. In the Cancel event if this condition is True: field, optionally enter an expression that evaluates to either .T. (TRUE) or .F. (FALSE). If necessary, click the button to use the Expression Builder. Leave the field blank to always cancel the event.

  2. Check the Suppress the Alpha Five error message box if hide any Alpha Five error message that may appear.

  3. Check the Display a message box if the event is cancelled? box if you would like to display a message when the event is cancelled.

  4. Enter the message you would like displayed in the box labelled Message Text.

  5. Click Next >.

  6. Optionally modify the comment that describes the action.

  7. Click Finish.
