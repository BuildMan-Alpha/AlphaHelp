Close Connection to SQL Database

The Close Connection to SQL Database action allows you to close a connection to a SQL database. This action must be preceded by the Connect to SQL Database action.

Procedure

  1. Enter the name of the variable, created by the Connect to SQL Database action, that references the open connection.

  2. Click Next >.

  3. Optionally, enter a comment that describes the action.

  4. Click Finish.

Limitations

Desktop applications only.
