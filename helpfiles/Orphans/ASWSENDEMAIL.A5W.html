ASWSENDEMAIL.A5W

Purpose

The application administrator would use the ASWSENDEMAIL.A5W page to send an email, presumably to someone in the customer table.

Description

The ASWSENDEMAIL.A5W page contains the DLG_IEMAIL dialog component. The Validate and AfterValidate events of the dialog contains all of the code necessary to send an email. The code tests for a valid email profile before sending the message. The dialog uses the Corporate style sheet, which is different from the main page.

Links

The Customers > Send Email entry of the NAV_INT navigation component displays the ASWSENDEMAIL.A5W page.

The Validate Event

The Validate event of the dialog component uses EMAIL_PROFILE_VALID()to determine the user has a valid email profile.

if email_profile_valid()= .F.

    CurrentForm.Has_Error = .T.

    CurrentForm.Error_Message = " Email Profile Not Defined"

end if

The AfterValidate Event

The AfterValidate event fires only if CurrentForm.Has_Error is set to FALSE. If the EMAIL_SEND() function returns 0 (zero), the script sets the CurrentForm.Has_Error property to TRUE. Otherwise, it erases the controls in preparation for sending another email.

if email_send(CurrentForm.Controls.sendto, CurrentForm.Controls.subject, CurrentForm.Controls.body,"",CurrentForm.Controls.cc, CurrentForm.Controls.bcc) = 0 'not sent

    CurrentForm.Has_Error = .T.

    CurrentForm.Error_Message = " Error: Email Not Sent"

else

    CurrentForm.Controls.sendto = ""

    CurrentForm.Controls.cc = ""

    CurrentForm.Controls.bcc = ""

    CurrentForm.Controls.subject = ""

    CurrentForm.Controls.body = ""

end if
