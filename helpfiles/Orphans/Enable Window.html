Enable Window

Overview

The Enable Window action makes a window able to receive focus. You can must have the window open to select it.

Description

  1. In the Specify the Window to enable list select either:

    • "Specific Window"

    • "Current Window"

    • "Prompt at Runtime"

  2. If you choose "Specific Window", select the window from the Window name list.

  3. Click Next >.

  4. Optionally modify the comment that describes the action.

  5. Click Finish.
