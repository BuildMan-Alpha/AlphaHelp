*TO_OCTAL Function

Syntax

Value as C = *To_Octal( Interger_Number as N )

Argument

Description

Interger_Number

An integer value to be converted to octal format.

Description

The *TO_OCTAL() function converts a number to an octal string.

Example

? *to_octal(123)

= "0000000173"
