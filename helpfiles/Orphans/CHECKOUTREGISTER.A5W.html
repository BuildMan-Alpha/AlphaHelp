CHECKOUTREGISTER.A5W

Purpose

The CHECKOUTREGISTER.A5W page is a fourth optional step in the checkout process. It allows a customer to enter security information to complete the registratin process.

Description

The CHECKOUTREGISTER.A5W page contains the NAV_MAIN navigation and DLG_REGCART dialog component. The dialog's AfterValidate code saves the dialog values to protected session variables. It also saves any edits made to an existing customer record.

A pg parameter is added to the page URL to show what page should be opened after this page is completed. For example: checkoutregister.a5w?pg=processing.htm. The variable pg is saved in a hidden field so it will be available when the page is submitted.

Links

The CHECKOUTCOMP.A5WCHECKOUTCART.A5W and CheckOutCompPayPal.A5W pages may display the CHECKOUTREGISTER.A5W page if the current customer is not found in the customer table. Errors can cause the CHECKOUTREGISTER.A5W page to redirect back to the CHECKOUTCART.A5W page.

Page Source

The first part of the page script runs before any HTML on the page. It checks the value of session.protectedchkout. If FALSE (.F.), the user is redirected to CHECKOUTCART.A5W. If the value of session.protectedchkout.firstname is NULL, the user has not finished the second checkout page, and is redirected to CHECKOUTCART.A5W. If the variable pg is not found or is NULL, the user is redirected to CHECKOUTCART.A5W

if eval_valid("session.protectedchkout") = .F. no values passed from page 3

    response.redirect("checkoutcart.a5w")

    end

elseif session.protectedchkout.firstname = "" no values passed from page 3 (first_name was required)

    response.redirect("checkoutcart.a5w")

    end

end if

if eval_valid("pg") = .F.  check if return page value passed

    response.redirect("checkoutcart.a5w")

    end

elseif pg = ""

    response.redirect("checkoutcart.a5w")

    end

end if

The variable session.protectedchkout.customer_id will have a value if the user is logged in and found in the customer table. If the user record already exists, they are redirected to the page saved in pg.

if session.protectedchkout.customer_id <> ""  User logged in, values passed from page 3, ok to complete sale

    response.redirect(pg)  redirect selected page

    end

end if
