Implementing Version 8 Security

To implement a security framework, you first need a web project with a suitable selection of pages. If you don't have a suitable project ready for experimentation, here is a solution.

  1. Select Web > Application Server to display the Application Server Settings dialog.

  2. On the General tab check Enable Security Framework.

  3. Click Save.

  4. Click Restart Server.

  5. Click Close to return to the Web Projects Control Panel.

  6. Create a new project named "Demo".

  7. Create four simple A5W pages. In normal applications, these pages would include a variety of components. However, since Version 8 controls access at the page level, not the component level, the goal here is to simply demonstrate how to control access to pages.

    8. Page Name

    Contents

    public.a5w

    Public

    Available to all users

    private.a5w

    Private

    Available to members of accounting and marketing

    accounting.a5w

    Accounting

    Available to members of accounting only

    marketing.a5w

    Marketing

    Available to members of marketing only

  8. Create a fifth menu page. This page will have hyperlinks to the previous four pages.

  9. The following tutorial steps will complete the security configuration.

Limitations
