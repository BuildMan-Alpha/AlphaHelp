Alpha Five Version 10 Feature Peek #28: Foreign character support
In July, I told you about one of the new features of Alpha Five Version 10, its foreign language support for labels and instructions in web apps built by Version 10. Well, we've one upped ourselves by adding international character support for the DATA (UTF8 is the technical term.)
Now, Web apps can be designed to support virtually any language, using that language's characters.
Here's an example of an Alpha Five Version 10 app using Chinese characters.
What does it mean?
It means that if you are building Codeless AJAX (TM) Web applications against SQL databases in Alpha Five v10 - you will be able to serve people worldwide with the data in their native tongue
