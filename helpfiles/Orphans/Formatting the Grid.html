Formatting the Grid

Changing the Stylesheet

  1. Display the Grid > Properties page.

  2. Selecting Layout Options > Style name > > Select style displays the Select style dialog box. You can preview and select the Cascading Style Sheet that you would like to apply to the grid.

  3. Select "WinXP" and click OK.

Selecting a Different Record Navigator

The <span class=Screen>Record Navigator Designer</span> displays a large number of pre-defined navigator styles for you to choose from. This dialog also allows you to modify the default navigator styles and create new navigator styles.

  1. Click the button in the Record Navigator > Record navigator layout field to display the Record Navigator Designer.

  2. Select the Pre-Defined Format and click OK.

Adding Freeform Text

Freeform text is ordinary HTML content that is placed above, below, to the left, or to the right of the grid. The advantage of freeform text is that it is always with the grid, no matter which page you place it on.

  1. Click the button in the Freeform Edit Regions > Above grid field to display the HTML Editor.

  2. Enter "Customer Contact Information".

  3. Select the text and change the size to "4 (14pt)".

  4. Click to apply a color to the text.

  1. Click Save. The grid should now look something like this.
