ASWUSERGET.A5W

ASWUSERGET.A5W contains logic to determine which page should be opened for the logged in user.

The NAV_INT navigation component calls the ASWUSERGET.A5W page.

Page Source

The system variable session.protectedpUser.ulink is populated by the security system and contains the value in the 'ulink' security field in the current user security record. If the user has an employee id saved in the security 'ulink' field, they are sent to the ASWUSERONE.A5W page. The value is sent in the URL in the emp_id variable. Otherwise, the user sees the ASWINDEX.A5W page.

' 'type=1' tells page to open for current logged in user

 

if session.protectedpUser.ulink <> ""

    response.redirect("aswuserone.a5w?emp_id="+session.protectedpUser.ulink+"&type=1") '

else

    response.redirect("aswindex.a5w")

end if
