Control Panel Methods

Alpha Five provides the following methods for manipulating the Control Panel.

Function

Description

.ACTIVATE()

Sets focus to the object.

.ACTIVE()

Returns the name of the control that has focus.

.ACTIVE_PAGE()

Returns the active page of the Control Panel.

.HIDE()

Hides the Control Panel window.

.ISVISIBLE()

Reports whether or not the Control Panel window is visible.

.MAXIMIZE()

Maximizes the Control Panel window.

.MINIMIZE()

Minimizes the Control Panel window.

.NAME()

Returns the name of the object (I.e. "").

.REFRESH()

Force repaint of control.

.RESTORE()

Restores the Control Panel window.

.SESSIONHANDLE()

Returns the session handle of the Control Panel window.

.SETFOCUS()

Sets focus to the Control Panel object.

.SHOW()

Shows the Control Panel window.

.SIZE_GET()

Gets window size ("Minimized", "Maximized" or "XY").

.VARIABLES()

Returns a pointer to the variables defined at the local, shared, or global levels.
