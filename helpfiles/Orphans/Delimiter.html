<<%DELIMITER% ... %DELIMITER%

Syntax

List as C = <<%DELIMITER%

string1
string2
...
stringN

%DELIMITER%

Argument

Description

string

 Any character string.

Description

<<%DELIMITER% ... %DELIMITER% allows you to define a quoted character string. This command is typically used when the string includes carriage return-line feed characters (CR-LF).

Note : You must not have any space characters between <<%DELIMITER% and string.

Note : <<%code% and <<%dlg% are special cases. The first defines a block of Xbasic code and turns on the Xbasic autohelp and color coding within the string. The second defines a block of Xdialog code and turns on the Xdialog autohelp and color coding within the string. Additionally, <<%html% is special in version 6. It defines a block of HTML code and turns on the HTML autohelp and color coding within the string.

For example:

X = <<%a%Alpha Software%a%

Y = <<%b%Alpha Five%b%

The above two commands are equivalent to:

X = "Alpha Software"

Y = "Alpha Five"

Example

The following script initializes an array.

dim colors10 as C

' First, create a string with a list of choices. Each choice is delimited with CR-LF

color_string = <<%choices%

Red

Green

Blue

Orange

Purple

%choices%

' Then, initialize the array with the CR-LF delimited string.

colors.initialize(color_string)
