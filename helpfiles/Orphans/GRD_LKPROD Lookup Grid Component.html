GRD_LKPROD Lookup Grid Component

Purpose

The GRD_INTSPEC grid component (on the ASWSPECLIST.A5W page) uses the GRD_LKPROD grid component to lookup and retrieve product details.

Description

The GRD_INTSPEC component is a read-only tabular grid based on a view that combines fields from the product and product_category tables.

Container

GRD_INTSPEC, ASWSPECLIST.A5W
