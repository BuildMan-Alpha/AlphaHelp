How to use this site

  • Tip: When searching, right-click on a found link to open it in a new tab or page, so that it will be easy to return to the list of links.
  • Video (3 minutes): How to search Alphapedia
  • Search by page titles:

    Search Wiki PageName

    Exact match
    Use the Search Wiki Page box at the upper right to find a page by title. Use the letter buttons on the resulting page list to restrict the search to pages starting with a single letter.
  • Search full text: Use the Find box at the top right to search by words in the body text of the pages. On the search results page, check "Advanced" if you want to refine the search by requiring terms (+term), excluding terms (-term), and/or requiring an exact match ("multi-word search term").

    Find



  • Browse: Use the Contents box at the top left and follow the links and browse arrows to the correct topic.
  • Give us feedback: Add a comment to the bottom of a page to let us know about improvements we could make to that page. You'll need to be logged in to do this.
