+ (Date Time Addition)

Syntax

Result as T = Operand1 as T + Seconds as N

Argument

Description

Operand1

A date time value.

Seconds

A number.

Description

Date time addition produces a date time value by adding an integral or fractional number of Seconds to Operand1.

Examples

The following example shows how to modify a date time variable (type T) by adding seconds.

t1 = now()

? t1

= 10/19/2005 11:07:07 53 am

t2 = ctodt("10/20/05 12:07:07 53 am")

? t2

= 10/20/2005 12:07:07 53 am

t2 = t2 + 60

? t2

= 10/20/2005 12:08:07 53 am

t1 = t1 + 0.47

? t1

= 10/19/2005 11:07:08 00 am
