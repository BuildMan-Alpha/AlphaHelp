Define Text Dialog Box

  1. Enter text in the text box, optionally with Xdialog formatting.

  2. If you are not familiar with Xdialog formatting commands, click the Use Genie to Define Formatted Text button to display the Static Text Genie.

  3. Click Preview to see what the text will look like.

  4. Click OK to continue.
