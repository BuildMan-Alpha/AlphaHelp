.PLAIN_TEXT

Syntax

RTF_Data as C = .PLAIN_TEXT

Description

The .PLAIN_TEXT property returns the unformatted text value of the RTF text.

Example

dim ptr as P

dim cdata as C

ptr = rtf.create("Alpha!")

cdata = ptr.plain_text

cdata -> "Alpha!"
