Close Form/Browse

Overview

The Close Form/Browse action closes the current window, or another form or browse window.

Procedure

  1. Make a selection in the Close which Form orBrowse window? field. This field will be grayed out unless this action is based on a Form or Browse. The choices are:

    • "Current Form or Browse Window".

    • "Another Form or Browse Window".

  2. In the How will you specify which Form or Browse window to close? field select either:

    • "By specifying the name of the Form or Browse window"

    • "By specifying a Form or Browse Pointer variable"

  3. If you selected "By specifying the name of the Form or Browse window", select a layout in the Window Name field. You may also type in the name of a form or browse.

  4. If you selected "By specifying a Form or Browse Pointer variable", enter the pointer name into the Pointer Name field.

  5. Optionally, check the box to automatically save the current record, if the user has not already saved it.

  6. Click Next >.

  7. Optionally modify the comment that describes the action.

  8. Click Finish.
